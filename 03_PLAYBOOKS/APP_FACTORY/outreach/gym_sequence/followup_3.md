## Usage notes

**Timing:** 14 days after initial email

**Why breakup emails work:**
- Creates closure
- Removes pressure
- Often triggers "wait, I meant to respond" replies
- Clears dead leads from pipeline

**Expected response:** 5-8% reply rate

**After this sequence:**
- If no response: add to 6-month re-engagement list
- If "not interested": remove from active list
- If "maybe later": add to 3-month follow-up list
- If interested: book the call
