## A/B test recommendations

**Test order:**

1. Run A vs B (engagement vs differentiation)
2. Winner vs C (distraction/data focus)
3. Winner vs D (retention angle)
4. Final winner vs E (corporate angle)

**Sample sizes:**

- 50 sends per variant minimum
- Track open rate, reply rate, positive reply rate
- Run each test for 2 weeks

**Expected benchmarks:**

| Metric | Target | Good | Excellent |
|--------|--------|------|-----------|
| Open rate | 50%+ | 55% | 65%+ |
| Reply rate | 6%+ | 10% | 15%+ |
| Positive reply rate | 4%+ | 6% | 10%+ |

## Gym type variations

**Boutique fitness (F45, OrangeTheory, etc.):**
- Emphasize community features
- Leaderboard/challenge angle
- Higher engagement expectation

**Big box gyms (Planet Fitness, Gold's):**
- Emphasize scale and ease
- Corporate partnership potential
- Member retention focus

**CrossFit boxes:**
- Competition/leaderboard angle
- Community challenges
- Box vs box competitions

**Personal training studios:**
- Client accountability tool
- Trainer can monitor client activity
- Premium add-on to training packages
