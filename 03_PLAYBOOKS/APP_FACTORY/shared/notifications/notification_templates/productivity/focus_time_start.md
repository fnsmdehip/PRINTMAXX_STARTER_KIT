# Focus time start notifications

Productivity apps: StudyLock, PromptVault

Sent when it's time to start a scheduled focus session.

---

## Primary variants

**Variant A - Direct**
- Title: "Focus time"
- Body: "Your [X] minute session is ready."

**Variant B - Question**
- Title: "Ready to focus?"
- Body: "Tap to start your session."

**Variant C - Task-aware**
- Title: "[Task name]"
- Body: "Time to start. [X] minutes blocked."

---

## Scheduled session reminders

**5 minutes before**
- Title: "Session in 5"
- Body: "Finish up what you're doing."

**At session time**
- Title: "Session starting"
- Body: "Tap to begin [X] minute focus."

---

## Recurring session

**Daily at same time**
- Title: "Daily focus"
- Body: "Same time, same progress. Let's go."

**Study group sync**
- Title: "[X] others starting"
- Body: "Your study group is going live."

---

## Pre-session checklist

Consider including in notification:
- "Got water? Phone charged?"
- "Materials ready? Tap to start."
- "Bathroom break first? Start in 5."

---

## Smart triggers

1. If user typically starts at this time, send reminder 5 min early
2. If user missed last 3 sessions at this time, ask if they want to reschedule
3. Link to task/topic if user set one
4. Include estimated end time

---

## Timing rules

- Respect user's scheduled times exactly
- No early morning focus notifications (before 6 AM)
- Allow snooze option (5, 15, 30 min)
- If snoozed twice, offer reschedule
