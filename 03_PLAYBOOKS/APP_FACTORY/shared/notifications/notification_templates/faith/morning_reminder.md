# Morning reminder notifications

Faith apps: PrayerLock, DailyAnchor, DailyDevotion

---

## Primary variants

**Variant A - Simple**
- Title: "Start your day"
- Body: "A few minutes of quiet before the noise begins."

**Variant B - Streak context**
- Title: "Day [X] waiting"
- Body: "Your morning reading is ready."

**Variant C - Time-based**
- Title: "5 minutes"
- Body: "That's all. Your verse is ready."

## Timing

- Default: 6:30 AM
- Allow user to set custom time
- Skip if user already opened app today
- Weekend timing: 7:30 AM default (later sleep)
