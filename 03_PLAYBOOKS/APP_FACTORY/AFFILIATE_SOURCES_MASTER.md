## Direct affiliate programs (no network)

Many apps run their own affiliate programs. Search:
- "[app name] affiliate program"
- "[app name] referral program"
- "[app name] partner program"

### How to find them

1. **Google search:** "[competitor] affiliate" or "[competitor] partner program"
2. **Website footer:** Look for "Affiliates" or "Partners" link
3. **Email founder:** Small apps often create affiliate deals on request
4. **Twitter DM:** Many indie makers respond to partnership requests

### Cold outreach template for affiliate deals

```
Subject: Partnership opportunity - [your app/audience]

Hey [Name],

I run [your app/channel] with [audience size/description].

My audience is perfect for [their product] because [specific reason].

Open to setting up an affiliate deal? I'm thinking:
- [X]% commission per sale
- Custom tracking link
- Happy to create dedicated content

Let me know if you're interested.

[Your name]
[Your app/channel link]
```

## Affiliate program evaluation checklist

Before promoting, check:

- [ ] Commission rate (aim for 15%+ for digital products)
- [ ] Cookie duration (30+ days preferred)
- [ ] Payment threshold (lower is better)
- [ ] Payment frequency (monthly standard)
- [ ] Tracking reliability (read reviews)
- [ ] Product quality (would you use it?)
- [ ] Refund rate (high refunds = clawbacks)
- [ ] Recurring commission (for subscriptions)
- [ ] Marketing materials provided
- [ ] Dedicated affiliate manager

## Integration with your apps

### How to add affiliate links in apps

**In-app recommendations:**
```
User completes workout
-> App: "Great workout! Want to optimize recovery?"
-> Recommended supplements (affiliate links)
-> Opens Safari/external browser
```

**Email sequences:**
```
User signs up for app
-> Day 3 email: "Tools that help with [problem]"
-> Curated list with affiliate links
-> Disclosure at bottom
```

**Resource pages:**
```
In-app "Resources" section
-> Curated tools/products
-> Affiliate links
-> Clear disclosure
```

### Disclosure requirements

**In-app:**
```
"We may earn a commission from purchases made through these links at no extra cost to you."
```

**Email:**
```
"Disclosure: This email contains affiliate links."
```

**Website:**
```
Full affiliate disclosure page + brief note near links
```
