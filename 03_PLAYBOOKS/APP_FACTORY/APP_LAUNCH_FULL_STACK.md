# App Launch Full Stack Checklist

Complete checklist for launching an app: code, marketing, content, ads, email, social, affiliates.

---

## Pre-launch phase (2-4 weeks before)

### Technical

- [ ] App passes all automated tests
- [ ] Tested on real devices (iPhone 11+, Pixel 5+)
- [ ] Tested on IPv6 network
- [ ] RevenueCat integrated and tested
- [ ] Analytics integrated (Mixpanel/Amplitude)
- [ ] Crash reporting enabled (Sentry/Crashlytics)
- [ ] Privacy policy live at valid URL
- [ ] Terms of service live at valid URL
- [ ] Support email configured
- [ ] TestFlight/internal testing complete (20+ testers)

### App Store assets

- [ ] App icon (1024x1024, unique design)
- [ ] Screenshots (6.5" iPhone, 5.5" iPhone, iPad if applicable)
- [ ] App preview video (15-30 seconds, optional but recommended)
- [ ] App name (unique, keyword-rich)
- [ ] Subtitle (30 chars, main benefit)
- [ ] Description (first 3 lines are crucial)
- [ ] Keywords (100 char limit, comma-separated)
- [ ] Category selected
- [ ] Age rating configured
- [ ] Privacy nutrition labels complete

### Google Play assets

- [ ] Feature graphic (1024x500)
- [ ] Screenshots (phone + tablet)
- [ ] App icon
- [ ] Short description (80 chars)
- [ ] Full description (4000 chars)
- [ ] Content rating questionnaire
- [ ] Data safety form complete

## Marketing infrastructure

### Landing page

- [ ] Domain purchased and configured
- [ ] Landing page live with:
  - [ ] Hero with app screenshots
  - [ ] 3 key benefits
  - [ ] Social proof (testimonials, ratings)
  - [ ] App Store + Google Play badges
  - [ ] Email capture for updates
  - [ ] Privacy policy link
  - [ ] Support contact

### Email setup

- [ ] Email service configured (Instantly/Smartlead for cold, ConvertKit for nurture)
- [ ] Welcome sequence ready (7 emails)
- [ ] Launch sequence ready (5 emails)
- [ ] Email templates match brand
- [ ] SPF/DKIM/DMARC configured
- [ ] Test email deliverability

### Social accounts

For each niche app, create:

**Primary:**
- [ ] TikTok account (@appname)
- [ ] Instagram account (@appname)
- [ ] X/Twitter account (@appname)

**Bio template:**
```
[App benefit in 1 line]
[Social proof: "10k+ users" or "4.8 stars"]
[CTA: "Download free 👇"]
[Link: linktr.ee/appname or direct app store link]
```

### Affiliate program

- [ ] Decide commission structure (15-30% recommended)
- [ ] Set up tracking (FirstPromoter, Rewardful, or RevenueCat referrals)
- [ ] Create affiliate landing page with:
  - [ ] Commission details
  - [ ] Sign-up form
  - [ ] Marketing assets for affiliates
  - [ ] Tracking link generator

## Cold outreach setup

### Target list building

- [ ] Identify 1000 potential users/customers
- [ ] Sources:
  - [ ] Reddit community members
  - [ ] Twitter followers of competitors
  - [ ] LinkedIn groups
  - [ ] App review commenters
- [ ] Clean and verify emails (Apollo, Hunter)

### Outreach sequences

**User acquisition (if B2C with discoverable emails):**
```
Subject: Quick question about [problem your app solves]

Hey [Name],

Saw you [context from their social/post].

We built [app name] specifically for [target audience] who struggle with [problem].

Would love your feedback - happy to give you lifetime free access.

[Your name]
```

**Partnership outreach:**
```
Subject: Collab idea for [their audience]

Hey [Name],

Love your content about [topic]. Your [specific content] was helpful.

We built [app name] that your audience might love - it helps [benefit].

Open to:
- Affiliate deal (20% commission)
- Sponsored post
- Content collab

Interested?

[Your name]
```

**Podcast/press outreach:**
```
Subject: Story idea: How [your app] is helping [audience]

Hey [Name],

Big fan of [podcast/publication]. Your episode with [guest] was great.

Quick pitch: [App name] has helped [number] people [achieve result].

I could share:
- The story behind building it
- What's working/not working
- Lessons for other builders

Worth 15 mins to explore?

[Your name]
```

## Analytics and tracking

### Key metrics to track

**Acquisition:**
- Downloads per day/week
- Download sources (organic, paid, referral)
- Cost per install (CPI) by channel
- App Store conversion rate

**Activation:**
- Onboarding completion rate
- Time to first value action
- Trial start rate

**Revenue:**
- Trial to paid conversion rate
- MRR / ARR
- Average Revenue Per User (ARPU)
- Lifetime Value (LTV)

**Retention:**
- Day 1, 7, 30 retention
- Monthly churn rate
- Feature usage rates

### Dashboard setup

- [ ] RevenueCat dashboard for revenue
- [ ] Mixpanel/Amplitude for behavior
- [ ] App Store Connect/Play Console for acquisition
- [ ] Custom dashboard (Notion/Sheets) for unified view

## Launch sequence

### Week -1 (7 days before)

- [ ] Final TestFlight build approved
- [ ] Submit to App Store review
- [ ] Submit to Google Play review
- [ ] All content scheduled
- [ ] Email sequence queued
- [ ] Landing page live
- [ ] Social accounts active with pre-launch content

### Day -3 to -1

- [ ] Confirm app approved
- [ ] Set release date
- [ ] Notify affiliates
- [ ] Notify beta testers
- [ ] Prepare launch day posts

### Launch day

- [ ] Release app on stores
- [ ] Send launch email
- [ ] Post launch announcement on all platforms
- [ ] Message warm contacts personally
- [ ] Start paid ads (small budget)
- [ ] Monitor for bugs/crashes
- [ ] Respond to all reviews

### Week 1 post-launch

- [ ] Daily content posting
- [ ] Respond to all user feedback
- [ ] Fix any bugs reported
- [ ] A/B test paywall
- [ ] Scale ads that work
- [ ] Kill ads that don't

### Week 2-4 post-launch

- [ ] Analyze conversion data
- [ ] Optimize onboarding based on data
- [ ] Create case studies from early users
- [ ] Expand to new channels
- [ ] Plan first feature update

## Affiliate integration checklist

### RevenueCat affiliate tracking

```swift
// iOS - Set affiliate attribution
Purchases.shared.attribution.setAttributes([
    "affiliate_code": affiliateCode
])
```

### Affiliate networks to join

| Network | Commission | Best for |
|---------|------------|----------|
| FirstPromoter | Custom | SaaS/apps |
| Rewardful | Custom | SaaS |
| Impact | Varies | Brands |
| ShareASale | Varies | General |
| PartnerStack | Custom | B2B |

### Affiliate program structure

**Standard tier:**
- 15% commission
- 30-day cookie
- Monthly payouts

**VIP tier (top performers):**
- 25% commission
- 90-day cookie
- Weekly payouts
- Custom codes
