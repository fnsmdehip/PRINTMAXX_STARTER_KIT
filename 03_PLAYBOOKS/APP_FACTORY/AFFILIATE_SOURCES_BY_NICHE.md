## Best Practices

### Placement Strategy

1. **After achievement:** Recommend products when user hits milestone
2. **Resource page:** Curated list in settings
3. **Email sequences:** Day 7, Day 30 emails
4. **Content marketing:** Blog posts, social content

### Disclosure Requirements

**In-app:**
```
"We may earn commission from purchases at no extra cost to you."
```

**Email:**
```
"Disclosure: This email contains affiliate links."
```

### Tracking

Use UTM parameters:
```
?utm_source=[app]&utm_medium=app&utm_campaign=[placement]&utm_content=[product]
```

Track in spreadsheet:
- Date
- Product
- Placement
- Clicks
- Conversions
- Revenue
