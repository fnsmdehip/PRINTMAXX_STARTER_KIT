# GlowMaxx ASO Launch Checklist

## Pre-Launch (1-2 weeks before)

### App Store Connect / Google Play Console

- [ ] Create app listing in App Store Connect
- [ ] Create app listing in Google Play Console
- [ ] Upload app icon (1024x1024)
- [ ] Upload screenshots for all device sizes
  - [ ] iPhone 6.7" (1290 x 2796)
  - [ ] iPhone 6.5" (1284 x 2778)
  - [ ] iPhone 5.5" (1242 x 2208)
  - [ ] iPad Pro 12.9" (2048 x 2732)
  - [ ] Android Phone (1080 x 1920 min)
  - [ ] Android Tablet (1200 x 1920 min)
- [ ] Create app preview video (15-30 seconds)
- [ ] Write app description (see app_store_description.md)
- [ ] Add keywords (see keywords.txt)
- [ ] Set up promotional text
- [ ] Configure age rating
- [ ] Set up privacy policy URL
- [ ] Set up support URL
- [ ] Configure pricing and availability

### RevenueCat Setup

- [ ] Create GlowMaxx project in RevenueCat
- [ ] Configure iOS products
  - [ ] Monthly subscription ($9.99)
  - [ ] Annual subscription ($49.99)
- [ ] Configure Android products
- [ ] Set up entitlements ("premium")
- [ ] Create offerings with packages
- [ ] Configure 7-day free trial
- [ ] Test purchases in sandbox
- [ ] Add API keys to app

### Legal

- [ ] Privacy policy published at glowmaxx.app/privacy
- [ ] Terms of service published at glowmaxx.app/terms
- [ ] Medical disclaimer in Terms of Service
- [ ] Subscription terms clearly stated

## Launch Day

### App Submission

- [ ] Submit iOS build for review
- [ ] Submit Android build for review
- [ ] Monitor review status
- [ ] Respond to any reviewer questions

### Social Launch

- [ ] Post launch announcement on all platforms
- [ ] Share App Store/Play Store links
- [ ] Reach out to influencers (see GTM doc)
- [ ] Enable link in bio everywhere

### Tracking Setup

- [ ] RevenueCat dashboard bookmarked
- [ ] App Store Connect analytics enabled
- [ ] Google Play Console analytics enabled
- [ ] Set up daily metrics review

## Post-Launch (Week 1)

### Monitor

- [ ] Daily installs
- [ ] Trial start rate
- [ ] Trial conversion rate
- [ ] Reviews and ratings
- [ ] Crash reports
- [ ] Customer support emails

### Optimize

- [ ] Respond to all reviews (especially negative)
- [ ] Note feature requests for v1.1
- [ ] A/B test screenshots if conversion low
- [ ] Adjust keywords based on search data

### Marketing Push

- [ ] Post daily content (see social_posts.md)
- [ ] Engage with looksmaxxing communities
- [ ] DM micro-influencers
- [ ] Consider small ad spend test ($50-100)

## Metrics Targets (Week 1)

| Metric | Target | Action if Below |
|--------|--------|-----------------|
| Installs | 100+ | More social content |
| Trial starts | 25%+ | Improve onboarding |
| Trial conversion | 15%+ | Optimize paywall |
| Rating | 4.5+ | Fix bugs, respond to reviews |
| Crashes | <1% | Priority bug fixes |

## Week 2+ Roadmap

### A/B Tests to Run
1. Paywall design (current vs minimal)
2. Trial length (7 days vs 3 days)
3. Price points ($9.99 vs $7.99 monthly)
4. Onboarding length (4 steps vs 6 steps)

### Features for v1.1
- Widget for home screen
- Apple Watch complications
- More exercise routines
- Community features (optional)
- Streak restoration IAP

### ASO Iterations
- Test new keyword sets monthly
- Update screenshots based on best performers
- Refresh promotional text seasonally
- Add localization for top markets (ES, DE, FR)
