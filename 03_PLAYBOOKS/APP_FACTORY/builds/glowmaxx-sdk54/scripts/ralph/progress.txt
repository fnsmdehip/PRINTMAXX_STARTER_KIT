### Key Learnings for SDK 54

1. **Notification API Breaking Changes**
   - The trigger input API changed significantly
   - Must use enum values from `Notifications.SchedulableTriggerInputTypes`
   - Calendar-based triggers use `DAILY`, `WEEKLY`, `MONTHLY`, `YEARLY`
   - Time-interval triggers use `TIME_INTERVAL`

2. **React Version Conflicts**
   - Use `--legacy-peer-deps` flag when installing to resolve React 19.1.0 vs 19.2.3 conflicts
   - Engine warnings about Node >= 20.19.4 can be ignored for development

3. **TypeScript Config**
   - SDK 54 uses `expo/tsconfig.base` as the base config
   - Must include `.expo/types/**/*.ts` and `expo-env.d.ts` in includes
