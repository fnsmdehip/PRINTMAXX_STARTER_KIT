# FocusPrayer Assets

Add the following image assets to this folder:

## Required Assets

1. **icon.png** (1024x1024)
   - App icon for iOS and Android
   - No transparency, rounded corners applied automatically

2. **splash.png** (1284x2778)
   - Splash screen image
   - Use #FAFAFA background to match app

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground
   - Icon should be centered with padding

4. **favicon.png** (48x48)
   - Web favicon

## Quick Generation

Use an online tool or design software to create:
- Simple prayer/hands icon
- Calming blue color (#4A90A4)
- Clean, minimal design

Or use Expo's icon generator: https://expo.dev/tools
