## Emergency Procedures

### If app crashes on launch day

1. Check Sentry for crash reports
2. Identify affected OS/device
3. Push hotfix if critical
4. Post status update on social
5. Email affected users with ETA

### If App Store rejects update

1. Read rejection reason carefully
2. Fix cited issues
3. Write appeal if unjustified
4. Resubmit within 24 hours
5. Document for future reference

### If reviews are negative

1. Respond to each review professionally
2. Identify patterns in complaints
3. Prioritize fixes in next update
4. Ask satisfied users to leave reviews
5. Update app description if needed

### If competitor launches similar product

1. Stay focused on your users
2. Accelerate unique features
3. Double down on community
4. Don't engage in public comparison
5. Use it as motivation
