# StepUnlock - ASO Checklist

## App Store Optimization Checklist

### App Name & Subtitle
- [x] App name includes main keyword (StepUnlock: Walk to Use Apps)
- [x] Subtitle explains core function (Block Apps Until You Walk)
- [ ] Test name variations in A/B tests

### Keywords (iOS - 100 chars)
- [x] Primary: app blocker, step counter, screen time
- [x] Secondary: walk tracker, phone addiction, digital detox
- [x] Long-tail: step goal, health app, focus app, habit tracker
- [x] Total: 99 characters used
- [ ] Review competitor keywords monthly

### Screenshots (Required)
- [ ] Screenshot 1: Progress ring at 60% with step count
- [ ] Screenshot 2: Blocked apps screen with lock icons
- [ ] Screenshot 3: Unlock animation (apps becoming available)
- [ ] Screenshot 4: Streak counter showing 14-day streak
- [ ] Screenshot 5: Settings with step goal slider
- [ ] Screenshot 6: Stats page with weekly chart

### Screenshot Guidelines
- Use iPhone 15 Pro Max frames
- Green/teal color scheme for health vibes
- Show realistic step counts (not round numbers)
- Include blocked app icons (Instagram, TikTok, etc.)
- Captions: "Walk First" / "Apps Unlock at 5,000 Steps" / "Build the Habit"

### App Preview Video (Optional but recommended)
- [ ] 15-30 second video
- [ ] Show: Open app > Try blocked app > Walk > Progress updates > Unlock
- [ ] Include satisfying unlock animation
- [ ] Upbeat background music

### Description Optimization
- [x] First line is the hook (blocks apps until you walk)
- [x] Problem clearly stated
- [x] Solution explained simply
- [x] Features in scannable list format
- [x] Results/social proof included
- [x] Subscription terms disclosed
- [ ] Localize for top 5 markets

### Ratings & Reviews
- [ ] In-app review prompt after first successful unlock
- [ ] Respond to reviews within 48 hours
- [ ] Address sync issues in responses with solutions
- [ ] Never incentivize reviews

### Category Selection
- Primary: Health & Fitness
- Secondary: Productivity
