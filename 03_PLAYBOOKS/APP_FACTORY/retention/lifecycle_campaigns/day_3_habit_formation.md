# Day 3 habit formation campaign

Goal: Get users to return 3 days in a row. This is the habit formation threshold.

Users with 3-day streaks retain 2.5x better than users with 1-day streaks.

---

## The habit formation window

Days 1-3 are critical. Users are still deciding if your app fits their life.

**Key insight:** By day 3, users have formed an opinion. Either:
- This app is useful and I'll keep using it
- This app is not for me and I'll delete it

Your job: Make the value obvious before they decide.

## Targeting and segmentation

### Segment A: Strong start (high priority)

**Criteria:**
- Completed D1 core action
- Opened app on D2
- Notifications enabled

**Approach:** Light touch. Just remind and celebrate.

### Segment B: Needs nudge

**Criteria:**
- Completed D1 core action
- Did NOT open on D2

**Approach:** Extra push on D2 evening. Email with specific value prop.

### Segment C: Slipping away

**Criteria:**
- Did NOT complete D1 core action
- Opened app but bounced

**Approach:** Value-focused messaging. What are they missing?

### Segment D: At risk

**Criteria:**
- Installed but minimal engagement
- Notifications disabled

**Approach:** Reduce frequency. Focus on one compelling reason to return.

## The 3-day rule

Why day 3 matters:

1. **Cognitive commitment** - User has invested enough time to feel ownership
2. **Routine formation** - Brain starts associating time/context with app
3. **Value realization** - User has seen enough to judge value
4. **Social proof** - User starts identifying as "someone who does X"

If you can get someone to day 3, D7 and D30 retention improve dramatically.
