## App Store Connect (iOS)

*Example: PrayerLock - Focus Timer*

### Subtitle (30 chars)

*Example: Faith-based productivity timer*

### Keywords (100 chars)
[keyword1,keyword2,keyword3,...]

**Rules:**
- No spaces after commas
- No duplicates from title
- Include competitor names (carefully)
- Long-tail variations
- Singular, not plural

## App Preview Video Script Template

**Duration:** 15-30 seconds

```
[0-3s] Problem statement or hook
Text: "[Problem question or statement]"

[3-7s] App opens, main action shown
Text: "[Primary action]"

[7-11s] Core feature in action
Text: "[Feature description]"

[11-17s] User flow / benefit demonstration
Text: "[Benefit statement]"

[17-22s] Result / achievement / outcome
Text: "[Outcome]"

[22-26s] Stats or social proof
Text: "[Proof point]"

[26-30s] Logo and CTA
Text: "[AppName - Tagline]"
```

**Video Rules:**
- No audio required (80% watch muted)
- Portrait orientation for App Store
- Show real app usage
- Fast pacing, no slow transitions
- Text overlays essential

## ASO Optimization Checklist

### Before Launch
- [ ] App name includes primary keyword
- [ ] Subtitle includes secondary keyword
- [ ] All 100 keyword characters used
- [ ] Description includes keywords naturally
- [ ] 6 screenshots minimum
- [ ] Screenshot text is readable
- [ ] App preview video (optional but recommended)
- [ ] Promotional text is compelling

### After Launch (First Week)
- [ ] Monitor keyword rankings
- [ ] Check conversion rate
- [ ] Respond to all reviews
- [ ] Note any indexing issues

### Ongoing (Monthly)
- [ ] A/B test screenshots
- [ ] Update keywords based on rankings
- [ ] Refresh promotional text
- [ ] Add new features to What's New
- [ ] Analyze competitor keywords

## Keyword Research Process

### Finding Keywords

1. **Brainstorm:** List all relevant terms
2. **Competitor analysis:** Check top 10 apps in category
3. **Autocomplete:** Use App Store search suggestions
4. **Tools:** AppTweak, Sensor Tower, AppFollow
5. **Long-tail:** Combine keywords for specific phrases

### Prioritizing Keywords

| Factor | Weight |
|--------|--------|
| Relevance | High |
| Search volume | Medium |
| Competition | Medium |
| Current ranking | Low |

### Keyword Placement Priority

1. **App Name** - Strongest weight
2. **Subtitle** - Strong weight
3. **Keyword field** - Medium weight (iOS only)
4. **Description** - Low weight (mainly Android)

---

## Common ASO Mistakes

1. **Keyword stuffing** - Makes copy unreadable
2. **Ignoring subtitle** - Wasted keyword opportunity
3. **Bad screenshots** - Showing splash, not features
4. **No A/B testing** - Guessing instead of testing
5. **Forgetting localization** - Missing non-English markets
6. **Ignoring reviews** - Unanswered complaints hurt ranking
7. **Static listings** - Never updating promotional text
