## Production Notes

### File Naming
- `prayerlock_ss_01_hero_6.5.png`
- `prayerlock_ss_02_blocking_6.5.png`
- `prayerlock_ss_03_timer_6.5.png`
- `prayerlock_ss_04_scripture_6.5.png`
- `prayerlock_ss_05_streak_6.5.png`

### Export Settings
- Format: PNG (no transparency)
- Color profile: sRGB
- Compression: Optimized for web (< 500KB each)

### Safe Areas
- Top: 44px (notch area)
- Bottom: 34px (home indicator)
- Left/Right: 20px minimum padding

### Device Frame
- Use official Apple device frames
- No third-party mockup templates with visible branding
- Shadow: soft, 20px blur, 40% opacity black
