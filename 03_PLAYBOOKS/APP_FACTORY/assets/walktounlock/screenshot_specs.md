## Production Notes

### File Naming
- `walktounlock_ss_01_hero_6.5.png`
- `walktounlock_ss_02_goal_6.5.png`
- `walktounlock_ss_03_progress_6.5.png`
- `walktounlock_ss_04_locked_6.5.png`
- `walktounlock_ss_05_celebrate_6.5.png`

### Export Settings
- Format: PNG (no transparency)
- Color profile: sRGB
- Compression: Optimized (< 500KB each)

### Safe Areas
- Top: 44px (notch area)
- Bottom: 34px (home indicator)
- Left/Right: 20px minimum padding

### Device Frame
- Use official Apple device frames
- Shadow: soft, 20px blur, 40% opacity black
- Frame color: Space Gray or Silver
