# FemFit - App Icon Specifications

## Primary Icon Concept

**Design:** Luna the cat face or stylized dumbbell with soft feminine aesthetic

**Visual Description Option A (Luna):**
- Simplified Luna cat face
- Cute, friendly expression
- Small dumbbell or weight element
- Soft coral and cream color scheme

**Visual Description Option B (Dumbbell):**
- Stylized feminine dumbbell
- Rounded edges, soft shapes
- Coral and lavender gradient
- Heart or paw print detail

**Recommended:** Option A (Luna face) for brand recognition

## Style Guidelines

### Do
- Make Luna recognizable and cute at small sizes
- Keep design soft and feminine (rounded shapes)
- Use the coral-cream color scheme consistently
- Test on light and dark backgrounds
- Ensure Luna's expression is cheerful

### Don't
- Make Luna too detailed (simplify for small sizes)
- Use aggressive/masculine fitness imagery
- Include text or letters
- Use harsh angles or sharp corners
- Make the fitness element (dumbbell) dominant

## Luna Design Reference

**Luna Character Traits:**
- Species: Cat (tabby or solid white/cream)
- Eyes: Large, friendly, green/mint colored
- Expression: Always cheerful, supportive
- Style: Chibi/simplified cartoon
- Accessories: None (or small sweatband for workout themes)

**Simplification for Icon:**
- Reduce to face only
- 2-3 colors maximum
- Eyes as curved happy lines (closed)
- Simple triangle ears
- Optional: tiny paw or dumbbell
