## Background Music Mood

**Style:** Modern tech ambient, subtle electronic
**Tempo:** 100-120 BPM (moderate, not distracting)
**Instruments:** Soft synth pads, subtle beats, clean tones
**Avoid:** Dramatic buildups, dubstep drops, corporate generic

**Royalty-Free Sources:**
- Epidemic Sound (search: "tech", "modern ambient")
- Artlist (search: "technology", "startup")
- YouTube Audio Library

**Track Characteristics:**
- Clean and professional
- Suggests innovation without being overwrought
- Low volume (30-40%) so text is readable

## Text Overlay Styling

**Font:** SF Pro Display

## Production Checklist

### Pre-Production
- [ ] Populate app with 500+ prompts for demo
- [ ] Set up ChatGPT comparison screens
- [ ] Prepare good/bad prompt examples
- [ ] Configure Pro features for demo

### Recording
- [ ] Screen record app navigation
- [ ] Capture ChatGPT before/after
- [ ] Record AI Improver in action
- [ ] Get clean copy animation
- [ ] 60fps for smooth playback

### Post-Production
- [ ] Assemble scenes
- [ ] Add text overlays
- [ ] Add background music
- [ ] Color grade for consistency
- [ ] Export 1080x1920, H.264

### Quality Check
- [ ] All text readable at phone size
- [ ] Prompt text legible in screenshots
- [ ] Before/after difference is clear
- [ ] Audio levels consistent
- [ ] Under 30 seconds
- [ ] Appeals to AI power users

---

## Messaging Priorities

1. **Quantity:** 500+ prompts (establishes value)
2. **Simplicity:** Copy and paste (reduces friction)
3. **Results:** Better AI output (the real benefit)
4. **Pro value:** AI improvement (reason to upgrade)

Don't try to explain everything. Show the core flow: browse, copy, paste, better results.
