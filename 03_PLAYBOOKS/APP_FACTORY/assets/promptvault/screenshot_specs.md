## Production Notes

### File Naming
- `promptvault_ss_01_hero_6.5.png`
- `promptvault_ss_02_categories_6.5.png`
- `promptvault_ss_03_copy_6.5.png`
- `promptvault_ss_04_improver_6.5.png`
- `promptvault_ss_05_generator_6.5.png`

### Export Settings
- Format: PNG (no transparency)
- Color profile: sRGB
- Compression: Optimized (< 500KB each)

### Design Notes
- Target audience: AI power users (25-45)
- Aesthetic: Dark mode, tech-forward, professional
- Show actual prompt text (make it realistic and useful)
- Pro features should look premium
