# DailyAnchor - App Icon Specifications

## Primary Icon Concept

**Design:** Anchor symbol with subtle sunrise/light element

**Visual Description:**
- Simplified anchor silhouette at center
- Subtle sunrise glow at top (representing morning/daily)
- Clean, minimal design
- Gradient background from dark gray to emerald green
- Not overly nautical, more symbolic

## Style Guidelines

### Do
- Keep anchor simple and iconic (no rope details)
- Subtle sunrise glow suggests "daily" and "morning"
- Modern flat design with minimal depth
- Test visibility on all backgrounds
- Make it feel calm and grounded

### Don't
- Make it look like a marine/sailing app
- Add cross or other religious symbols (too literal)
- Include text or letters
- Use complex anchor designs with rope/chain
- Make sunrise too prominent (it's a subtle accent)

## Icon Variants

### Primary (Default)
- Full gradient background
- White anchor with sunrise glow
- Use for App Store, home screen

### Notification Icon (Android)
- White anchor silhouette only
- Transparent background
- No gradient or glow

### Dark Mode Variant
- Slightly lighter gradient for dark backgrounds
- Test on iOS dark mode wallpapers

## File Specifications

### Format
- **iOS:** PNG with no transparency, sRGB
- **Android:** PNG with transparency
