## Quality Requirements (ALL Icons)

Every icon MUST have:
1. **Depth/dimension** - 3D, isometric, or layered elements (not flat)
2. **Gradient** - Rich color gradient (not solid fills)
3. **Relevant symbol** - Instantly communicates app purpose
4. **Glow/reflection** - Subtle light effects for premium feel
5. **1024x1024** - Generate at max resolution, resize down
6. **No text** - App name appears below icon on device, never on icon itself
7. **Simple composition** - One central element, clean background
8. **Rounded square** - iOS auto-crops to squircle, design within safe zone (leave 10% margin)

**Anti-patterns (avoid):**
- Single letter on colored background (lazy, generic)
- Too many small details (unreadable at 60x60 display size)
- Realistic photography (doesn't scale down well)
- Flat design without depth (looks cheap)
- Neon/harsh colors without subtlety

## Batch Generation Script (Gemini API)

For automated generation via Gemini API:

```python
import google.generativeai as genai
import os

genai.configure(api_key=os.environ.get("GEMINI_API_KEY"))

# Model for image generation
model = genai.GenerativeModel("gemini-2.0-flash-exp")  # or imagen-3 when available

PROMPTS = {
    "prayerlock": "3D app icon for a prayer accountability app...",  # full prompt from above
    "biomaxx": "3D app icon for a biohacking health tracker...",
    "stepunlock": "3D app icon for a walking fitness app...",
    # ... add all prompts
}

OUTPUT_DIR = "MONEY_METHODS/APP_FACTORY/builds"

for app_name, prompt in PROMPTS.items():
    print(f"Generating icon for {app_name}...")
    response = model.generate_content(prompt)

    # Save the generated image
    if response.candidates and response.candidates[0].content.parts:
        for part in response.candidates[0].content.parts:
            if hasattr(part, 'inline_data'):
                img_data = part.inline_data.data
                output_path = f"{OUTPUT_DIR}/{app_name}-sdk54/assets/icon.png"
                os.makedirs(os.path.dirname(output_path), exist_ok=True)
                with open(output_path, 'wb') as f:
                    f.write(img_data)
                print(f"  Saved to {output_path}")
```

**Alternative: Manual generation via Gemini browser UI**
1. Go to https://aistudio.google.com/
2. Select Gemini model with image generation
3. Paste each prompt
4. Download 1024x1024 PNG
5. Copy to `builds/{app}-sdk54/assets/icon.png`

## Adaptive Icon (Android)

Android requires separate foreground and background layers:

```json
// app.json
{
  "expo": {
    "icon": "./assets/icon.png",
    "android": {
      "adaptiveIcon": {
        "foregroundImage": "./assets/adaptive-icon.png",
        "backgroundColor": "#1a1a2e"
      }
    }
  }
}
```

Generate foreground layer (transparent background, centered symbol only):

**Template prompt suffix for Android foreground:**
```
... [same prompt] ... Transparent background. Only the central icon element,
no background gradient. Centered in frame with 30% padding on all sides.
PNG with alpha transparency. For Android adaptive icon foreground layer.
```
