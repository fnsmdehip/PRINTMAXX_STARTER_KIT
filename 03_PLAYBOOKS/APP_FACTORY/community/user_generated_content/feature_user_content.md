# Feature user content

How to showcase UGC across channels for maximum impact.

---

## Where to feature UGC

### Website

| Location | UGC type | Impact |
|----------|----------|--------|
| Homepage hero | Best testimonial | High |
| Landing page | 3-5 testimonials | High |
| Pricing page | ROI/value testimonials | Very high |
| App store page | Star ratings, quotes | Very high |
| Blog sidebar | User photos/quotes | Medium |
| Footer | Logos, trust badges | Medium |

### Social media

| Platform | Best UGC format | Posting frequency |
|----------|-----------------|-------------------|
| Instagram | Repost stories, carousel testimonials | 2-3x/week |
| Twitter/X | Quote RT wins, screenshot threads | Daily |
| TikTok | Duet user videos, compilations | 2-3x/week |
| LinkedIn | Case study summaries, quotes | 1-2x/week |
| YouTube | Full testimonial videos, compilations | Weekly |

### In-app

| Location | UGC type | Purpose |
|----------|----------|---------|
| Onboarding | Success stories | Motivation |
| Paywall | Testimonials | Conversion |
| Progress screen | Community wins | Social proof |
| Empty states | User tips | Education |

### Email

| Email type | UGC use |
|------------|---------|
| Welcome sequence | Featured member story |
| Weekly digest | Community highlights |
| Re-engagement | Success stories from similar users |
| Upgrade prompts | Premium user testimonials |

## Website showcase formats

### Testimonial card

```html
<div class="testimonial-card">
  <div class="quote">"Finally an app that actually works..."</div>
  <div class="result">Lost 15 lbs in 8 weeks</div>
  <div class="author">
    <img src="avatar.jpg" alt="Sarah M." />
    <div>
      <strong>Sarah M.</strong>
      <span>Member since 2024</span>
    </div>
  </div>
</div>
```

**Elements:**
- Direct quote (keep short, 1-2 sentences)
- Specific result (number if possible)
- Photo (real face if permitted)
- Name (first name + last initial minimum)
- Context (how long they've used it)

### Testimonial carousel

Best for landing pages. Show 5-7 testimonials.

**Order strategy:**
1. Strongest testimonial first
2. Mix of use cases (different goals)
3. Variety of demographics
4. End with aspirational success story

### Video testimonial embed

```html
<div class="video-testimonial">
  <video poster="thumbnail.jpg" controls>
    <source src="testimonial.mp4" type="video/mp4">
  </video>
  <div class="video-caption">
    <strong>John's story:</strong> "From skeptic to 200-day streak"
  </div>
</div>
```

**Best practices:**
- Show face in thumbnail
- Keep videos under 90 seconds
- Add captions for accessibility
- Use compelling thumbnail frame

### Results grid

For visual transformations or achievements:

```
[Before photo] [After photo]
"8 weeks of consistency"
- Sarah M., Teacher

[Before photo] [After photo]
"Never thought I could do this"
- Mike R., Software Engineer
```

### Logo wall

If B2B or course:

```
"Trusted by teams at:"
[Logo] [Logo] [Logo] [Logo] [Logo]
```

### Numbers banner

Aggregate UGC into stats:

```
10,000+ active members | 500,000 habits tracked | 4.8 avg rating
```

## Social media showcase formats

### Instagram

**Story reshare:**
1. Screenshot user's story (with permission)
2. Add frame/branding
3. Tag original creator
4. Add "See more stories" link sticker

**Feed carousel:**
```
Slide 1: User photo/screenshot
Slide 2: Their quote
Slide 3: Their results
Slide 4: CTA ("Start your journey")
```

**Highlight bubbles:**
Create highlights for:
- "Wins" - Success stories
- "Reviews" - Testimonials
- "Tips" - User-submitted tips

### Twitter/X

**Quote RT format:**
```
[Quote RT user's post]

"This is what we build for."

[App name] user since Day 1. Their journey inspires us every day.
```

**Thread format:**
```
1/ Our users are incredible. Here are 5 wins from this week:

2/ @user1 hit 100 days. "I never thought I'd make it this far."

3/ @user2 lost 20 lbs. "The accountability feature changed everything."

[etc.]

6/ What's YOUR win this week? Reply below and we might feature you next.
```

### TikTok

**Duet format:**
- Duet user's video
- Add your reaction/commentary
- Keep it authentic, not salesy

**Compilation format:**
- Collect 5-10 short clips
- Edit into montage
- Add trending audio
- Text overlay with results

### LinkedIn

**Case study post:**
```
[User name] had a problem: [problem statement]

They tried [previous solutions]. Nothing worked.

Then they found [your product].

The result:
- [Metric 1]
- [Metric 2]
- [Metric 3]

"[Direct quote from user]"

Here's what made the difference: [key insight]

---

Have you experienced something similar? I'd love to hear your story.
```

## Curation workflow

### Weekly process

**Monday:**
1. Review all #wins posts from past week
2. Flag top 10 for potential featuring
3. Check permissions status

**Tuesday:**
1. Reach out for permissions if needed
2. Select top 5 with permissions
3. Add to content calendar

**Wednesday:**
1. Design/format selected UGC
2. Schedule for social platforms
3. Queue for email digest

**Friday:**
1. Feature "win of the week"
2. Update website testimonials (monthly)
3. Archive used content

### Quality criteria

Rate each piece before featuring:

| Criteria | Weight | Score 1-5 |
|----------|--------|-----------|
| Specific result mentioned | 25% | |
| Emotional impact | 25% | |
| Visual quality (if applicable) | 20% | |
| Relatable story | 20% | |
| Unique angle | 10% | |

**Threshold:** Feature if total score > 3.5

### Content rotation

Don't overuse same testimonials:

| Platform | Rotate every |
|----------|--------------|
| Website hero | Quarterly |
| Landing page | Monthly |
| Social media | Weekly |
| App Store | Quarterly |
| Email | Per campaign |

## Attribution best practices

### Always include

- Creator's name (as they prefer)
- Context (role, location, or time using product)
- Link back to original (if public)

### Format examples

```
"[Quote]" - Sarah M., Portland OR

"[Quote]" - @username on Twitter

"[Quote]" - Mike, Premium member since 2024

"[Quote]" - Anonymous member
```

### Tagging etiquette

**Social media:**
- Always tag creator if featuring their content
- Ask first if they want to be tagged
- Some prefer anonymity despite public post

**Website:**
- Link to their social profile (with permission)
- Or keep it simple: First name, last initial

## Measuring feature impact

### Track per featured UGC

| Metric | Track |
|--------|-------|
| Engagement rate | Likes, comments, shares vs average |
| Click-through | If CTA included |
| Conversion | UTM tags for landing pages |
| Sentiment | Comments positive/negative |

### Compare UGC vs branded content

| Content type | Avg engagement | Avg CTR | Conversion |
|--------------|----------------|---------|------------|
| Branded | X | X% | X% |
| UGC | Y | Y% | Y% |
| Difference | +/-% | +/-% | +/-% |

### Attribution tracking

For website:
- Add UTM parameters to UGC campaigns
- Track which testimonials appear on converting pages
- A/B test testimonial placement

```
utm_source=instagram
utm_medium=ugc
utm_campaign=sarah-testimonial
```
