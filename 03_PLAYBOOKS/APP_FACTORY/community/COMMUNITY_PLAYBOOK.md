# Community playbook

Why most apps fail at retention: no community. Users download, try once, forget. Community changes this.

---

## Why community matters for app retention

### The numbers

- Apps with communities: 2-3x higher 30-day retention
- Community members: 6x more likely to convert to paid
- Word-of-mouth from community: $0 CAC
- Churn for community members: 40-60% lower than non-members

### The psychology

Users stay for:
1. **Identity** - "I'm a person who does X"
2. **Accountability** - Others expect them to show up
3. **Social proof** - Seeing others succeed
4. **Belonging** - Part of something bigger

Solo app usage = easy to quit. Community = social cost to leaving.

---

## Community vs audience

| Audience | Community |
|----------|-----------|
| One-way (you to them) | Multi-directional (they talk to each other) |
| Content consumers | Content creators |
| Follow you | Know each other |
| Leave when bored | Stay for relationships |
| CAC for each conversion | Members bring members |

**Test:** If you disappeared, would they keep talking? That's community.

### Audience to community evolution

1. **Broadcast** - You post, they read
2. **Interaction** - They reply to you
3. **Discussion** - They reply to each other
4. **Creation** - They create without prompting
5. **Identity** - They identify as part of the group

Most apps stuck at stage 1-2. The money is at stage 4-5.

## Platform comparison

### Discord

**Best for:** Tech-savvy users, gamers, younger demos
**Pros:**
- Free to run
- Highly customizable (bots, channels, roles)
- Real-time chat feels alive
- Great for live events

**Cons:**
- Learning curve for non-gamers
- Can feel chaotic without moderation
- Notification fatigue risk

**Cost:** Free (Nitro boosts for perks)

### Circle

**Best for:** Professional communities, courses, coaches
**Pros:**
- Clean, professional interface
- Spaces/threads organize discussions
- Built-in events and courses
- Better for async discussions

**Cons:**
- $49-99/month
- Less "alive" feeling than Discord
- Fewer integrations

**Cost:** $49/mo (Basic) - $99/mo (Professional)

### Skool

**Best for:** Course creators, coaching programs
**Pros:**
- Gamification built-in (leaderboards, levels)
- Clean, simple interface
- Course hosting included
- Strong viral mechanics

**Cons:**
- $99/month
- Limited customization
- Less suited for casual communities

**Cost:** $99/month flat

### Comparison matrix

| Feature | Discord | Circle | Skool |
|---------|---------|--------|-------|
| Monthly cost | Free | $49-99 | $99 |
| Real-time chat | Excellent | Good | Basic |
| Async discussions | Basic | Excellent | Good |
| Gamification | Via bots | Limited | Built-in |
| Course hosting | No | Yes | Yes |
| Mobile experience | Great | Good | Good |
| Learning curve | Medium | Low | Low |

**Recommendation by niche:**
- Faith apps: Discord (live prayer, Bible studies)
- Fitness apps: Skool (gamification, challenges)
- Productivity apps: Circle (async discussions, professionals)

## Community flywheel

```
New user joins
       |
       v
Gets quick win (first post welcomed)
       |
       v
Sees others like them succeeding
       |
       v
Participates (asks question, shares win)
       |
       v
Gets value (answer, encouragement)
       |
       v
Helps someone else
       |
       v
Feels ownership (this is MY community)
       |
       v
Brings friends (tells others about it)
       |
       v
New user joins...
```

### Flywheel levers

**Speed up the flywheel:**
1. Fast response time (< 30 min for new member posts)
2. Celebrate every win publicly
3. Match new members with veterans
4. Create easy first contribution opportunities
5. Highlight member success in app

**What slows the flywheel:**
1. Ignored posts (dead community feeling)
2. No visible activity
3. Complicated onboarding
4. No clear path to participate
5. Toxic members

## Community economics

### Build vs buy

**Build in-app community:**
- Full control
- Seamless UX
- Expensive to build ($50k-200k)
- Must maintain

**Use external platform:**
- Fast to launch (1 day)
- Proven tools
- Less control
- Platform risk

**Recommendation:** Start external, bring in-app once validated.

### Revenue opportunities

1. **Premium community tier** - Paid access to inner circle
2. **Cohort programs** - Paid challenges with accountability
3. **Merch** - Community identity items
4. **Events** - Paid live workshops
5. **Certifications** - Paid ambassador/coach programs

### Cost structure (1,000 member community)

| Platform | Monthly cost | Per-member |
|----------|--------------|------------|
| Discord | $0 | $0 |
| Circle Basic | $49 | $0.05 |
| Skool | $99 | $0.10 |
| Custom (Firebase) | $20-50 | $0.02-0.05 |

## Launch sequence

### Week 1: Foundation

- [ ] Choose platform based on audience
- [ ] Set up basic structure (see discord_setup/)
- [ ] Create welcome sequence
- [ ] Seed with 10-20 founding members
- [ ] Write community guidelines

### Week 2: Activation

- [ ] Run first challenge (see challenges_events/)
- [ ] Post daily prompts
- [ ] Respond to every post within 30 min

- [ ] Launch ambassador program
- [ ] Run first live event
- [ ] Feature first user-generated content
- [ ] Create first leaderboard
- [ ] Send first community newsletter

### Month 2+: Scale

- [ ] Automate onboarding
- [ ] Train moderators
- [ ] Systematize content calendar
- [ ] Launch paid tier
- [ ] Build referral loops

## Common mistakes

### 1. Building before validating
You need 10 engaged members before building anything. Test in DMs first.

### 2. Over-engineering structure
Start with 3 channels max. Add more only when existing ones overflow.

### 3. Being the only voice
If you post 80% of content, it's a blog, not a community. Target < 30% you, 70% members.

### 4. No clear purpose
"Networking" isn't a purpose. "Help each other hit 100 days of meditation" is.

### 5. Ignoring bad actors
One toxic member drives away 10 good ones. Remove fast.

### 6. Measuring vanity metrics
Member count means nothing. Track active participation rate.

---

## Key takeaways

1. Community = retention insurance
2. Start external, bring in-app later
3. The flywheel is everything (speed it up)
4. Activity beats size (100 engaged > 10,000 lurkers)
5. You're building a culture, not a chat room
