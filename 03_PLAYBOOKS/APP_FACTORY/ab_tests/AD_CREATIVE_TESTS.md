## Creative Format Tests

### Test 1: Video vs Static vs Carousel

**Hypothesis:** UGC video outperforms other formats for app installs.

```

**Variant D: Video (Polished)**
```
15-30 second video

## Budget Allocation for Testing

### Initial Testing Phase

```
Total weekly budget: $1,000

Split:
- 40% Control (best performing creative)
- 60% Test variations (15% each for 4 tests)

Run for: 7 days minimum
Required: $250+ spend per variant for significance
```

### Scaling Phase

```
Total weekly budget: $5,000

Split:
- 70% Winners (proven creative)
- 20% Iterations (variations of winners)
- 10% New concepts
```

## Measurement Framework

### Primary Metrics

| Metric | Definition | Target |
|--------|------------|--------|
| CTR | Clicks / Impressions | >1% (feed) |
| CPI | Cost per install | <$2 (varies by app) |
| ROAS | Revenue / Ad spend | >1.0 |

### Secondary Metrics

| Metric | Definition | Why It Matters |
|--------|------------|----------------|
| Hook rate | 3-second views / impressions | Video quality |
| Hold rate | 15-second views / 3-second views | Content quality |
| IPM | Installs per 1000 impressions | Overall efficiency |
| D7 retention | Users active on day 7 | Targeting quality |

## Creative Testing Process

### Step 1: Hypothesis
```
"[Format/angle] will outperform [current] because [reason]"
```

### Step 2: Production
```
- Brief creative team/creator
- Produce variants
- QA all assets
```

### Step 3: Setup
```
- Create ad sets (one per variant)
- Equal budget split
- Same audience targeting
- Set up tracking
```

### Step 4: Run
```
- 7 days minimum
- $250+ per variant
- Monitor for issues daily
```

### Step 5: Analyze
```
- Compare primary metrics
- Check for statistical significance
- Segment by placement/audience
```

### Step 6: Action
```
Winner: Scale budget
Loser: Kill
Inconclusive: Extend or iterate
```

---

## Creative Iteration Framework

### When Winner Found:

1. **Variation testing:** Tweak elements of winner
   - Different hook
   - Different CTA
   - Different thumbnail

2. **Audience expansion:** Same creative, new audiences

3. **Platform expansion:** Adapt winner for other platforms

### When No Winner:

1. **Check sample size:** Need $250+ per variant
2. **Check targeting:** Audience may be too broad
3. **Try new angle:** Current concept may not resonate
4. **Test format change:** Video vs static vs carousel
