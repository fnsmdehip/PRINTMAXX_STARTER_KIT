## Headline Tests

### Test 1: Benefit vs Feature vs Social Proof

**Hypothesis:** Benefit headlines convert better than feature or social proof.

**Variant A: Feature-Focused**
```
AI-Powered Writing Assistant
Generate content with advanced language models.
```

**Variant B: Benefit-Focused**
```
Write 10x Faster
Create blog posts, emails, and social content in seconds.
```

**Variant C: Social Proof**
```
The Writing Tool 50,000+ Creators Use
Join the community creating better content faster.
```

## Measurement Framework

### Primary Metrics

| Metric | Definition | Target |
|--------|------------|--------|
| Conversion rate | Signups / Visitors | 3-5% for cold, 15-25% for warm |
| Bounce rate | Single page visits / All visits | <60% |

### Secondary Metrics

| Metric | Definition | Insight |
|--------|------------|---------|
| Time on page | Average time spent | Content engagement |
| Scroll depth | How far users scroll | Content interest |
| Form abandonment | Started but not completed | Form friction |
| CTA click rate | CTA clicks / page views | CTA effectiveness |

### Segment Analysis

- Traffic source (organic, paid, social, email)
- Device type (desktop, mobile, tablet)
- Geographic region
- New vs returning visitors

## Testing Priority Framework

### Start Here (Highest Impact)

1. **Headline** - Biggest impact, easy to test
2. **CTA copy and placement** - Direct conversion impact
3. **Social proof presence** - Trust building

### Next Level

4. **Hero image/video** - Engagement driver
5. **Form length** - Friction point
6. **Page length** - Content strategy

### Advanced

7. **Color scheme** - Usually low impact
8. **Font changes** - Rarely significant
9. **Minor copy tweaks** - Diminishing returns

---

## Common Mistakes to Avoid

1. **Testing too many elements** - One variable per test
2. **Not waiting for significance** - Minimum 100 conversions per variant
3. **Mobile neglect** - Test mobile separately if needed
4. **Ignoring page speed** - Slow variants will always lose
5. **Confusing correlation with causation** - Consider external factors
