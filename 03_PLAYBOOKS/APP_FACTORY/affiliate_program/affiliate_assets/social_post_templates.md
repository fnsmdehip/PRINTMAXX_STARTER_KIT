## Platform-specific notes

### Twitter/X

**Best practices:**
- Keep under 280 characters for single tweets
- Thread for longer content (hook in tweet 1, details in 2-5)
- Include link directly in tweet
- #ad or (affiliate) works for disclosure

**Thread structure:**
```
Tweet 1: Hook + main point
Tweet 2-4: Details/tips/results
Tweet 5: CTA with link
```

### Instagram

**Best practices:**
- First 125 characters show before "more" (put hook here)
- Link goes in bio (say "link in bio")
- 3-5 hashtags (not 30)
- Disclosure visible before "more" cutoff

**Caption structure:**
```
Line 1: Hook
Line 2-3: Key content
Line 4: CTA + disclosure
Line 5: Hashtags (optional)
```

### TikTok

**Best practices:**
- Video content outperforms text
- Caption supports video, doesn't replace it
- Link in bio (mention verbally in video)
- #ad or verbal disclosure required

**Video structure:**
```
0-3s: Hook (problem or curiosity)
3-20s: Show app in use
20-30s: Result + CTA
```

### LinkedIn

**Best practices:**
- Longer form acceptable (up to 1,300 chars)
- Professional angle
- No hashtags needed
- Disclosure at end

### Pinterest

**Best practices:**
- Image-focused
- Title and description for SEO
- Link directly in pin
- Disclosure in description

## Disclosure requirements reminder

**Minimum disclosure:**
- "#ad" visible in post
- OR verbal disclosure in video
- OR "affiliate link" mentioned

**Placement:**
- Before the fold (visible without clicking "more")
- At start of video (first 30 seconds)
- Not buried after 10 hashtags

**Examples of compliant disclosures:**
```
#ad
#affiliate
(affiliate link)
"I'm an affiliate and earn commission"
"This is a paid partnership"
"I may earn a commission"
```
