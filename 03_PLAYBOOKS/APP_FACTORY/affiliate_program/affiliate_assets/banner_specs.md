## Design specifications

### Safe zones

```
728x90 (Leaderboard):
- Logo: Left 20%
- Copy: Center 50%
- CTA: Right 25%
- Margin: 10px all sides

300x250 (Medium Rectangle):
- Logo: Top 15%
- Copy: Middle 50%
- CTA: Bottom 25%
- Margin: 15px all sides

320x50 (Mobile):
- Logo: Left 25%
- Copy: Center 45%
- CTA: Right 25%
- Margin: 5px all sides
```

### Typography

```
Headline:
- Font: [Brand font] or Inter Bold
- Size: 18-32px (varies by banner size)
- Color: [Brand primary] or White

Body:
- Font: [Brand font] or Inter Regular
- Size: 12-16px
- Color: [Brand secondary] or #333333

CTA:
- Font: [Brand font] or Inter Semi-Bold
- Size: 14-18px
- Color: White on [Brand primary]
```

### Colors

```
Primary background options:
1. Solid brand color
2. Gradient (brand primary to secondary)
3. White with colored accents
4. Dark mode (near black)

CTA button:
- Background: [Brand accent] or contrasting color
- Text: White
- Border radius: 4-8px

Avoid:
- Pure black backgrounds
- Low contrast text
- Clashing colors
- Too many colors (3 max)
```

### Logo placement

```
Minimum size:
- 728x90: 100px wide
- 300x250: 80px wide
- 320x50: 60px wide

Position:
- Top-left or centered
- Clear space around logo = logo height
- Never below CTA button
```

## File format specifications

### Static banners

```
Format: PNG-24
Resolution: 1x (standard), 2x (retina)
Max file size: 150KB
Color profile: sRGB
Background: Solid or transparent (if needed)
```

### Animated banners

```
Format: GIF or HTML5
Animation length: 15-30 seconds
Loop: 3 times, then stop on final frame
Max file size: 200KB (GIF), 100KB (HTML5)
Frame rate: 24fps max
```

### HTML5 specifications

```
Initial load: <150KB
Polite load: <2.2MB total
Clicktag: Standard IAB format
Border: 1px solid #ccc required
```

## Testing checklist

Before releasing banners:

- [ ] Text is readable at actual size
- [ ] CTA button is prominent
- [ ] Logo is visible and clear
- [ ] Colors match brand guidelines
- [ ] File size under limit
- [ ] Looks good on both light and dark sites
- [ ] Animation stops after 3 loops
- [ ] All copy is accurate (pricing, claims)
- [ ] Clickthrough URL is correct
- [ ] Tracking parameters work

## Quick reference

### Must-haves for every banner

1. Logo (visible, properly sized)
2. Headline (clear, benefit-focused)
3. CTA button (contrasting color)
4. Affiliate tracking link

### Never include

1. Misleading claims
2. Fake buttons or interactions
3. Strobing or flashing animation
4. Auto-playing audio
5. Competitor logos (without comparison context)
