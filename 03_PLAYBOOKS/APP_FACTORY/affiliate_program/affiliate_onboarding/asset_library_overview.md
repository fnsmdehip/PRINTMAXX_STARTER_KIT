## Banner ads

### Available sizes

| Size | Name | Best for |
|------|------|----------|
| 728x90 | Leaderboard | Blog headers, above content |
| 300x250 | Medium Rectangle | Sidebars, in-content |
| 160x600 | Wide Skyscraper | Sidebars |
| 300x600 | Half Page | High-impact sidebar |
| 320x50 | Mobile Leaderboard | Mobile sites |
| 320x100 | Large Mobile | Mobile apps |
| 970x250 | Billboard | Homepage takeover |

### Banner variations

Each size includes:

**Version A: Feature-focused**
- Headline: Key benefit
- Visual: App screenshot
- CTA: "Try Free" or "Get Started"

**Version B: Offer-focused**
- Headline: Discount/promo
- Visual: Price crossed out
- CTA: "Save 10%" or "Get Deal"

**Version C: Social proof**
- Headline: User testimonial snippet
- Visual: Star rating or user count
- CTA: "See Why" or "Join [X]k Users"

### File formats

- PNG: Static, high quality
- GIF: Animated (subtle animation)
- HTML5: Interactive (advanced users)

### Usage guidelines

Do:
- Use as provided
- Place on your own website/content
- A/B test different versions
- Track performance with your affiliate link

Don't:
- Edit text or imagery
- Remove branding
- Use on adult/illegal content
- Misrepresent the product

## Logo files

### Versions available

| Type | Use |
|------|-----|
| Primary logo | Full color on light backgrounds |
| Reversed logo | White/light on dark backgrounds |
| Icon only | App icon, favicons |
| Wordmark | Text only, no icon |
| Monochrome | Single color applications |

### File formats

- SVG: Vector, scalable
- PNG: Transparent background
- EPS: Print/design software
- PDF: Universal

### Color codes

```
Primary: #[HEX]
Secondary: #[HEX]
Accent: #[HEX]
Text: #[HEX]
Background: #[HEX]
```

### Logo usage rules

Do:
- Use official files only
- Maintain clear space around logo
- Keep proportions (don't stretch)
- Use on contrasting backgrounds

Don't:
- Change colors
- Add effects (shadows, gradients)
- Rotate or distort
- Combine with other logos
