# Platform Specifications

Quick reference for format requirements across platforms.

---

## Video Platforms

### TikTok

| Spec | Requirement |
|------|-------------|
| Aspect ratio | 9:16 (vertical) |
| Resolution | 1080x1920 px |
| Max duration | 10 minutes |
| Optimal duration | 15-60 seconds |
| File size | Max 287 MB (iOS), 72 MB (Android) |
| File format | MP4, MOV |
| Captions | Auto-generated, editable |
| Safe zones | Top 150px (username), Bottom 270px (UI) |

**Best practices:**
- Hook in first 1-2 seconds
- Text overlays for sound-off viewing
- Trending audio boosts discovery
- Native editing features help reach
- Post 1-3x daily for growth

### Instagram Reels

| Spec | Requirement |
|------|-------------|
| Aspect ratio | 9:16 (vertical) |
| Resolution | 1080x1920 px |
| Max duration | 90 seconds |
| Optimal duration | 15-30 seconds |
| File format | MP4, MOV |
| Cover image | Separate upload, 1080x1920 |
| Safe zones | Bottom 20% for UI elements |

**Best practices:**
- Cover image matters for profile grid
- Captions essential (80% watch muted)
- First 3 seconds determine retention
- Cross-post TikToks (remove watermark)
- Hashtags in caption, not on video

### YouTube Shorts

| Spec | Requirement |
|------|-------------|
| Aspect ratio | 9:16 (vertical) |
| Resolution | 1080x1920 px |
| Max duration | 60 seconds |
| File format | MP4, MOV, WebM |
| Title | Max 100 characters |
| Safe zones | Top/bottom 10% for UI |

**Best practices:**
- Add #Shorts to title or description
- Vertical only (no letterboxing)
- Loop potential increases watch time
- Can link to long-form content
- Subscribe CTA in video works

### YouTube Long-form

| Spec | Requirement |
|------|-------------|
| Aspect ratio | 16:9 (horizontal) |
| Resolution | 1920x1080 px (1080p) or 3840x2160 (4K) |
| Max duration | 12 hours (verified), 15 min (unverified) |
| Optimal duration | 8-15 minutes |
| Thumbnail | 1280x720 px, max 2MB |
| File format | MP4, MOV, AVI, WMV |

**Best practices:**
- Thumbnail = 50% of click decision
- First 30 seconds = viewer retention hook
- Chapters improve watch time
- End screens last 20 seconds
- Cards for linking related content

### X/Twitter Video

| Spec | Requirement |
|------|-------------|
| Aspect ratio | 16:9, 1:1, or 9:16 |
| Resolution | 1920x1080 (landscape), 1080x1080 (square), 1080x1920 (portrait) |
| Max duration | 2 minutes 20 seconds |
| File size | Max 512 MB |
| File format | MP4, MOV |
| Recommended | 1:1 for mobile feed |

**Best practices:**
- Square (1:1) gets most real estate
- Captions essential (autoplay muted)
- First 3 seconds = thumbnail
- Native upload outperforms links
- Thread + video combo performs well

## Image Platforms

### Instagram Feed

| Spec | Requirement |
|------|-------------|
| Square | 1080x1080 px (1:1) |
| Portrait | 1080x1350 px (4:5) |
| Landscape | 1080x566 px (1.91:1) |
| Max images | 10 (carousel) |
| File format | JPG, PNG |
| Caption | 2200 characters max |
| Hashtags | Max 30, optimal 5-10 |

**Best practices:**
- 4:5 portrait = most feed space
- First image = scroll-stopper
- Carousel = higher engagement
- Alt text helps accessibility + SEO
- Consistent grid aesthetic

### Instagram Stories

| Spec | Requirement |
|------|-------------|
| Aspect ratio | 9:16 |
| Resolution | 1080x1920 px |
| Duration | 15 seconds per story |
| File format | JPG, PNG, MP4, MOV |
| Safe zones | Top/bottom 250px for UI |

**Best practices:**
- Interactive stickers boost engagement
- Poll/quiz = 2x engagement
- Swipe-up (or link sticker) for 10k+
- Highlight covers matter for profile
- Post 3-7 stories daily

### LinkedIn

| Spec | Requirement |
|------|-------------|
| Image post | 1200x627 px (1.91:1) |
| Square post | 1200x1200 px (1:1) |
| Portrait | 1080x1350 px (4:5) |
| Carousel | PDF, up to 300 pages |
| Video | 16:9 or 1:1, max 10 min |
| Text post | 3000 characters, 1300 optimal |

**Best practices:**
- PDFs for carousels (better quality)
- First line = hook (shows in preview)
- Line breaks every 1-2 sentences
- Personal stories outperform corporate
- Engagement within first hour matters

### X/Twitter Images

| Spec | Requirement |
|------|-------------|
| Single image | 1200x675 px (16:9) or 1200x1200 (1:1) |
| Two images | 700x800 px each |
| Three images | 700x800 (left), 1200x686 (right stack) |
| Four images | 1200x600 px each |
| File format | JPG, PNG, GIF |
| File size | Max 5MB (images), 15MB (GIFs) |

**Best practices:**
- Single image outperforms multi
- Text on image = more saves
- 16:9 for link previews
- Alt text required for accessibility
- GIFs autoplay, catch attention

### Pinterest

| Spec | Requirement |
|------|-------------|
| Standard pin | 1000x1500 px (2:3) |
| Square pin | 1000x1000 px (1:1) |
| Long pin | 1000x2100 px (max) |
| Video pin | 9:16, 4:5, 1:1, or 16:9 |
| File format | JPG, PNG, MP4, MOV |

**Best practices:**
- Vertical pins dominate feed
- Text overlay = more clicks
- Rich pins for articles/products
- Board organization matters
- Consistent posting > sporadic

## Audio Platforms

### Podcast (Apple/Spotify)

| Spec | Requirement |
|------|-------------|
| Audio format | MP3 (preferred), M4A, WAV |
| Sample rate | 44.1 kHz |
| Bit rate | 128 kbps (mono), 256 kbps (stereo) |
| Cover art | 3000x3000 px (min 1400x1400) |
| Episode length | 20-60 min optimal |

**Best practices:**
- Mono for solo, stereo for interviews
- Consistent intro/outro branding
- Show notes with timestamps
- Chapters for navigation
- Transcripts improve SEO

### Audiogram/Waveform Video

| Spec | Requirement |
|------|-------------|
| Square | 1080x1080 px |
| Portrait | 1080x1350 px |
| Format | MP4 with waveform overlay |
| Duration | 30-60 seconds |
| Tools | Headliner, Wavve, Descript |

## Text Platforms

### Blog/Website

| Spec | Requirement |
|------|-------------|
| Word count | 1500-2500 (SEO optimal) |
| Featured image | 1200x630 px (OG image) |
| Meta description | 155-160 characters |
| Title length | 50-60 characters |
| URL slug | Lowercase, hyphens, no stop words |

**Best practices:**
- H2/H3 structure for scanning
- Internal links (3-5 per post)
- External links to authoritative sources
- Alt text on all images
- Schema markup for rich snippets

### Email Newsletter

| Spec | Requirement |
|------|-------------|
| Subject line | 40-50 characters optimal |
| Preview text | 40-130 characters |
| Body length | 500-1000 words |
| Image width | 600px max |
| CTA buttons | 44px minimum height |

**Best practices:**
- Mobile-first design (60%+ open mobile)
- Single column layout
- One primary CTA
- Plain text version for deliverability
- Unsubscribe link required

## Export Settings Cheatsheet

### Video Export (Premiere/Final Cut)

```
Codec: H.264
Container: MP4
Frame rate: 30 fps (60 for action)
Bitrate: 10-15 Mbps (1080p)
Audio: AAC, 128-256 kbps
```

### Image Export (Photoshop/Canva)

```
Format: PNG (graphics), JPG (photos)
Color: sRGB
Quality: 80-90%
Resolution: 72 PPI (web)
```

### Audio Export (Podcast)

```
Format: MP3
Bitrate: 128 kbps mono / 256 kbps stereo
Sample rate: 44.1 kHz
Loudness: -16 LUFS (Spotify), -19 LUFS (Apple)
```
