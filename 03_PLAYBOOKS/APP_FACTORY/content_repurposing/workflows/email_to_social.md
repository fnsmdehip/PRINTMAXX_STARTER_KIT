# Email to Social Workflow

Turn newsletter content into social posts across platforms.

---

## When to Use

- Newsletter issue performed well (high opens/clicks)
- Contains shareable insights
- Topic has broader social appeal
- You want to grow email list from social

---

## Why This Works

| Newsletter Strengths | Social Strengths |
|---------------------|------------------|
| Deep relationship | Broad discovery |
| High engagement | Viral potential |
| Owned audience | Algorithm amplification |
| Long-form okay | Forces clarity |

**The flywheel:**
Newsletter insight → Social post → New followers → Subscribe → Newsletter reader

---

## The Extraction Framework

### Newsletter Anatomy

Typical newsletter structure:

| Section | Social Potential |
|---------|-----------------|
| Subject line | Tweet/hook |
| Opening hook | Thread opener |
| Main insight | Core social post |
| Supporting points | Individual posts |
| Story/example | Story thread |
| CTA | Promotional post |

### What to Extract

From one newsletter, pull:

1. **1 main insight** → Full social post
2. **2-3 supporting points** → Individual posts or thread
3. **1 story** → Narrative thread
4. **3-5 quotable lines** → Quote cards or tweets
5. **Key stats** → Data graphics

---

## The Process

### Step 1: Review Performance

Before repurposing, check:
- Open rate (above your average?)
- Click rate (what links got clicks?)
- Replies (what sparked response?)

**High-performers get priority repurposing.**

### Step 2: Extract Atomic Content

Read through and highlight:
- Bold statements
- Specific numbers
- Actionable tips
- Personal stories
- Surprising insights

Mark each with target platform.

### Step 3: Transform for Platform

| Source Content | Transform Into |
|---------------|----------------|
| Opening hook | X post / thread hook |
| Main point | LinkedIn text post |
| Step-by-step | Carousel |
| Story | Thread or Reel |
| Stat | Quote card |
| CTA | Promotional post |

### Step 4: Schedule Across Week

Don't dump everything at once. Spread:

| Day | Post Type |
|-----|-----------|
| Send day | Tease newsletter, push subscribe |
| Day +1 | Main insight as X post |
| Day +2 | Story as thread |
| Day +3 | Supporting point on LinkedIn |
| Day +4 | Quote card on IG |
| Day +5 | Carousel from tips |

## Platform Adaptations

### X/Twitter

| Newsletter Element | X Format |
|-------------------|----------|
| Subject line | Tweet hook |
| Main insight | Standalone tweet |
| Multiple points | Thread |
| Story | Narrative thread |
| Stats | Data tweet with visual |

**X-specific tips:**
- First line = hook
- One idea per tweet
- Numbers perform well
- End with insight, not CTA

### LinkedIn

| Newsletter Element | LinkedIn Format |
|-------------------|-----------------|
| Full newsletter | Condensed post |
| Story | Personal narrative post |
| Tips | How-to post |
| Data | Results-focused post |

**LinkedIn-specific tips:**
- Professional but human
- First line shows in preview
- Personal stories 2x engagement
- End with question

### Instagram

| Newsletter Element | IG Format |
|-------------------|-----------|
| Tips/steps | Carousel |
| Quote | Quote card |
| Story | Reel |
| Data | Infographic |

**IG-specific tips:**
- Visual-first
- Captions secondary
- Save-worthy content
- Hashtags in caption

### TikTok/Reels

| Newsletter Element | Video Format |
|-------------------|--------------|
| Main insight | Talking head |
| Story | Narrative video |
| Tips | Quick-tip video |

**Video-specific tips:**
- Hook in 2 seconds
- Conversational delivery
- Captions required
- 45-60 seconds optimal

## List-Building Loop

### Social → Newsletter Growth

Every social post from newsletter should:
1. Deliver standalone value
2. Hint at deeper content
3. Include soft CTA to newsletter

**Soft CTA examples:**
- "Full breakdown in my newsletter"
- "I share stuff like this every [day]"
- "Link in bio for the complete guide"

**Not:**
- "Subscribe to my newsletter!!!"
- "Go sign up now!"
- Spammy link dumps

### Tracking the Loop

Monitor:
- Which social posts drive subscribes
- Which newsletter topics perform on social
- Conversion rate: social view → subscribe

## Repurposing Old Newsletters

### Mining the Archive

Your old newsletters are content gold:
1. Filter by open rate (top 20%)
2. Check if topics still relevant
3. Update stats/examples if needed
4. Repurpose as new social content

**Evergreen topics:**
- Frameworks
- Personal stories
- Lessons learned
- How-tos
- Mindset pieces

**Skip:**
- Time-sensitive news
- Outdated tool recommendations
- Event announcements
- Seasonal content (unless in season)

### Refresh Strategy

Every quarter:
1. Review top 5 newsletters from last year
2. Update any outdated information
3. Create fresh social content
4. Track as new repurposed content

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Copy-paste newsletter to social | Adapt format per platform |
| No standalone value | Social post must work alone |
| Spammy CTAs | Soft CTA or none |
| One-and-done | Spread content across week |
| Ignoring archive | Mine old newsletters |
| Same post everywhere | Adapt voice/format |
| No tracking | Log in tracker |
