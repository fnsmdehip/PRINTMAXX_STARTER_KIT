# Video to Blog Workflow

Transform videos into SEO-optimized articles.

---

## When to Use

- Video got strong engagement
- Topic has search volume
- Content is evergreen (not news/trends)
- Video explains a process or concept

---

## Why This Works

| Video Strengths | Blog Strengths |
|----------------|----------------|
| Engagement, watch time | SEO, long-term traffic |
| Platform algorithm | Google search |
| Personality, trust | Scannable, reference |
| Discovery | Deep-linking |

Combining both = maximum reach.

---

## The Process

### Step 1: Get the Transcript

**Free options:**
- YouTube auto-captions (download as .txt)
- Descript (free tier: 1 hour/month)
- Whisper (open source, local)
- CapCut (export captions as text)

**Paid options:**
- Descript Pro ($12/mo)
- Otter.ai ($16/mo)
- Rev ($1.50/min for human)

### Step 2: Clean the Transcript

Raw transcripts are messy. Clean up:

| Issue | Fix |
|-------|-----|
| Filler words | Delete "um," "uh," "like," "you know" |
| Run-on sentences | Add periods, break into shorter sentences |
| Verbal tics | Remove repeated phrases |
| Spoken grammar | Convert to written grammar |
| Tangents | Cut or move to sidebar |

**Example:**

*Raw transcript:*
> "So like, um, the first thing you want to do is, you know, open up your dashboard and like click on the settings button which is, um, in the top right corner..."

*Cleaned:*
> "First, open your dashboard and click Settings in the top right corner."

### Step 3: Structure for SEO

Transform cleaned transcript into blog structure:

```
# [Keyword-Rich Title] (H1)

[Hook paragraph - answer the question immediately]

## What You'll Learn (optional overview)
- Point 1
- Point 2
- Point 3

## [First Main Section] (H2)

[Content from video section 1]

### [Subsection if needed] (H3)

[Detailed content]

## [Second Main Section] (H2)

[Content from video section 2]

## [Continue for each section]

## Conclusion

[Summary + CTA]

## FAQ (schema-friendly)

### [Question from comments/video]
[Answer]
```

### Step 4: Add SEO Elements

**Required additions:**

| Element | What to Add |
|---------|-------------|
| Meta title | 50-60 chars, keyword front-loaded |
| Meta description | 155 chars, includes keyword + benefit |
| URL slug | lowercase-hyphenated-keywords |
| Alt text | Describe images with keywords |
| Internal links | 3-5 links to related content |
| External links | 1-2 authoritative sources |

**Keyword placement:**
- Title (H1): Primary keyword
- First paragraph: Primary keyword
- Subheadings: Secondary keywords
- Throughout: Natural variations

### Step 5: Enhance with Blog-Only Content

Videos can't do everything. Add:

| Addition | Example |
|----------|---------|
| Code blocks | Actual code snippets viewers can copy |
| Tables | Comparison charts |
| Checklists | Downloadable action items |
| Screenshots | Step-by-step visuals |
| Links | Resources mentioned |
| Updates | New info since video |

## SEO Research Before Writing

**Don't skip this.** Check before converting:

1. **Search volume**: Use Ahrefs, Ubersuggest, or Google Keyword Planner
2. **Competition**: Can you rank? Check top 10 results
3. **Intent match**: Are searchers looking for what your video covers?
4. **Related keywords**: Find secondary terms to include

**Quick check (free):**
- Google: Search your topic, see "People also ask"
- AnswerThePublic: Related questions
- AlsoAsked: Question clusters

|--------------|------------------|

## Time Investment

| Task | Time |
|------|------|
| Transcription | 10 min (automated) |
| Cleaning transcript | 20-30 min |
| SEO research | 15 min |
| Restructuring | 30 min |
| Adding enhancements | 30-45 min |
| Final editing | 15 min |
| **Total** | **2-2.5 hours** |

For a 10-minute video, expect 1500-2500 word article.

## Publishing Checklist

### Before Publishing

- [ ] Keyword in title, URL, first paragraph
- [ ] Meta description written (155 chars)
- [ ] All images have alt text
- [ ] Internal links added (3-5)
- [ ] External links added (1-2)
- [ ] Mobile preview checked
- [ ] Reading time appropriate (7-10 min optimal)

### On Publish

- [ ] Social share image set (1200x630)
- [ ] Published to blog
- [ ] Submitted to Google Search Console
- [ ] Shared on social with video link

### After Publishing

- [ ] Tracked in repurposing_tracker.csv
- [ ] Added to internal linking rotation
- [ ] Update video description with blog link
- [ ] Pinned comment on video linking to blog

---

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Direct transcript dump | Restructure for reading |
| No SEO research | Check search volume first |
| Missing visuals | Add screenshots, diagrams |
| No internal links | Link to related content |
| Duplicate title | Create unique blog title |
| No enhancement | Add tables, code, downloads |
