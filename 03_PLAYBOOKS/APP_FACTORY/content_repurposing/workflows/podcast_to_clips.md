# Podcast to Clips Workflow

Extract short clips from podcast episodes for social distribution.

---

## When to Use

- Podcast episode has strong moments
- Guests said quotable things
- Topic resonates with social audience
- You want to promote full episode

---

## The Clip Extraction Framework

### What Makes a Good Clip

| Element | Why It Works |
|---------|--------------|
| Strong opinion | Stops scroll, sparks reaction |
| Specific insight | Delivers immediate value |
| Story moment | Emotional hook |
| Surprising stat | Pattern interrupt |
| Quotable line | Shareable, memorable |
| Controversy | Drives engagement |

### Clip Length by Platform

| Platform | Optimal Length | Max Length |
|----------|---------------|------------|
| TikTok | 30-60 sec | 10 min |
| Reels | 15-30 sec | 90 sec |
| Shorts | 30-60 sec | 60 sec |
| X | 30-60 sec | 2:20 |
| LinkedIn | 30-90 sec | 10 min |

**Sweet spot: 45-60 seconds works everywhere.**

## The Process

### Step 1: Timestamp While Recording

During recording or editing, mark:
- Strong quotes (exact timestamp)
- Story beginnings
- Key insights
- Laugh moments
- Controversial takes

**Notation format:**
```
12:45 - Guest quote about failure
23:18 - Story: lost first client
34:02 - Hot take on AI
45:30 - Actionable tip on pricing
```

### Step 2: Transcribe the Episode

**Tools:**
- Descript ($12/mo) - Transcription + editing
- Otter.ai ($16/mo) - Live transcription
- Whisper (free) - Local, open source
- Riverside (if recorded there) - Built in

### Step 3: Select Clip Moments

From 60-minute episode, target:
- 5-10 potential clips
- 3-5 final clips after review
- 1 "hero" clip for main promotion

**Selection criteria:**

| Criteria | Weight |
|----------|--------|
| Standalone value (makes sense without context) | High |
| Scroll-stopping hook | High |
| Social-friendly length | Medium |
| Represents episode well | Medium |
| Guest looks/sounds good | Low (but matters) |

### Step 4: Edit the Clips

**Basic edit checklist:**
- [ ] Clean start (no "umm" or trailing words)
- [ ] Clean end (complete thought)
- [ ] Captions added
- [ ] Aspect ratio correct (9:16 for short-form)
- [ ] Audio levels normalized
- [ ] Visual interest (wave, captions, B-roll)

**Advanced edits:**
- Add progress bar
- Split screen (two people)
- B-roll overlay
- Animated captions (word-by-word)
- Brand intro/outro (2-3 seconds max)

### Step 5: Optimize for Each Platform

**TikTok:**
- Vertical crop (focus on speaker)
- Auto-captions (CapCut style)
- Hook in first 2 seconds
- No watermarks from other platforms

**Reels:**
- Cover image for grid
- Hashtags in caption
- Different hook than TikTok

**Shorts:**
- #Shorts in title
- More educational framing
- Subscribe CTA

**X:**
- Square (1:1) or horizontal works
- Quote pull in tweet text
- Thread potential

**LinkedIn:**
- More professional framing
- Context in post text
- Tag guest

### Original Podcast Moment

```

Guest: "They build in secret. They think they need a perfect
```

**Cut points:**
- Start: 34:22 (after "biggest mistake")
- End: 35:40 (after "solving their problem")
- Duration: 78 seconds

**Trim decision:**
- Cut host question (self-explanatory in context)
- Start with "They build in secret"
- Or add text overlay: "Biggest founder mistake?"

## Clip Editing Tools

### Free Options

| Tool | Best For | Limitations |
|------|----------|-------------|
| CapCut | Auto-captions, templates | Some features paid |
| DaVinci Resolve | Pro-level editing | Steep learning curve |
| Canva | Simple clips | Limited video features |
| Headliner | Audiograms | Watermark on free |

### Paid Options

| Tool | Price | Best For |
|------|-------|----------|
| Descript | $12/mo | Transcription + edit |
| Opus Clip | $19/mo | AI clip extraction |
| Riverside | $15/mo | Record + clip |
| Premiere Pro | $23/mo | Full production |

### AI-Assisted Clipping

**Opus Clip, Vizard, and similar tools:**
- Upload full episode
- AI identifies "viral" moments
- Outputs ready-to-post clips
- Saves 2-3 hours per episode

**Pros:**
- Fast (10 min vs 2 hours)
- Finds moments you'd miss
- Auto-captions

**Cons:**
- AI picks aren't always best
- Generic feel without customization
- Cost adds up

**Recommendation:** Use AI for first pass, manually select and polish final clips.

## Clip Types to Extract

### 1. The Hot Take

Strong opinion that sparks reaction.

**Example:** "Email marketing is dead for B2B. Here's what's working instead."

**Platform:** X, LinkedIn

### 2. The Tactical Tip

Specific, actionable advice.

**Example:** "Here's the exact cold email script that gets 40% replies."

**Platform:** TikTok, Reels, Shorts

### 3. The Story

Personal narrative with lesson.

**Example:** "I lost my first client because I made this mistake..."

**Platform:** All platforms (stories work everywhere)

### 4. The Quotable Line

Memorable, shareable statement.

**Example:** "The market doesn't care about perfect. They care about solved."

**Platform:** Quote cards (static), all video

### 5. The Data Point

Surprising stat or result.

**Example:** "We A/B tested this for 6 months. Version B won by 340%."

**Platform:** LinkedIn, X (data performs well)

### 6. The Disagreement

Guest challenges common wisdom.

**Example:** "Everyone says niched down. I think that's terrible advice for beginners."

**Platform:** X (drives debate), TikTok (controversy)

## Tracking Performance

### Log Each Clip

In `templates/repurposing_tracker.csv`:
- Episode source
- Clip timestamp
- Platform posted
- Views/engagement
- Click-throughs to full episode

### Identify Patterns

After 10+ episodes:
- Which topics make best clips?
- Which guests drive engagement?
- Which clip types perform per platform?
- What length works best?

Use patterns to inform future recording.

## Promotion Strategy

### Cross-Platform

1. Post hero clip everywhere on release day
2. Stagger other clips over the week
3. Reshare best performers after 2-4 weeks
4. Create compilation clips monthly

### Guest Amplification

**Make clips easy for guests to share:**
- Send clips directly (WeTransfer, DM)
- Provide suggested captions
- Tag them when posting
- Ask for repost after 24 hours

Guest shares = 2x reach minimum.

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Clip needs context | Must stand alone |
| Too long | 45-60 sec sweet spot |
| No hook | First 2 sec = grab attention |
| No captions | 85% watch muted |
| Same clip everywhere | Adapt per platform |
| Guest not tagged | Tag for amplification |
| No CTA | Point to full episode |
