# Blog to Carousel Workflow

Transform blog posts into Instagram/LinkedIn carousels.

---

## When to Use

- Blog post has clear sections
- Content can be broken into visual points
- Topic appeals to social audience
- You want to drive traffic to full article

---

## Why Carousels Work

| Metric | Single Image | Carousel |
|--------|-------------|----------|
| Engagement rate | 1.74% | 3.15% |
| Save rate | Low | High |
| Time on post | 2 sec | 15+ sec |
| Algorithm favor | Standard | Boosted |

Carousels = higher engagement + saves = more reach.

---

## The Process

### Step 1: Identify Carousel-Worthy Posts

**Good candidates:**
- Listicles (5 ways to..., 7 mistakes...)
- How-to guides with steps
- Comparison posts (X vs Y)
- Framework explanations
- Stat-heavy content

**Skip:**
- Opinion pieces
- News/time-sensitive content
- Posts without clear structure
- Complex technical deep-dives

### Step 2: Extract Core Points

From a 2000-word blog, pull:

| Slide | Content |
|-------|---------|
| 1 | Hook (title + visual interest) |
| 2-8 | One point per slide |
| 9 | Summary or key takeaway |
| 10 | CTA (save, follow, comment) |

**Rule:** If a point needs more than 30 words, split it.

### Step 3: Write Slide Copy

**Per slide:**
- Headline: 5-8 words max
- Body: 15-25 words max
- Total text: Under 50 words

**Example transformation:**

*Blog paragraph:*
> "The first step in building an email list is creating a lead magnet that solves a specific problem for your audience. This could be a PDF checklist, a mini-course, a template, or a resource list. The key is making it specific and immediately actionable."

*Carousel slide:*
> **Create One Lead Magnet**
>
> Solve one specific problem.
> PDF, template, or checklist.
> Make it immediately actionable.

### Step 4: Design the Carousel

**Design principles:**
- Consistent template across all slides
- Brand colors and fonts
- Readable on mobile (text size 24pt+)
- Visual hierarchy (headline > body)
- "Swipe" indicator on early slides

**Layout options:**

```
LAYOUT 1: Centered
+------------------+
|     HEADLINE     |
|                  |
|    Body text     |
|    goes here     |
|                  |
|    [visual]      |
+------------------+

LAYOUT 2: Top-Aligned
+------------------+
| HEADLINE         |
| Body text here   |
|                  |
|                  |
|    [visual]      |
+------------------+

LAYOUT 3: Split
+------------------+
| [icon]  HEADLINE |
|         -------- |
|         Body     |
|         text     |
|         here     |
+------------------+
```

## Platform-Specific Adjustments

### Instagram Carousel

| Spec | Recommendation |
|------|----------------|
| Slides | 5-10 (10 max) |
| Size | 1080x1350 (4:5 portrait) |
| Format | PNG or JPG |
| Caption | 300-500 words, CTA, hashtags |

**Caption formula:**
```
[Hook that creates curiosity]

[Brief summary of carousel value]

[Bullet points of what they'll learn]

[CTA: Save this | Share with someone | Follow for more]

#hashtags (5-10 relevant)
```

### LinkedIn Carousel

| Spec | Recommendation |
|------|----------------|
| Slides | 8-15 |
| Size | 1080x1350 (4:5) or 1920x1080 (PDF) |
| Format | PDF (better quality) |
| Text post | 200-300 words with key insight |

**LinkedIn text formula:**
```
[Bold hook statement]

[Why this matters to your audience]

[Key insight from carousel]

[CTA: Thoughts? | Share if useful | Follow for more]
```

**PDF carousel tips:**
- Create in Canva, export as PDF
- PDFs render crisper than images
- Add page numbers for reference
- Include clickable links (LinkedIn supports)

## Design Templates

### Canva Templates

Search for:
- "Instagram carousel template"
- "LinkedIn carousel template"
- "Educational carousel"

**Free templates work fine.** Customize colors/fonts to brand.

### DIY Design Rules

| Element | Guideline |
|---------|-----------|
| Font size | 24pt minimum (body), 36pt+ (headlines) |
| Colors | Max 3 colors per carousel |
| Contrast | Dark text on light, or vice versa |
| Whitespace | 20% minimum margins |
| Images | Optional, don't overcrowd |

### Brand Consistency

Create a template with:
- Your brand colors
- 2 fonts (headline + body)
- Logo placement (corner, last slide)
- Consistent layout per slide type

Save as template for future carousels.

## Carousel Writing Tips

### Headlines That Stop Scroll

- Use numbers: "5 ways to..." "The 3-step..."
- Create curiosity: "The trick no one talks about"
- Make a promise: "Double your [result] with this"
- Call out audience: "Founders: stop doing this"

### Body Copy That Flows

- One idea per slide
- Short sentences (8 words max ideal)
- Action verbs
- Specific > vague

### CTAs That Convert

| Goal | CTA |
|------|-----|
| Engagement | "Comment your biggest challenge" |
| Saves | "Save this for your next [task]" |
| Shares | "Tag someone who needs this" |
| Traffic | "Full guide: link in bio" |
| Follows | "Follow for daily tips on [topic]" |

## Publishing Strategy

### Instagram Best Times

- Tuesday-Friday: 11 AM - 1 PM
- Avoid: Saturday morning, Sunday evening
- Test your specific audience

### LinkedIn Best Times

- Tuesday-Thursday: 8 AM - 10 AM
- Avoid: Weekends, late night
- Early morning catches commuters

### Posting Checklist

- [ ] Carousel exported correctly (check file size)
- [ ] Cover slide is scroll-stopping
- [ ] Caption written with CTA
- [ ] Hashtags researched (IG)
- [ ] Posted at optimal time
- [ ] Engaged with comments in first hour
- [ ] Tracked in repurposing_tracker.csv

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Too much text per slide | Max 50 words per slide |
| Inconsistent design | Use templates |
| Weak first slide | Hook must stop scroll |
| No CTA | Always include clear CTA |
| Posting and ghosting | Engage with comments |
| Not repurposing winners | Turn top carousels into threads |
