# Thread to Video Workflow

Transform X threads into TikTok/Reels scripts.

---

## When to Use

- Thread got 50+ likes (validated content)
- Educational or how-to content
- Contains clear steps or list
- Has storytelling arc

---

## The Process

### Step 1: Analyze the Thread

Pull metrics from your thread:
- Which tweets got most engagement?
- Where did people reply/quote?
- What questions came up?

**High engagement = high interest = video priority**

### Step 2: Extract the Core

From a 10-tweet thread, identify:

| Element | Source | Use In Video |
|---------|--------|--------------|
| Hook | Tweet 1 | First 2 seconds |
| Main promise | Tweet 1-2 | Opening statement |
| Key points | Tweets 3-8 | Body content |
| Proof/example | Any tweet | B-roll or text overlay |
| CTA | Tweet 10 | Closing |

### Step 3: Write the Script

**Script Template:**

```
HOOK (0-2 sec):
[Pattern interrupt or bold claim from thread]

SETUP (2-5 sec):
[Why should viewer care? Stakes or benefit]

POINT 1 (5-12 sec):
[First key insight from thread]
[Supporting detail or example]

POINT 2 (12-20 sec):
[Second key insight]
[Quick proof or visual]

POINT 3 (20-30 sec):
[Third key insight]
[Application or result]

CLOSE (30-40 sec):
[Summary of value]
[CTA: follow, comment, save]
```

## Format Variations

### Talking Head (Easiest)

- You on camera
- Text overlays for key points
- Good for authority building
- Tools: Phone camera, CapCut

### Voiceover + B-Roll

- Your voice over relevant footage
- Stock video or screen recordings
- Good for tutorials
- Tools: Descript, CapCut, Pexels

### Text-Only

- No voice, just text on screen
- Trending audio in background
- Good for shy creators
- Tools: CapCut templates

### Split Screen

- You + visuals side by side
- Good for reactions or comparisons
- Tools: CapCut, Premiere

---

## Platform Adjustments

### TikTok Version

- Use trending audio if fits
- More casual, faster pacing
- Hook must stop scroll immediately
- CTAs: "Follow for part 2"

### Instagram Reels Version

- Cover image matters (shows on grid)
- Slightly more polished vibe
- Hashtags in caption
- CTAs: "Save this for later"

### YouTube Shorts Version

- Can be more educational
- Title with keywords
- End with subscribe CTA
- Link to long-form in comments

## Production Checklist

### Pre-Production
- [ ] Thread identified with good engagement
- [ ] Core points extracted (3-5 max)
- [ ] Script written and timed
- [ ] Hook tested aloud (does it grab?)

### Production
- [ ] Good lighting (face visible)
- [ ] Clean audio (no echo/background)
- [ ] Filmed in vertical (9:16)
- [ ] Multiple takes of hook

### Post-Production
- [ ] Captions added (CapCut auto)
- [ ] Text overlays for key points
- [ ] Cuts to remove "ums" and pauses
- [ ] Music/audio added (if using)
- [ ] Exported at 1080x1920

### Publishing
- [ ] Platform-specific caption written
- [ ] Hashtags added (TikTok/Reels)
- [ ] Posted at optimal time
- [ ] Tracked in repurposing_tracker.csv

## Time Investment

| Task | Time |
|------|------|
| Thread analysis | 5 min |
| Script writing | 15 min |
| Recording | 15-30 min |
| Editing | 20-30 min |
| Publishing | 10 min |
| **Total** | **65-90 min** |

With practice, this drops to 45-60 min per video.

---

## Thread Selection Criteria

**Prioritize threads that:**
- Got 50+ likes (content validated)
- Have clear numbered points
- Include specific numbers/results
- Tell a story with before/after
- Answer common questions

**Skip threads that:**
- Are mostly opinions/hot takes
- Require extensive visuals to explain
- Are time-sensitive (news, trends)
- Need heavy context from replies

## Quick Script Templates

### The "How I Did X" Format

```
Hook: "I [achieved result] in [timeframe]."
Setup: "Here's exactly how, step by step."
Points: [3-5 numbered steps]
Close: "That's it. Simple but effective. Follow for more."
```

### The "Mistakes to Avoid" Format

```
Hook: "Stop doing this. It's killing your [metric]."
Setup: "I made these mistakes for [time]. Don't repeat them."
Points: [3 mistakes with fixes]
Close: "Now you know. Save this and share with someone who needs it."
```

### The "Comparison" Format

```
Hook: "X vs Y. Which is better?"
Setup: "I tested both for [timeframe]."
Points: [Head-to-head comparisons]
Close: "Winner: [X/Y] for [specific use case]. Comment which you prefer."
```
