# PromptVault - Competitive Analysis

---

## Direct Competitors

### 1. PromptBase

**Model:** Marketplace for buying/selling prompts
**Pricing:** Individual prompt purchases ($2-10 each)
**Traffic:** ~500k monthly visitors

**What they do well:**
- First-mover advantage
- Large prompt variety
- Seller ecosystem
- DALL-E, Midjourney focus

**What they miss:**
- Inconsistent quality (user-submitted)
- No AI enhancement features
- Pay-per-prompt friction
- No organization tools
- Image prompts dominate (text prompts sparse)

**Our differentiation:**
- Subscription model (unlimited access)
- Curated, tested quality
- AI improvement features
- Better for text/chat prompts

---

### 2. AIPRM (Chrome Extension)

**Model:** Freemium Chrome extension
**Pricing:** Free + $9/mo Pro
**Users:** 1M+ extension installs

**What they do well:**
- Deep ChatGPT integration
- One-click prompt insertion
- Community templates
- Large user base

**What they miss:**
- Chrome only (no mobile)
- Community prompts = noise
- No AI generation/improvement
- Only works in ChatGPT
- Extension-dependent

**Our differentiation:**
- Cross-platform (web, mobile, extension)
- Works with any AI (Claude, Gemini, etc.)
- AI-powered features
- Curated quality over community noise

---

### 3. FlowGPT

**Model:** Community prompt sharing
**Pricing:** Free (ad-supported)
**Traffic:** ~2M monthly visitors

**What they do well:**
- Large community
- Social features (likes, comments)
- Persona/character prompts
- High traffic

**What they miss:**
- Quality control issues
- Cluttered, noisy experience
- Character/roleplay focus (not business)
- No premium features
- Ad-supported = user is product

**Our differentiation:**
- Professional/business focus
- Curated, not crowdsourced
- Clean, organized UX
- Premium tier with AI features

---

### 4. PromptHero

**Model:** Prompt gallery (image AI focus)
**Pricing:** Free
**Traffic:** ~1M monthly visitors

**What they do well:**
- Beautiful image prompt gallery
- Midjourney/Stable Diffusion focus
- Search and filter

**What they miss:**
- Almost entirely image-focused
- Text prompts are afterthought
- No AI enhancement
- No organization

**Our differentiation:**
- Text/chat prompt focus
- Business use cases
- AI improvement features
- Organization tools

---

### 5. ChatGPT Prompt Library (Various)

**Model:** Various free websites/PDFs
**Pricing:** Free
**Examples:** GitHub repos, Notion templates, PDFs

**What they do well:**
- Free
- Some high-quality collections
- Community contributions

**What they miss:**
- Scattered, hard to find
- No search/organization
- Static (not updated)
- No AI features
- Not productized

**Our differentiation:**
- Centralized, searchable
- Regularly updated
- AI enhancement features
- Proper product experience

---

## Adjacent Competitors

### 6. ChatGPT Plus (with Custom GPTs)

**Model:** OpenAI's premium tier
**Pricing:** $20/mo

**What they do well:**
- Native integration
- Custom GPTs = specialized prompts
- GPT store
- OpenAI brand trust

**What they miss:**
- Custom GPTs are complex to use
- No prompt library per se
- Locked to OpenAI ecosystem
- No prompt improvement features

**Our differentiation:**
- Works with any AI
- Simpler prompt library format
- AI improvement for any prompt
- Lower price for prompt access

---

### 7. Copy.ai / Jasper

**Model:** AI writing tools with templates
**Pricing:** $49-$125/mo

**What they do well:**
- Templates are prompts
- Polished output
- Business focus
- Team features

**What they miss:**
- Very expensive
- Closed system (can't use elsewhere)
- Overkill for many use cases
- Templates vs prompts different

**Our differentiation:**
- 10x cheaper
- Use prompts anywhere
- More flexible
- AI improvement feature

## Market Gap Analysis

### Unserved needs:

1. **Curated text prompt library**
   - PromptBase is marketplace (inconsistent)
   - FlowGPT is community (noisy)
   - No premium curated option

2. **AI prompt enhancement**
   - No competitor offers AI improvement
   - No competitor offers prompt generation
   - Unique feature opportunity

3. **Cross-platform prompt access**
   - AIPRM is Chrome-only
   - Others are web-only
   - Mobile is ignored

4. **Business/professional prompts**
   - FlowGPT is character/persona focused
   - PromptHero is image focused
   - Business use cases underserved

### Against FlowGPT:
"Professional prompts, not roleplay characters. Curated quality, not community noise."

## Competitive Response Plan

- Double down on multi-platform
- Build superior mobile experience
- Emphasize curated quality

### If PromptBase goes subscription:
- Compete on curation (not marketplace)
- Highlight AI features
- Focus on UX

### If major player enters (OpenAI, Notion, etc.):
- Already have user base as moat
- Niche down if needed
- Consider acquisition opportunity

---

## Key Insights

1. **No one owns "curated + AI-enhanced" position**
   - Clear white space
   - Unique value proposition

2. **Image prompts dominate, text underserved**
   - Midjourney/DALL-E get most attention
   - ChatGPT/Claude prompts less organized

3. **Quality over quantity wins**
   - Community platforms = noise
   - Curation is a feature

4. **AI features are differentiator**
   - No competitor has prompt improvement
   - Build moat with AI capabilities

5. **Cross-platform is rare**
   - Most are web-only or extension-only
   - Mobile is opportunity
