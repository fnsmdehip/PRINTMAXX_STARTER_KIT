## VA instructions

For VAs doing gym outreach:

1. **Find gyms:**
   - Google "[City] gyms"
   - Yelp gym listings
   - Google Maps scraping
   - Focus on independent gyms first (easier decision makers)

2. **Find contact info:**
   - Website "Contact" or "About" page
   - Social media profiles
   - Google Business listing

3. **Personalize each email:**
   - Mention the gym name
   - Reference something specific if possible
   - Keep it short

4. **Track everything:**
   - Log in spreadsheet immediately after sending
   - Set follow-up reminders
   - Update status after each response

5. **Escalate positive responses:**
   - Forward to [Name] immediately
   - Provide all context
