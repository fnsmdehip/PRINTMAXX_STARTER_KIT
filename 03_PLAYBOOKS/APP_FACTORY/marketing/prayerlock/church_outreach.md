## VA instructions

For VAs doing church outreach:

1. **Find churches:**
   - Google "[City] churches"
   - Check church directories
   - Look for active social media presence (means they're engaged)

2. **Find contact info:**
   - Website "Contact" page
   - Staff directory
   - Social media profiles

3. **Personalize each email:**
   - Mention something specific from their website/social
   - Use correct names
   - Never send generic emails

4. **Track everything:**
   - Log in spreadsheet immediately after sending
   - Set follow-up reminders in calendar
   - Update status after each response

5. **Escalate positive responses:**
   - Forward to [Name] immediately
   - Don't make commitments without approval
   - Provide all context in escalation
