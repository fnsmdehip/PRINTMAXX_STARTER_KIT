### Section 1: Quick Background (150 words)
- Zapier: Pioneer, most integrations, higher price
- Make (formerly Integromat): Visual builder, cheaper, steeper learning

### Section 2: Interface Comparison (400 words)

**Zapier Interface**
- Linear workflow (trigger → action → action)
- Clean, simple UI
- Quick to learn
- Screenshot: Simple automation

**Make Interface**
- Visual canvas with branching
- More complex at first
- More powerful once learned
- Screenshot: Complex workflow

**Verdict:** Zapier easier to start, Make better for complex flows

### Section 3: Integrations (300 words)

| Platform | Native Integrations | Webhooks | API |
|----------|---------------------|----------|-----|
| Zapier | 5,000+ | Yes | Yes |
| Make | 1,000+ | Yes | Yes |

**Key differences:**
- Zapier has more obscure app connections
- Make's integrations often have more features per app
- Both support webhooks for custom connections

**Verdict:** Zapier wins on quantity, Make often wins on depth

### Section 4: Pricing Deep Dive (500 words)

**Zapier Pricing**
- Free: 100 tasks/month, 5 Zaps
- Starter: $19.99/mo - 750 tasks
- Professional: $49/mo - 2,000 tasks
- Team: $69/mo - 2,000 tasks + collaboration

**Make Pricing**
- Free: 1,000 ops/month
- Core: $9/mo - 10,000 ops
- Pro: $16/mo - 10,000 ops + more features
- Teams: $29/mo - 10,000 ops + collaboration

**The Math:**
- 1 Zapier task = roughly 1-3 Make operations
- For same workflow: Zapier ~3x more expensive
- Example: 5,000 monthly runs
  - Zapier: $49/mo (Professional)
  - Make: $9/mo (Core)

**Verdict:** Make significantly cheaper at scale

### Section 5: Features Comparison (400 words)

| Feature | Zapier | Make |
|---------|--------|------|
| Multi-step workflows | Yes | Yes |
| Branching/routers | Basic | Advanced |
| Error handling | Basic | Advanced |
| Scheduling | Yes | Yes |
| Filters | Yes | Yes |
| Data stores | No | Yes |
| Iterations | Limited | Full |
| AI actions | Yes (new) | Yes |

**Key advantages:**
- Zapier: Simpler, faster setup, more apps
- Make: Branching, iterations, data stores, cheaper

### Section 6: Real-World Tests (400 words)

**Test 1: Simple Email to Spreadsheet**
- Zapier: 5 minutes to set up
- Make: 8 minutes to set up
- Winner: Zapier (simpler task)

**Test 2: Complex Lead Routing**
- Zapier: Couldn't do conditional branching easily
- Make: Native router module, 15 minutes
- Winner: Make (complex logic)

**Test 3: High-Volume Social Posting**
- Zapier: $49/mo for volume needed
- Make: $9/mo for same volume
- Winner: Make (cost)

### Section 7: When to Choose Which (300 words)

**Choose Zapier if:**
- You're non-technical
- You need obscure app integrations
- Simple workflows only
- Time matters more than money

**Choose Make if:**
- You have complex branching needs
- High volume of operations
- You're comfortable with visual programming
- Budget is a concern

### Section 8: Migration Tips (150 words)

- Rebuild vs migrate considerations
