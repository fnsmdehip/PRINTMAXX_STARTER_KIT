# AFFILIATE_SITES

SEO-optimized sites with affiliate links. Product reviews, comparisons.

## Status: Planning

## Revenue Model
- Commission per sale (3-50% depending on program)
- Target: $500+/month per site at scale

## Execution Steps

1. Pick niche with high-commission affiliate programs
2. Keyword research for buyer-intent terms
3. Build site with Next.js (fast, SEO-friendly)
4. Create comparison and review content
5. Build backlinks through outreach
6. Monitor rankings and optimize

## Folder Contents

- products/ - Individual site projects
- marketing/ - Link building strategies, social promotion
- research/ - Keyword research, competitor analysis
- gtm/ - Content calendars, launch plans
- assets/ - Site mockups, logos, featured images

## Key Metrics

- Organic traffic
- Click-through rate to affiliate links
- Conversion rate
- Revenue per 1000 visitors

## Affiliate Programs to Consider

- Amazon Associates (low commission, high volume)
- SaaS affiliate programs (high commission)
- Finance products (high commission)
- Course platforms (moderate commission)

## Tools

- Ahrefs/SEMrush for keyword research
- Next.js for site development
- ThirstyAffiliates for link management
- Google Search Console for tracking
