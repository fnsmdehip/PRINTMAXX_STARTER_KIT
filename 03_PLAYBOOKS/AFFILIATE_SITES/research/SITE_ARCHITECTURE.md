# Affiliate Site Architecture

**Updated:** 2026-01-20
**Purpose:** Optimal site structure for SEO rankings and affiliate conversions

---

## Architecture Overview

### The Hub-and-Spoke Model

```
                    [Homepage]
                        |
        ┌───────────────┼───────────────┐
        |               |               |
   [Category Hub]  [Category Hub]  [Category Hub]
        |               |               |
   ┌────┴────┐    ┌────┴────┐    ┌────┴────┐
   |    |    |    |    |    |    |    |    |
[Post][Post][Post][Post][Post][Post][Post][Post][Post]
```

**Why this works:**
- Clear topical hierarchy for Google
- Internal linking builds authority
- Users navigate naturally
- Each hub can rank for competitive terms
- Posts capture long-tail keywords

## Essential Page Types

### 1. Homepage

**Purpose:** Brand introduction, category navigation, authority signals

**Required elements:**
- Clear value proposition (1 sentence)
- Category navigation (3-5 main topics)
- Featured/popular content
- Trust signals (featured in, credentials)
- Email capture (if list building)

**Template structure:**
```
[Hero: Value prop + main CTA]
[Category cards: 3-5 main topics]
[Featured posts: Best performers]
[Trust bar: Featured in logos / social proof]
[Email signup: Above footer]
[Footer: Navigation + legal pages]
```

### 2. Category Hubs (Pillar Pages)

**Purpose:** Rank for main keywords, organize content, build topical authority

**Required elements:**
- Comprehensive overview of topic (2000+ words)
- Table of contents with jump links
- Internal links to all related posts
- External links to authoritative sources
- Key takeaways/summary box
- Multiple affiliate links woven naturally

**Example structure:**
```

[Updated date + reading time]

[Quick answer box: Top pick + why]

## Table of Contents
1. Our Top Picks
2. How We Evaluated
3. Detailed Reviews
4. Buying Guide
5. FAQs

## Our Top Picks
[Comparison table with affiliate links]

## How We Evaluated
[Methodology explanation]

## Detailed Reviews
[Each product: 300-500 words + pros/cons + affiliate link]

## Buying Guide
[What to consider when choosing]

## Frequently Asked Questions
[Schema-optimized Q&A]

## Related Reading
[Links to spoke content]
```

## URL Structure

### Recommended Format

```
yourdomain.com/                           # Homepage
yourdomain.com/category-name/             # Category hub
yourdomain.com/category-name/post-slug/   # Individual posts
yourdomain.com/reviews/                   # Review hub (optional)
yourdomain.com/reviews/product-name/      # Individual reviews
```

### URL Best Practices

- Use lowercase
- Use hyphens, not underscores
- Keep URLs short (3-5 words)
- Include primary keyword
- No dates in URLs (evergreen)
- No stop words unless natural

**Good:** `/best-crm-software/`
**Bad:** `/the-10-best-crm-software-solutions-for-2026/`

## Internal Linking Strategy

### Link Hierarchy

```
Homepage
  ↓ links to
Category Hubs
  ↓ links to
Individual Posts
  ↓ links back to
Category Hubs (and to related posts)
```

### Link Rules

1. **Every post links to its category hub**
2. **Category hubs link to all child posts**
3. **Related posts link to each other**
4. **Use descriptive anchor text** (not "click here")
5. **Limit to 3-5 internal links per 1000 words**
6. **Most important pages get most internal links**

### Anchor Text Examples

**Good:**
- "Check out our [full comparison of CRM tools]"
- "See our [HubSpot review] for more details"

**Bad:**
- "Click [here] to learn more"
- "Read more [about this topic]"

## Content Hierarchy

### Authority Distribution

```
Homepage (Domain authority)
    ↓
Category Hubs (Primary keywords, high competition)
    ↓
Comparison Posts (VS keywords, medium competition)
    ↓
Review Posts (Product keywords, medium competition)
    ↓
Tutorial Posts (How-to keywords, low competition)
    ↓
Supporting Content (Long-tail, very low competition)
```

### Content Volume Recommendations

| Site Stage | Category Hubs | Posts per Hub | Total |
|------------|---------------|---------------|-------|
| Launch | 3 | 5 | 18 |
| Month 3 | 3 | 15 | 48 |
| Month 6 | 5 | 20 | 105 |
| Month 12 | 5 | 40 | 205 |

## Essential Site Pages

### Required (Legal/Trust)

1. **About Page**
   - Who you are
   - Why you're qualified
   - Your review process
   - Contact info

2. **Affiliate Disclosure**
   - FTC-required disclosure
   - How you make money
   - Link on every page with affiliate links

3. **Privacy Policy**
   - Required for AdSense, many affiliate programs
   - Cookie notice if using EU traffic
   - Generate at termly.io or similar

4. **Terms of Service**
   - Liability limitations
   - Content usage rights

5. **Contact Page**
   - Form or email
   - Increases trust

### Recommended (Conversion)

1. **Resource Hub**
   - Free tools
   - Templates
   - Guides (lead magnets)

2. **Newsletter Signup**
   - Dedicated landing page
   - Capture emails for remarketing

## Technical SEO Requirements

### Site Speed

- Target: <3 second load time
- Use caching plugin (if WordPress)
- Optimize images (WebP format, lazy loading)
- Use CDN for static assets
- Minimize plugins/scripts

### Mobile Optimization

- Responsive design required
- Mobile-first indexing is default
- Test with Google Mobile-Friendly Test
- Thumb-friendly navigation
- No horizontal scrolling

### Schema Markup

**Implement on:**
- Reviews (Review schema)
- Products (Product schema)
- FAQ sections (FAQ schema)
- How-to guides (HowTo schema)
- Breadcrumbs (BreadcrumbList schema)

**Example Review Schema:**
```json
{
  "@context": "https://schema.org/",
  "@type": "Review",
  "itemReviewed": {
    "@type": "Product",
    "name": "Product Name"
  },
  "reviewRating": {
    "@type": "Rating",
    "ratingValue": "4.5",
    "bestRating": "5"
  },
  "author": {
    "@type": "Person",
    "name": "Your Name"
  }
}
```

## CMS Recommendations

### For Beginners: WordPress

**Pros:**
- Easy setup
- Huge plugin ecosystem
- Lots of tutorials
- Good SEO plugins (Yoast, RankMath)

**Recommended plugins:**
- RankMath or Yoast (SEO)
- WP Rocket or LiteSpeed (Caching)
- ShortPixel (Image optimization)
- ThirstyAffiliates (Link management)
- TablePress (Comparison tables)

### For Developers: Next.js + Headless CMS

**Pros:**
- Fast performance
- Modern stack
- Full control
- Easy to scale

**Stack:**
- Next.js (frontend)
- MDX or headless CMS (content)
- Vercel (hosting)
- Custom affiliate link tracking

### For Speed: Static Site Generators

**Options:**
- Astro (recommended for content sites)
- Hugo
- 11ty

**Pros:**
- Fastest possible load times
- Cheapest hosting
- Most secure

## Scaling Your Site

### Month 1-3: Foundation
- Focus on 1 category
- Publish 3-5 posts/week
- Build initial backlinks
- Start email list

### Month 3-6: Expansion
- Add 1-2 more categories
- Maintain publishing cadence
- Update existing content
- Build more backlinks

### Month 6-12: Optimization
- Analyze what's working
- Double down on winners
- Prune/update losers
- Add conversion optimization

### Year 1+: Scale
- Consider writers
- Expand categories
- Add monetization channels
- Explore adjacent niches
