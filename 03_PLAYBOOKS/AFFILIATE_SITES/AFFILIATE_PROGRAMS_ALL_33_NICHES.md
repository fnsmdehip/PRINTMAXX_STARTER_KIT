## Affiliate Strategy & Best Practices

### Content Types That Convert

| Content Type | Conversion Rate | Best For |
|--------------|-----------------|----------|
| Comparison posts ("X vs Y") | 5-8% | High-intent buyers |
| "Best of" roundups | 3-5% | Discovery phase |
| Tutorials with tools | 2-4% | Problem solvers |
| Reviews (honest) | 4-7% | Near-purchase |
| Case studies | 6-10% | High-ticket items |
| Email sequences | 2-5% | Nurturing leads |
| YouTube reviews | 3-6% | Visual products |

### Link Placement Best Practices

1. **In-content links** - Place naturally within valuable content
2. **Comparison tables** - High conversion, easy scanning
3. **Call-to-action buttons** - Use sparingly, test colors
4. **Resource pages** - Comprehensive tool lists
5. **Email newsletters** - Warm audience, higher trust
6. **YouTube descriptions** - Link first tool mentioned

### Tracking Setup

**Essential tracking:**
- UTM parameters on all links
- Sub-ID tracking per content piece
- Click-through rate by placement
- Conversion rate by traffic source
- Revenue per click (EPC)

**Recommended tools:**
- ThirstyAffiliates (WordPress)
- Pretty Links (link management)
- Google Analytics 4 (attribution)
- Affiliate network dashboards

## FTC Compliance Requirements

### Disclosure Requirements

**Required disclosure language:**
- "This post contains affiliate links. If you purchase through these links, I may earn a commission at no extra cost to you."
- Must be **clear and conspicuous**
- Must appear **before** affiliate links
- Must be visible on **all platforms** (blog, social, email)

### Placement Rules

| Platform | Disclosure Location |
|----------|---------------------|
| Blog posts | Top of post, above first link |
| YouTube | Video description + verbal mention |
| Instagram | #ad or #affiliate in caption |
| TikTok | Verbal disclosure + text overlay |
| Email | Header or footer disclosure |
| Podcasts | Verbal disclosure before/after |

### What NOT to Do

- Never hide disclosures in terms pages
- Never use tiny text or hidden elements
- Never claim "reviews are unbiased" if compensated
- Never make false income claims
- Never guarantee results from products

### Documentation

Keep records of:
- All affiliate agreements
- Commission statements
- Promotional materials used
- Disclosure compliance checks
