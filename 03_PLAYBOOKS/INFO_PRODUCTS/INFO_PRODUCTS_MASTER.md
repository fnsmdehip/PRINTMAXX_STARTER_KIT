# Info Products Master Framework

**Last Updated:** 2026-01-24
**Status:** Ready for implementation
**Platform:** Gumroad (primary), Loom (delivery), Notion (templates)

---

## Core Philosophy

no guru energy. no fake scarcity. no countdown timers.

the play: document what you actually do. package it. sell it once. let it compound.

people pay for:
1. time saved (give them the shortcut)
2. mistakes avoided (show what doesn't work)
3. working systems (not theory, actual files they can use)

---

## Funnel Architecture (All Products)

```
Free Content (Twitter/TikTok/YouTube)
    ↓
Lead Magnet (Notion template, checklist, mini-guide)
    ↓
Email Sequence (3-5 emails, value-first)
    ↓
Core Product ($27-97)
    ↓
Upsell (Done-with-you or premium tier)
    ↓
Backend (Consulting, agency, higher-ticket)
```

### Conversion Benchmarks

| Stage | Target Rate |
|-------|-------------|
| Content → Lead Magnet | 2-5% |
| Lead Magnet → Email Sub | 30-50% |
| Email → Core Product | 2-5% |
| Core → Upsell | 10-20% |

## Product Tiers (Use Across All Methods)

### Tier 1: Lead Magnet (Free)
- Notion template
- Checklist PDF
- Swipe file
- Mini Loom walkthrough (5-10 min)

**Goal:** Capture email, demonstrate value

### Tier 2: Core Product ($27-97)
- Full system documentation
- All templates/files
- Loom walkthroughs (30-60 min total)
- Community access (optional)

**Goal:** Deliver transformation, build trust

### Tier 3: Premium ($197-497)
- Everything in Core
- 1:1 onboarding call (30 min)
- Private Slack/Discord access
- Monthly group call
- Updates for life

**Goal:** Higher touch, recurring relationship

### Tier 4: Done-With-You ($997-2500)
- Full implementation support
- Weekly calls for 4-8 weeks
- Custom setup
- Priority support

**Goal:** High-ticket, limited spots

## Gumroad Setup (Standard for All)

### Product Page Structure

```
[Hero]
one line: what they get + result

[Social Proof]
3-5 real results (screenshots, numbers)

[The Problem]
2-3 sentences on pain point

[The Solution]
what's inside (bullet list, no fluff)

[What You Get]
- Item 1 (specific deliverable)
- Item 2 (specific deliverable)
- Item 3 (specific deliverable)

[Pricing]
single price, no fake discounts

[FAQ]
3-5 real questions

[Guarantee]
30-day refund, no questions
```

### Pricing Psychology

- $27: impulse buy, low friction
- $47: still impulse, slightly more committed
- $97: considered purchase, needs clear ROI
- $197+: needs proof, testimonials, maybe call

**Rule:** Price = 10x the value of time saved

## Loom Delivery System

### Why Loom
- Personal feel without live calls
- Async = scales infinitely
- Can update/replace without breaking links
- Embedded in Notion = clean delivery

### Loom Structure (Per Module)

```
1. Context (30 sec) - why this matters
2. The system (2-5 min) - show don't tell
3. Walk through (5-10 min) - do it live
4. Common mistakes (1-2 min) - what to avoid
5. Next step (30 sec) - what to do now
```

### Total Runtime Targets

| Product Tier | Loom Time |
|--------------|-----------|
| Lead Magnet | 5-10 min |
| Core ($27-47) | 30-45 min |
| Core ($67-97) | 45-90 min |
| Premium | 2-3 hours |

## Email Sequence Framework

### Sequence 1: Post Lead Magnet (3 emails)

**Email 1 (Immediate):** Deliver the thing
- Subject: here's [thing]
- Body: link + 1 quick win they can get today

**Email 2 (Day 2):** Quick win story
- Subject: [specific result] in [timeframe]
- Body: mini case study + link to lead magnet again

**Email 3 (Day 4):** Soft pitch
- Subject: the full system
- Body: problem → solution → what's in product → link

### Sequence 2: Post Purchase (3 emails)

**Email 1 (Immediate):** Welcome + quick start
- Subject: you're in. start here.
- Body: access link + first thing to do

**Email 2 (Day 3):** Check in + support
- Subject: how's it going?
- Body: common sticking point + how to fix + reply if stuck

**Email 3 (Day 7):** Upsell (if applicable)
- Subject: ready for the next level?
- Body: what they've learned → what's next → premium offer

## Copy Templates

### Product Title Formula
```
[Method] + [Result] + [Timeframe/Qualifier]

Examples:
- App Factory Blueprint: Launch Your First App in 30 Days
- Content Farm System: 10 Accounts, 600 Posts/Day, Zero Burnout
- Roblox Game Factory: Build Games With AI, Earn Robux Passively
```

### Hero Line Formula
```
the exact [system/templates/process] i use to [specific result]

Examples:
- the exact system i use to launch apps that make $2-5k/mo
- the templates i use to run 10 content accounts in 2 hours/day
- the process i use to build roblox games without coding
```

### Bullet Formula
```
[Specific deliverable] - [what it does for them]

Examples:
- 47 pre-written cold email templates - copy, paste, send
- Complete Notion dashboard - track all 10 accounts in one view
- 5 game templates - clone and customize in under an hour
```

## Anti-Guru Checklist

Before publishing any product:

- [ ] No fake countdown timers
- [ ] No "only 10 spots left" (unless true)
- [ ] No income screenshots without context
- [ ] No "this will change your life" claims
- [ ] Price is fair (10x value rule)
- [ ] 30-day refund, no friction
- [ ] Actually delivers what it promises
- [ ] You use this system yourself

## Product Launch Checklist

### Pre-Launch (Week -1)
- [ ] Product page live (unlisted)
- [ ] Looms recorded and uploaded
- [ ] Templates tested and cleaned
- [ ] Email sequences written
- [ ] Lead magnet ready
- [ ] 3-5 beta testers have used it

### Launch Day
- [ ] Make product public
- [ ] Post on Twitter/X
- [ ] Post on relevant subreddits
- [ ] Email list announcement
- [ ] Update link in bio

### Post-Launch (Week 1)
- [ ] Collect testimonials
- [ ] Answer support questions
- [ ] Note FAQ for product page
- [ ] Plan first update/improvement
