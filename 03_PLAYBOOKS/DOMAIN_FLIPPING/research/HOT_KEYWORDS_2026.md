## Domain Name Testing

### Quick Brandability Check

Score each 1-5:
1. **Pronounceable:** Can say it once and repeat?
2. **Spellable:** Can type without seeing?
3. **Memorable:** Remember tomorrow?
4. **Positive:** Good associations?
5. **Unique:** Stands out?

- **20+:** Premium brandable
- **15-19:** Good brandable
- **10-14:** Average
- **<10:** Pass

### Radio Test

Would this work in a podcast ad?

"Visit [domain].com to sign up"

If you'd need to spell it, it fails.
