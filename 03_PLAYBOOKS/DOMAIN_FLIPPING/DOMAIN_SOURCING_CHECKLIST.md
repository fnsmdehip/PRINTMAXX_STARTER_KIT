**Time required:** 30-60 min/day

## Daily Expired Domain Check (15 min)

### ExpiredDomains.net Filters

```
Extension: .com
Length: 3-12 characters
Majestic TF: 10+
Archive.org: Has data
```

- [ ] Check "Deleted Domains" tab
- [ ] Check "Pending Delete" tab
- [ ] Export promising domains to spreadsheet
- [ ] Cross-reference with NameBio for comparable sales

### Quick Validation (per domain)

- [ ] Wayback Machine - what was it? (avoid adult/spam)
- [ ] Google site:domain.com - any indexing? (avoid penalized)
- [ ] Majestic - clean backlinks? (avoid PBN/spam profiles)
- [ ] USPTO - trademark clear? (avoid legal issues)

## Monthly Portfolio Review

### Keep/Drop Decision Matrix

| Criteria | Keep | Drop |
|----------|------|------|
| Inquiries last 6 months | 1+ | 0 |
| Time listed | <12 months | 18+ months |
| Renewal cost vs value | <5% | >20% |
| Trend relevance | Growing/stable | Declining |
| Comparable sales | Found | None |

### Actions

- [ ] Review all domains expiring in 60 days
- [ ] Drop bottom 10-20% performers
- [ ] Reinvest savings into better inventory
- [ ] Update pricing based on market comps
- [ ] Send outreach for stale inventory
