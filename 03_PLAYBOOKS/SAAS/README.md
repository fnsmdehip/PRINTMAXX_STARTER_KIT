# SAAS

Software as a service. Recurring revenue products.

## Status: Planning

## Revenue Model
- Monthly subscriptions: $9-$99/month
- Annual plans at discount
- Target: 100+ paying customers

## Execution Steps

1. Identify repeatable pain point
2. Validate with landing page and waitlist
3. Build MVP (focus on core value prop)
4. Launch to waitlist
5. Iterate based on usage data
6. Scale with content marketing and referrals

- products/ - Individual SaaS projects
- marketing/ - Landing pages, email sequences, docs
- research/ - Customer interviews, competitor analysis
- gtm/ - Launch strategies, pricing experiments
- assets/ - Screenshots, demo videos, logos

## Key Metrics

- MRR (Monthly Recurring Revenue)
- Churn rate
- CAC (Customer Acquisition Cost)
- LTV (Lifetime Value)
