## Common questions during demos

### "Can I import from Notion/Google Docs?"
"Yes. Export as text or CSV, upload to StackPilot. Most people migrate in under 30 minutes."

### "Do I need API keys?"
"For running prompts: No. We handle the API connection. You pay us, we pay the AI providers.

For Team plan: Optional. You can use your own keys for higher limits or specific model access."

### "What about prompt security?"
"Your prompts are yours. We don't train models on them. We don't share them. You can export or delete anytime."

### "Can I run prompts in my own apps?"
"Pro plan includes basic API access. You can trigger prompts programmatically. Full API docs available."

### "What if the AI gives bad output?"
"Same as using the AI directly. StackPilot doesn't change the AI's responses. It just makes it easier to run good prompts consistently."

### "Is there a desktop app?"
"Web app only right now. Works great in the browser. Desktop app is on the roadmap if there's demand."
