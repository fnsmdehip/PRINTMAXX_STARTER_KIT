# PROMPT_ENGINEERING (MM033)

Sell prompts as products and prompt expertise as services.

## Quick Start

1. Pick one niche (faith, fitness, cold email, app marketing)
2. Write 10 prompts with proven outputs
3. Create Gumroad account
4. Price first pack at $37
5. Post 3x/week about prompt tips
