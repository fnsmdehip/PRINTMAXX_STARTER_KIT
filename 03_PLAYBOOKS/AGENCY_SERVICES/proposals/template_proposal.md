# Notes for using this template

**Before sending:**

1. Replace all [BRACKETS] with specific information
2. Remove sections that don't apply
3. Add specifics from discovery call
4. Include sample work if relevant
5. Proofread for client name consistency
6. Set realistic timeline based on your capacity

**Best practices:**

- Send within 24-48 hours of discovery call
- Keep it to 2-3 pages max
- Be specific about what's not included
- Price confidently (don't apologize)
- Include a clear next step

**Follow up:**

- If no response in 3 days: brief check-in
- If no response in 7 days: final follow-up
- After 14 days: proposal expires, requires new discussion
