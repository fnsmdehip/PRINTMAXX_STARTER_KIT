## Deliverable specifications

### Content transformation standards

Each transformation includes:
- Source content extraction
- AI processing with custom prompt
- Format-specific output
- Quality check step
- Delivery to target (or draft)

### Prompt standards

Every prompt includes:
- Voice/tone instructions
- Format requirements (length, structure)
- Platform-specific guidelines
- Example outputs
- Edge case handling

### Output format specifications

**Twitter/X threads:**
- Hook-first opening
- 5-10 tweets typical
- Thread numbering
- CTA in final tweet
- Character count optimized

**LinkedIn posts:**
- Opening hook
- Line break formatting
- 150-300 words
- Professional tone
- Optional hashtags

**Newsletter sections:**
- Conversational tone
- 200-500 words
- Scannable formatting
- Internal links preserved

**Instagram captions:**
- Emoji usage (if brand-appropriate)
- Hashtag block
- Line break formatting
- CTA included

**Quote graphics (text extraction):**
- 3-5 quotable snippets
- Under 280 characters each
- Standalone meaning
- Engagement-optimized
