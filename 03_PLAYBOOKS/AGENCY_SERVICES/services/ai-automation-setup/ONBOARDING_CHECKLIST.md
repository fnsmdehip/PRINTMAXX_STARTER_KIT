# Client Onboarding Checklist - AI Automation Setup

## Pre-kickoff (before first meeting)

### Client tasks
- [ ] Sign proposal or SOW
- [ ] Pay 50% deposit
- [ ] Complete client intake form
- [ ] Identify team members who need access

### Your tasks
- [ ] Send welcome email with next steps
- [ ] Create client folder in project management tool
- [ ] Set up communication channel (Slack/email thread)
- [ ] Schedule kickoff call
- [ ] Review intake form before kickoff

---

## Kickoff call (30-45 min)

### Agenda
1. Introductions (if new contacts)
2. Confirm project scope and deliverables
3. Walk through timeline
4. Discuss access requirements
5. Identify primary contact and escalation path
6. Schedule check-in points
7. Answer questions

### After kickoff
- [ ] Send meeting notes within 24 hours
- [ ] Send access request list with instructions
- [ ] Confirm communication preferences
- [ ] Set internal deadlines in your calendar

---

## Access collection

### Common access needed

**n8n/Make:**
- Self-hosted: Server access or admin login
- Cloud: Invite to workspace

**Email (Gmail/Outlook):**
- App password (not main password)
- OAuth connection (preferred)

**CRM (HubSpot, Pipedrive, etc.):**
- API key
- Admin access for integration setup

**Spreadsheets:**
- Shared folder access
- Service account for Google Sheets

**AI tools:**
- API keys (OpenAI, Anthropic, etc.)
- Or use client's existing accounts

### Access security
- [ ] Use secure sharing method (1Password, encrypted)
- [ ] Document all access received
- [ ] Confirm access works before building
- [ ] Note any permissions limitations

---

## Build phase checklist

### For each workflow:
- [ ] Create in development environment first
- [ ] Test with sample data
- [ ] Document all steps and logic
- [ ] Create error handling
- [ ] Test edge cases
- [ ] Move to production
- [ ] Test in production with real data

### Documentation to create:
- [ ] Workflow diagram or description
- [ ] Trigger explanation
- [ ] What to do if it fails
- [ ] How to modify common settings
- [ ] Video walkthrough (Silver/Gold)

---

## Review phase

### Send to client:
- [ ] Summary of what was built
- [ ] How to test each workflow
- [ ] Questions or clarifications needed
- [ ] Request for feedback within 48 hours

### Revision handling:
- [ ] Log all feedback received
- [ ] Clarify anything unclear
- [ ] Implement within scope
- [ ] Note any out-of-scope requests for separate quote

---

## Training session

### Preparation:
- [ ] All workflows tested and working
- [ ] Documentation complete
- [ ] Screen share setup ready
- [ ] Recording ready (Loom or Zoom)

### During training:
- [ ] Overview of all workflows
- [ ] Live demonstration of each
- [ ] Show how to monitor for errors
- [ ] Walk through documentation
- [ ] Q&A time
- [ ] Confirm they can run it themselves

### After training:
- [ ] Send recording within 24 hours
- [ ] Send final documentation package
- [ ] Send final invoice (50%)

---

## Project close

### Final delivery:
- [ ] All workflows live and tested
- [ ] Documentation delivered
- [ ] Training completed
- [ ] Final payment received

### Support handoff:
- [ ] Confirm support period length
- [ ] Set support end date reminder
- [ ] Explain how to reach you during support
- [ ] Define what's included vs. new project

### Post-project:
- [ ] Request testimonial (after 2 weeks)
- [ ] Schedule optimization check-in (Gold)
- [ ] Add to case study pipeline
- [ ] Note upsell opportunities
- [ ] Send thank you note
