# Client Onboarding Checklist - Cold Email Setup

## Pre-kickoff

### Client tasks
- [ ] Sign proposal/SOW
- [ ] Pay 50% deposit
- [ ] Complete client intake form
- [ ] Provide ICP details
- [ ] Share existing email copy (if any)
- [ ] Confirm budget for tools

### Your tasks
- [ ] Send welcome email
- [ ] Create client project folder
- [ ] Research domain availability
- [ ] Prepare domain name options
- [ ] Schedule kickoff call

---

## Kickoff call (45 min)

### Agenda
1. Confirm ICP and targeting criteria
2. Discuss value proposition and angles
3. Review domain name options
4. Walk through timeline and warmup process
5. Discuss sending tool preferences
6. Explain what's needed from them
7. Answer questions

### Key decisions needed
- [ ] Domain names approved
- [ ] ICP confirmed
- [ ] Primary angle/offer confirmed
- [ ] Sending tool selected
- [ ] Meeting booking process confirmed

### After kickoff
- [ ] Send meeting notes
- [ ] Purchase approved domains
- [ ] Send invoice for domain/tool costs
- [ ] Begin DNS setup

---

## Week 1: Infrastructure setup

### Domain setup
- [ ] Purchase domains (immediately after approval)
- [ ] Add to Google Workspace or Microsoft 365
- [ ] Create email accounts
- [ ] Set up forwarding to main inbox (for replies)
- [ ] Add profile photos
- [ ] Create email signatures

### DNS configuration (per domain)
- [ ] SPF record added
- [ ] DKIM enabled and verified
- [ ] DMARC policy configured
- [ ] Custom tracking domain (if using)
- [ ] Allow 24-48 hours for propagation
- [ ] Verify all records with MXToolbox

### Sending tool setup
- [ ] Create account
- [ ] Connect all email accounts
- [ ] Configure sending limits
- [ ] Set up inbox rotation
- [ ] Test connection for each account

### Documentation
- [ ] Record all login credentials securely
- [ ] Document DNS settings
- [ ] Note any issues encountered

---

## Week 2-3: Warmup period

### Warmup configuration
- [ ] Connect warmup tool (Instantly, Warmbox, etc.)
- [ ] Set initial volume (low)
- [ ] Enable reply matching
- [ ] Configure spam rescue (if available)

### Warmup monitoring
Daily checks during warmup:
- [ ] All accounts active
- [ ] No deliverability warnings
- [ ] Warmup emails being received
- [ ] Replies being sent

### Parallel work during warmup
- [ ] Write email sequences
- [ ] Source and verify leads
- [ ] Build campaign in sending tool (paused)
- [ ] Set up tracking and analytics

---

## Email sequence development

### Research
- [ ] Review client's existing messaging
- [ ] Research ICP pain points
- [ ] Analyze competitor emails (if available)
- [ ] Note any compliance requirements

### Sequence 1 draft
- [ ] Subject line A/B variants
- [ ] Email 1: Opening hook and value prop
- [ ] Email 2: Follow-up angle
- [ ] Email 3: Breakup or new angle
- [ ] Email 4 (if applicable)

### Sequence 2 draft (Silver/Gold)
- [ ] Different angle than Sequence 1
- [ ] Same structure

### Copy review
- [ ] Send to client for review
- [ ] Incorporate feedback
- [ ] Finalize within 48 hours
- [ ] Load into sending tool

---

## Lead sourcing

### List building
- [ ] Define search criteria from ICP
- [ ] Build list in [Apollo/LinkedIn/tool]
- [ ] Export initial list
- [ ] Verify all emails
- [ ] Remove bounces and catch-alls
- [ ] Dedupe against existing contacts (if CRM connected)

### Quality check
- [ ] Spot check 10-20 contacts
- [ ] Verify titles match criteria
- [ ] Confirm company fit
- [ ] Check email validity

### List delivery
- [ ] Format for sending tool
- [ ] Upload to campaign
- [ ] Assign to sequences
- [ ] Document list source and criteria

---

## Week 4: Campaign configuration

### Campaign setup
- [ ] Create campaign in sending tool
- [ ] Load email sequences
- [ ] Import lead list
- [ ] Configure sending schedule
- [ ] Set daily limits per inbox
- [ ] Enable inbox rotation
- [ ] Set up A/B tests

### Testing
- [ ] Send test emails to yourself
- [ ] Check formatting and links
- [ ] Verify tracking works
- [ ] Test on different email clients
- [ ] Confirm booking link works

### Integrations (if applicable)
- [ ] Connect CRM
- [ ] Set up lead routing
- [ ] Configure reply detection
- [ ] Test full flow

---

## Launch

### Pre-launch checklist
- [ ] All inboxes warmed (2+ weeks)
- [ ] DNS verified one more time
- [ ] Copy approved by client
- [ ] Leads loaded and verified
- [ ] Sending limits appropriate
- [ ] Warmup still running in background
- [ ] Client ready to handle replies

### Launch day
- [ ] Activate campaign
- [ ] Monitor first hour
- [ ] Check for bounces
- [ ] Verify emails landing in inbox (test accounts)
- [ ] Document launch time

### First week monitoring
Daily:
- [ ] Check open rates
- [ ] Monitor bounce rates
- [ ] Review any negative replies
- [ ] Adjust if needed
- [ ] Report to client

---

## Training session

### Agenda
1. Dashboard overview
2. How to read metrics
3. How to pause/resume campaigns
4. How to handle replies
5. How to add new leads
6. What to do if deliverability drops
7. Q&A

### Materials to deliver
- [ ] Recording of training
- [ ] Written documentation
- [ ] Login credentials (securely)
- [ ] Contact for support

## Project close

### Final handoff
- [ ] All campaigns running
- [ ] Documentation delivered
- [ ] Training completed
- [ ] Final invoice paid
- [ ] Support period communicated

### Post-launch
- [ ] Weekly check-in for first month (Gold)
- [ ] Request testimonial at week 2-3
- [ ] Note upsell opportunities
