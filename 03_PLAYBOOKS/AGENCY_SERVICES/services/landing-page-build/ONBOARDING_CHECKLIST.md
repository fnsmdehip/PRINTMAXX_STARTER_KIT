# Client Onboarding Checklist - Landing Page Build

## Pre-kickoff

### Client tasks
- [ ] Sign proposal/SOW
- [ ] Pay 50% deposit
- [ ] Complete client intake form
- [ ] Gather brand assets (logo, colors, fonts)
- [ ] Collect testimonials/social proof
- [ ] Confirm hosting preference

### Your tasks
- [ ] Send welcome email
- [ ] Create client project folder
- [ ] Review their existing materials
- [ ] Research competitors/similar pages
- [ ] Schedule kickoff call

---

## Kickoff call (30 min)

### Agenda
1. Confirm offer and target audience
2. Discuss page goals and success metrics
3. Review brand assets received
4. Discuss tone and style preferences
5. Walk through proposed structure
6. Clarify timeline and feedback process
7. Answer questions

### Key information to gather
- Primary conversion goal
- Secondary goals (if any)
- Key objections to address
- Unique selling points
- Competitor pages they like/dislike

### After kickoff
- [ ] Send meeting notes
- [ ] Request any missing assets
- [ ] Begin wireframe and copy

---

## Asset collection

### Brand assets needed

**Logo:**
- [ ] SVG or PNG (transparent background)
- [ ] Light and dark versions (if available)

**Colors:**
- [ ] Primary color (hex code)
- [ ] Secondary color
- [ ] Accent color
- [ ] Background preference

**Fonts:**
- [ ] Heading font
- [ ] Body font
- [ ] Or permission to select

**Images:**
- [ ] Product shots
- [ ] Team photos
- [ ] Hero image preference

### Social proof needed
- [ ] Testimonials (with names, titles, photos if possible)
- [ ] Company logos (for "as seen in" or client logos)
- [ ] Statistics or numbers
- [ ] Reviews or ratings
- [ ] Case study excerpts

### Copy inputs
- [ ] Offer description
- [ ] Target audience description
- [ ] Key benefits (in client's words)
- [ ] FAQs (common questions)
- [ ] Any existing copy or taglines

---

## Wireframe and copy phase

### Wireframe creation
- [ ] Create page structure outline
- [ ] Define each section's purpose
- [ ] Note visual requirements per section
- [ ] Identify CTA placements

### Copywriting
- [ ] Headline options (3-5 variants)
- [ ] Subheadline
- [ ] Problem/pain section
- [ ] Solution/offer section
- [ ] Features/benefits copy
- [ ] Social proof framing
- [ ] FAQ questions and answers
- [ ] CTA copy
- [ ] Meta title and description (Gold)

### Send for review
- [ ] Wireframe document
- [ ] Full copy draft
- [ ] Questions or options needing decision
- [ ] Request feedback within 48 hours

---

## Design and build phase

### Design
- [ ] Create visual design based on wireframe
- [ ] Apply brand colors and fonts
- [ ] Source or create images
- [ ] Design mobile version
- [ ] Internal design review

### Development
- [ ] Build on chosen platform
- [ ] Implement responsive design
- [ ] Add form integration
- [ ] Set up email automation (if applicable)
- [ ] Add analytics tracking
- [ ] Speed optimization
- [ ] Test all links and forms

### Quality checks
- [ ] Desktop browser testing (Chrome, Safari, Firefox)
- [ ] Mobile testing (iOS, Android)
- [ ] Form submission test
- [ ] Email integration test
- [ ] Page speed test (<3 seconds)
- [ ] Spelling and grammar check
- [ ] Link verification

---

## Review phase

### Send to client
- [ ] Preview link
- [ ] How to provide feedback
- [ ] What to check (functionality, copy, design)
- [ ] Timeline for feedback (48 hours)

### Revision handling
- [ ] Log all feedback
- [ ] Clarify unclear requests
- [ ] Implement within scope
- [ ] Note out-of-scope requests for separate quote

---

## Launch

### Pre-launch checklist
- [ ] All revisions implemented
- [ ] Client final approval received
- [ ] Form tested with client's email
- [ ] Analytics verified
- [ ] Custom domain connected (if applicable)
- [ ] SSL certificate active
- [ ] OG image and meta tags set
- [ ] Favicon added

### Launch day
- [ ] Publish page
- [ ] Verify live URL
- [ ] Test form one more time
- [ ] Send live link to client
- [ ] Confirm everything works

### Post-launch
- [ ] Send final invoice
- [ ] Schedule walkthrough call
- [ ] Document any known issues
- [ ] Share analytics access

---

## Walkthrough session

### Agenda
1. Walk through live page
2. Show how to access and edit
3. Explain analytics dashboard
4. Review form submissions location
5. Discuss A/B testing (if applicable)
6. Q&A

### Materials to deliver
- [ ] Recording of walkthrough
- [ ] Login credentials (if applicable)
- [ ] Documentation for edits
- [ ] Analytics access
- [ ] Support contact info

---

## Project close

### Final handoff
- [ ] Page live and verified
- [ ] Documentation delivered
- [ ] Final payment received
- [ ] Support period communicated

### Post-project
- [ ] Request testimonial (after 2 weeks)
- [ ] Check conversion performance (30 days)
- [ ] Note optimization opportunities
- [ ] Add to portfolio (with permission)
