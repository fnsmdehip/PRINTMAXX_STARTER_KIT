## UNIVERSAL CONTRA SETUP (All Niches)

### Profile Structure (Order Matters)

```
1. HEADLINE (80 chars max)
   Format: [SPECIALTY] for [TARGET] seeking [OUTCOME]

2. BIO (800 chars soft limit)
   - Lead: Specific result (not generic intro)
   - Proof: Numbers, case study, credential
   - Method: How you do it
   - Niche focus: Who you work with

3. PORTFOLIO (3-5 projects, quality > quantity)
   - Project case study format (most important)
   - Screenshot + description
   - Specific result metrics

4. RATES
   - Day rate: 6-8 hours of focused work (highest value)
   - Project rate: Fixed scope (most common)
   - Retainer: Monthly (premium positioning)

5. AVAILABILITY
   - Set "selective" or "available" not "actively looking"
   - Limited availability = higher perceived value
```

### Portfolio Project Formula (Apply to Each)

```
PROJECT TITLE: [Result] for [Client Type]

CHALLENGE:
{2-3 sentences of what client struggled with}

SOLUTION:
{What you did, specific tools/process}

RESULTS:
- Metric 1: X% improvement / $X revenue
- Metric 2: Time saved / efficiency gain
- Metric 3: Client satisfaction / next step

TIMELINE: X weeks/months
TEAM SIZE: You + [if applicable]
```

4. **Results** (3-5 bullet points)

   - Revenue impact if possible
   - Client satisfaction/transformation

### Pricing Strategy on Contra

**Key Insight:** Contra clients have budgets and don't shop on price.

**Day Rate (Highest Value):**
- Set 6-8 hours of focused work
- Price higher than hourly equivalent
- Example: $150/hour × 8 = $1,200, but set day rate at $1,500-2,000
- Signals serious clients, not hour-choppers

**Project Rate (Most Common):**
- Fixed scope, fixed price
- Always higher than hourly equivalent
- Get 50% upfront, 50% on delivery
- Reduces scope creep, clear expectations

**Monthly Retainer (Highest Margin):**
- 15-20 hours/month work at premium rate
- Example: $150/hour × 20h = $3,000, set retainer at $3,500-4,000
- Build in buffer for small requests
- Lock in recurring revenue

**Tiered Pricing Example:**

```
PRICING:
- Day rate: $1,500 (6-8h focused work)
- Project: $3,500 (complete automation system)
- Retainer: $3,500/month (20h/month, ongoing optimization)

Note: All work billed on results, not hours. Retainer includes
3x optimization calls + unlimited small requests (under 1h).
```
