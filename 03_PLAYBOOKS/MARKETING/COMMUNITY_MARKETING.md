# SECTION 1: DISCORD SERVER

## Why Build a Discord

**Benefits:**
- Direct access to users
- Beta testing pool
- Feature feedback
- Community-driven growth
- Superfans for referrals

**Best for:**
- Apps with engaged users
- Products needing feedback loops
- Building brand loyalty

---

## Discord Setup

### 1. Create Server

```
Server name: [Product] Community
Template: Community server
```

### 2. Channel Structure

```
📢 ANNOUNCEMENTS
├── #announcements (admin only)
├── #changelog (admin only)
└── #roadmap (admin only)

💬 COMMUNITY
├── #general
├── #introductions
├── #wins (share progress)
└── #feedback

🆘 SUPPORT
├── #help
├── #bugs
└── #feature-requests

🎁 EXCLUSIVE
├── #beta-access (role-gated)
└── #affiliate-chat (role-gated)
```

### 3. Roles

| Role | Access | Who |
|------|--------|-----|
| @Admin | All channels | Team |
| @Moderator | Moderate | Trusted users |
| @Beta Tester | Beta channel | Early adopters |
| @Affiliate | Affiliate channel | Affiliates |
| @Member | Public channels | Everyone |

### 4. Bot Setup

**Essential bots:**
- **MEE6** - Moderation, levels, welcome
- **Carl-bot** - Reaction roles, logging
- **Ticket Tool** - Support tickets

---

## Growing Discord

### Invite Strategies

1. **In-app prompt**
   - After positive action (streak, milestone)
   - "Join our community for tips and support"

2. **Email footer**
   - All emails include Discord link
   - "Got questions? Join our Discord"

3. **Social bio**
   - Link in all platform bios
   - Pin invite tweet

4. **Content CTA**
   - End videos/posts with Discord invite
   - "Continue the conversation in Discord"

### Engagement Tactics

1. **Welcome new members**
   - Automated welcome message
   - Ask one question to start conversation

2. **Weekly challenges**
   - Faith: 7-day prayer challenge
   - Fitness: Step challenges
   - Study: Focus time competition

3. **AMAs**
   - Monthly founder AMA
   - Answer all questions live

4. **Exclusive drops**
   - Early feature access
   - Discount codes
   - Behind-the-scenes

# SECTION 3: HACKER NEWS

## When to Post

**Best for:**
- Technical products
- AI/ML tools
- Developer-focused apps
- Interesting launches

**Show HN criteria:**
- Something people can try
- Not a landing page
- Technical or novel angle

---

## Post Types

### Launch Post

```
Title: Show HN: [Product] – [One-line description]

Body:
Hey HN,

I built [Product] to solve [problem].

[2-3 sentences on why/how]

Tech stack: [What you used]

Would love feedback on [specific question].

[Link]
```

### Side Project Post

```
Title: I built [thing] in [time] using [tech]

Body:
[Story of why you built it]

[Technical details]

[What you learned]

[Link if relevant]
```

---

## HN Etiquette

**Do:**
- Respond to every comment
- Be humble and curious
- Share technical details
- Acknowledge valid criticism

**Don't:**
- Astroturf with fake accounts
- Ignore criticism
- Be defensive
- Use marketing speak

---

## Best Times

- **Post at:** 9 AM EST / 6 AM PST
- **Best days:** Tuesday-Thursday
- **Avoid:** Weekends, holidays

# SECTION 5: QUORA

## Why Quora

**Benefits:**
- Long-tail SEO value
- Evergreen traffic
- Establish expertise
- Direct product mentions (when relevant)

**Best for:**
- Products solving specific problems
- Building thought leadership
- Passive traffic over time

---

## Strategy

### Profile Setup

```
Bio: [Role] at [Company]. Building [Product] to help [audience] with [problem].
Credentials: Add relevant expertise
```

### Finding Questions

Search for:
- "[Problem your product solves]"
- "Best [category] app"
- "[Competitor] alternative"
- "How to [thing your product helps with]"

### Answer Formula

```
HOOK:
[Direct answer to question in first sentence]

BODY:
[Expand with personal experience or expertise]
[Include specific examples]
[Add value beyond the question]

MENTION (if natural):
"I actually built [Product] to solve exactly this problem..."
OR
"Tools like [Product] can help with this..."

CTA:
[Ask for follow-up or upvote if helpful]
```

---

## Best Practices

1. **Answer thoroughly** - Long answers rank better
2. **Add images** - Screenshots, diagrams
3. **Be first or best** - Early answers get more views
4. **Update old answers** - Refresh with new info
5. **Subtle product mentions** - Only when genuinely relevant

**Best for:**

## Pin Strategy

### Pin Types

1. **Quote cards** - Scripture, motivational quotes
2. **Infographics** - Tips, how-tos, lists
3. **Product pins** - App screenshots, features
4. **Blog pins** - Link to content

### Pin Specs

- **Size:** 1000x1500 (2:3 ratio)
- **Format:** PNG or JPG
- **Text:** Large, readable
- **Branding:** Subtle logo/watermark

### Pin Description

```

[Value statement]

[CTA with link]

#hashtag1 #hashtag2 #hashtag3
```

## Posting Schedule

**Frequency:** 5-10 pins/day
**Tool:** Tailwind for scheduling
**Best times:** Evenings and weekends

# SECTION 7: SMS/WHATSAPP MARKETING

## When to Use

**SMS best for:**
- Cart abandonment (apps with purchases)
- Flash sales
- Important updates
- Re-engagement

**Caution:**
- Requires explicit opt-in
- Can burn goodwill if overused
- Legal requirements (TCPA, GDPR)

---

## SMS Setup

### Platforms

| Platform | Cost | Best For |
|----------|------|----------|
| Twilio | $0.0075/msg | Developers |
| Postscript | $25/mo+ | E-commerce |
| SimpleTexting | $29/mo | Small business |

### Compliance

**Required:**
- Explicit opt-in ("Text JOIN to...")
- Easy opt-out ("Reply STOP")
- Clear purpose disclosure
- No spam

---

## SMS Campaigns

### Welcome

```
[Brand]: Thanks for signing up! 🎉
Reply HELP for support or STOP to unsubscribe.
```

### Re-engagement

```
[Brand]: We miss you!
Your [streak/progress] is waiting.
Open the app to continue: [link]
```

### Flash Sale

```
[Brand]: 24 HOURS ONLY ⚡
50% off annual plan.
Use code FLASH50: [link]
Reply STOP to opt out.
```

---

## WhatsApp Business

### Setup

1. Download WhatsApp Business app
2. Verify business phone number
3. Create business profile
4. Set up quick replies

### Use Cases

- Customer support
- Broadcast updates (requires opt-in)
- Personal outreach to affiliates

---

## Metrics

| Metric | Target |
|--------|--------|
| Opt-in rate | 5%+ |
| Open rate | 98%+ (SMS) |
| Click rate | 15%+ |
| Opt-out rate | <2% per campaign |
