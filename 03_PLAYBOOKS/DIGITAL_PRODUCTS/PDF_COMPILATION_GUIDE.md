## Compilation Workflow (For Human)

### Step 1: Tool Selection (5 min)
Pick PDF creation tool:
- **Canva** (fastest, free tier works) - drag and drop, templates
- **Google Docs → Export PDF** (simplest) - just write and export
- **Typst/LaTeX** (best formatting) - if comfortable with markup
- **Notion → Export PDF** (decent) - familiar interface

**Recommendation:** Google Docs for speed. Canva for visual polish.

### Step 2: Per-Product Assembly (2-4 hours each)

1. Open source files listed in compilation table above
2. Copy content in chapter order into your PDF tool
3. Format headings, tables, code blocks per formatting instructions
4. Add page breaks between chapters
5. Create cover page per design instructions
6. Create table of contents
7. Add worksheets (fill-in-the-blank sections)
8. Add final page with upsell CTA + Calendly link
9. Add header (product name) and footer (page number + printmaxx.ai)
10. Export as PDF

### Step 3: Upload to Platforms (20 min per product)

**Whop:**
1. Go to https://whop.com/creator
2. Create product → Digital download
3. Upload PDF
4. Copy title, description, price, tags from listing spec
5. Add thumbnail/cover image
6. Set PWYW minimum
7. Publish

**Gumroad:**
1. Go to https://gumroad.com/products/new
2. Upload PDF
3. Copy title, description, price from listing spec
4. Set PWYW minimum
5. Add tags
6. Add thumbnail
7. Publish

### Step 4: Bundle Creation (10 min per bundle)

1. Create bundle product on Whop/Gumroad
2. Add component products
3. Set bundle price (discounted)
4. Cross-link from individual product pages

### Step 5: Distribution (Ongoing)

Per the listing specs, each product has a distribution map:
- Twitter/X thread announcing launch
- Reddit post in relevant subreddit (value-first, product link in comments)
- Newsletter issue featuring the product
- Medium/Substack article teasing key insights
- Buffer CSV posts (already scheduled)
