# UGC quality checklist

Evaluate content before paying. Use this checklist for every delivery.

---

## Technical quality (non-negotiable)

### Video requirements

| Criteria | Pass | Fail |
|----------|------|------|
| Resolution | 1080p minimum, 4K preferred | 720p or lower |
| Orientation | Vertical 9:16 for social | Horizontal unless specified |
| Frame rate | 24-60fps, consistent | Choppy, inconsistent |
| File format | MP4, MOV | Compressed, web-only |
| Audio | Clear, no background noise | Echo, wind, distortion |
| Lighting | Even, face visible | Dark, harsh shadows |
| Focus | Sharp, stable | Blurry, shaky |

### Audio requirements

| Criteria | Pass | Fail |
|----------|------|------|
| Voice clarity | Every word audible | Mumbling, unclear |
| Background noise | None or minimal | Traffic, TV, others talking |
| Echo/reverb | None | Bathroom echo |
| Sync | Audio matches video | Lip sync issues |
| Volume | Consistent throughout | Sudden changes |

### Lighting requirements

| Criteria | Pass | Fail |
|----------|------|------|
| Face visibility | Clearly lit, no shadows | Half face dark |
| Color accuracy | Natural skin tones | Orange, green, washed out |
| Consistency | Same throughout | Changing mid-video |
| Source | Natural or ring light | Harsh overhead only |

## Common issues and fixes

### Technical issues

| Issue | Fix | Prevention |
|-------|-----|------------|
| Dark video | Request reshoot or adjust in edit | Include lighting specs in brief |
| Bad audio | Use voiceover or captions | Require audio test first |
| Wrong orientation | Re-edit or reshoot | Specify 9:16 in brief |
| Low resolution | Request original files | Specify 1080p minimum |

### Performance issues

| Issue | Fix | Prevention |
|-------|-----|------------|
| Reading script | Request more natural delivery | Ask for memorization |
| Low energy | Request more enthusiasm | Provide energy reference |
| Too fast | Ask them to slow down | Include pacing notes |
| Not believable | Major revision or new creator | Better casting |

### Content issues

| Issue | Fix | Prevention |
|-------|-----|------------|
| Messy background | Request reshoot | Show example backgrounds |
| Product not visible | Request product close-ups | Include framing guide |
| Missed talking points | Request revision with checklist | Numbered brief |
| Wrong tone | Provide reference videos | Clearer creative direction |

## Pre-delivery checklist (for creators)

Send this to creators before they submit:

```
Before submitting, please verify:

Technical:
[ ] Filmed in 1080p or higher
[ ] Vertical format (9:16)
[ ] Clear audio, no background noise
[ ] Good lighting on face
[ ] Stable footage (no shake)

Content:
[ ] All talking points covered
[ ] Product name mentioned correctly
[ ] Call to action included
[ ] Within time limit (+/- 5 seconds)

Files:
[ ] Raw footage included
[ ] Final edited version
[ ] Multiple takes if applicable
```

## Revision request template

When content needs fixes:

```
Hi [Name],

Thanks for the video! A few things need adjustment before we can use this:

1. [Specific issue] - [Specific fix needed]
2. [Specific issue] - [Specific fix needed]

Everything else looks good. Let me know if you have questions about any of these.

Need revised version by [date] please.
```

Be specific. "Make it better" is not actionable feedback.

---

## When to reject and move on

Reject content (and potentially creator) if:

- Multiple reshoots don't fix fundamental issues
- Creator argumentative about reasonable feedback
- Quality significantly below their portfolio samples
- Repeated deadline misses
- Dishonest about capabilities

Don't waste time trying to fix unfixable situations.
