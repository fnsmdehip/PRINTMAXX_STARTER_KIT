# Delivery process

How to deliver content to clients professionally.

---

## Delivery preparation

### Pre-delivery checklist

Before sending anything to client:

**Content verification:**
- [ ] All videos QC approved
- [ ] All hooks recorded
- [ ] Raw footage available
- [ ] File quality verified

**File preparation:**
- [ ] Correct naming convention
- [ ] Proper folder structure
- [ ] All files uploaded
- [ ] Links tested

**Documentation:**
- [ ] Delivery email drafted
- [ ] Final invoice prepared
- [ ] Any notes documented

## File organization

### Folder structure

```
[Client]_[Project]_[Date]/
├── FINAL/
│   ├── Video1_Hook1_Final.mp4
│   ├── Video1_Hook2_Final.mp4
│   ├── Video1_Hook3_Final.mp4
│   ├── Video2_Hook1_Final.mp4
│   └── ...
├── RAW/
│   ├── Video1_RawFootage.mov
│   ├── Video2_RawFootage.mov
│   └── ...
└── README.txt
```

### File naming convention

```
[Brand]_[ContentType]_[Hook#]_[Version]_[Date].mp4

Examples:
GlowSerum_Testimonial_Hook1_v1_20240115.mp4
GlowSerum_Testimonial_Hook2_v1_20240115.mp4
TechApp_Tutorial_Hook1_Final_20240115.mp4
```

## Delivery methods

### Option 1: Google Drive (recommended)

**Setup:**
1. Create organized folder
2. Upload all content
3. Set sharing to "Anyone with link can view"
4. Copy link

**Pros:** Free, easy, organized
**Cons:** Requires Google account, storage limits

---

### Option 2: Dropbox

**Setup:**
1. Create folder
2. Upload content
3. Create shared link
4. Set link permissions

**Pros:** Professional, reliable
**Cons:** Storage limits on free plan

---

### Option 3: WeTransfer

**Setup:**
1. Go to wetransfer.com
2. Add files
3. Enter recipient email
4. Add message
5. Send

**Pros:** No account needed, simple
**Cons:** Link expires (7 days free), less organized

---

### Option 4: Frame.io (premium)

**Setup:**
1. Upload to project
2. Invite client as reviewer
3. Enable comments

**Pros:** Built for video, commenting features
**Cons:** Paid service

## Delivery email template

### Standard delivery

```
Subject: Your UGC Content is Ready - [Project Name]

Hi [Name],

Great news - your UGC content is ready for review!

ACCESS YOUR CONTENT:
[Link to folder]

WHAT'S INCLUDED:
- [X] final videos
- [X] hook variations ([total] cuts)
- Raw footage files

QUICK OVERVIEW:
Video 1: [Brief description - e.g., "Testimonial focusing on problem-solution"]
Video 2: [Brief description]
...

NEXT STEPS:
1. Review the content
2. Let me know if you need any revisions (1 round included)
3. Once approved, I'll send the final invoice

Please share your feedback within [X] business days so we can stay on schedule.

Looking forward to hearing your thoughts!

[Your name]
```

## Handling delivery feedback

### Client approves

```
"Great to hear! I'll send the final invoice now.

Payment is due within [X] days. Once received, I'll follow up about your next batch.

Thanks for the great project!"
```

### Client requests revisions

```
"Thanks for the feedback! I'll get these revisions to you by [date].

To confirm, you're requesting:
1. [Revision 1]
2. [Revision 2]

Is that correct? I'll get started once confirmed."
```

### Client has concerns

```
"I appreciate you sharing that. Let me address each point:

[Point 1]: [Your response]
[Point 2]: [Your response]

What would be the best way to resolve this to your satisfaction?"
```

## Delivery SLAs

### Standard delivery

- Initial delivery: Within agreed timeline
- Client feedback window: 3-5 business days
- Revision turnaround: 2-3 business days
- Final approval: Within 7 days of revision

### Rush delivery

- Initial delivery: Within 48-72 hours
- Client feedback: 24 hours
- Revision turnaround: 24-48 hours
- Premium pricing applies
