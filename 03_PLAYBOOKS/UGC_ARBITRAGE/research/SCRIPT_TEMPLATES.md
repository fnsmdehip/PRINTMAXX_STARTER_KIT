# UGC Script Templates

**Updated:** 2026-01-20
**Purpose:** Ready-to-use script frameworks for different UGC content types

---

## Script Writing Fundamentals

### The UGC Formula

```
HOOK (0-3 seconds): Stop the scroll
PROBLEM (3-8 seconds): Relatable pain point
SOLUTION (8-20 seconds): Product introduction
PROOF (20-25 seconds): Why it works
CTA (25-30 seconds): What to do next
```

### Script Writing Tips

**Do:**
- Write how people talk (conversational)
- Keep sentences short
- Use "you" language
- Include natural pauses
- Write for the viewer, not the product

**Don't:**
- Use marketing jargon
- Make it sound scripted
- Include too many features
- Forget the emotional hook
- Make it too long

## Category 1: Problem-Solution Scripts

### Template 1A: The Frustration Hook

**Best for:** Products solving daily annoyances

```
SCRIPT:

[HOOK - frustrated tone]
"Okay, can we talk about [common frustration]?"

[PROBLEM - relatable]
"Every time I [action], it's like [exaggerated comparison].
I've tried [previous solutions], and nothing worked."

[SOLUTION - discovery tone]
"Then I found [Product].
It [main benefit] in [timeframe/ease]."

[DEMO - show product]
"Look, you just [simple action], and [result]."

[PROOF - genuine reaction]
"I've been using it for [time] and honestly?
[Specific result or feeling]."

[CTA]
"Link's in [bio/description] if you need this in your life."
```

**Example filled in (skincare):**
```
"Okay, can we talk about morning routines taking forever?"

"Every time I do my skincare, it's like a 15-step process.
I've tried streamlining it, skipping steps, nothing worked."

"Then I found this 3-in-1 serum.
It does moisturizer, SPF, and vitamin C all at once."

"Look, you just put it on, and you're done. One product."

"I've been using it for 3 weeks and honestly?
My skin looks better than when I used 7 products."

"Link's in my bio if you need this in your life."
```

## Script Customization Guide

### Variables to Personalize

```
[PRODUCT] - Product name
[BENEFIT 1-3] - Main benefits
[PROBLEM] - Pain point it solves
[TIMEFRAME] - How long to see results
[OLD WAY] - What they did before
[RESULT] - Specific outcome
[EMOTION] - How it makes them feel
[TARGET] - Who it's for
[LOCATION] - Where link is (bio/description)
```

### Brief Template for Creators

```markdown
## UGC Brief

### Product Info
- Product: [Name]
- What it does: [1 sentence]
- Main benefit: [Key takeaway]
- Target customer: [Who]
- Key message: [What we want conveyed]

### Script
[Include full script or template]

### Dos
- [Specific thing to do]
- [Another thing]

### Don'ts
- [Thing to avoid]
- [Another thing to avoid]

### Visual Requirements
- Format: [9:16, 1:1, etc.]
- Length: [X seconds]
- Setting: [Where to film]
- Lighting: [Preference]
- Show product: [When/how]

### Deliverables
- [Number] of videos
- Due date: [Date]
- Raw files: [Yes/No]
- Revisions included: [Number]
```

## Script Do's and Don'ts

### Do's

- Sound natural (read aloud to test)
- Include specific details (numbers, timeframes)
- Show genuine emotion
- Match script to creator's style
- Keep hooks under 3 seconds
- Include visual direction
- Give permission to improvise

### Don'ts

- Use corporate language
- Include competitor bashing
- Make medical/legal claims
- Over-script (kills authenticity)
- Include too many talking points
- Forget the CTA
- Make it too long

## Industry-Specific Scripts

### Skincare/Beauty

**Focus on:** Before/after, texture, application, feeling
**Language:** "Skin feels," "looks," "texture," "glow"
**Proof:** Show on skin, close-ups, natural lighting

### Tech/Gadgets

**Focus on:** Problem solved, ease of use, time saved
**Language:** "Works with," "connects to," "automatically"
**Proof:** Demo in real use case, comparison to old way

### Food/Supplements

**Focus on:** Taste, convenience, results felt
**Language:** "Tastes like," "energy," "feel the difference"
**Proof:** Reaction to taste, lifestyle integration

### Fitness

**Focus on:** Results, convenience, motivation
**Language:** "Gains," "recovery," "performance"
**Proof:** Before/after, during workout use

### Productivity/SaaS

**Focus on:** Time saved, simplification, organization
**Language:** "Saves me," "all in one," "finally"
**Proof:** Screen share, workflow demonstration
