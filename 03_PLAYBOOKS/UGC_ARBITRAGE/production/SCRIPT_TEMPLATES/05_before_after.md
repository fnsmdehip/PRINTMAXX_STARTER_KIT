# Script template: Before/after transformation

Visual proof of results. High converting for results-based products.

---

## Structure

```
[HOOK - 2-3 seconds]
Tease the transformation

[BEFORE STATE - 8-12 seconds]
Show/describe the problem state

[TRANSITION - 3-5 seconds]
What they did (product introduction)

[PROCESS - 8-12 seconds]
Brief mention of using product

[AFTER STATE - 10-15 seconds]
Show/describe the result

[CTA - 3-5 seconds]
How to get same result
```

---

## Example script: Skincare (30 seconds)

```
HOOK:
"30 days using [Product]. Here's what happened."

BEFORE STATE:
[Shows before photo/footage]
"This was my skin a month ago. Breakouts here, here, and here. Texture was rough. I hated taking photos."

TRANSITION:
"I started using [Product] every night."

PROCESS:
"Just applied it after washing my face. Nothing else changed in my routine."

AFTER STATE:
[Shows current skin]
"This is today. Same lighting, no filter. The breakouts are gone and my skin actually feels smooth."

CTA:
"Link below if you want to try it."
```

## Before/after visual requirements

### Photography/video standards
- Same lighting in both shots
- Same angle/distance
- Same background (or similar neutral)
- No filters that alter appearance
- Clear, in-focus images
- Similar clothing (for body transformations)

### What makes compelling before/after
- Obvious visual difference
- Believable timeline
- Authentic conditions (not studio vs phone)
- Relatable starting point
- Aspirational but achievable end point

---

## Types of transformations

### Physical/visual
- Skin condition
- Hair
- Body composition
- Home/space organization
- Product appearance (cleaning, etc.)

### Digital/screen-based
- Inbox/dashboard before and after
- App screens
- Design improvements
- Process simplification

### Lifestyle/intangible
- Energy levels (harder to show)
- Stress reduction
- Time savings
- Confidence

For intangible transformations, use:
- Verbal description
- Metaphors
- Specific scenarios
- Body language changes

---

## Timeline frameworks

### Short timeline (7-14 days)
Best for: Quick-acting products, simple solutions
Risk: May seem too good to be true

### Medium timeline (30-60 days)
Best for: Most products, believable transformation
Sweet spot: Most credible

### Long timeline (90+ days)
Best for: Major transformations, lifestyle changes
Benefit: More believable for big changes

Always state timeline clearly.

---

## Credibility boosters

### Make it believable
- "I was skeptical too"
- "Same lighting, no filter"
- Mention what DIDN'T change
- Acknowledge it's not magic
- Note any other factors

### Specific details
- Exact timeframe
- Exact routine/usage
- Before measurements (if applicable)
- What they were doing before that didn't work

---

## FTC compliance for before/after

Required considerations:
- Results must be genuine
- Typical results should be disclosed if atypical
- Testimonials reflect honest experience
- No misleading editing or filters
- Timeframe must be accurate

Add disclaimer if needed:
"Results may vary. [Product] worked for me with [conditions]."
