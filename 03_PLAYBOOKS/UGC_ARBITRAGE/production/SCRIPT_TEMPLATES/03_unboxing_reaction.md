# Script template: Unboxing reaction

First impressions and product reveal. Great for physical products.

---

## Structure

```
[SETUP - 3-5 seconds]
Context for why they're excited

[UNBOXING - 15-20 seconds]
Opening package, reacting to contents

[FIRST IMPRESSION - 10-15 seconds]
Initial thoughts, touch, feel, look

[STANDOUT FEATURE - 8-10 seconds]
What makes this different

[VERDICT - 5-8 seconds]
Overall reaction and CTA
```

---

## Example script: Tech gadget (45 seconds)

```
SETUP:
"Okay this just arrived and I've been waiting for this all week. Let's see what we got."

UNBOXING:
[Opens package]
"Oh, the packaging is actually really nice. They didn't cheap out on this."
[Pulls out product]
"Okay, first thing - it's smaller than I expected. In a good way."

FIRST IMPRESSION:
"The build quality is solid. Like, this doesn't feel cheap at all."
[Examines product]
"I love that they included [accessory]. A lot of brands make you buy that separately."

STANDOUT FEATURE:
"The main thing I wanted to see was [feature]. And yeah, it actually does [function] really well."

VERDICT:
"Honestly impressed. I'll do a full review after using it for a week but first impression is really good. Link below if you want to check it out."
```

## Example script: Beauty product (30 seconds)

```
SETUP:
"My order from [Brand] finally came. I've been hearing so much about this."

UNBOXING:
[Opens package]
"Cute packaging. Okay let's see..."
[Takes out product]
"Ooh, the bottle is heavy. Feels premium."

FIRST IMPRESSION:
"The texture is... actually really nice. Not too thick."
[Applies small amount]
"It absorbs super fast. No sticky feeling."

STANDOUT FEATURE:
"And it's fragrance-free which was the main reason I wanted to try it."

VERDICT:
"I'll update you guys after a week but so far I'm loving this. Link in bio."
```

## Reaction cues (for creators)

### Positive reactions
- "Oh wow"
- "Okay, this is nice"
- "I didn't expect this"
- "Look at that"
- "This is quality"
- Raised eyebrows
- Nodding
- Smile/laugh of pleasant surprise

### Neutral/honest reactions
- "Interesting..."
- "Let me see..."
- "So this is..."
- Examining closely
- Turning product around

### Things to comment on
- Packaging quality
- Product weight/feel
- Size compared to expectations
- Included accessories
- Smell (if applicable)
- Color/finish
- First touch impression

---

## Unboxing flow tips

### Pacing
- Don't rush - let moments breathe
- Pause to show details
- Natural reactions > scripted reactions

### Camera work
- Start with package visible
- Show each step of opening
- Get close-ups of product details
- Show product in hand for scale

### Audio
- Box/packaging sounds are good
- Talk through what you're seeing
- Genuine reactions matter more than perfect delivery

## Product categories and focus areas

### Tech/Electronics
- Build quality
- Size/weight
- Included accessories
- Buttons/ports
- Display quality
- First power-on

### Beauty/Skincare
- Packaging aesthetics
- Product texture
- Smell
- Application test
- Skin feel after

### Fashion/Accessories
- Material quality
- Stitching/construction
- Fit (if wearable)
- Color accuracy vs photos
- Versatility

### Food/Supplements
- Packaging integrity
- Portion size
- Smell
- Taste (first bite reaction)
- Texture

### Home goods
- Assembly ease
- Material quality
- Size accuracy
- Functionality test
- Aesthetic fit

## Timing breakdown (45 second version)

| Section | Duration | Focus |
|---------|----------|-------|
| Setup | 4-5s | Build anticipation |
| Unboxing | 15-18s | Visual reveal |
| First impression | 12-15s | Touch/feel commentary |
| Standout feature | 8-10s | Key differentiator |
| Verdict | 5-6s | Summary + CTA |
| **Total** | ~45s | |

## B-roll opportunities

Capture extra footage:
- Close-up of logo/branding
- Product details (texture, buttons, etc.)
- Size comparison (hand, common objects)
- Product in use
- Packaging details

---

## Creator brief notes

For unboxing content, specify:
- Don't open package before filming
- Film in good lighting
- Have backup takes ready
- Capture genuine first reaction
- Note specific features to highlight
- Mention if product needs to be shown working
