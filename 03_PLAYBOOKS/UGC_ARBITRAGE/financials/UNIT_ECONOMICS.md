# UGC arbitrage unit economics

Understand your numbers. Know your margins.

---

## Core unit economics

### Single video economics

**Revenue:**
Client pays: $150 (30-second video)

**Costs:**
- Creator fee: $25
- Payment processing (3%): $4.50
- Time cost (0.5 hr @ $30/hr): $15
- Tool allocation: $2

**Total cost:** ~$46.50
**Gross profit:** $103.50
**Margin:** 69%

---

### Retainer economics (monthly)

---

### What affects margins

**Factors that increase margin:**
- Lower-cost creators
- Efficient systems (less time)
- Higher prices (value positioning)
- Bulk creator discounts
- Less revision work

**Factors that decrease margin:**
- Premium/specific creator needs
- Scope creep
- High revision rates
- Client management time
- Tool costs scaling

## Revenue models comparison

### Project-based

```
Revenue: Variable, project by project
Pros: Flexibility, no commitment
Cons: Inconsistent, more sales effort

Example month:
- 3 projects @ $800 avg = $2,400
- Costs: ~$800
- Net: ~$1,600
```

---

### Retainer-based

```
Revenue: Predictable monthly
Pros: Stable, less sales, deeper relationships
Cons: Commitment, capacity planning

Example month:
- 2 retainers @ $2,500 = $5,000
- Costs: ~$1,500
- Net: ~$3,500
```

---

### Hybrid model (recommended)

```
Revenue: Base from retainers + project upside
Pros: Stability + growth potential
Cons: More to manage

Example month:
- 1 retainer @ $3,000 = $3,000
- 2 projects @ $1,000 = $2,000
- Total: $5,000
- Costs: ~$1,600
- Net: ~$3,400
```

## Pricing math

### How to set prices

**Cost-plus pricing:**
```
Price = (Creator cost + Time cost + Overhead) / (1 - Target margin)
Example: ($25 + $30 + $5) / (1 - 0.7) = $200
```

**Market pricing:**
- Research competitor rates
- Position relative to value
- Adjust for your market

**Value pricing:**
- What's it worth to client?
- How much revenue can it generate?
- Price based on outcomes

Sweet spot: Price where (volume x margin) is maximized.

## Break-even analysis

### Monthly break-even

**Fixed costs:**
- Tools: $75
- Your minimum income need: $3,000
- Total: $3,075

**Per-video contribution:**
- Average price: $150
- Variable cost: $45
- Contribution margin: $105

**Break-even videos:**
$3,075 / $105 = ~30 videos/month

---

### Project break-even

For each project:
```
Break-even point = Fixed project costs / Contribution margin %

Example:
- Fixed: $50 (time spent on sales)
- Margin: 70%
- Break-even: $50 / 0.7 = $71

Any project over $71 is profitable.
```

## Cash flow considerations

### Payment timing

**Cash inflow:**
- 50% deposit on start
- 50% on delivery
- Net 7-30 on invoices

**Cash outflow:**
- Creator payment: Upon delivery
- Tools: Monthly subscription
- Your draw: Monthly

**Gap management:**
- Deposits cover creator costs
- Maintain cash reserve (1-2 months)
- Invoice promptly
- Follow up on late payments

## Profitability improvement

### Quick wins

1. **Reduce creator costs** - Negotiate bulk rates, find better sources
2. **Increase prices** - Test 10-15% increases
3. **Reduce revisions** - Better briefs, better QC
4. **Systematize** - Templates, processes, automation

### Long-term optimization

1. Build creator loyalty (better rates over time)
2. Increase average project size (packages, retainers)
3. Reduce time per video through systems
4. Add complementary services (strategy, editing)
