# UGC_ARBITRAGE

Source cheap UGC content, resell at markup to brands.

## Status: Planning

## Revenue Model
- Source UGC: $20-$100 per video
- Sell to brands: $100-$500 per video
- Target: 3-5x markup on content

## Execution Steps

1. Build network of UGC creators (Fiverr, Upwork, direct)
2. Create brand client pipeline
3. Match brand briefs with creators
4. Quality control and delivery
5. Scale with SOPs and VAs

## Folder Contents

- products/ - UGC packages and offerings
- marketing/ - Portfolio, case studies, outreach
- research/ - Creator databases, brand lists
- gtm/ - Sales scripts, pricing strategies
- assets/ - Portfolio videos, creator contracts

## Key Metrics

- Number of active creators
- Number of brand clients
- Average order value
- Profit margin per video

## Creator Sources

- Fiverr (low cost, variable quality)
- Upwork (mid-tier)
- Direct outreach to micro-influencers
- UGC platforms (Billo, Trend)

## Tools

- Notion for creator and client management
- Loom for briefs and feedback
- Dropbox for file delivery
- Stripe for invoicing

## Compliance Notes

- Clear contracts with creators
- Usage rights documentation
- FTC disclosure requirements for brands
