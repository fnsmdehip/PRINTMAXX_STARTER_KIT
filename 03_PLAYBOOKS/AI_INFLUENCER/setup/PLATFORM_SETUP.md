# Platform Setup for AI Influencers

How to set up accounts with proper disclosure on each major platform.

---

## Pre-launch checklist

Before creating any account:

- [ ] Character design complete
- [ ] Avatar assets ready
- [ ] Voice established
- [ ] 10 pieces of content queued
- [ ] Disclosure language finalized
- [ ] Understand platform TOS on AI content

---

## TikTok setup

### Account creation

1. Create standard creator account
2. Switch to Business Account for analytics
3. Username: @[charactername] or @[charactername]_ai

### Bio requirements

**Template:**
```
[Tagline - what you help with]
[I'm an AI] or [AI-powered] - must be visible
[Link to linktree/website]
```

**Example:**
```
Daily motivation for your fitness journey
AI coach here to help you move
linktr.ee/maxfit
```

### TikTok-specific disclosure

- Bio must mention AI nature
- Videos with AI avatars: Label using TikTok's AI label feature
- Synthetic media disclosure tool available in upload flow

### Content settings

- Enable comments (for engagement)
- Enable duets (for growth)
- Enable stitch (for reach)
- Reply to comments as character

## Instagram setup

### Account creation

1. Create Professional Account (Creator)
2. Category: "Digital Creator" or "Entrepreneur"
3. Username: @[charactername] (keep consistent with TikTok)

### Bio setup

**Template:**
```
[One-line value proposition]
[AI-generated content]
[Call to action]
[Link]
```

**Example:**
```
Your daily reminder to show up for yourself
AI-powered devotionals
New content every morning
linktr.ee/gracedaily
```

### Instagram-specific requirements

- Profile clearly indicates AI/synthetic nature
- Use Instagram's AI label on synthetic content
- Stories: Add "AI" sticker or text overlay
- Reels: Disclosure in caption

### Content format priorities

1. Reels (primary growth driver)
2. Carousels (education/value)
3. Stories (engagement/personality)
4. Feed posts (polish/aesthetic)

## YouTube setup

### Channel creation

1. Create brand account (not personal)
2. Channel name: [Character Name] - optional "AI" suffix
3. Handle: @[charactername]

### Channel description

**Template:**
```
[What this channel offers]

[Clear AI disclosure statement]

[Posting schedule]

[How to connect]
```

**Example:**
```
Quick productivity tips to help you build more with less time.

I'm Aria, an AI productivity guide. All content on this channel
is created using AI tools including avatar, voice, and scripts.

New tips every Monday, Wednesday, and Friday.

Follow on Twitter: @AriaProductivity
```

### YouTube-specific setup

- Add AI disclosure to channel About section
- Include disclosure in video descriptions
- Consider adding to channel trailer
- Pin disclosure comment on videos

### Content strategy

- Shorts for growth (TikTok repurposed)
- Long-form for depth (5-15 min)
- Community tab for engagement
- Playlists for organization

## X (Twitter) setup

### Account creation

1. Standard account
2. Apply for creator if eligible
3. Username: @[charactername]

### Profile setup

**Bio template:**
```
[Value proposition]
[Clear AI indicator]
[What to expect]
```

**Example:**
```
Building in public as an AI
Productivity experiments + what's working
Posts Mon-Fri
```

### X-specific practices

- Bio must indicate AI nature
- Consider pinned tweet with longer explanation
- Don't fake human interactions
- Transparent about AI nature in replies when asked

### Content mix

- Original posts (value-driven)
- Threads (in-depth content)
- Quote tweets (opinions/commentary)
- Replies (engagement)

## LinkedIn setup

### Account considerations

LinkedIn is tricky for AI personas:
- Personal profiles require human identity
- Company pages work better for AI projects
- Clear disclosure is essential

### Recommended approach

Create a Company Page:
1. Company name: [Character Name] AI
2. Industry: Technology / Education
3. Tagline includes "AI-powered"

**Alternative:** Human operator shares AI content with clear attribution

### LinkedIn-specific

- More conservative platform - disclosure must be prominent
- Focus on value over personality
- Professional tone required
- Long-form posts perform well

## Cross-platform consistency

### Username strategy

Ideal: Same handle everywhere
Reality: Grab what's available, stay close

Priority order:
1. @charactername
2. @charactername_ai
3. @thecharactername
4. @charactername[number]

### Bio consistency

Core elements on all platforms:
- Same value proposition
- Same AI disclosure approach
- Same link destination
- Same visual branding

### Content adaptation

Same content, different formats:
- TikTok: Vertical video, fast-paced
- Instagram: Polished reels + carousels
- YouTube: Longer, more detailed
- X: Text-first, conversation-ready

## Account security

### For AI accounts

- Use unique email per platform
- Strong, unique passwords
- 2FA enabled everywhere
- Document recovery methods
- Don't link to personal accounts

### Operational security

- Create business email: [character]@domain.com
- Consider separate Google/iCloud for AI accounts
- Keep login credentials documented securely
- Have recovery plans for locked accounts

## Pre-launch content queue

Before going live, have ready:

- [ ] 10 posts/videos minimum
- [ ] Bio and profile complete
- [ ] Disclosure language consistent
- [ ] Response templates for common questions
- [ ] Plan for first week engagement

### Soft launch strategy

1. Post first 3 pieces of content
2. Engage for 24 hours
3. Assess response
4. Adjust if needed
5. Continue posting cadence
6. Scale engagement gradually

## Platform-specific TOS notes

### TikTok
- Requires labeling AI-generated content
- Synthetic media policy applies
- Don't impersonate real people

### Instagram
- AI content labels available
- Authentic content policies apply
- Impersonation prohibited

### YouTube
- Disclosure in description required
- AI labels may be mandatory for synthetic content
- Adhere to community guidelines on synthetic media

### X
- Synthetic and manipulated media policy
- Disclosure recommended
- Don't mislead users about content origin

## Troubleshooting

### Account suspended?
1. Review platform's specific AI/synthetic media policy
2. Check if disclosure was sufficient
3. Appeal with clear explanation of AI nature
4. Document for future prevention

### Low engagement?
1. Check content quality
2. Verify posting times
3. Increase engagement activities
4. Adjust content based on analytics

### Negative comments about AI nature?
1. Don't hide or delete
2. Respond professionally as character
3. Reinforce disclosure and value
4. Some users will always object - focus on your audience
