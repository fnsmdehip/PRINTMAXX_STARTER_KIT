## Overview

## Step-by-step implementation

### Day 1-2: Create persona identity (4-6 hours)

**Task 1: Define persona profile**

Choose niche and create detailed persona:

| Element | Example (Faith Niche) | Example (Fitness Niche) |
|---------|----------------------|------------------------|
| Name | Grace Faithwell | Maya Fitwell |
| Age | 28 | 25 |
| Niche | Christian faith + productivity | Women's fitness + wellness |
| Tone | Warm, encouraging, scripture-based | Motivating, no-BS, science-backed |
| Visual style | Modest, warm lighting, earth tones | Athletic wear, gym/outdoor settings |
| Platform focus | Instagram + TikTok | TikTok + YouTube Shorts |
| Revenue path | PrayerLock app + Bible study course | WalkToUnlock app + supplement affiliate |

**Task 2: Generate consistent visual identity**

Use Leonardo.ai to create 20-30 base images of your persona:

Prompt template:
```
[ethnicity] [gender], [age] years old, [body type], [hair description],
wearing [outfit], in [setting], [activity], [lighting style],
consistent character, photorealistic, 8k quality
```

Example:
```
young woman, 25 years old, athletic build, brown hair in ponytail,
wearing black athletic wear, in modern gym, doing yoga pose,
natural lighting, consistent character, photorealistic, 8k quality
```

Generate in batches:
- 10 photos: different outfits, same face
- 5 photos: action shots (working out, cooking, reading)
- 5 photos: lifestyle (cafe, outdoor, home)
- 5 photos: close-up (for profile pics, story content)

Save all images in consistent folder: `AI_INFLUENCER/assets/{persona_name}/`

Consistency tip: Use the same seed number in Leonardo to maintain face consistency across images.

**Task 3: Create voice identity**

1. Go to ElevenLabs, create new voice
2. Select or clone a voice that matches persona (warm female, authoritative male, etc.)
3. Save voice ID for all future content
4. Test with 5 sample scripts, adjust settings until natural

Voice settings reference:
```
Stability: 0.5-0.7 (lower = more variation, more natural)
Similarity: 0.7-0.85 (higher = more consistent)
Style: 0.3-0.5 (moderate expressiveness)
```

### Day 2-3: Set up accounts (2-3 hours)

**Task 4: Create social accounts**

For each persona, create accounts on primary platforms:
1. Open GoLogin, create new browser profile
2. Assign unique SOAX mobile proxy
3. Create accounts with persona name, bio, profile pic
4. Set up bio with niche-relevant keywords
5. Add link-in-bio (Beacons or Linktree)

Bio formula:
```
[What you help with] | [Credibility element]
[Content type] posted daily
[CTA] -> Link below
```

Example:
```
Helping women build faith + focus | 100-day prayer streak
Daily prayer prompts + productivity tips
Download PrayerLock -> link below
```

### Day 3-7: Content creation pipeline (8-12 hours)

**Task 5: Batch create 30 posts**

Content types per persona:
1. AI UGC talking head (D-ID/HeyGen) - highest engagement
2. Quote cards with persona image background
3. Tutorial/tip slideshows (Canva)
4. AI voiceover on stock video (ElevenLabs + CapCut)
5. Reaction/commentary on trending topics

Production workflow:
```
Day 3: Write 30 scripts (hooks + body + CTA)
Day 4: Generate talking head videos in D-ID (batch of 10)
Day 5: Create quote cards + slideshows in Canva (batch of 10)
Day 6: Edit voiceover videos in CapCut (batch of 10)
Day 7: Schedule all 30 posts across platforms
```

**Task 6: Set up FTC-compliant disclosure**

CRITICAL: AI-generated content must be disclosed. FTC synthetic media rules apply.

Required disclosures:
- Bio: "#AIcreated" or "AI-generated persona" in bio
- Posts: Affiliate links need "#ad" or "#sponsored"
- Testimonials: Cannot use AI to fake testimonials
- Before/after: HIGH RISK with AI (triple legal jeopardy, avoid)

Reference: `FACELESS_ARMY_LEGAL_ANALYSIS.md` and `compliance/`

### Week 2-4: Grow and monetize

**Task 7: Daily posting + engagement**

Daily routine per persona (20-30 min):
```
1. Post scheduled content (pre-batched)
2. Reply to all comments as persona (stay in character)
3. Engage with 10-15 niche accounts
4. Check analytics: top performing content
5. Save trending topics for next content batch
```

Post 2-3x/day per account. Schedule for peak times.

**Task 8: Monetize with affiliate links**

At ANY follower count, add affiliate links:
- Amazon Associates (immediate approval)
- Niche-specific programs (apply directly)
- Own products (app links, Gumroad)

Place links in:
- Bio link page (Beacons/Linktree)
- Comment replies ("Link in bio!")
- Video descriptions
- Story/highlight links

**Task 9: Pitch micro-sponsorships at 1K+ followers**

Brand deal pricing tiers:
```
1K-10K followers:  $50-200/post
10K-50K followers: $200-1,000/post
50K-250K followers: $1,000-5,000/post
250K+ followers:    $5,000-50,000+/post
```

Find brands via:
- DM brands in your niche directly
- AspireIQ, CreatorIQ, Grin platforms
- Affiliate networks (ShareASale, Impact)

## Risk management

- **FTC compliance:** Always disclose AI-generated content. Follow `FACELESS_ARMY_LEGAL_ANALYSIS.md`. $51,744 per violation for undisclosed synthetic testimonials.
- **Platform bans:** Use anti-detect browser + proxies. Keep personas believable. Don't claim to be a real human when specifically asked.
- **Face consistency:** Use same Leonardo seed, save all base images, maintain style consistency. Followers notice inconsistency.
- **Brand deal risk:** Disclose AI nature to sponsors upfront. Most accept it if engagement is real.
- **Voice cloning legal:** Only clone voices you have rights to. ElevenLabs default voices are safe.
- **Platform disclosure rules evolving:** Monitor TikTok, IG, YouTube AI content policies quarterly.
