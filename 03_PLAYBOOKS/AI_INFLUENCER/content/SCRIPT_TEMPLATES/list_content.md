# List Content Script Template

Numbered lists, rankings, and roundup content that drives saves and shares.

---

## Format specs

- Length: 30-120 seconds
- Structure: Hook > List items > Wrap
- Best number: 3, 5, 7, or 10 items
- Value: Each item standalone useful

---

## Template 1: Top [X] list

```
HOOK (3 sec):
"Top [number] [things] for [outcome]:"

LIST (40-90 sec):
"Number [X]: [Item]
[One sentence why/how]

Number [X-1]: [Item]
[One sentence why/how]

[Continue...]

Number 1: [Best item]
[One sentence why/how]"

WRAP (10 sec):
"[Optional: What they have in common]
[Optional: Which to start with]"

CTA (5 sec):
"Save this list."
```

**Example:**
```
"Top 5 apps for focus:

Number 5: Forest
Gamifies phone-free time by growing virtual trees.

Number 4: Freedom
Blocks distracting sites across all devices.

Number 3: Brain.fm
AI-generated music designed for focus, not entertainment.

Number 2: Centered
Coaches you through deep work sessions with gentle nudges.

Number 1: Your phone's Do Not Disturb
Free. Built-in. Most people forget it exists.

Notice the pattern: The best focus tools remove options, not add them."

"Save this list."
```

## Template 3: Types/categories list

```
HOOK (3 sec):
"The [number] types of [thing]:"

LIST (50-90 sec):
"Type 1: The [Name]
[Description]
[How to identify/work with]

Type 2: The [Name]
[Description]
[How to identify/work with]

Type 3: The [Name]
[Description]
[How to identify/work with]"

APPLICATION (15 sec):
"Which type are you?
[Optional: Brief quiz or self-identification help]"

CTA (5 sec):
"Comment your type."
```

**Example:**
```
"The 3 types of procrastinators:

Type 1: The Perfectionist
Delays because nothing feels ready. Waits for the 'right moment.'
Fix: Set a 'good enough' threshold and ship.

Type 2: The Overwhelmed
Delays because the task feels too big. Doesn't know where to start.
Fix: Break it into 15-minute chunks. Start with just one.

Type 3: The Avoider
Delays because the task is uncomfortable. Prefers easier things.
Fix: Do the uncomfortable thing first. Get it over with.

Which type are you? Be honest."

"Comment your type. I'll share specific strategies."
```

Schedule downtime this week. Not when you're done. Now."

## List content best practices

### Number psychology:
- 3: Quick, digestible, memorable
- 5: Comprehensive but manageable
- 7: Feels substantial
- 10: Authoritative, "complete" feeling

### Item ordering:
- Best item first OR last (for retention)
- Worst/weakest in middle
- Surprising item for interest spike

### Each item needs:
- Clear identification
- Brief explanation
- Standalone value

### What makes lists shareable:
- Useful reference
- Identity-related ("which type are you")
- Surprising inclusions
- Practical application

---

## Common list mistakes

**Too many items:**
50 tips overwhelm. Pick the best 5-7.

**No explanation:**
Just naming things without why.

**Generic items:**
"Work hard" isn't useful. Be specific.

**Wrong ordering:**
Burying the best content in the middle.

**No application:**
Lists without "what to do with this."
