# Tip of the Day Script Template

Quick, actionable tips that deliver immediate value.

---

## Format specs

- Length: 15-45 seconds
- Structure: Problem > Tip > Application
- Value: Actionable in under 5 minutes
- CTA: Save or share focus

---

## Template 1: The quick win

```
HOOK (3 sec):
"[Number]-second tip that [specific result]:"

BODY (15-30 sec):
"[The tip in one sentence]

Here's how:
- Step one: [action]
- Step two: [action]
- Step three: [action]

[Why this works in one sentence]"

CTA (5 sec):
"Save this. Try it today."
```

**Example:**
```
"30-second tip that stops procrastination:

Start with 2 minutes.

Here's how:
- Pick the task you're avoiding
- Set a timer for 2 minutes
- Work until it beeps. Then decide to continue or stop.

You'll continue 80% of the time. Starting is the hard part."

"Save this. Try it today."
```

## Template 2: The mistake fix

```
HOOK (3 sec):
"Stop doing this with [topic]:"

BODY (20-35 sec):
"[Common mistake]

Why it's a problem:
[Brief explanation]

What to do instead:
[Better approach]

[Quick proof or logic]"

CTA (5 sec):
"Share with someone who needs this."
```

**Example:**
```
"Stop doing this with your to-do list:

Writing everything down in one list.

Why it's a problem:
Your brain sees 47 items and shuts down before starting.

What to do instead:
Write tomorrow's list tonight. Three things max. The rest waits.

Progress happens when you focus. A smaller list finishes faster."

"Share with someone who needs this."
```

## Template 3: The swap

```
HOOK (3 sec):
"Replace [common thing] with [better thing]."

BODY (15-25 sec):
"Most people [common approach].

Instead, try [alternative].

The difference:
[Old way result] vs [New way result]

[One sentence on why]"

CTA (5 sec):
"Comment if you're trying this."
```

**Example:**
```
"Replace 'I don't have time' with 'It's not a priority.'

Most people say they don't have time for things they want to do.

Instead, try being honest about priorities.

The difference:
'I don't have time' removes your power. 'It's not a priority' shows you're choosing.

You have the same 24 hours as everyone. You're choosing how to spend them."

"Comment if you're trying this."
```

## Template 4: The one thing

```
HOOK (3 sec):
"If you only do one thing for [area] today:"

BODY (15-25 sec):
"[The one thing]

Ignore [other things for now].

This moves the needle because [reason].

[How to actually do it]"

CTA (5 sec):
"Follow for daily tips."
```

**Example:**
```
"If you only do one thing for your health today:

Drink water first thing. Before coffee, before phone, before anything.

Ignore the supplements. Ignore the workout you'll skip anyway.

This works because you're dehydrated from sleep. Everything functions better hydrated.

Glass by your bed. Drink it before your feet hit the floor."

"Follow for daily tips."
```

## Template 5: The tool tip

```
HOOK (3 sec):
"[Tool] hack you're not using:"

BODY (20-35 sec):
"[The feature or technique]

How to set it up:
[Brief steps]

This saves [time/effort/money] by [mechanism].

[Quick example of the result]"

CTA (5 sec):
"Save for later."
```

**Example:**
```
"Google Calendar hack you're not using:

Create separate calendars for different life areas. Color code them.

How to set it up:
Settings > Add new calendar > Name it (Work, Personal, Health) > Pick color

This saves mental energy by showing you balance at a glance.

One look tells you if your week is all work and no life."

"Save for later."
```

## Template 6: The format tip

```
HOOK (3 sec):
"The [format] for [task]:"

BODY (20-30 sec):
"[Present the format/template]

Why this format:
[Brief benefit]

Example:
[Quick demonstration]"

CTA (5 sec):
"Screenshot this for next time."
```

**Example:**
```
"The format for saying no:

'Thanks for thinking of me. I'm not able to take this on right now.'

Why this format:
Appreciation + clear no + no excuse to argue with.

Example:
'Thanks for the invite to coffee. I'm not able to take this on right now.'

No explanation needed. No negotiation opened."

"Screenshot this for next time."
```

## Template 7: The timing tip

```
HOOK (3 sec):
"When to [task] for best results:"

BODY (15-30 sec):
"[The timing]

Why this timing:
[Science or logic]

How to apply:
[Practical implementation]"

CTA (5 sec):
"Try this tomorrow."
```

**Example:**
```
"When to make decisions for best results:

Morning. Before you've made other decisions.

Why this timing:
Decision fatigue is real. Your brain makes worse choices as the day goes on.

How to apply:
Big decisions before lunch. Put them on tomorrow's calendar now."

"Try this tomorrow."
```

## Niche variations

### Fitness tips:
- "30-second stretch that fixes desk posture"

- "Replace static stretching with this before lifting"

### Faith tips:
- "When to pray for maximum focus"
- "Scripture memorization technique that sticks"
- "Replace worry with this 60-second practice"

### Productivity tips:
- "Email setting that saves 30 min daily"
- "Meeting mistake wasting your afternoons"
- "Calendar hack for protecting deep work"

### Business tips:
- "Pricing format that increases conversions"
- "Follow-up timing that gets responses"
- "Email subject line structure that gets opens"

---

## Production notes

### Pacing:
- Fast but clear
- No filler words
- Slight pause before tip reveal

### Visuals:
- Text overlays for key points
- Steps can be numbered on screen
- Tool tips benefit from screen recording

### Best posting times:
- Morning: Productivity/health tips
- Lunch: Business tips
- Evening: Personal development tips
