# POD Design Prompts Library

Quick-reference prompts for AI design generation across PRINTMAXX niches.

---

## Master Prompt Template

```
[PRODUCT] design for print-on-demand,
"[TEXT/PHRASE]" in [FONT STYLE] font,
[VISUAL ELEMENTS],
[COLOR SCHEME],
transparent background PNG,
print-ready [DIMENSIONS],
[AESTHETIC] aesthetic,
no watermarks, no mockups
```

**Dimensions by Product:**
- T-Shirts: 4500x5400px (300dpi)
- Mugs: 3300x1300px (wrap design)
- Posters: 4500x6000px (18x24 at 300dpi)
- Stickers: 3000x3000px (die-cut)
- Phone Cases: 2400x4000px
- Tote Bags: 4200x4200px

## Batch Generation Workflow

1. Pick niche (Faith/Fitness/Tech)
2. Select 5 phrases from category
3. Apply 2 style variations each = 10 designs
4. Generate all in Midjourney/DALL-E
5. Select top 5 performers
6. Create product mockups
7. List on Etsy with full SEO

**Daily Target:** 10 designs generated, 5 listed

---

## Quality Checklist

Before using any AI-generated design:

- [ ] Text is legible and spelled correctly
- [ ] No AI artifacts or distortions
- [ ] Colors are print-appropriate (no pure black/white if avoidable)
- [ ] Resolution is 300dpi minimum
- [ ] Background is truly transparent
- [ ] Design is centered properly
- [ ] No copyrighted elements
- [ ] No trademarked phrases
- [ ] Passes "would I wear/use this?" test
