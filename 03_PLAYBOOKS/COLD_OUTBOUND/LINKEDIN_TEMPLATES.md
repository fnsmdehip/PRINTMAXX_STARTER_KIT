# LinkedIn Outreach Templates

Direct, short, no-fluff templates. Connection requests under 300 chars. Follow-ups that get replies.

---

## Connection Request Messages (Under 300 chars)

LinkedIn connection requests have a 300 character limit. Every word counts.

```
{{firstName}}, your {{content_type}} on {{topic}} caught my attention. Building something similar. Would love to exchange ideas.

```

## Follow-Up Sequences After Connection

Start after they accept your connection. Don't pitch immediately. Build the conversation first.

### Message 1: After They Accept (24-48 hours)

**Template A: Simple thanks + question**
```
Hey {{firstName}}, thanks for connecting.

Quick question: how's {{company}} handling {{problem_you_solve}} right now?

Curious because we see a lot of companies at your stage dealing with this.
```

**Template B: Observation-based**
```
Hey {{firstName}}, appreciate the connect.

Noticed {{company}} just {{trigger: launched something, made a hire, raised funding}}. Congrats.

How's that going so far?
```

**Template C: Common ground**
```
Hey {{firstName}}, thanks for connecting.

Saw we're both in the {{industry/niche}} space. Always good to know more people doing similar work.

What's keeping you busy right now?
```

**Template D: For creators**
```
Hey {{firstName}}, thanks for connecting.

Been following your {{platform}} content. The {{specific_content_type}} posts do well.

How are you thinking about growing from here?
```

### Message 2: Value-Add (3-4 days later)

**Template A: Resource share**
```
Hey {{firstName}},

Thought you might find this useful: we put together a {{guide/checklist/template}} on {{topic}}.

{{Link or offer to send}}

No pitch, just figured it's relevant to what you're working on.
```

**Template B: Insight share**
```
Hey {{firstName}},

Something I've been seeing a lot: {{insight relevant to their situation}}.

Companies like {{similar_company}} are handling it by {{specific tactic}}.

Thought you'd find that useful.
```

**Template C: Case study tease**
```
Hey {{firstName}},

We just wrapped a project with a company similar to {{company}}.

They went from {{before state}} to {{after state}} in {{timeframe}}.

Happy to share how they did it if you're curious.
```

**Template D: For creators**
```
Hey {{firstName}},

Wanted to share something: we've seen creators at your stage grow 3-5x by {{specific tactic}}.

{{Creator_name}} did exactly this and hit {{milestone}}.

Want me to break down how it works?
```

### Message 3: Soft Pitch (1 week later)

**Template A: Direct ask**
```
Hey {{firstName}},

I'll be direct: we help {{type of company}} with {{what you do}}.

Companies like {{similar_company}} have seen {{specific result}}.

Worth a 15-min call to see if we can do the same for {{company}}?
```

**Template B: Permission-based**
```
Hey {{firstName}},

Based on our conversation, I think there's a fit here.

We help {{type of company}} {{achieve outcome}}.

Would you be open to a quick call to explore this? No pressure either way.
```

**Template C: Problem-specific**
```
Hey {{firstName}},

If {{problem}} is still on your radar, we might be able to help.

We've helped {{X}} companies solve exactly this. Average result: {{specific outcome}}.

15 minutes to see if it makes sense?
```

**Template D: For creators**
```
Hey {{firstName}},

Here's the deal: we handle {{content distribution/clips/posting}} for creators.

You focus on content. We handle distribution. Free trial if you want to test it.

Interested?
```

### Direct Pitch Template

```

Is this something {{company}} is focused on?

{{Your name}}

```

```
Companies at this stage usually hit a wall with {{common problem}}.

We've helped {{X}} companies avoid that. Curious if it's relevant to you.

Worth a quick chat?

{{Your name}}
```


## Voice Note Scripts (Video Messages)

LinkedIn video messages get 3x the response rate. Keep them under 60 seconds.

### 30-Second Intro Script

```
Hey {{firstName}}, quick video for you.

I noticed {{observation about them or company}}.

We help {{type of company/creator}} with {{what you do}}.

Thought there might be a fit. Would love to chat if you're open to it.

Let me know.
```
*Delivery: casual, like texting a friend. Not salesy.*

### Quick Value Proposition (45 seconds)

```
Hey {{firstName}}.

Three things:

One: We help {{type of company}} with {{problem}}.

Two: {{Similar company}} saw {{specific result}} in {{timeframe}}.

Three: I think we could do the same for {{company}}.

Worth a quick call to see if there's a fit? No pressure either way.

Let me know what you think.
```
*Delivery: numbered points keep you focused. Pause between each.*

### Creator Outreach Script (30 seconds)

```
Hey {{firstName}}, quick one.

I've been following your content on {{platform}}. The {{specific content type}} stuff is good.

We help creators like you with distribution. Basically, you make the content, we get it everywhere.

Free trial if you want to test it. Let me know.
```
*Delivery: genuine appreciation, then straight to the offer.*

### Follow-Up Video (20 seconds)

```
Hey {{firstName}}, following up.

Sent you a message about {{topic}} a few days ago.

Still think there's a fit. Let me know if you're open to a quick chat.

No worries either way.
```
*Delivery: brief, not pushy. Shows persistence without annoyance.*

## Timing and Volume

**Connection requests:**
- 20-30 per day max
- Morning (8-9am) or lunch (12-1pm) their timezone
- Weekdays only

**Follow-ups:**
- Message 1: 24-48 hours after connect
- Message 2: 3-4 days after Message 1
- Message 3: 7 days after Message 2
- Stop after 3 unanswered messages

**InMails:**
- Tuesday-Thursday only
- 9-11am their timezone
- Save for high-value prospects

**Video messages:**
- Stand out more, use sparingly
- 2-3 per day max
- For prospects you really want

## Response Handling

**Positive response:**
```
Great. Here's my calendar: {{calendly_link}}

Or let me know what times work for you.
```

**"Not now" response:**
```
Totally understand. When would be a better time to circle back?
```

**Negative response:**
```
No problem. Thanks for letting me know.

If things change, feel free to reach out.
```

**Questions about pricing/details:**
```
Happy to get into that.

Quick summary: {{2-3 sentences on what you do and results}}

Easier to explain on a call. Do you have 15 minutes this week?
```

## Quick Personalization Variables

- {{firstName}} - Their first name
- {{company}} - Their current company
- {{trigger}} - Recent news, hire, funding, launch
- {{problem}} - Problem you solve
- {{similar_company}} - Company like theirs you've helped
- {{specific_result}} - Concrete outcome (numbers, timeframes)
- {{platform}} - For creators: TikTok, YouTube, Twitch, etc.
- {{follower_count}} - Their current audience size
- {{milestone}} - Next level they want to reach

---

## What Not to Do

- Don't pitch in connection request
- Don't send walls of text
- Don't use "I hope this message finds you well"
- Don't ask multiple questions at once
- Don't give up after one message
- Don't send the same template to everyone without personalization
- Don't use obviously automated messages
- Don't connect and immediately pitch

## LinkedIn Content Templates

### Post Format 1: Story Framework

```
[Hook: Outcome achieved]

[Blank line]

Here's what happened:

[Context: Starting point]

[The problem: What went wrong/challenge faced]

[The pivot: What changed]

[The result: Specific numbers]

[The lesson: Takeaway]

What's your experience with [topic]?
```

**Example:**
```
I booked 47 meetings in 30 days using cold LinkedIn DMs.

Here's what happened:

Started with the usual spray-and-pray approach. 200 connection requests/day. Generic template. 3% accept rate. Zero replies.

The problem: I was treating LinkedIn like email. Volume over value.

The pivot: Flipped to quality. 20 connections/day. Personalized notes. Engaged with their content first.

The result: 65% accept rate. 23% reply rate. 47 booked calls.

The lesson: LinkedIn rewards relationships. Build them before you pitch.

What's working for your LinkedIn outbound right now?
```

### Post Format 2: Contrarian Take

```
Unpopular opinion: [Contrarian statement]

Here's why I believe this:

1. [Point with evidence]

2. [Point with evidence]

3. [Point with evidence]

What should you do instead?

[Alternative approach]

Agree? Disagree? I want to hear it.
```

### Post Format 3: How-To List

```
How to [achieve outcome] in [timeframe]:

1. [Step] - [Brief explanation]
2. [Step] - [Brief explanation]
3. [Step] - [Brief explanation]
4. [Step] - [Brief explanation]
5. [Step] - [Brief explanation]

The key: [Main insight]

Which step are you starting with?
```

### Post Format 4: Data/Results Share

```
[Number] [metric] in [timeframe].

Here's the breakdown:

What worked:
- [Tactic 1]: [Result]
- [Tactic 2]: [Result]
- [Tactic 3]: [Result]

What didn't:
- [Failed tactic]: [Why it failed]

The takeaway: [Lesson learned]

DM me "template" if you want the exact sequence I used.
```

### Post Format 5: Carousel Structure (10 slides)

```
Slide 1: [Hook - problem or outcome]
Slide 2: [Why this matters - context]
Slide 3: [Point 1 with detail]
Slide 4: [Point 2 with detail]
Slide 5: [Point 3 with detail]
Slide 6: [Point 4 with detail]
Slide 7: [Point 5 with detail]
Slide 8: [Common mistake to avoid]
Slide 9: [Summary of key points]
Slide 10: [CTA - follow, comment, save]
```

## Advanced Connection Request Sequences

### Multi-Touch Sequence A: Content Engagement First

**Day 0:** Engage with 2-3 of their posts (meaningful comments)
**Day 3:** Like their most recent post
**Day 5:** Send connection request with note:

```
{{firstName}}, been following your content on {{topic}}. Really liked your take on {{specific post topic}}.

I'm building {{your thing}} in a similar space. Would love to connect and swap notes.
```

**Day 6 (if accepted):** Message 1
**Day 10:** Message 2 (value add)
**Day 17:** Message 3 (soft pitch)

### Multi-Touch Sequence B: Trigger-Based

**Trigger:** Company raised funding / hired for role / launched product

**Day 0:** Connection request:
```
{{firstName}}, congrats on {{trigger}}. We help companies at exactly this stage with {{problem}}.

Thought it might be worth connecting.
```

**Day 2 (if accepted):** Message 1:
```
Hey {{firstName}}, thanks for connecting.

Saw the news about {{trigger}}. When companies hit this stage, they usually face [problem 1] or [problem 2].

Is that something you're thinking about?
```

**Day 5:** Message 2 (case study relevant to trigger)
**Day 10:** Message 3 (direct ask)

### Multi-Touch Sequence C: Mutual Connection Warm Intro

**Day 0:** Ask mutual connection for warm intro
**Day 3:** If no intro, send connection request:
```
{{firstName}}, we're both connected to {{mutual_connection}}. Noticed you're working on {{their thing}}.

I help with {{your thing}} - thought there might be overlap. Worth connecting?
```

**Follow sequence from there**

## LinkedIn Comment Templates

### Value-Add Comment

```
[Agree/acknowledge] + [Additional insight with data] + [Question]
```

**Examples:**
- "Spot on about personalization. We saw 40% higher reply rates when we added specific company context vs just first name. What's your optimal personalization level?"
- "This is exactly right. I'd add that timing matters too - Tuesday 10am consistently outperforms for us. When do you typically send?"

### Challenge Comment

```
[Acknowledge point] + [Different perspective] + [Your data/experience]
```

**Examples:**
- "Interesting take. I've actually seen the opposite - shorter DMs (under 50 words) consistently outperform longer ones in my campaigns. Might depend on the industry?"
- "Good framework. Curious about the no-link approach though - we've found links in follow-up messages work well once trust is built. Different experience?"

### Question Comment

```
[Specific question about their method] + [Context for why you're asking]
```

**Examples:**
- "What's your accept rate with the no-note approach? Debating whether to test this on my campaigns."
- "How long do you wait before the first follow-up? I've been doing 48 hours but wondering if that's too soon."

## Objection Handling Templates

### "Not interested"

```
Totally understand. If anything changes, feel free to reach out.

One quick question though - what are you using for [problem] currently? Always curious what's working for teams like yours.
```

### "We already have a solution"

```
Makes sense. Who are you working with?

[If they answer]

Got it. How's that going? Always curious what's working well and what could be better.
```

### "Not the right time"

```
Completely understand. When would be a better time to check back in?

[Or]

No worries. I'll follow up in [timeframe]. In the meantime, I'm going to send you a [resource] that might be useful regardless.
```

### "Send me more info"

```
Happy to. Quick question so I can send the right stuff - what's your main priority with [problem] right now?

[Or]

Sure thing. I put together a quick overview here: [link]. After you check it out, are you open to a 15-min call to see if it makes sense?
```

### "What's the price?"

```
Depends on a few things - mainly [factor 1] and [factor 2].

To give you accurate pricing, I'd need to understand your current [situation]. Do you have 15 minutes this week?
```
