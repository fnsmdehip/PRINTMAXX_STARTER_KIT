# LinkedIn message templates

20 message templates for different situations. Customize for your offer and recipient.

## Rules for LinkedIn DMs

1. **Keep it short** - Under 100 words, ideally under 50
2. **Don't pitch immediately** - Start a conversation first
3. **Be human** - Casual tone, not corporate
4. **Give before asking** - Value first, ask second
5. **One CTA** - Don't overwhelm with options
6. **Mobile-first** - Most people read on phone, brevity wins

## Post-connection openers (no pitch)

### Template 1: Simple thanks
```
Hey {{firstName}}, thanks for connecting!

Noticed you're at {{company}}. How's the [industry/role] treating you these days?
```

### Template 2: Common ground
```
Hey {{firstName}}, glad to connect!

Saw we're both in [industry/role/location]. Always good to know more people in the space.

What's keeping you busy these days?
```

### Template 3: Content compliment
```
Hey {{firstName}}, thanks for connecting.

Been seeing your posts about [topic]. Good stuff.

How long have you been focused on that?
```

### Template 4: Mutual connection
```
Hey {{firstName}}, glad we connected!

I see we both know [mutual connection]. Small world.

How do you two know each other?
```

### Template 5: Background interest
```
Hey {{firstName}}, thanks for connecting.

Noticed your background in [previous company/role]. That must have been interesting.

What brought you to {{company}}?
```

## Message sequence example

**Day 1 (post-connection):**
```
Hey Sarah, thanks for connecting!

Noticed you're leading marketing at Acme. How's Q1 shaping up?
```

**Day 3 (if they reply):**
```
That's great to hear!

Question: how are you all handling [problem]? Curious because we've been helping companies like yours with exactly that.
```

**Day 3 (if no reply):**
```
Hey Sarah, hope things are good!

Wanted to share something we put together on [topic]: [link]

Thought it might be useful given what Acme is working on.
```

**Day 7 (soft pitch):**
```
Hey Sarah, quick thought:

We help [type of company] with [what you do]. Companies like [similar company] have seen [result].

Worth a 15-min call to see if we can help Acme?
```

**Day 14 (breakup):**
```
Hey Sarah,

Reached out a few times. Totally understand if timing isn't right.

If [topic] becomes a priority, I'm here. Otherwise, wishing you a great year!
```

## Personalization tips

### What to reference:
- Their recent posts or comments
- Company news (funding, launches, hires)
- Mutual connections
- Shared background (school, previous company, location)
- Content they've created (articles, podcasts)

### What to avoid:
- Generic compliments ("Love your work!")
- Overly familiar tone if you don't know them
- Long messages on first touch
- Multiple questions in one message
- Anything that sounds automated

## InMail templates (Sales Navigator)

InMail allows contacting non-connections. Same rules apply, but you pay per message (or use credits), so be selective.

### Template: Direct InMail
```
Hi {{firstName}},

I help [type of company] with [problem you solve].

[Similar company] recently saw [result] working with us.

Is this something {{company}} is focused on? Happy to share how they did it.

[Your name]
```

### Template: Question InMail
```
Hi {{firstName}},

Quick question: How is {{company}} currently handling [problem]?

I ask because we specialize in [what you do] and I think there might be a fit.

Let me know if you're open to a quick chat.

[Your name]
```

**InMail tips:**
- Response rate is lower than connection message
- Subject line matters (keep it short and relevant)
- Save for high-value prospects you can't reach otherwise

## Response handling

### If they respond positively:
```
Great to hear! Here's my calendar to grab 15 minutes: [link]

Or if easier, just let me know your availability.
```

### If they respond "maybe later":
```
No problem at all. I'll circle back in [timeframe].

In the meantime, if anything changes, just shoot me a message.
```

### If they respond negatively:
```
Totally understand. Thanks for letting me know.

If anything changes down the road, feel free to reach out.
```

### If they ask for more info:
```
Happy to share more.

Quick summary: [2-3 sentences about what you do and results]

Would it help to do a quick call to go deeper? Here's my calendar: [link]
```

## Quick reference: Do's and Don'ts

**Do:**
- Keep messages short (50-100 words)
- Personalize with something specific
- Ask questions to start conversations
- Provide value before asking
- Follow up (most replies come after 2-3 touches)

**Don't:**
- Send long paragraphs
- Pitch in first message
- Use generic templates verbatim
- Send the same message to everyone
- Give up after one message
