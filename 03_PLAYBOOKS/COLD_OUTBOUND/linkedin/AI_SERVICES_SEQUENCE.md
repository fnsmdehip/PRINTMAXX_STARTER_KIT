# LinkedIn sequence: AI services (content + UGC + compliance)

Connection request + 3 follow-up messages selling AI-powered services to SMBs. Works for content agency, UGC production, and compliance audit offers.

---

## Connection request (under 300 chars)

**Variant A: Observation-based**
```
{{firstName}}, noticed {{company}} is growing fast. We help companies like yours with AI-powered content/automation. Would love to connect and share some insights.
```
*Characters: ~140*

**Variant B: Trigger-based**
```
{{firstName}}, saw {{company}} is {{trigger}}. We've helped similar companies handle that growth with AI automation. Worth connecting?
```
*Characters: ~115*

**Variant C: Mutual ground**
```
{{firstName}}, we're both in the {{industry}} space. We build AI content and automation systems for companies like {{company}}. Let's connect.
```
*Characters: ~120*

---

## Message 1: After accept (24-48 hours)

**Template A: Question opener**
```
hey {{firstName}}, thanks for connecting.

quick question: how is {{company}} handling content creation right now? doing it in-house or outsourcing?

asking because most companies at your stage are spending 15-20 hrs/week on content. curious if that's the case for you.
```

**Template B: Observation opener**
```
hey {{firstName}}, appreciate the connect.

spent a minute looking at {{company}}'s content. you're doing solid work on {{platform}} but I noticed you're only on 1-2 platforms.

how are you thinking about expanding distribution?
```

**Template C: Congrats opener**
```
hey {{firstName}}, thanks for connecting.

saw the news about {{trigger}} at {{company}}. congrats.

companies at this stage usually need to scale content fast. how are you handling that?
```

thought that might be relevant to {{company}}.
```

**Template B: Case study tease**
```
hey {{firstName}},

they were spending $6k/mo on content. we cut that to $1,500 using AI while tripling their output from 30 to 90+ posts/month.

happy to share how they did it if you're curious.

## Response handling

**Positive:**
```
great. here's my calendar: {{calendlyLink}}

or let me know what times work for you this week.
```

**"Not now":**
```
totally understand. when would be better to circle back? i'll set a reminder.
```

**"Send more info":**
```
sure thing. quick question so I send the right stuff: what's your biggest bottleneck right now - content creation, distribution, or compliance?
```

**"What's the price?":**
```
depends on scope. content packages start at $1,500/mo. UGC videos $50-150 each. compliance audits $2,500 flat.

easiest to explain on a quick call. do you have 15 minutes this week? {{calendlyLink}}
```

**Negative:**
```
no problem. appreciate you letting me know.

if things change, feel free to reach out anytime. good luck with {{company}}.
```

## Timing

- Connection request: Mon-Thu, 8-10am their timezone
- Message 1: 24-48 hours after accept
- Message 2: 3-4 days after Message 1
- Message 3: 7 days after Message 2
- Stop after 3 unanswered messages

## Volume

- 20-25 connection requests/day
- Personalize every request (reference something specific)
- Engage with 2-3 of their posts before connecting (warm the intro)
