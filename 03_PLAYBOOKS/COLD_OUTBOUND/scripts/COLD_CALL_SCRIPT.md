# Cold call script

Phone script for cold calling prospects. Use as a framework, not a robotic script.

## Before calling

### Prep checklist:
- [ ] Know their name and company
- [ ] Know their role
- [ ] Have a relevant pain point or trigger
- [ ] Know what you want to accomplish (meeting)
- [ ] Have calendar open

### Mindset:
- You're calling to help, not to sell
- Expect rejection (it's normal)
- Be conversational, not scripted
- Smile while talking (they can hear it)

## The cold call framework

### 1. Opening (5-10 seconds)

**Goal:** Get permission to continue

```
"Hi [Name], this is [Your name] from [Company].

Did I catch you at a bad time?"
```

**Why this works:**
- Acknowledges you're interrupting
- Gives them control
- Ironically, most people say "no, what's up?"

**Alternative openers:**

```
"Hi [Name], [Your name] here. Got 30 seconds?"
```

```
"[Name]? This is [Your name]. Quick call, I promise."
```

### 2. Pattern interrupt (5 seconds)

**Goal:** Say something unexpected so they don't auto-reject

```
"I know you weren't expecting this call, and I'll be quick."
```

**Why this works:**
- Acknowledges reality
- Builds trust through honesty
- Differentiates from typical sales calls

### 3. Reason for calling (15-20 seconds)

**Goal:** Establish relevance quickly

**Template:**
```
"Reason I'm calling: I work with [type of company] who typically
struggle with [pain point].

We help them [outcome].

[Name], is that something you're dealing with at [Company]?"
```

**Example:**
```
"Reason I'm calling: I work with SaaS companies around your size
who struggle to book enough sales meetings.

We help them build outbound systems that consistently fill their pipeline.

Sarah, is pipeline generation something you're focused on right now?"
```

### 4. Handle the response

**If they say "Yes, we're dealing with that":**
```
"Great. Most companies I talk to are trying [their attempt].
What's working for you so far?"

[Listen]

"Got it. What we typically do is [brief value prop].
Would it make sense to schedule 15 minutes to see if we can help?"
```

**If they say "No, we're all set":**
```
"Fair enough. Just curious - what's working for you on that front?"

[Listen - they might reveal a gap]

If still no: "Got it. Mind if I reach out in a few months in case things change?"
```

**If they say "What is this about?":**
```
"Totally fair. In short: we help [type of company] [outcome].

Companies like [relevant company] have seen [result].

Is [their pain point] something on your radar?"
```

**If they say "I'm busy" or "Not a good time":**
```
"Completely understand. When would be a better time for a quick 2-minute
conversation? I can call you back."

[Get specific day/time]
```

### 5. Qualification (30-60 seconds)

**Goal:** Determine if they're worth a meeting

**Questions:**
```
"Just so I don't waste your time - are you the one who handles [area]?"

"What's your biggest challenge with [problem area] right now?"

"Have you looked at solutions for this before?"

"What would need to happen for this to become a priority?"
```

### 6. Close for the meeting (15 seconds)

**Goal:** Book the meeting while you have them

```
"Here's what I suggest: Let's do a quick 15-minute call where I can
learn more about [Company] and share what's worked for similar companies.

Does [specific day] work for you?"
```

**If they hedge:**
```
"I get it. How about this - you pick a time that works, and if it's
not valuable, you can hang up anytime. Fair?"
```

**Lock it in:**
```
"Great. I've got [day] at [time]. I'll send a calendar invite.
What email should I send it to?"
```

### 7. Close the call

```
"Perfect, [Name]. You'll get that invite in a few minutes.
Looking forward to [day]. Thanks for your time."
```

## Objection handling

### "We're not interested"
```
"Totally understand. Usually when I hear that, it's because [assumption].
Is that the case, or something else?"

[Listen]

If real objection surfaces: Address it
If they just want off: "Got it. Mind if I follow up via email in case
things change?"
```

### "Send me some information"
```
"Happy to. What specifically would be helpful?

Actually, it might be easier if we do a quick 10-minute call and I can
walk you through exactly what applies to [Company].

Would [time] work?"
```

### "We use [competitor/solution]"
```
"How's that going for you?

[Listen]

Interesting. A lot of our clients actually came from [competitor] when
they realized [common gap]. Has that been an issue for you?"
```

### "I'm not the right person"
```
"Got it. Who handles [decision area] at [Company]?

Would you be comfortable making an intro? Or I can mention you sent me."
```

### "Call me back in [timeframe]"
```
"Will do. Just so I'm prepared - what would need to be different in
[timeframe] for this to make sense?"

[Document and actually call back]
```

### "How much does it cost?"
```
"Depends on what we'd be doing together. Typically [range] depending
on scope.

But the better question is whether we can deliver ROI worth that
investment. That's what I'd want to figure out on a quick call.

Can we do [time]?"
```

## Voicemail (if they don't answer)

See VOICEMAIL_SCRIPT.md for dedicated voicemail templates.

Quick version:
```
"Hi [Name], this is [Your name] from [Company].

Calling about [one sentence value prop].

Call me back at [number] or I'll try you again [day].

Again, [Your name], [number]."
```

## Call cadence

### How many calls per contact:
- Attempt 1: Day 1
- Attempt 2: Day 3 (different time of day)
- Attempt 3: Day 7
- Attempt 4: Day 14 (then move to email/nurture)

### Best times to call:
- 8-9 AM (before the day gets busy)
- 11:30 AM - 12:00 PM (before lunch)
- 4-5 PM (end of day, wrapping up)
- Avoid Monday mornings and Friday afternoons

### Volume targets:
- 30-50 dials per day for SDR
- 50-80 dials per day aggressive
- Expect 10-20% connect rate
- Expect 10-20% conversion to meeting (of connections)

## Call tracking

### Log for each call:
- Date/time
- Outcome (connected, voicemail, no answer, meeting booked)
- Notes from conversation
- Follow-up scheduled
- Objections heard

### Metrics to track:
- Dials per day
- Connect rate
- Conversation to meeting rate
- Meeting show rate
- Objections (categorize and learn)

## Mindset and energy

### Before the call block:
- Get in a good mental state
- Stand up while calling
- Have water nearby
- Turn off distractions

### During calls:
- Smile (changes your voice)
- Speak slowly and clearly
- Use their name 2-3 times
- Mirror their energy level
- Listen more than talk

### After rejection:
- Don't take it personally
- Log and move on
- Learn from objections
- Remember: 90%+ won't book. That's normal.

## Practice

### Before going live:
1. Practice script out loud 10+ times
2. Record yourself and listen back
3. Role play with colleague
4. Practice handling objections
5. Have script visible but don't read robotically

### Improvement loop:
1. Make 20 calls
2. Review what worked/didn't
3. Adjust script
4. Repeat
