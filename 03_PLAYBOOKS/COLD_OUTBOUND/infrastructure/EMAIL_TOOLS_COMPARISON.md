## Decision Matrix

### Choose Instantly if:
- Starting out
- Want simple UI
- Need unlimited inboxes
- Don't want to think about warmup

### Choose Smartlead if:
- Scaling past 1000 emails/day
- Want better analytics
- Running multiple clients/campaigns
- Need API access

### Choose Emailbison if:
- Budget conscious
- Comfortable with newer tool
- Want aggressive pricing
- Don't need hand-holding

### Choose Apollo if:
- Need lead data + sending
- Want one tool for everything
- Doing LinkedIn + email together
- B2B SaaS focus

### Choose Lemlist if:
- Heavy personalization (images, video)
- Multichannel sequences
- Willing to pay more
- Want pretty templates

## Red Flags to Avoid

**Skip these:**
- Tools with per-email pricing at scale (gets expensive)
- Anything requiring annual contracts upfront
- Tools without built-in warmup (unless you have a reason)
- Enterprise tools (Outreach, Salesloft) - overkill and overpriced

**Watch for:**
- Hidden inbox limits
- Warmup that doesn't actually work
- Data that's outdated
- Support that disappears
