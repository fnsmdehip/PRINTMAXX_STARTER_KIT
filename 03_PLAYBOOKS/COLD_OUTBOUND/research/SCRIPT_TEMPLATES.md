# Cold Email Script Templates

**Updated:** 2026-01-20
**Purpose:** 20 proven cold email templates for B2B outreach

---

## Template Structure Framework

### The AIDA Formula

```
ATTENTION: Subject line + first line (hook)
INTEREST: Why they should care (problem/observation)
DESIRE: Your solution (brief, specific)
ACTION: Clear CTA (low commitment)
```

### Email Anatomy

```
Subject: [50 chars max, personalized]

[Line 1: Hook/observation - proves you researched]

[Line 2-3: Problem statement/bridge]

[Line 4: Your offer/solution - specific value]

[Line 5: Soft CTA - easy to respond to]

[Signature: Name only, no fluff]
```

### Universal Rules

1. Under 100 words total
2. One clear ask per email
3. No attachments in first email
4. No HTML/images
5. Mobile-friendly formatting
6. Personalization that proves research
7. Specific, not generic

## Category 1: Observation-Based Templates

### Template 1: The Tech Stack Observation

**Best for:** SaaS companies, tech products

```
Subject: Noticed you're using {competitor}

Hey {FirstName},

Saw {Company} is using {Competitor} for {function}.

Most teams I talk to using {Competitor} waste {X hours/week} on {specific pain}.

We help companies like {Similar Company} cut that by {X%} without switching everything.

Worth a 15-min look?

{YourName}
```

**Variables:**
- competitor: Their current tool
- function: What the tool does
- specific pain: Known competitor weakness
- Similar Company: Social proof
- X%: Specific result

---

### Template 2: The Job Posting Observation

**Best for:** Services, agencies, tools that replace hires

```
{YourName}
```

## Category 2: Problem-Based Templates

**Best for:** Clear pain points, known issues

```

{FirstName},

Quick question - is {specific problem} something you're dealing with at {Company}?

Most {their role}s I talk to spend {X hours/week} on this.

We've helped {similar companies} cut that to {Y hours} with {brief solution description}.

If that's relevant, happy to show you how in 15 minutes.

{YourName}
```

---

### Template 7: The Question Hook

**Best for:** Discovery, qualifying interest

```
Subject: Quick question

{FirstName},

How are you currently handling {specific process} at {Company}?

Asking because most {role}s tell me it's their biggest time sink.

If you're happy with your current setup, ignore this. But if you're spending more than {X hours/week} on it, might be worth a chat.

What do you think?

{YourName}
```

```
{YourName}
```

## Category 4: Follow-Up Templates

**Best for:** 3-5 days after no response

```
Worth chatting?

{YourName}
```

**Best for:** 5-7 days after no response

```

{FirstName},

Still happy to show you how {brief value prop} if timing is better now.

{YourName}
```

### Template 18: The Breakup Email

**Best for:** Final email in sequence

```

If that changes, you know where to find me.

{YourName}
```

```
{YourName}
```

```
{YourName}
```

## Subject Line Bank

### Observation-Based
- "Noticed you're using {competitor}"
- "Quick note about {Company}"
- "Re: your {role} opening"
- "Saw your post about {topic}"
- "Congrats on the {milestone}"

### Question-Based
- "Quick question"
- "How do you handle {process}?"
- "Who handles {function} at {Company}?"
- "Thoughts on {topic}?"

### Value-Based
- "Idea for {Company}"
- "{Resource} for {Company}"
- "How {Similar Company} {result}"
- "{Mutual connection} suggested I reach out"

### Follow-Up
- "Re: {original subject}"
- "Bumping this"
- "Should I close your file?"
- "Things change in {X months}"

### Avoid
- ALL CAPS anything
- "Quick favor"
- "Touching base"
- "Just following up"
- "Hope you're well"
- Fake Re:/Fwd:

## Personalization Checklist

### Before Sending, Verify:

- [ ] Company name is correct
- [ ] First name is spelled right
- [ ] Role/title is accurate
- [ ] Observation is actually true
- [ ] Competitor/tool mention is verified
- [ ] Similar company is relevant
- [ ] Results/numbers are accurate

### Personalization Sources

| Source | What to Find |
|--------|--------------|
| LinkedIn | Role, posts, activity |
| Company website | Tech stack, team, news |
| Twitter/X | Recent thoughts, interests |
| Press releases | Funding, launches, hires |
| G2/Capterra | Tool reviews |
| Crunchbase | Funding, investors |
| BuiltWith | Tech stack |

## Template Testing Framework

### How to A/B Test

1. Pick one variable to test
2. Create two versions
3. Send 50+ each
4. Measure response rate
5. Winner becomes control
6. Test next variable

### Variables to Test

| Variable | Impact | Test Frequency |
|----------|--------|----------------|
| Subject line | Highest | Every campaign |
| First line | High | Monthly |
| CTA | High | Monthly |
| Length | Medium | Quarterly |
| Personalization depth | Medium | Quarterly |
| Send time | Low-Medium | Quarterly |

### Tracking Template

```markdown
## Template Test: {Name}

**Variable tested:** {Subject line / CTA / etc.}

**Version A:** {Description}
**Version B:** {Description}

**Sample size:** {X emails each}

**Results:**
- Version A: {X}% response rate
- Version B: {X}% response rate

**Winner:** {A/B}
**Action:** {Update template}
```

## Common Mistakes to Avoid

### Mistake 1: Too Long
**Problem:** Walls of text
**Fix:** Under 100 words, 5-6 lines max

### Mistake 2: Too Generic
**Problem:** Could send to anyone
**Fix:** Add specific observation/personalization

### Mistake 3: Weak CTA
**Problem:** "Let me know your thoughts"
**Fix:** Specific ask: "Worth a 15-min call Tuesday?"

### Mistake 4: Multiple Asks
**Problem:** Call + download + visit website
**Fix:** One clear action per email

### Mistake 5: All About You
**Problem:** "We are... We do... We help..."
**Fix:** Lead with their problem, not your solution

### Mistake 6: No Value
**Problem:** Just asking for a meeting
**Fix:** Offer insight, resource, or specific benefit
