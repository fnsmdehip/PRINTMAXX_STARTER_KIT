# Web development agency 5-touch sequence

Use this for: Web dev agencies, Webflow agencies, WordPress developers, custom development shops.

## Sequence overview

| Email | Timing | Goal |
|-------|--------|------|
| 1 | Day 0 | Site observation |
| 2 | Day 3 | Performance angle |
| 3 | Day 7 | Before/after showcase |
| 4 | Day 12 | Quick win offer |
| 5 | Day 17 | Breakup |

Expected reply rate: 4-8%
Expected meeting rate: 2-4%

---

## Email 1: Site observation

**Subject line variants:**
- A: {{company}} website
- B: Quick observation
- C: Your site's [specific issue]

**Body:**

```
{{firstName}},

Took a look at {{company}}'s website and noticed [specific observation].

[1-2 sentences explaining why this matters - conversion impact, user experience, etc.]

We've helped [similar company] fix this exact issue and they saw [specific result].

Worth a quick call to discuss? Or I can send over a brief audit if that's easier.

[Your name]
[Your agency]
```

**Example observations:**
- "Your site loads in 6 seconds on mobile. Industry benchmark is under 3."
- "Your checkout flow has 5 steps. Reducing to 3 typically lifts conversion 15%+."
- "Your mobile navigation covers the CTA button on scroll."
- "Your contact form has 8 fields. Reducing to 4 usually doubles submissions."

---

## Email 2: Performance impact

**Subject line variants:**
- A: Re: {{company}} website (reply thread)
- B: Performance = revenue
- C: What slow sites cost

**Timing:** 3 days after Email 1

**Body:**

```
{{firstName}},

Quick follow-up with some context on why [the issue] matters:

Google research shows:
- 53% of mobile users abandon sites that take >3 seconds to load
- Every 1-second delay reduces conversions by 7%
- Site speed directly impacts search rankings

For a site like {{company}}, this could mean [estimated impact].

We've helped [X] companies improve their scores from [before] to [after].

Would you be open to a free performance audit?

[Your name]
```

**Why it works:**
- Data-backed claims
- Quantifies the problem
- Free offer reduces friction

**Subject line variants:**

**Body:**

```

Let us fix [one specific issue] for {{company}}.

**The fix:** [Specific change]
**Timeline:** [X] days
**Investment:** [$X] flat rate
**Expected impact:** [Improvement estimate]
```

- Specific deliverable
- Proves value before bigger projects

## Site audit checklist

### Before emailing, check:

**Performance:**
- [ ] PageSpeed Insights score (Google)
- [ ] Mobile loading time
- [ ] Core Web Vitals (LCP, FID, CLS)
- [ ] Image optimization

**User experience:**
- [ ] Mobile responsiveness
- [ ] Navigation clarity
- [ ] CTA visibility
- [ ] Form complexity

**Technical:**
- [ ] SSL certificate
- [ ] Broken links (use screaming frog)
- [ ] Meta tags and SEO basics
- [ ] Schema markup

**Conversion:**
- [ ] Clear value proposition above fold
- [ ] Trust signals (testimonials, logos)
- [ ] Friction in conversion flow

### Tools:
- **Google PageSpeed Insights:** Performance scores
- **GTmetrix:** Detailed performance analysis
- **Mobile-Friendly Test:** Mobile issues
- **Screaming Frog:** Technical SEO audit
- **BuiltWith:** What tech stack they use

## Service positioning

### Full redesign (big project)
- Complete site rebuild
- $10K-$50K+ depending on scope
- 2-4 month timeline
- Best for: Outdated sites, rebrands

### Performance optimization (quick win)
- Speed improvements only
- $2K-$5K
- 1-2 week timeline
- Best for: Good design, poor performance

### Landing page (foot in door)
- Single high-converting page
- $1K-$3K
- 1-2 weeks
- Best for: Testing relationship

### Ongoing maintenance
- Monthly retainer for updates/support
- $500-$2K/month
- Best for: After project completion

## Vertical-specific angles

### Ecommerce:
```
Your product page load time is costing you sales. Shopify benchmarks show
every second of delay reduces conversion by 7%. At your traffic level,
that's potentially $[X] in lost monthly revenue.
```

### SaaS:
```
Your marketing site isn't matching the polish of your product. First
impressions matter. Visitors decide in 3 seconds if you're credible
enough to try.
```

### Professional services:
```
Your competitors' sites look more modern. For [lawyers/accountants/etc.],
trust is everything. Your site should signal "established and professional,"
not "we built this in 2015."
```

### Local business:
```
80% of your visitors are on mobile. Your site isn't mobile-optimized,
which means you're losing potential customers before they even call.
```

## Objection handlers

**"We just redesigned"**
```
Good to know. When was that? A lot changes in 2-3 years (Core Web Vitals
requirements, mobile standards, conversion best practices). Might be worth
a quick audit to see if there are easy wins without a full redesign.
```

**"We're using [Squarespace/Wix/etc.]"**
```
Those platforms are fine for starting out. Most companies outgrow them when
they need [specific thing: custom functionality, better performance, integrations].
When that happens, we're here.
```

**"It's not a priority"**
```
Fair enough. One question: What would have to happen for it to become a priority?
A competitor launching a better site? A big traffic spike coming? Good to know
for when I follow up later.
```

**"Too expensive"**
```
We can scale the scope. Instead of a full redesign, what if we tackled just
the [highest impact area] for [smaller amount]? Prove the value before
investing more.
```
