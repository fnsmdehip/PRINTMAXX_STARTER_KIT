# Account Naming Guide

Strategic naming conventions for content farm accounts across niches and platforms. Good names increase discoverability, memorability, and trust.

---

## Naming principles

### Core rules

1. **Memorable** - Easy to remember and spell
2. **Searchable** - Contains niche keywords
3. **Scalable** - Works as you grow
4. **Available** - Check all platforms first
5. **Authentic** - Matches content style

### Name types

| Type | Example | Best for | Risk level |
|------|---------|----------|------------|
| Personal brand | @alexbuilds | Authority accounts | Low |
| Niche keyword | @productivityhacks | SEO-focused | Medium |
| Brandable | @hustleforge | Long-term brand | Low |
| Theme-based | @dailystoic | Content aggregators | Medium |
| Persona | @techmomreviews | Character accounts | Low |
| Descriptive | @5minworkouts | How-to content | Medium |

## Platform-specific considerations

### X/Twitter

**Constraints:**
- Max 15 characters
- Underscores allowed
- Case insensitive

**Best practices:**
- Shorter is better (easier to mention)
- Include niche keyword
- Avoid numbers at end (looks spammy)

**Examples:**
```
Good: @productivitylab (14 chars)
Good: @alexbuilds (10 chars)
Bad: @productivityhacks123 (too long, numbers)
```

### TikTok

**Constraints:**
- Max 24 characters
- Periods and underscores allowed
- No spaces

**Best practices:**
- Memorable over keyword-stuffed
- Personality shows through
- Can be more playful

**Examples:**
```
Good: @fitwithjess
Good: @the.money.guy
Bad: @fitness_tips_and_workout_ideas
```

### Instagram

**Constraints:**
- Max 30 characters
- Periods, underscores, numbers allowed
- Case insensitive

**Best practices:**
- Brand consistency with other platforms
- Clean and professional
- Easy to tag

**Examples:**
```
Good: @alexbuilds
Good: @the.productivity.lab
Bad: @alex.builds.stuff.for.entrepreneurs
```

### YouTube

**Constraints:**
- Max 100 characters (but shorter better)
- Spaces allowed in display name
- Handle separate from name

**Best practices:**
- Display name: Full, readable
- Handle: Short, memorable
- Keywords help search

**Examples:**
```
Display: Alex Builds | Productivity Tips
Handle: @alexbuilds
```

## Naming process

### Step 1: Brainstorm (15 minutes)

Generate 20+ names using these prompts:
- [Your name] + [action verb]
- [Niche keyword] + [modifier]
- [Number] + [topic]
- [Aspiration] + [identity]
- Invented/brandable word

### Step 2: Filter (10 minutes)

Remove names that:
- Are hard to spell
- Sound like something else
- Are too long
- Are too generic
- Feel inauthentic

### Step 3: Check availability (15 minutes)

Use namecheckr.com or manual search:
- [ ] Twitter/X available
- [ ] Instagram available
- [ ] TikTok available
- [ ] YouTube available
- [ ] .com domain (optional)

### Step 4: Test (5 minutes)

Ask yourself:
- Can I say this name 1,000 times?
- Will I want this name in 5 years?
- Does it fit the content I'll make?
- Would I follow an account with this name?

## Common mistakes

### Avoid these patterns

**Too generic:**
- @fitness
- @money
- @productivity
(Usually taken and not memorable)

**Too clever:**
- @pr0ductiv1ty
- @fitne$$
(Hard to find, looks spammy)

**Too long:**
- @productivitytipsandtricksfordaily
- @fitnessworkoutsandmealprep
(Gets truncated, hard to remember)

**Numbers at end:**
- @alexbuilds123
- @fitnessguy99
(Looks like backup account)

**Underscores everywhere:**
- @_alex_builds_
- @fitness__tips
(Hard to type, looks odd)

## Name change considerations

### When to keep your name
- Already have 1K+ followers
- Getting brand recognition
- Name still fits content
- No major issues

### When to change
- Pivoting niches completely
- Name has negative associations
- Consistently misspelled
- Much better name available

### How to change
1. Update name across all platforms same day
2. Announce change to audience
3. Pin post explaining new name
4. Update all bio links
5. Maintain content consistency
