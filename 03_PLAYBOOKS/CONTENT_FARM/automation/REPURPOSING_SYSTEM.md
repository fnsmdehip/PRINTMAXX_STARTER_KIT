# Content Repurposing System

Turn 1 piece of content into 10+ pieces across platforms. Maximize ROI on content creation.

---

## The repurposing philosophy

**Core principle:** Create once, distribute everywhere

Every piece of content should become:
- Multiple formats (video, text, image)
- Multiple lengths (short, medium, long)
- Multiple platforms (X, IG, TikTok, YT)
- Multiple angles (educational, entertainment, promotional)

## Platform-specific formatting

### From video to shorts

**Vertical video requirements:**

| Platform | Aspect | Max length | Min length |
|----------|--------|------------|------------|
| TikTok | 9:16 | 10 min | 5s |
| IG Reels | 9:16 | 90s | 5s |
| YT Shorts | 9:16 | 60s | 15s |

**Conversion process:**
1. Identify 3-5 compelling segments
2. Crop/reframe to vertical
3. Add captions
4. Add hook at beginning
5. Add CTA at end
6. Adjust for platform

### From long-form to threads

**Conversion formula:**
```
Introduction → Tweet 1 (hook)
Main point 1 → Tweet 2
Main point 2 → Tweet 3
Main point 3 → Tweet 4
Main point 4 → Tweet 5
Conclusion → Tweet 6 (summary)
CTA → Tweet 7 (action)
```

**Thread writing rules:**
- Each tweet stands alone
- Numbers or bullets help
- Keep under 280 characters
- End with clear CTA

### From video to carousel

**Conversion process:**
1. Identify 8-10 key points
2. One point per slide
3. Add visuals/icons
4. Keep text minimal
5. Strong hook on slide 1
6. CTA on last slide

**Carousel template:**
```
Slide 1: Hook/title
Slide 2: The problem
Slides 3-8: Key points (1 each)
Slide 9: Summary
Slide 10: CTA
```

## The repurposing workflow

### Phase 1: Create pillar content

**Weekly:**
- 1 long-form video OR
- 1 in-depth thread OR
- 1 carousel guide OR
- 1 podcast episode

### Phase 2: Extract clips/segments

**Same day or next day:**
- Identify 5-10 repurposable segments
- Note timestamps or sections
- Rank by potential virality

### Phase 3: Format for platforms

**Days 2-3:**
- Create vertical clips
- Write platform-specific captions
- Design supporting graphics
- Generate text versions

### Phase 4: Schedule distribution

**Days 3-4:**
- Add to content queues
- Stagger across week
- Different platforms, different days
- Same content, different times

### Phase 5: Track and optimize

**Ongoing:**
- Monitor performance
- Note what works where
- Adjust future repurposing
- Double down on winners

## Platform-specific rules

### TikTok repurposing rules

**Do:**
- Remove other platform watermarks
- Use trending sounds
- Add TikTok-native captions
- Post natively (not shared)

**Don't:**
- Post with YouTube watermark
- Use same caption everywhere
- Ignore trending sounds
- Post too similar content same day

### Instagram repurposing rules

**Do:**
- Optimize for Reels algorithm
- Use carousel for educational
- Vary content formats
- Add hashtags strategically

**Don't:**
- Post same Reel 2 days in a row
- Ignore Stories opportunity
- Use same hashtags always
- Neglect caption quality

### YouTube Shorts rules

**Do:**
- Strong hook in 1 second
- Vertical 9:16 format
- Under 60 seconds
- Loop potential

**Don't:**
- Post horizontal videos
- Exceed 60 seconds
- Ignore first frame
- Forget end screen

### X/Twitter rules

**Do:**
- Vary formats (text, image, video)
- Thread educational content
- Use quote tweets strategically
- Engage after posting

**Don't:**
- Post same tweet twice
- Ignore character limits
- Forget to add media
- Post and ghost

## Measuring repurposing ROI

### Track per pillar piece

| Metric | Measurement |
|--------|-------------|
| Creation time | Hours to create original |
| Repurposing time | Hours to create all versions |
| Total pieces created | Number of content pieces |
| Total reach | Sum of impressions |
| Total engagement | Sum of engagements |
| Leads generated | Sign-ups attributed |
| Revenue attributed | Sales from content |

### Calculate efficiency

```
Content ROI = Total reach / Creation hours
Repurposing ROI = Total reach / (Creation + repurposing hours)

Example:
- 1 video: 10 hours to create
- Repurposed to 10 pieces: 5 additional hours
- Total reach: 500,000 impressions
- Without repurposing: 50,000 reach per hour
- With repurposing: 33,333 reach per hour
- BUT: Total reach 10x higher

Winner: Repurposing (10x reach for 1.5x time)
```
