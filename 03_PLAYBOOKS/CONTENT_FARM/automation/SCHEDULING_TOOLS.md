## Multi-account setup

### Strategy 1: Single tool for all

**Pros:**
- One dashboard
- Unified analytics
- Easier to manage

**Cons:**
- More expensive
- May lack platform-specific features

**Best tool: Hootsuite or Metricool**

### Strategy 2: Platform-specific tools

**Pros:**
- Best features per platform
- Often cheaper combined
- Platform-optimized

**Cons:**
- Multiple dashboards
- More login management

**Recommended stack:**
- X: Hypefury ($29)
- Instagram: Later ($18)
- TikTok: Later (included) or native
- YouTube: Native

**Total: $47/month** (vs $99+ for Hootsuite)

### Strategy 3: Hybrid approach

**Core tool:** Buffer or Publer (all platforms, basic)
**Power tools:** Add Hypefury for X, Later for IG/TT

## Content batching workflow

### Weekly batching process

**Day 1: Content creation**
```
Morning: Film all video content (2-3 hours)
Afternoon: Write all text content (2-3 hours)
Output: 20-30 pieces of content
```

**Day 2: Editing and formatting**
```
Morning: Edit videos (2-3 hours)
Afternoon: Create graphics (2-3 hours)
Output: All content ready for scheduling
```

**Day 3: Scheduling**
```
Morning: Schedule all platforms (1-2 hours)
Afternoon: Set up engagement reminders
Output: Week fully scheduled
```

### Monthly batching process

**Week 1:** Create content for month
**Week 2:** Edit and prepare all content
**Week 3:** Schedule first 2 weeks
**Week 4:** Schedule last 2 weeks + plan next month

### Batch scheduling tips

1. **Create content categories**
   - Educational
   - Entertainment
   - Promotional
   - Engagement

2. **Use content buckets**
   - Evergreen (reusable)
   - Timely (date-specific)
   - Trending (flexible)

3. **Schedule in order**
   - Main feed content first
   - Stories/ephemeral second
   - Engagement posts last

## Queue management

### Queue structure

**Primary queue:** Regular content
- Automatically posts at scheduled times
- Refill weekly

**Promotional queue:** Sales/CTA content
- Interspersed with primary
- 10-20% of content

**Evergreen queue:** Reusable content
- Auto-cycles through best performers
- Set and forget

### Queue ratios

**Recommended mix:**
```
60% Educational/Value content
20% Entertainment/Engagement
10% Personal/Behind-scenes
10% Promotional/CTA
```

### Queue maintenance

**Weekly:**
- Review upcoming week's queue
- Adjust for current events
- Add trending content

**Monthly:**
- Review queue performance
- Remove underperformers
- Add new evergreen content
- Update promotional content

## Automation rules

### Auto-scheduling rules

**Rule 1: Optimal time slots**
```
Platform determines best times based on:
- Your historical engagement
- Audience activity
- Competition
```

**Rule 2: Content spacing**
```
Minimum 2 hours between posts
No more than X posts per day
Vary content types
```

**Rule 3: Engagement windows**
```
Schedule posts when you can engage
First 30-60 minutes matter most
Plan around your availability
```

### Auto-recycling rules

**Which content to recycle:**
- Evergreen educational content
- Top-performing posts (3%+ engagement)
- Timeless tips and hacks

**Recycling schedule:**
- Wait 30+ days before recycling
- Update hook/image slightly
- Track performance of recycled content

**What NOT to recycle:**
- News/trending content
- Time-specific posts
- Underperforming content
- Promotional posts

## Troubleshooting

### Common issues

**Posts not publishing:**
1. Check account connection
2. Verify permissions
3. Check for platform rate limits
4. Review content for violations

**Wrong posting times:**
1. Verify timezone settings
2. Check daylight saving
3. Confirm schedule in tool

**Poor engagement on scheduled posts:**
1. Test different times
2. Engage immediately after posting
3. Check content quality
4. Verify not shadowbanned

### Platform-specific issues

**Twitter/X:**
- API rate limits (check with tool)
- Thread posting issues
- Image upload failures

**Instagram:**
- Auto-publish limitations
- Hashtag limits
- Account connection drops

**TikTok:**
- Limited third-party support
- Video format requirements
- Sound/music restrictions
