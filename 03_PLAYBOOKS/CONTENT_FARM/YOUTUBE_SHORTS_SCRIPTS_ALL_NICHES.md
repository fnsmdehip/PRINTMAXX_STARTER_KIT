## Template Customization Guide

Each niche has 5 scripts with these consistent elements:

**Hook Formula (0-2 seconds):**
- Start with consequence or confession
- Use "I/we" not generic advice
- Create curiosity or emotional resonance

**Content (2-50 seconds):**
- Specific examples, not vague claims
- Show actual numbers when possible
- 3-5 key points max
- Visual breaks (B-roll, on-screen text)

**CTA (50-60 seconds):**
- Offer specific value (link to what?)
- Make it easy (one click)
- Create follow-through (subscribe, check link bio)

**Text Overlay Formatting:**
- All CAPS for emphasis
- 3-5 words max per line
- Place on screen 1-2 seconds
- High contrast (white/yellow on dark background)

**Visual Format Selection:**
- **Talking head:** Confessional, emotional, trust-building
- **Screen record:** How-to, dashboard, app demos
- **B-roll:** Lifestyle, transformation, narrative
- **Split screen:** Comparison, before/after, alternatives

**Sound Selection Strategy:**
- Match energy level (upbeat for productivity, calm for wellness)
- Use trending TikTok sounds when applicable
- Niche-specific music (worship for faith, gym beats for fitness)
- 8-bit notification sounds for tech niches
