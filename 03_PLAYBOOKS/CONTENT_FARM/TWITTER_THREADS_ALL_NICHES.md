## Thread Architecture Framework

### Three Core Structures Per Niche

1. **How-To Thread** - Step-by-step actionable advice (best for algorithm: 7-10 tweets)
2. **Story Thread** - Personal narrative with lesson (best for engagement: 5-8 tweets)
3. **Listicle Thread** - Ranked/categorized options (best for shares: 6-9 tweets)

### Universal Thread Success Metrics

- **Hook strength:** First tweet must stop scroll (consequence-first, no explanation)
- **Thread length:** 6-9 tweets optimal (not 15+, that's dead TikTok energy)
- **CTA placement:** End of thread only, soft angle ("reply 'X' for Y" or just "link in bio")
- **Engagement tactic:** Self-reply funnel OR reply-bait question OR contrarian take
- **Pacing:** One core idea per tweet. No fluff tweets.

### Global CTA Options

**Option 1 (Soft, No Link):**
```

```


**Option 4 (Authority):**
