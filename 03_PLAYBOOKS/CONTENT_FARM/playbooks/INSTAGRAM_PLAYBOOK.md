## Account infrastructure

### Profile isolation requirements

Instagram is owned by Meta and has sophisticated tracking. Isolation is critical.

| Component | Requirement |
|-----------|-------------|
| Browser profile | GoLogin or unique device |
| Proxy | Residential, unique IP per account |
| Phone number | Unique (for verification) |
| Email | Unique email per account |
| Device | Ideally separate phone per account |
| Facebook | Never link to same FB account |

**Meta's tracking methods:**
- Device fingerprint
- IP address
- Email patterns
- Phone numbers
- Facebook pixel/cookies
- Behavioral patterns

### Setup options

**Option 1: Mobile devices (safest)**
- Used iPhone/Android: $80-150 each
- Separate SIM or Wi-Fi
- One account per device
- Cost: $100-200 per account setup

**Option 2: Browser profiles (moderate risk)**
- GoLogin: $49/month for 100 profiles
- Residential proxy: $5-10/month per IP
- Mobile user agent spoofing
- Cost: $5-10/month per account

**Option 3: Emulators (highest risk)**
- BlueStacks or LDPlayer
- Only for testing
- High ban rate
- Not recommended for main accounts

### Account creation checklist

- [ ] Create unique email (ProtonMail, Tutanota)
- [ ] Set up isolated environment (device or GoLogin)
- [ ] Configure residential proxy (match target location)
- [ ] Create account (use phone number if possible)
- [ ] Complete profile immediately
- [ ] Add profile photo
- [ ] Write bio (simple, no links yet)
- [ ] Don't switch to business account yet
- [ ] Browse explore for 30 minutes

## 14-day warmup protocol

Instagram bans new accounts aggressively. Patience here saves accounts.

### Days 1-3: Passive browsing

**Actions per day:**
- Browse explore/feed: 30-45 minutes
- Like posts: 10-15 (in your niche)
- Watch stories: 10-15 accounts
- Watch Reels: 15-20 minutes
- No follows
- No posting
- No DMs

**Goal:** Establish normal user behavior pattern

### Days 4-7: Light engagement

**Actions per day:**
- Browse: 30 minutes
- Like posts: 15-25
- Follow accounts: 5-10 (mix of sizes)
- Watch stories completely
- Save posts: 5-10
- First story: Day 6 (simple photo/text)

**Follow rules:**
- Max 10 follows/hour
- Mix followers and non-followers
- Some big accounts, some small
- Wait 30 seconds between actions

### Days 8-10: Start posting

**Actions per day:**
- Engagement: Continue previous levels
- First feed post: Day 8 (no hashtags)
- Story: 1-2 per day
- Reply to any comments within 1 hour
- Follow: 10-15/day
- Unfollow: 0 (not yet)

**First post guidelines:**
- Single image or carousel
- Native Instagram filter
- Simple caption
- No hashtags (or max 3)
- No external links
- Post at peak hours

### Days 11-14: Ramp up

**Actions per day:**
- Posts: 1 feed post + 2-3 stories
- Engagement: 30-40 likes, 10-20 comments
- Follow: 15-20/day
- Can start unfollowing: 5-10/day
- Can use more hashtags: up to 15

**Day 14 milestones:**
- Switch to Business/Creator account
- First Reel post
- Add link in bio
- Full hashtag strategy

### Warmup metrics

| Day | Followers | Following | Posts | Reach |
|-----|-----------|-----------|-------|-------|
| 1 | 0 | 0 | 0 | 0 |
| 7 | 20-50 | 50-100 | 2-3 | 500-1K |
| 14 | 100-300 | 150-300 | 8-12 | 3K-10K |

Red flag if: reach stays under 500 by day 10

## Reels algorithm tactics

### How the algorithm works

Instagram Reels algorithm scores:
1. **Watch time %** - Most important
2. **Rewatches** - Loops count
3. **Shares** - To DMs and stories
4. **Saves** - Strongest signal after watch time
5. **Comments** - Length matters
6. **Likes** - Weakest signal
7. **Follows from reel** - Very strong

### Reel structure

```
0:00-0:01 - Hook (stop the scroll)
0:01-0:03 - Context (what this is about)
0:03-0:08 - Value/entertainment
0:08-0:12 - Payoff or surprise
0:12-0:15 - CTA or loop point
```

**Optimal length:**
- Educational: 15-30 seconds
- Entertainment: 7-15 seconds
- Product: 10-20 seconds
- Trending: Match original sound length

### Hook strategies

**Visual hooks:**
- Movement in first frame
- Bright colors
- Text on screen immediately
- Face close to camera
- Pointing at something

**Audio hooks:**
- Start mid-sentence
- Trending sound (recognizable)
- Shocking statement
- Question

**Text hooks:**
- "Wait for the end"
- "You're doing [X] wrong"
- "Save this for later"
- Numbers and stats
- "POV:" format

### Hashtag strategy 2024

Instagram reduced hashtag importance but they still help.

**Formula: 5-15 hashtags**
```
- 2-3 broad niche (#fitness, #productivity)
- 3-5 specific niche (#homeworkout, #morningroutine)
- 2-3 content type (#reelsinstagram, #motivationalquotes)
- 1-2 trending/seasonal
```

**Where to place:**
- In caption (visible): Max 3-5
- First comment (hidden): Rest of them
- Never use banned hashtags

### Posting times

**Best times (EST):**
| Day | Best times |
|-----|------------|
| Monday | 6am, 10am, 10pm |
| Tuesday | 2am, 4am, 9am |
| Wednesday | 7am, 8am, 11pm |
| Thursday | 9am, 12pm, 7pm |
| Friday | 5am, 1pm, 3pm |
| Saturday | 11am, 7pm, 8pm |
| Sunday | 7am, 8am, 4pm |

**Note:** Test and adjust based on your audience insights.

### Audio strategy

**Trending sounds:**
- Check Reels tab for recurring audio
- Use sounds before they peak (< 10K uses ideal)
- Original audio for educational content
- Save trending sounds when browsing

**Finding sounds:**
- Browse Reels for 15 minutes daily
- Check competitors' recent Reels
- Use Tokboard or TrendTok (works for IG too)
- Meta's Sound Collection for commercial use

## Risk management

### Shadowban detection

**Signs:**
- Hashtag reach drops to zero
- Explore reach disappears
- Only followers see content
- Engagement drops 70%+

**Testing:**
1. Search hashtag you used
2. Check if your post appears
3. Ask follower to search
4. Check Insights for "From hashtags" = 0

**Recovery:**
- Stop posting for 48-72 hours
- Remove all hashtags from recent posts
- Delete any flagged content
- Resume without hashtags
- Gradually reintroduce (3, then 5, then 10)

### Action blocks

**What triggers blocks:**
- Too many follows/unfollows (50+/hour)
- Too many likes (200+/hour)
- Repetitive comments
- Same DM to multiple people
- Using automation incorrectly

**Types of blocks:**
- Soft block: Action blocked for 24 hours
- Hard block: Action blocked for 7+ days
- Permanent: Feature disabled forever

**Prevention:**
- Stay under 50 actions/hour
- Vary action types
- Take 5-minute breaks hourly
- Don't use cheap automation
- Mix manual and automated actions

### Ban prevention checklist

**Never do:**
- Mass follow/unfollow
- Use banned hashtags
- Post copyrighted content
- Use aggressive automation
- Buy followers
- Post same content on multiple accounts
- Excessive link posting

**Always do:**
- Space out actions naturally
- Unique content per account
- Engage before posting
- Respond to comments
- Use Instagram native features
- Follow community guidelines

### Account recovery

**If account disabled:**
1. Request review immediately (in-app)
2. Submit ID verification if asked
3. Wait 24-48 hours
4. Submit appeal via help form
5. Check email for responses
6. If denied, submit one more appeal
7. After 2 denials, account is likely gone

**Prevention backups:**
- Export follower list monthly
- Save all content locally
- Document account stats
- Have backup accounts warming
