## Automation Best Practices

### Do's

- Start simple, add complexity gradually
- Always review AI-generated content
- Maintain authentic engagement (don't automate replies)
- Keep backups of automation configs
- Test before going live

### Don'ts

- Don't automate engagement (comments, likes)
- Don't post exact same content everywhere
- Don't trust AI output without review
- Don't over-automate (lose authenticity)
- Don't neglect platform-native features

### Red Flags

**Your automation is too aggressive if:**
- Accounts getting flagged/restricted
- Engagement dropping despite posting
- Getting spam complaints
- Content feels generic
- No genuine community building

## Platform-Specific Limitations

### TikTok

- No official scheduling API (use workarounds)
- Scheduling tools post via notifications
- Auto-posting can feel less native
- Best: Plan content, post manually

### Instagram

- Auto-posting Reels limited
- Some features need Business account
- Carousel scheduling varies by tool
- Stories scheduling limited

### YouTube

- Scheduling built into Studio
- Shorts can be scheduled
- Third-party tools for analytics mainly
- Premieres for engagement

### X (Twitter)

- Good scheduling support
- Threads can be scheduled
- Some tools better than others
- Native scheduling available

### LinkedIn

- Good third-party support
- Document posts may need manual
- Newsletter integration limited
- Native scheduling available

## Workflow Templates

### Weekly Content Workflow

```markdown
## Monday: Planning

1. Review last week's analytics (30 min)
2. Generate content ideas with AI (30 min)
3. Plan weekly content calendar (30 min)

## Tuesday-Thursday: Creation

1. Batch record video content (2-3 hours)
2. Edit and prepare (2-3 hours)
3. Write captions/copy (1 hour)

## Friday: Scheduling

1. Schedule all content for next week
2. Set up cross-posting
3. Prepare engagement plan

## Daily: Engagement

1. Respond to comments (30 min)
2. Engage with others' content (30 min)
3. Post stories/ephemeral content
```

### Content Repurposing Workflow

```markdown
## Input: One Long-Form Video

1. Upload to YouTube (primary)
2. Auto-clip with Opus Clip → 5-10 shorts
3. Export audio → Podcast
4. Transcribe → Blog post
5. Pull quotes → X threads
6. Key points → LinkedIn carousel
7. Behind-scenes → Stories

## Output: 15-20 Pieces of Content
```
