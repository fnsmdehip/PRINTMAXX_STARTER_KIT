**Updated:** 2026-01-20
**Purpose:** Proven content structures and hooks that drive engagement and growth

## Universal Content Principles

### The Viral Equation

```
Viral potential = (Hook strength × Value delivery × Emotional resonance) / Friction

Where:
- Hook strength = First 1-3 seconds
- Value delivery = Useful, entertaining, or novel
- Emotional resonance = Makes them feel something
- Friction = Confusion, length, poor quality
```

### The 3 Value Types

**1. Educational** - Teaches something useful
- How-to content
- Explanations
- Tips and tricks
- Behind-the-scenes

**2. Entertainment** - Makes them feel something
- Humor
- Surprise
- Drama
- Relatability

**3. Inspirational** - Motivates action
- Success stories
- Transformations
- Possibilities
- Challenges overcome

**Best content combines 2-3 types.**

## Hook Formulas (First 1-3 Seconds)

### Category 1: Curiosity Hooks

**Formula: Create information gap**

| Template | Example |
|----------|---------|
| "This {thing} changed everything" | "This one automation changed everything" |
| "Nobody talks about this" | "Nobody talks about the real cost of SaaS" |
| "I discovered something" | "I discovered why most emails fail" |
| "The secret to {result}" | "The secret to 10x content output" |
| "What happens when {action}" | "What happens when you stop using Notion" |

### Category 2: Contrarian Hooks

**Formula: Challenge common belief**

| Template | Example |
|----------|---------|
| "{Common advice} is wrong" | "Working harder is wrong" |
| "Stop doing {thing}" | "Stop posting every day" |
| "Why {popular thing} doesn't work" | "Why funnels don't work anymore" |
| "{Thing} is dead" | "Email marketing is dead" |
| "Unpopular opinion: {statement}" | "Unpopular opinion: followers don't matter" |

### Category 3: Promise Hooks

**Formula: Specific outcome + timeframe**

| Template | Example |
|----------|---------|
| "How I {result} in {time}" | "How I hit 100K in 90 days" |
| "I made ${X} doing this" | "I made $10K from one thread" |
| "{Number} ways to {result}" | "3 ways to double your reach" |
| "The exact {thing} I used to {result}" | "The exact system I used to quit my job" |
| "Do this for {time} and {result}" | "Do this for 30 days and watch what happens" |

### Category 4: Pattern Interrupt Hooks

**Formula: Start unexpectedly**

| Template | Example |
|----------|---------|
| "Wait, don't scroll" | "Wait, don't scroll - you need to see this" |
| "I was today years old when..." | "I was today years old when I learned this" |
| "This will sound crazy but..." | "This will sound crazy but I deleted half my content" |
| Start mid-action | *Opens with typing/building/doing* |
| Visual surprise | Unusual image/transition/movement |

### Category 5: Relatable Hooks

**Formula: Share common frustration/experience**

| Template | Example |
|----------|---------|
| "POV: {relatable situation}" | "POV: You just hit publish for the 100th time" |
| "When {common experience}" | "When the algorithm ghosts your best content" |
| "Me trying to {struggle}" | "Me trying to stay consistent" |
| "That feeling when {situation}" | "That feeling when your post finally takes off" |
| "Can we talk about {issue}?" | "Can we talk about creator burnout?" |

## Content Structure Formulas

### Formula 1: Problem-Agitation-Solution (PAS)

**Structure:**
```
Hook: Identify the problem
Problem: State it clearly
Agitation: Make it painful (show consequences)
Solution: Your answer
CTA: What to do next
```

**Example script:**
```
"Most people waste 3 hours a day on email.
[PROBLEM]

That's 15 hours a week. 60 hours a month.
Think about what you could do with that time.
[AGITATION]

Here's the system I use to hit inbox zero in 15 minutes:
[SOLUTION - 3 steps]

Save this for later.
[CTA]"
```

### Formula 2: Before-After-Bridge (BAB)

**Structure:**
```
Before: Where they are now (pain state)
After: Where they could be (desire state)
Bridge: How to get there (your solution)
```

**Example script:**
```
"6 months ago, I was posting daily with zero growth.
[BEFORE]

Today, I've added 50K followers and make $10K/month from content.
[AFTER]

Here's the exact change I made:
[BRIDGE - the tactical shift]"
```

### Formula 3: Story Structure

**Structure:**
```
Hook: Attention-grabbing start
Setup: Context (quick)
Conflict: The challenge/problem
Resolution: What you did
Lesson: What they can learn
CTA: Application for them
```

**Example script:**
```
"I almost quit content creation last month.
[HOOK]

After 6 months of daily posting with no results...
[SETUP]

I was about to delete my accounts when I discovered something.
[CONFLICT]

I changed one thing about my content strategy.
[RESOLUTION - explain what]

Here's what I learned that could help you:
[LESSON - 2-3 key points]

Try this for the next 30 days.
[CTA]"
```

### Formula 4: List Structure (Listicle)

**Structure:**
```
Hook: Promise of value
Setup: Context for why this matters
Items: 3-7 specific points (numbered)
Summary: Quick recap
CTA: Save/share/follow
```

**Example script:**
```
"5 tools that replaced my entire marketing team:
[HOOK]

Each one saves me at least 5 hours per week.
[SETUP]

1. [Tool + what it does + result]
2. [Tool + what it does + result]
3. [Tool + what it does + result]
4. [Tool + what it does + result]
5. [Tool + what it does + result]
[ITEMS]

Total savings: 25+ hours per week.
[SUMMARY]

Save this for later.
[CTA]"
```

### Formula 5: Tutorial/How-To

**Structure:**
```
Hook: Outcome promise
Context: What this helps with
Steps: Clear numbered actions (3-7)
Result: What they'll achieve
CTA: Try it/follow for more
```

**Example script:**
```
"How to write a week of content in 2 hours:
[HOOK]

This is the exact system I use every Monday.
[CONTEXT]

Step 1: [Action + brief explanation]
Step 2: [Action + brief explanation]
Step 3: [Action + brief explanation]
Step 4: [Action + brief explanation]
[STEPS]

Done. That's your entire week scheduled.
[RESULT]

Follow for more content systems.
[CTA]"
```

### Formula 6: Compare/Contrast

**Structure:**
```
Hook: Two options teased
Option A: What most do (cons)
Option B: What works (pros)
Winner: Clear recommendation
CTA: Which do you do?
```

**Example script:**
```
"Posting daily vs. posting strategically:
[HOOK]

Daily posting:
- Burns you out
- Quality suffers
- Algorithm notices inconsistency
[OPTION A]

Strategic posting (3-4x/week):
- Better content
- Sustainable pace
- Higher engagement
[OPTION B]

Quality beats quantity every time.
[WINNER]

Which approach do you use? Comment below.
[CTA]"
```

## Platform-Specific Formulas

### TikTok/Reels (Short-Form Video)

**30-second formula:**
```
0-1s: Visual hook (movement, text, face)
1-3s: Verbal hook (statement, question)
3-20s: Core content (value delivery)
20-28s: Summary/lesson
28-30s: CTA + loop potential
```

**60-second formula:**
```
0-3s: Strong double hook (visual + verbal)
3-15s: Setup/context
15-45s: Main content (3-5 points)
45-55s: Key takeaway
55-60s: CTA + potential loop
```

### YouTube Long-Form

**8-15 minute formula:**
```
0-30s: Hook + promise
30-60s: Intro (who you are, why listen)
1-2min: Context/overview
2-10min: Main content (chapters)
10-12min: Summary/recap
12-14min: CTA + next steps
14-15min: End screen engagement
```

### X/Twitter Thread

**7-tweet thread formula:**
```
Tweet 1: Hook (stops scroll) + promise
Tweet 2: Context/setup
Tweet 3-5: Main points (one per tweet)
Tweet 6: Summary/key insight
Tweet 7: CTA + retweet request
```

### LinkedIn Post

**Post formula:**
```
Line 1: Hook (attention-grabber)
Line 2: [blank - forces "see more"]
Lines 3-5: Story/context setup
Lines 6-10: Core insight (short paragraphs)
Lines 11-12: Takeaway/lesson
Line 13: Question for engagement
```

## Content Testing Framework

### A/B Test Elements

| Element | Impact | Test Method |
|---------|--------|-------------|
| Hook | Highest | Same content, different hooks |
| Thumbnail | High | Different images, same video |
| Length | Medium | Long vs short versions |
| CTA | Medium | Different calls to action |
| Format | Medium | Video vs carousel vs static |

### Test Protocol

1. Create hypothesis ("Hook A will outperform Hook B because...")
2. Create two versions (change ONE element)
3. Post at similar times/days
4. Wait for minimum views (1000+)
5. Compare metrics (engagement rate, not just views)
6. Document winner
7. Apply to future content

### Content Scorecard

| Metric | Weight | Scoring |
|--------|--------|---------|
| Hook strength | 30% | Did it stop scroll? View retention 0-3s |
| Value delivery | 30% | Did people finish? Completion rate |
| Engagement | 25% | Comments, shares, saves |
| Conversion | 15% | Did they take action? |

## Repurposing Playbook

### One Piece → Multiple Formats

```
Long-form YouTube video
    ↓
Cut into 5-10 short clips (TikTok/Reels/Shorts)
    ↓
Extract audio for podcast clip
    ↓
Pull quotes for X posts
    ↓
Create carousel from key points (Instagram/LinkedIn)
    ↓
Write thread from script (X)
    ↓
Create blog post from transcript
    ↓
Email newsletter from highlights
```

### Repurposing Rules

1. Native-first: Adapt format to platform
2. No watermarks: Remove or recreate
3. Best parts: Don't just cut, curate
4. Different hooks: Same content, platform-specific hooks
5. Space it out: Don't post everywhere same day
