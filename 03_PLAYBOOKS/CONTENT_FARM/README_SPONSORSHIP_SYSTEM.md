## Pricing Formula

### Calculate Your Base Rate

```
Base Rate = Followers × Engagement Rate × Niche Multiplier

Example:
- 50,000 followers
- 4% engagement rate
- 1.8 niche multiplier (tech niche)
- 50,000 × 0.04 × 1.8 = $3,600 base rate
```

### Set Your Tiers

From base rate:
- **TIER 1** (single post): $1,200-2,000 (30-55% of base)
- **TIER 2** (3-4 videos): $3,000-4,500 (80-125% of base, show savings)
- **TIER 3** (monthly retainer): $2,500-3,600/month (70-100% of base)
