## Automation Pipeline

```
1. SCRAPE: Monitor news sources every 5-15 min
2. FILTER: AI identifies newsworthy items
3. VERIFY: Cross-reference multiple sources
4. PACKAGE: Generate post in viral format
5. POST: Immediately to all platforms
6. ITERATE: Track what performs, adjust criteria
```

### Viral Formats

**X/Twitter:**
```
BREAKING: [Headline]

[Key detail 1]
[Key detail 2]
[Key detail 3]

[Source link or "developing..."]
```

**TikTok:**
- Face-to-camera or text overlay
- 30-60 seconds
- Hook in first 3 seconds
- "What just happened with [topic]"
