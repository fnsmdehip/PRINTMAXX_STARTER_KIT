## Overview

Run multiple niche social accounts across TikTok, Instagram Reels, YouTube Shorts, and Facebook Reels. Post 3x/day per account using templates, AI-generated content, and repurposed clips. Revenue from creator funds, ad revenue share, affiliate links in bio, and cross-selling to apps/products/newsletters.

Key insight: Facebook Reels pays $4.40/1K views (4-440x more than TikTok or YouTube Shorts). Cross-post everything to FB Reels first.

Proof: @Jonnyvandel runs 200+ accounts posting 600 posts/day. Forest app (CF vertical) gets 44M downloads. Motivation quote accounts routinely hit 100K+ followers in 3-6 months.

## Step-by-step implementation

### Day 1: Account setup (3-4 hours)

**Task 1: Choose 3 niches**

Start with proven niches that have high engagement and monetization paths:

| Niche | Account Name Pattern | Content Type | Monetization Path |
|-------|---------------------|--------------|-------------------|
| Faith | @prayerdaily, @faithfulfinds | Prayer prompts, Bible verses, testimonials | PrayerLock app, faith courses |
| Fitness | @gainzguide, @fitcheck365 | Workout clips, transformations, tips | WalkToUnlock app, supplements affiliate |
| Tech/Productivity | @buildfast, @solodevtips | Coding tips, tool reviews, indie hacking | Apps, SaaS, info products |

**Task 2: Create accounts with anti-detect browser**

For each niche, create accounts on 4 platforms:

1. Open GoLogin, create new profile per account
2. Assign SOAX mobile proxy (different IP per profile)
3. Create TikTok account (phone number verification)
4. Create Instagram account (email verification)
5. Create YouTube channel (Google account)
6. Create Facebook page (personal FB needed)

Total: 12 accounts across 4 platforms for 3 niches.

Safe limits per platform (reference `PLATFORM_AUTOMATION_LIMITS_2026.md`):
```
TikTok:    3 posts/day, 50 follows/day, 100 likes/day
Instagram: 3 posts/day, 30 follows/day, 50 likes/day
YouTube:   1-2 Shorts/day
Facebook:  3 Reels/day (NO follow limits for pages)
```

**Task 3: Set up content templates**

For each niche, create 5 content templates in Canva:

1. Quote card (text overlay on gradient/image background)
2. Listicle slide (numbered tips, swipe format)
3. Before/after comparison
4. Tutorial/how-to (step numbered overlay)
5. Hook + reveal (curiosity gap format)

Save templates for reuse. Each template produces 10-20 unique posts by swapping text/images.

### Day 2-3: Content batch creation (6-8 hours)

**Task 4: Batch create 30 posts per niche (90 total)**

Production workflow per niche:
```
Step 1: Write 30 hooks (use WINNING_CONTENT_STRUCTURES.csv)
Step 2: Generate images/backgrounds in Leonardo.ai or Canva
Step 3: Add text overlays using templates
Step 4: Generate voiceover in ElevenLabs (for video posts)
Step 5: Edit in CapCut (add trending audio, captions, transitions)
Step 6: Export in platform-specific formats:
        - TikTok: 1080x1920, 15-60 sec
        - IG Reels: 1080x1920, 15-90 sec
        - YT Shorts: 1080x1920, under 60 sec
        - FB Reels: 1080x1920, 15-90 sec
```

This gives you 10 days of content at 3 posts/day per niche.

Content sources (not plagiarizing, adapting):
- Reddit top posts in niche subreddits (rewrite as video)
- Twitter viral threads (convert to slideshow)
- YouTube long-form (clip key moments into Shorts)
- Books/courses (summarize key insights)

**Task 5: Schedule all content**

Use Buffer (free: 3 channels, 10 posts/queue) or Publer ($12/mo: unlimited):

Optimal posting times:
```
Post 1: 7-9 AM (morning commute)
Post 2: 12-1 PM (lunch break)
Post 3: 6-9 PM (evening scroll)
```

Schedule 10 days out for each account.

### Day 3-7: Launch and engage (2-3 hours/day)

**Task 6: Post and engage daily**

Daily routine (per account, 15-20 min each):
```
1. Check scheduled posts went live
2. Reply to ALL comments (algorithm boost)
3. Engage with 10-15 accounts in your niche (comment, not just like)
4. Follow 10-20 relevant accounts
5. Check analytics: which posts performed best?
```

Engagement is critical the first 30 days. Algorithm rewards accounts that engage with others.

**Task 7: Cross-post everything to Facebook Reels**

FB Reels pays $4.40/1K views vs $0.01-1.00/1K on TikTok. This is the single biggest arbitrage opportunity.

Cross-posting workflow:
1. Download TikTok video without watermark (snaptik.app)
2. Upload to Facebook page as Reel
3. Add FB-specific hashtags
4. Done. Same content, 4-440x more revenue.

### Week 2-4: Monetize and optimize

**Task 8: Add affiliate links to all bios**

For each niche, set up affiliate links:
```
Faith niche:
  - Bible app affiliate (YouVersion)
  - Prayer journal (Amazon Associates)
  - PrayerLock app (own product, direct link)

Fitness niche:
  - Supplement affiliate (Transparent Labs, Athletic Greens)
  - Workout app affiliate (own apps or partners)
  - Equipment affiliate (Amazon Associates)

Tech niche:
  - SaaS affiliate (Notion, Vercel, tools you use)
  - Course affiliate (Udemy, Skillshare)
  - Own products (Gumroad digital products)
```

Use Linktree or Beacons for multi-link bio pages.

**Task 9: Apply for creator funds/monetization**

Requirements vary by platform:
```
TikTok Creator Fund:     10K followers, 100K views in 30 days
TikTok Creativity Program: 10K followers, 100K views, 1+ min videos
Instagram Reels bonuses:  Invite-only (build engagement first)
YouTube Shorts fund:      1K subscribers, 10M Shorts views in 90 days
Facebook Reels monetization: 10K followers, 600K minutes watched in 60 days
```

Facebook Reels has the lowest effective bar and highest payout. Prioritize it.

**Task 10: A/B test content formats**

Run weekly tests:
- Test A: Quote cards vs talking head clips
- Test B: Trending audio vs original voiceover
- Test C: 15-second hooks vs 60-second deep dives
- Test D: Engagement bait CTA ("comment YES") vs no CTA

Track: views, engagement rate, follower growth, link clicks per format.

## Scaling path

### $0-500/mo (Month 1-2)
3 niches, 12 accounts, 3 posts/day. 0-5K followers per account. Focus: consistency, finding winning formats.

### $500-5K/mo (Month 2-4)
Scale to 5 niches. Hire clipper/editor VA ($100-300/mo). 5-50K followers. Accounts hit monetization thresholds. Focus: cross-posting to FB Reels, affiliate optimization.

### $5K-10K+/mo (Month 4-8)
10+ niches, 40+ accounts. Full VA team for posting ($500-1K/mo). Content repurposing pipeline automated. Cross-sell to own products (apps, courses). Focus: systemize, delegate, maximize RPM.

## Risk management

- **Account ban:** Use anti-detect browser + mobile proxies. Never access multiple accounts from same IP/device. Keep backup accounts warming.
- **Algorithm change:** Diversify across 4 platforms. Never depend on one. FB Reels arbitrage may close, so stack revenue sources.
- **Content burnout:** Batch create 2 weeks ahead. Use templates to reduce creative load. Hire editors early.
- **Copyright claims:** Only use royalty-free music, original footage, or properly licensed clips. CapCut has royalty-free audio library.
- **Engagement drop:** Test new formats weekly. Pivot quickly based on data. Watch competitors for trending formats.
