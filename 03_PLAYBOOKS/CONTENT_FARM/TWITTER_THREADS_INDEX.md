## Thread Types Reference

### How-To Thread (Best for: Algorithm boost + concrete value)
- **Length:** 8-9 tweets
- **Structure:** Problem → Method (3-4 steps) → Results → CTA
- **Engagement:** 100-200 replies average
- **Hook Type:** Consequence-first (what happened, not explanation)
- **CTA Placement:** End of thread with self-reply funnel
- **Expected Reach:** 3-5K (highest performing)

### Story Thread (Best for: Emotional resonance + shares)
- **Length:** 6-7 tweets
- **Structure:** Hook (personal struggle) → Context → Shift/Realization → Results → CTA
- **Engagement:** 80-160 replies average
- **Hook Type:** Relatable failure or transformation moment
- **CTA Placement:** End of thread, soft ask
- **Expected Reach:** 2-4K (highest share rate 20-30%)

### Listicle Thread (Best for: Viral potential + bookmarks)
- **Length:** 8 tweets
- **Structure:** Hook → Framework → Tier 1-4 ranking → What Doesn't Work → CTA
- **Engagement:** 100-180 replies average
- **Hook Type:** Ranked list promise (definitive ranking)
- **CTA Placement:** End of thread with resource promise
- **Expected Reach:** 2-4K (high bookmark rate)
