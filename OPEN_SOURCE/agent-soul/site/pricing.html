faq

common questions

is it really free?
yes. all 19 modules, MIT license, no usage limits, no feature gates. self-host on your machine or server. the paid tiers are for managed hosting and team features. the code is identical.
what's the difference between self-hosted and pro?
self-hosted: you run it, you maintain it, you configure connectors yourself. pro: we host it, we maintain it, connectors are pre-configured, you get a visual dashboard. same code either way.
do I need an LLM API key?
sovrun itself doesn't call LLMs. it's the cognitive and orchestration layer that wraps around whatever LLM you use. your agents call the LLM. sovrun makes those agents smarter (voice model, correction chains, drift scoring) and more reliable (DAG orchestration, crash recovery, handoffs).
can I use it with OpenAI / Gemini / local models?
yes. sovrun is model-agnostic. the voice model, correction chains, and drift scoring work with any LLM output. the memory import module reads ChatGPT, Claude, and Gemini exports.
what's the consulting for?
businesses that want sovrun deployed but don't have the technical team to do it. we come in, audit your systems, build custom agents for your specific workflows, configure everything, and optionally manage it ongoing. same system, professional installation.
how is this different from n8n or Zapier?
n8n and Zapier are workflow automation (when X happens, do Y). sovrun is an agent OS (agents that learn your patterns, orchestrate themselves, recover from crashes, and score their own output quality). sovrun can create n8n workflows programmatically. they solve different problems and work together.
