agents that learn your voice, orchestrate as DAGs, recover from crashes, create n8n workflows autonomously, and score their own output quality. 20 modules across 5 layers. import your ChatGPT/Claude/Gemini history. 126 MCP connectors with auto-setup + 1,200 more via n8n bridge. extracted from a production system running 33 agents 24/7.

what sovrun does

5 layers. each works standalone.

wire in one module or all 19. no framework lock-in, no vendor SDK, no dependency chain.

cognitive layer (7 modules)

voice modeling trained on YOUR prompts. correction chain learning that mines your feedback into rules. soul drift scoring. bias-null filtering. self-audit that catches slop in your own config.

agents learn your patterns, not LLM defaults

orchestration layer (4 modules)

DAG execution with Sequential, Parallel, and Loop agents. typed agent-to-agent handoffs with guardrails. closed-loop decision engine. loop closer that verifies decisions led to real outcomes.

compose agents into DAGs, not linear pipelines

memory layer (4 modules)

procedural memory with FTS5 search across past solutions. import history from ChatGPT, Claude, and Gemini. session briefings from git diffs. conversation logger with keyword search.

import from any LLM, remember what worked

infrastructure layer (3 modules)

crash recovery with deterministic replay (crash at step 14, replay from 14, not 1). circuit breaker state machine. file locking. loop detection for runaway agents. HTML tracing timeline.

zero wasted API calls on recovery

connectors layer (105 MCP)

auto-setup for Slack, GitHub, Notion, Stripe, Gmail, Postgres, and 99 others. agents build n8n workflows programmatically. one command to configure any connector.

sovrun-mcp --setup slack

zero or optional deps

2 dependencies (tenacity for retry, httpx for HTTP). runs on any machine with Python 3.10+. no vendor lock-in. n8n bridge and MCP connectors are optional.

pip install sovrun installs nothing extra

the problem

your AI agents run on default LLM knowledge. you correct them. they don't learn from the corrections. session 47, same correction you gave in session 3.

nobody checks if agent output matches your voice. nobody scores drift from directives. nobody closes the loop between "agent decided X" and "X actually happened." crash at step 14 of 16? replay from step 1. every API call wasted. need Slack + GitHub + Notion connected? configure each one manually for every project.

sovrun solves each of these with a dedicated module. 20 modules, 5 layers, 2 deps.

what's in the box

19 modules. 5 layers.

each module works standalone. from sovrun.core.voice_extractor import VoiceExtractor

cognitive (7 modules)
voice_extractor
reads your prompt history, extracts vocabulary, tone, banned words. outputs a voice model you inject into any agent prompt.
your prompts -> voice model -> agent injection
cognitive_engine
identifies correction chains. builds meta-rules that would have prevented each correction sequence.
168 chains, 6.9 avg corrections per chain
pattern_miner
finds correction markers, escalation triggers, satisfaction signals. mines transferable rules from what works and what doesn't.
correction + satisfaction + escalation signals
user_sim_refiner
simulates YOUR critique on any project. applies your thinking when you're not in the room.
autonomous critique using your patterns
self_audit
checks for AI slop in config files, context bloat, orphan scripts, missing protocols. the system that audits the system.
slop detection + context bloat + orphan scan
bias_null
5-point pre-output filter. catches legacy smuggling, appeasement, popular-default traps, training-bias contamination.
run before every major output
decision_engine
closed-loop autonomous decisions. scan, score, act, log. CSV audit trail for every decision with full reasoning chain.
rule-based + LLM-delegated judgment
orchestration (4 modules)
orchestration
SequentialAgent, ParallelAgent, LoopAgent primitives. compose agents into DAGs. non-dependent phases run in parallel.
DAG execution with typed state
handoff
typed agent-to-agent delegation with context transfer. agent A invokes B mid-task, passes state, receives result.
scoped guardrails per handoff
loop_closer
closes 4 loops: decision execution, feedback tracking, pipeline advancement, soul drift scoring (0-10 per output).
alerts when drift drops below 6/10
workflow_bridge
agents create n8n workflows programmatically. define triggers, nodes, connections in Python. export as n8n-compatible JSON.
sovrun-workflow --create "slack to sheets"
memory (4 modules)
procedural_memory
auto-generates skill docs from successful agent runs. FTS5-indexed. before each run, queries for relevant past solutions.
agents remember what worked
memory_import
import conversation history from ChatGPT, Claude, and Gemini exports. normalizes into sovrun's format for voice extraction.
sovrun-import --provider chatgpt --file export.zip
conversation_logger
extracts messages from session transcripts into searchable JSONL. incremental processing, keyword search, statistics.
search your entire prompt history
session_briefing
"what happened since last session" from git diffs, state files, task trackers. no LLM calls. under 30 seconds.
context continuity across sessions
infrastructure (3 modules)
resilience
retry with exponential backoff + jitter, file locking (fcntl), circuit breaker state machine, input sanitization, loop detection.
CLOSED -> OPEN -> HALF_OPEN -> CLOSED
durable
deterministic replay on crash. each step logged atomically. crash at step 14 replays from 14, not from 1. zero wasted API calls.
step-level checkpoint + replay
tracing
HTML timeline of LLM calls, tool executions, handoffs, failures. per-agent token attribution. cost tracking.
sovrun-trace --visualize
connectors (105 MCP + bridge)
mcp_bridge
auto-configures MCP servers for 126 connectors. one command per service. generates claude_desktop_config.json entries.
sovrun-mcp --setup slack github notion
deps
dependency manager. tenacity for retry, httpx for HTTP. auto-detects installed packages and routes to best available implementation.
sovrun-deps --install n8n-client

get started

running in 2 minutes.

terminal
pip install sovrun
# or: git clone https://github.com/fnsmdehip/sovrun.git && cd sovrun && pip install -e .

# import your history from any LLM provider
sovrun-import --provider chatgpt --file ~/Downloads/conversations.json
sovrun-import --provider claude --dir ~/.claude/projects

# build voice model from your prompt history
sovrun-voice --extract

# build correction chains + meta-rules
sovrun-engine --build-model

# set up your connectors
sovrun-mcp --setup slack github notion

# run orchestration DAG
sovrun-orchestrate --visualize

# close all loops (decisions, feedback, drift scoring)
sovrun-loops --cycle
