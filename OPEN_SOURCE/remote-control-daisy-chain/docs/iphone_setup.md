# iPhone Setup

## Agent-native path

1. Install the Tailscale iPhone app.
2. Log into the same tailnet as the Mac.
3. Open the MagicDNS URL for the published control app.
4. Use the browser control surface from there.

## Full desktop path

1. Install the RustDesk iPhone app.
2. Pair to the Mac using the host ID.
3. Use a permanent password only after you finish unattended setup on the Mac.

## Keep in mind

- The Tailscale app is what makes the private tailnet URL work abroad.
- RustDesk is optional. It is for GUI-level control, not the primary automation path.
