## FAQs

**Q: Is web scraping legal?**
```
Scraping publicly available data is legal in the US (hiQ v. LinkedIn, 2022). I only scrape public data. I don't bypass paywalls, break authentication, or access private information. If your request involves non-public data, I'll let you know upfront.
```

**Q: Can you scrape [specific website]?**
```
Message me the URL before ordering. I'll confirm within 2 hours whether I can scrape it, what fields are available, and how many records exist. Some sites are easier than others, so I want to set expectations before you pay.
```

**Q: What if the website changes after you deliver?**
```
Websites change layouts occasionally, which can break scrapers. Premium includes a reusable Python script with documentation so you can re-run it. If the site changes within 30 days of delivery, I'll update the scraper once at no charge.
```

**Q: Can you scrape emails?**
```
I can extract emails that are publicly displayed on websites (business directories, company about pages, contact pages). I don't scrape private emails, personal data behind logins, or data from platforms that explicitly prohibit it in ways that would create legal issues.
```

**Q: How fast can you deliver?**
```
Basic (500 records): 24-48 hours. Standard (5,000 records): 3-4 days. Premium (50,000 records): 5-7 days. Rush delivery available for +50% fee if you need it faster.
```
