## FAQs

**Q: What platform do you build on?**
```
I use modern web frameworks (Next.js, Astro) deployed on free-tier hosting. This means your site loads in under 2 seconds, costs $0/month to host (Basic/Standard), and doesn't need constant plugin updates like WordPress. If you specifically need WordPress, I can do that too. Just mention it in your order.
```

**Q: Do I need to buy a domain?**
```
You'll need your own domain ($10-15/year from Namecheap or Google Domains). Standard and Premium packages include domain connection and DNS setup at no extra charge. If you don't have a domain yet, I'll recommend the best option for your business name.
```

**Q: Can you add a booking system?**
```
Yes. Premium package includes integration with Cal.com (free) or Calendly. Visitors can book appointments directly on your site without calling. If you already use a scheduling tool, I'll integrate that instead.
```

**Q: Will it rank on Google?**
```
Standard and Premium include SEO fundamentals: proper title tags, meta descriptions, header structure, image optimization, sitemap, and Google Business Profile connection. This gets you indexed and ranking for "[your service] + [your city]" searches. Full SEO campaigns are a separate service if you want ongoing ranking work.
```

**Q: What if I need changes after delivery?**
```
Each package includes revisions (1-5 depending on tier). After final delivery, Premium clients get 30 days of support for minor text/image changes. For ongoing updates, I offer a $50/month maintenance retainer.
```
