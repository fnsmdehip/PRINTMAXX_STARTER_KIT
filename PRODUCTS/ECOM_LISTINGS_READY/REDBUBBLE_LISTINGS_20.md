## HOW TO USE THIS FILE

1. Go to redbubble.com/portfolio/images/new
2. Upload design PNG (4500x5400px, transparent background, 300 DPI)
3. Copy **Title** into title field
4. Copy **Description** into description field
5. Enter each **Tag** individually (15 max per design)
6. Select **Category** and **Media** as listed
7. Enable products listed under **Products to Enable**
8. Set markup to **35%** (Settings > Default Markup)
9. Click "Save Work"

**Design creation:** Use Canva (free) or Midjourney/Leonardo for graphics. Text-only designs can be made in Canva in 5-10 minutes.
