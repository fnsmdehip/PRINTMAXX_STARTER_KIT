## Steps

1. Go to whop.com and create account
2. Click "Create a Whop"
3. For each product: New Product > Digital Product
4. Copy title, description, price from each file
5. Upload the PDF (convert from md using `../GUMROAD_INSTANT_UPLOAD/convert_to_pdf.sh`)
6. Set thumbnail (specs in each listing file)
7. Publish
