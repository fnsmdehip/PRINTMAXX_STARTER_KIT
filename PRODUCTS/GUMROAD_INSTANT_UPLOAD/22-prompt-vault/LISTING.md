## DESCRIPTION (paste into Gumroad body)

You're using AI wrong.

Not because the technology is bad. Because your prompts are bad. You're typing "write me a blog post about marketing" and getting exactly what you deserve — generic, forgettable output that sounds like every other AI-generated piece of content on the internet.

The difference between a $0 prompt and a $1,000 prompt isn't vocabulary. It's structure. It's constraints. It's knowing exactly what to ask for and how to ask for it.

This vault contains 200 prompts tested across Claude, GPT-4o, and every major model. Organized into 5 categories covering every function of a one-person business. Each prompt produces output you can use immediately — not output you need to spend 45 minutes editing.

**5 categories. 40 prompts each.**

Content (40 prompts). Social post generation, thread writing, newsletter sections, blog post outlines, video scripts, repurposing sequences, hooks, CTAs, and engagement bait. Each prompt tested for output quality, not just novelty.

Outreach (40 prompts). Cold email first lines, follow-up sequences, LinkedIn messages, partnership pitches, podcast pitches, freelance proposals, and objection responses. Personalization templates for batch processing.

Code (40 prompts). App scaffolding, debugging assistance, code review, documentation generation, refactoring, test writing, and API integration. Prompts that produce working code, not pseudocode.

Research (40 prompts). Competitor analysis, market sizing, customer interview synthesis, trend identification, pricing research, and opportunity scoring. Structured output formats you can act on.

Strategy (40 prompts). Business model evaluation, launch sequencing, pricing strategy, positioning, GTM planning, and revenue projection. Decision frameworks that replace $500/hour consultants.

**What makes these different:**

Every prompt includes: the exact text, the variables to customize, an example output, and a note on which model handles it best. No vague "here's a template you can adapt." Copy, paste, customize the bracketed variables, get usable output.

Works with Claude, ChatGPT, GPT-4o, Gemini. Model-agnostic.

$27. Less than one hour of your time. More leverage than a month of prompt experimentation.
