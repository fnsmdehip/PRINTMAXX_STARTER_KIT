## HOW TO USE THIS FILE

For each book:
1. Create the interior pages in Canva (or BookBolt $10/mo for templates)
2. Create the cover in Canva (use KDP cover template generator)
3. Upload to KDP at kdp.amazon.com
4. Set pricing per the spec below
5. KDP review takes 24-72 hours, then live on Amazon

**Interior creation tools:**
- Canva (free) - use "Custom Size" to create pages at the trim size
- BookBolt ($10/mo) - has KDP-specific templates, faster
- Tangent Templates (free) - basic journal interiors

**Cover creation:**
- KDP Cover Calculator: kdp.amazon.com/en_US/cover-calculator
- Enter trim size + page count to get exact cover dimensions
- Create cover in Canva at those dimensions
