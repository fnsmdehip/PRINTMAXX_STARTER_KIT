## DAY 1: REDBUBBLE + TEESPRING (Fastest to Revenue)

### Step 1: Redbubble Account Setup (30 min)

- [ ] Go to redbubble.com and click "Sell your art"
- [ ] Sign up with email (use your business email)
- [ ] Complete artist profile:
  - [ ] Username: match your brand (e.g., PRINTMAXX or niche-specific)
  - [ ] Bio: "Faith, fitness, and tech designs for people who build things."
  - [ ] Profile picture: brand logo or clean headshot
  - [ ] Header image: banner with 3-4 sample designs
- [ ] Go to Account Settings > Artist Tools:
  - [ ] Set default markup to **35%** (not the default 20%)
  - [ ] Enable all product types by default
- [ ] Add payment method:
  - [ ] PayPal (fastest payouts, $20 minimum)
  - [ ] Or bank deposit (monthly, $20 minimum)
- [ ] Account is live immediately. No approval needed.

### Step 2: Upload First 20 Designs to Redbubble (2 hours)

- [ ] Open PRODUCTS/REDBUBBLE_LISTINGS.md in a separate window
- [ ] For each of the 20 listings:
  - [ ] Go to redbubble.com/portfolio/images/new
  - [ ] Upload design PNG (4500x5400px, transparent background)
  - [ ] Copy-paste title from REDBUBBLE_LISTINGS.md
  - [ ] Copy-paste description
  - [ ] Enter all 15 tags (one per tag field)
  - [ ] Select category as listed
  - [ ] Select media type: "Design & Illustration"
  - [ ] Click "Enable All" for products
  - [ ] Preview design on t-shirt, mug, and poster (check alignment)
  - [ ] Adjust placement if design is cut off or too small
  - [ ] Click "Save Work"
  - [ ] Repeat for all 20 listings

**Upload order priority (from REDBUBBLE_LISTINGS.md):**
1. Listings 3 + 11 (Islamic art - low competition)
2. Listings 1 + 17 (Christian women - huge market)
3. Listings 7 + 8 + 15 (Developer stickers - high conversion)
4. Remaining listings in any order

### Step 3: TeeSpring/Spring Account Setup (20 min)

- [ ] Go to teespring.com (now called spring.com)
- [ ] Sign up with email
- [ ] Set up your storefront:
  - [ ] Store name (match Redbubble or create separate brand)
  - [ ] Store URL: spring.com/your-store-name
  - [ ] Bio and banner
- [ ] Add payment: PayPal or Stripe (payouts after 14-day hold)
- [ ] Connect to YouTube if you have a channel (Spring integration)

### Step 4: Upload Top 10 Designs to TeeSpring (1 hour)

Pick the 10 strongest designs (highest sell-through potential):

- [ ] 1. Trust the Process (F6) - faith

- [ ] 6. Ctrl+Z My Life (T9) - universal humor
- [ ] 7. Revenue > Followers (T12) - solopreneur
- [ ] 8. Side Project Graveyard (T17) - dev humor

For each design on TeeSpring:
- [ ] Click "Create" > "Start designing"
- [ ] Upload design PNG
- [ ] Select products: T-shirt, hoodie, mug (minimum)
- [ ] Set pricing:
  - T-shirt: $24.99 (base ~$13, profit ~$12)

## DAY 2: AMAZON KDP + ETSY SETUP

### Step 5: Amazon KDP Account Setup (30 min)

- [ ] Go to kdp.amazon.com
- [ ] Sign in with Amazon account (or create one)
- [ ] Complete tax interview:
  - [ ] Enter your legal name and address
  - [ ] TIN/SSN (required for US sellers)
  - [ ] Complete the tax form (W-9 for US, W-8BEN for international)
- [ ] Set up payment:
  - [ ] Bank account for direct deposit (recommended)
  - [ ] Set royalty payment threshold ($10 minimum)
- [ ] KDP account is active immediately after tax interview

### Step 6: Create and Upload First 3 KDP Journals (4-5 hours)

**Create interiors in this order (simplest first):**

**Book 5: 30-Day Habit Tracker (2 hours to create interior)**
- [ ] Open Canva > Custom Size > 6" x 9" (1800 x 2700 px at 300 DPI)
- [ ] OR use BookBolt ($10/mo) for pre-made templates
- [ ] Create these page types:
  - [ ] Title page (1 page)
  - [ ] How to use (1 page)
  - [ ] Habit selection page (1 page, duplicate 6 times for 6 cycles)
  - [ ] 30-day grid (2 pages, duplicate 6 times)
  - [ ] Daily reflection (2 pages, duplicate 6 times)
  - [ ] Weekly review (1 page, duplicate 24 times = 4 per cycle x 6)
  - [ ] Cycle review (1 page, duplicate 6 times)
  - [ ] Bonus sections (8 pages)
- [ ] Export all pages as PDF (Print quality)
- [ ] Merge into single PDF (use Smallpdf.com or Preview on Mac)
- [ ] Create cover using KDP Cover Calculator dimensions
- [ ] Upload to KDP:
  - [ ] Title, subtitle, author from KDP_JOURNALS_5.md
  - [ ] 7 keywords from the spec
  - [ ] 2 categories
  - [ ] Upload interior PDF
  - [ ] Upload cover PDF
  - [ ] Preview in KDP Previewer (check margins and text alignment)
  - [ ] Set price: $5.99, 60% royalty
  - [ ] Publish

**Book 1: 90-Day Prayer Journal (3 hours to create interior)**
- [ ] Same process as above using KDP_JOURNALS_5.md specs
- [ ] Key pages: daily spread (90 copies), prayer tracker, answered prayers log
- [ ] Price: $7.99

**Book 4: Pomodoro Focus Planner (3 hours to create interior)**
- [ ] Key pages: daily pomodoro log (90 copies), weekly review (12 copies)
- [ ] Price: $7.99

- [ ] Upload all 3 to KDP
- [ ] KDP review takes 24-72 hours

### Step 7: Etsy Seller Account Setup (30 min)

- [ ] Go to etsy.com/sell
- [ ] Click "Open your Etsy shop"
- [ ] Set shop preferences:
  - [ ] Language: English
  - [ ] Country: United States
  - [ ] Currency: USD
- [ ] Name your shop (tips: include niche words, keep it memorable)
  - Examples: PrintMaxxDigital, FaithFitTech, DigitalToolsShop
- [ ] Create first listing (required to open shop - use Listing 11 from ETSY_LISTINGS_20.md)
- [ ] Set up payments:
  - [ ] Enter bank account for Etsy Payments
  - [ ] Credit card on file for Etsy fees ($0.20/listing + 6.5% + 3% payment)
- [ ] Set up billing:
  - [ ] Etsy charges listing fees ($0.20 per listing, renews every 4 months)
- [ ] Shop is live after first listing published

### Step 8: Upload 20 Etsy Listings (2-3 hours)

- [ ] Open PRODUCTS/ETSY_LISTINGS_20.md
- [ ] For each listing:
  - [ ] Click "Add a listing" in Etsy seller dashboard
  - [ ] Upload 5-7 listing images (mockups, screenshots, "what's included" graphic)
  - [ ] Title: copy from ETSY_LISTINGS_20.md (max 140 chars)
  - [ ] Category: select as listed
  - [ ] Set "Digital download" as product type
  - [ ] Price: as listed
  - [ ] Upload the actual product file (PDF/ZIP)
  - [ ] Description: copy-paste from file
  - [ ] Tags: enter all 13 tags
  - [ ] Publish

**Listing image creation (do this first):**
- [ ] Create 5 mockup images per product in Canva:
  1. Main product mockup (device/book showing the product)
  2. Inside pages / feature screenshots
  3. "What's included" summary graphic
  4. Testimonial or benefit callout
  5. Brand/trust graphic
- [ ] Use Canva's "Mockup" feature or search "digital product mockup"
- [ ] Export each at 2700 x 2025 px (Etsy's recommended ratio)
