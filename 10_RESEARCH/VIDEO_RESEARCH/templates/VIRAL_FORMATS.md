## hook structures (first 3 seconds)

### proven hooks by type

| hook type | template | engagement rate | best niche |
|-----------|----------|----------------|------------|
| shock stat | "97% of people don't know this..." | 8-12% | fitness, finance |
| controversy | "unpopular opinion: [contrarian take]" | 10-15% | any |
| curiosity gap | "I tried [X] for 30 days. here's what happened." | 7-10% | fitness, tech |
| direct address | "if you're [pain point], watch this" | 6-9% | fitness, self-help |
| before/after | [visual transformation, no text needed] | 12-18% | fitness, beauty |
| POV | "POV: you just [relatable situation]" | 8-12% | lifestyle, humor |
| tutorial tease | "the trick that [impressive result]" | 7-11% | tech, fitness |
| story hook | "I was [situation] when [unexpected event]" | 9-13% | personal brand |

### hook formulas
```
1. NUMBER + UNEXPECTED CLAIM
   "3 exercises that burn more fat than running"
   "1 ingredient that makes any meal healthy"

2. THIS vs THAT
   "gym vs calisthenics (honest comparison)"
   "$5 protein vs $50 protein"

3. STOP DOING X
   "stop doing crunches. do this instead."
   "stop scrolling. this will change your morning."

4. SECRET/HIDDEN
   "the feature Apple doesn't want you to know about"
   "hidden settings that double your phone's speed"

5. TIME-BOUND RESULT
   "do this for 7 days and watch what happens"
   "5 minutes every morning = completely different body"
```

## visual format templates

### format 1: SPLIT SCREEN COMPARISON
```
layout: top/bottom or left/right split
top/left: before / competitor / wrong way
bottom/right: after / our way / right way
text: large bold comparison labels
audio: trending sound or voice-over
duration: 15-30s
platforms: TikTok, Reels, Shorts
```

### format 2: TEXT OVERLAY CAROUSEL
```
layout: dark/gradient background, large white text
structure: statement per slide, 2-3 seconds each
text: 3-5 key points, one per "slide" (actually timed text)
audio: trending sound (low) + optional voice-over
duration: 15-30s
platforms: TikTok, Reels
```

### format 3: POV/GREEN SCREEN
```
layout: person reacting / pointing at content
structure: green screen background with relevant imagery
text: POV label + reaction captions
audio: original voice + trending sound
duration: 15-45s
platforms: TikTok, Reels
```

### format 4: FACELESS NARRATION
```
layout: stock/AI footage with text overlay
structure: narrator explains while relevant visuals play
text: key points as subtitles + highlighted keywords
audio: AI voice (ElevenLabs) or trending sound
duration: 30-60s
platforms: Shorts, Reels, TikTok
```

### format 5: PRODUCT SHOWCASE
```
layout: clean background, product center frame
structure: problem > product intro > demo > result > CTA
text: feature callouts, price comparison
audio: upbeat trending sound
duration: 15-30s
platforms: all
```

### format 6: DAY IN MY LIFE / ROUTINE
```
layout: multiple scenes, fast cuts
structure: morning > midday > evening, each 5-10s
text: minimal, time labels
audio: trending chill sound
duration: 30-60s
platforms: TikTok, Reels
```

### format 7: STORYTIME WITH HOOKS
```
layout: face-to-camera or AI avatar
structure: hook > buildup > twist/payoff > CTA
text: caption highlights for key moments
audio: original voice, dramatic pauses
duration: 30-90s
platforms: TikTok, Reels, Shorts
```

## CTA templates

### end-screen CTAs
```
1. SOFT: "follow for more [niche] tips"
2. ENGAGEMENT: "comment [X] if you want part 2"
3. SAVE: "save this before it disappears"
4. SHARE: "send this to someone who needs it"
5. LINK: "link in bio for the full guide"
6. PRODUCT: "use code [X] for 20% off" (affiliate)
```

### mid-video CTAs (for longer content)
```
1. "wait for the end - this is the best part"
2. "if you're still watching, you're already ahead"
3. "most people skip this part (don't be most people)"
```

## trending sound strategy

### how to find trending sounds
1. TikTok Creative Center > Trending Sounds (by region)
2. Instagram > Reels > save audio from viral reels
3. CapCut > trending templates (shows associated sounds)
4. Tokboard.com > trending sound tracker

### sound selection matrix
| content type | sound type | volume |
|-------------|-----------|--------|
| educational | trending lo-fi / chill | 15-25% |
| motivational | epic / builds up | 30-40% |
| humor/meme | trending meme sound | 50-70% |
| product demo | upbeat / modern | 20-30% |
| storytelling | cinematic / emotional | 25-35% |
| faceless narration | none (voice only) | 0% |
