## 4. KEY FRAMEWORKS

### Framework 1: The pSEO City Page Strategy

| Field | Detail |
|-------|--------|
| **What** | Generate 200+ pages targeting "[city name] + [product keyword]" long-tail searches |
| **How** | Template page with city-specific data. Programmatically generate for 200+ cities. Each page ranks for that city's variation |
| **Why** | One-time effort. Each page is a permanent traffic asset. Long-tail keywords = lower competition = faster ranking |
| **PRINTMAXX Application** | Generate city pages for app landing pages: "best prayer app [city]," "fitness lock screen app [city]," etc. Already have 300 longtail slugs |
| **Applies To** | MM001 (apps), MM003 (affiliate), CONTENT/ pages |
| **Priority** | HIGHEST |

### Framework 2: Affiliate-SEO Flywheel

| Field | Detail |
|-------|--------|
| **What** | Affiliates promote product -> create backlinks -> backlinks boost SEO -> higher SEO = more sales -> more affiliates join |
| **How** | Set up Rewardful ($50-200/mo). Recruit affiliates. They write blog posts/reviews. These create backlinks. Backlinks improve domain authority. Higher authority = higher rankings = more organic sales = more affiliates want to join |
| **Why** | Self-reinforcing loop. Affiliate marketing AND SEO boost each other |
| **PRINTMAXX Application** | Set up affiliate program for apps and digital products from Day 1. Let affiliates do the marketing and boost SEO simultaneously |
| **Priority** | HIGH |

### Framework 3: Acquisition Funnel Strategy

| Field | Detail |
|-------|--------|
| **What** | Buy smaller competing products to redirect their traffic to your main product |
| **How** | Danny acquired Dreamchanted and Hesti AI (smaller AI photo tools). Their existing traffic + backlinks now point to HeadshotPro |
