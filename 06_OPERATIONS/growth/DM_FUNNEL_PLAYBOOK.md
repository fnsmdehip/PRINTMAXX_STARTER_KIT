## Keyword DM Funnels

### How Keyword DM Funnels Work

1. Post high-value content with keyword CTA
2. User comments/DMs specific keyword
3. Send auto-reply with lead magnet link
4. Capture lead via landing page
5. Nurture via email or Telegram

### Platform-Specific Keyword Tactics

#### X/Twitter

**Manual DM Method (Recommended for Starting):**
- Post valuable content with keyword CTA
- Manually DM users who reply with keyword
- Takes 5-10 min/day for 20-50 DMs
- Highest trust, lowest ban risk

**Self-Reply Funnel (Proven High-Performer):**
```
Main Tweet: [Valuable insight or hook]
      |
Self-Reply: "DM me 'TOOLS' for the complete breakdown"
```

**Example from WINNING_CONTENT_STRUCTURES.csv:**
```
CS004: "Most people get X wrong"
CTA: "DM me 'FIX' for the solution"
Performance: HIGH
```

**X/Twitter Safe Limits (Jan 2026):**
- Manual DMs: 50-100/day (aged account)
- New accounts: Start with 20-30/day
- Spread across 8-12 hours

#### Instagram

**Comment-to-DM Flow:**
1. Post Reel with "Comment 'X' for the free guide"
2. ManyChat auto-DMs users who comment keyword
3. DM contains link to lead magnet

**Instagram DM Limits (Jan 2026):**

| Account Age | DMs/Day | Notes |
|-------------|---------|-------|
| <2 months | 10-20 | Manual only |
| 2+ months | 30-50 | Can use light automation |
| Established | 50-100 | ManyChat safe if warmed |

**ManyChat Setup for IG:**
- Cost: Free up to 1K contacts, then $15/mo
- Connect IG Business account
- Set up keyword triggers
- Create automated response flows

#### TikTok

**Bio Link Method (Only Reliable Option):**
- TikTok DM automation not recommended (detection aggressive)
- Use bio link for lead capture
- Comment CTA drives to bio

**TikTok Keyword CTA Examples:**
```
Video: [Value content]
Text Overlay: "Comment 'FREE' and check bio for the guide"
Bio Link: Landing page with email capture
```

### Telegram Channel Types

**Public Broadcast Channel:**
- One-way communication
- Unlimited subscribers
- Good for tips, content drops
- Can't see who's in channel

**Private Group:**
- Two-way communication
- Community interaction
- Limited to 200K members
- Requires invite link

**Recommended Structure:**
```

- Daily tips and content
- Tease premium content
- Link to products
      |
      v
Private Group (Paid or Engaged)
- Community discussion
- Direct access to you
- Exclusive content
```

## Compliance and TOS

### Platform Rules Summary

| Platform | Auto-DM Allowed | Keyword Trigger | Mass DM | Notes |
|----------|-----------------|-----------------|---------|-------|
| X/Twitter | Yes (careful) | Manual safe | No | Watch daily limits |
| Instagram | Yes (ManyChat) | Yes | No | Business account required |
| TikTok | No | No | No | Bio link only |
| LinkedIn | Yes (cloud tools) | Limited | No | Chrome extensions dead |

### X/Twitter DM Compliance

**Safe Practices:**
- Keep auto-DMs under 50/day for new accounts
- 100-200/day max for aged accounts
- Don't include links in auto-DMs (send after reply)
- Make messages feel personal
- Don't spam promotional content

**Risky Practices (Avoid):**
- Mass DM to non-followers
- Identical messages at scale
- Aggressive sales pitch in first DM
- Using broken automation tools

### Instagram DM Compliance

**ManyChat Safe Use:**
- Only respond to users who initiated (comment/DM)
- Include clear opt-out option
- Don't over-automate (feels spammy)
- Follow Meta's messaging policies

**Banned Practices:**
- DM users who didn't engage first
- Scrape and mass DM
- Misleading keyword triggers

### FTC Requirements for DM Marketing

**Required Disclosures:**
- If DM contains affiliate link: Disclose
- If promoting product you're paid for: Disclose
- Testimonials must be real and substantiated

**Disclosure Example:**
```
Here's the tool I mentioned: [LINK]
(affiliate link - I may earn commission)
```

### Spam Triggers to Avoid

| Trigger | Why It's Bad | Alternative |
|---------|--------------|-------------|
| Identical messages | Flagged as spam | Use 5-10 message variants |
| Links in first message | Often filtered | Send link after reply |
| All caps | Looks spammy | Normal capitalization |
| Too many emojis | Spam signal | 1-2 max or none |
| Immediate DM after follow | Feels automated | Wait 1-24 hours |

### Follow-Up Sequence

**24h Follow-Up (No Response):**
```
Hey, just making sure you got the [THING] I sent yesterday.

Here's the link again: [LINK]

Any questions?
```

**48h Follow-Up (Engaged but No Action):**
```
[NAME] - one more thing that might help:

[Additional tip or resource]

If you want to go deeper on [TOPIC], let me know.
```

**7-Day Follow-Up (Cold Lead):**
```
Hey [NAME], been a few days since I sent over [THING].

Curious: Did it help with [GOAL]?

No worries either way, just checking in.
```

**Interest Qualifier:**
```
Thanks for asking about [PRODUCT].

Quick question first: What's your main goal with [TOPIC]?

Want to make sure it's actually a fit before I explain more.
```

```
Based on what you said, [PRODUCT] could definitely help.

It's designed for [IDEAL CUSTOMER] who want [OUTCOME].

Would you want me to send more details?
```

```
I hear you on the price.

[PRODUCT] is [$X]. If it helps you [SPECIFIC OUTCOME], that's worth [VALUE COMPARISON].

Either way, no pressure. The free [LEAD MAGNET] I sent has a lot of the core concepts.
```
