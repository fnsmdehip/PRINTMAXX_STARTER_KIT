## Core Posting Tactics

### 1. Reply Bait Strategy

**How it works:** Post content designed to drive comments/replies which boost algorithmic distribution.

**Patterns:**
- "Which do you choose? A or B" (forces engagement)
- "Unpopular opinion: [hot take]" (forces agreement/disagreement)
- "Rate my [X] 1-10" (simple engagement)
- "Comment [word] if you [identify with X]" (DM funnel trigger)
- "Wrong answers only" (humor engagement)
- "I'll start: [example]" (thread starter)

**Reply Bait Templates:**
```
Tech niche:
- "You can only use one AI tool for the rest of your life. Which one?"
- "Claude vs GPT? Wrong answers only"
- "Your tech stack for 2026 in 3 words. Go."
- "Rate my app stack 1-10: [list]"

Faith niche:
- "One verse that changed your life. Go."
- "Morning prayer or night prayer person?"
- "Rate my prayer routine 1-10"

Fitness niche:
- "Push pull legs or bro split? Fight me."
- "Your pre-workout stack. Go."
- "Gym at 5am or 9pm? Only one right answer."
```

### 2. Greentext Satirical Take Format

**Source:** @anothercohen (119K views, 1.2K likes, Jan 24 2026)

**Format:** Use ">" prefix for each line, escalating absurdity, end with rhetorical question

**Why it works:** Exposes industry BS through satire. Greentext format familiar from Reddit/4chan. Each step builds absurdity. Rhetorical question invites engagement and shares.

**Structure:**
```
> action 1 (reasonable)
> action 2 (slightly questionable)
> action 3 (absurd consequence)
> action 4 (more absurd)
> action 5 (ridiculous outcome)
Why is no one doing this??
```

**Example (ARR Gaming - 119K views):**
```
> vibe code a CRM
> charge your own company a hefty subscription of $500k/year
> pay the invoice
> claim the $500k as net new ARR
> do this monthly
> after 4 months you're at $2m ARR
> raise a large Series A
Why is no one doing this??
```

**Best For:**
- Tech/startup niches (mocking VC metrics, funding, valuations)
- Exposing industry practices through satire
- Controversial takes delivered as "advice"
- Any topic where the absurdity reveals truth

**Greentext Template Ideas:**

**Tech/Startup:**
```
> build MVP in a weekend
> post "just shipped" on twitter
> get 50 likes
> claim product-market fit
> raise $2M seed
> hire 10 people
> still no customers
Why is no one doing this??
```

**AI Hype:**
```
> wrap an API
> add a landing page
> call it "AI-powered"
> charge 10x the API cost
> get featured on Product Hunt
> claim you're an AI startup
> sell to enterprise for 8 figures
Why is no one doing this??
```

**Indie Hacker:**
```
> buy a $9 domain
> spin up a landing page
> screenshot your own purchase
> post "first customer!"
> get 100 likes
> screenshot the likes
> "went viral"
Why is no one doing this??
```

**Key Elements:**
1. Start somewhat reasonable
2. Each step more absurd
3. Treat absurdity as normal
4. End with "Why is no one doing this??" (the hook)
5. Let readers realize the satire themselves

**Engagement Pattern:** High shares because people want to expose the thing being mocked. Comments split between "this is satire right" and "I've seen this happen."

### 3. A/B Post Style (Contrarian Double-Post)

**Format:** Post same statement twice with opposite takes
**Why it works:** Tests which take resonates, contrarian often wins, algorithm treats as 2 chances

**Example from @aftermagics (proven):**
```
Post 1: "start using ai to write your replies" → 1K views
Post 2: "stop using ai to write your replies" → 1.9K views

Key insight: Contrarian "stop" got 2x engagement
```

**Structure:**
```
Post A: [Affirmative statement]
Post B: [Same topic, opposite take]

Track which performs better. Double down on winning angle.
```

**A/B Post Ideas:**

**Tech/Solopreneur:**
- "start using claude code" vs "stop using claude code"
- "ship fast break things" vs "ship slow get it right"
- "raise VC" vs "stay bootstrapped"
- "build in public" vs "build in private"
- "use AI for everything" vs "AI is overrated"

**Format variations:**
- Vertical text (one word per line) like @aftermagics
- Side-by-side image comparison
- Before/after screenshots

**Faith:**
- "Sunday morning" vs "Sunday evening" (vibes comparison)
- "Phone first" vs "Prayer first" (morning routine)
- "Quiet prayer" vs "Worship music prayer" (style preference)

**Fitness:**
- "Monday motivation" vs "Friday energy" (relatable)
- "Home gym" vs "Commercial gym" (preference)
- "Bulk season" vs "Cut season" (aesthetic comparison)

## Scraping Workflow

### Daily Video/Image Scrape (15 min)

1. **Check trending in niche:**
   ```bash
   # Twitter trending search
   Search: [niche] min_faves:1000 filter:media since:2026-01-22
   ```

2. **Download viral content:**
   ```bash
   # Twitter video download
   yt-dlp "https://twitter.com/user/status/xxxxx"

   # TikTok download
   yt-dlp "https://tiktok.com/@user/video/xxxxx"
   ```

3. **Save to media library:**
   ```
   CONTENT/media/
   ├── memes/
   ├── infographics/
   ├── reaction_clips/
   └── scrape_log.csv
   ```

4. **Log scrape:**
   - Source URL
   - Content type
   - Views/engagement at scrape time
   - Potential use (which niche account)
