## IMPORT TO GOOGLE SHEETS

### Step-by-Step

1. **Create New Sheet**
   - Go to sheets.google.com
   - Create new spreadsheet
   - Name it: "PRINTMAXX Control Center"

2. **Import Each CSV**
   - File → Import → Upload
   - Select CSV file
   - Import location: "Insert new sheet(s)"
   - Separator type: Comma
   - Click "Import data"
   - Repeat for all 9 CSV files

3. **Rename Tabs**
   - Remove ".csv" extension from tab names
   - Order tabs as listed above for logical flow

4. **Format Sheets**
   - Freeze top row on each tab: View → Freeze → 1 row
   - Add color coding to status columns:
     - Green: active, ready, implemented, closed_won
     - Yellow: warmup, discovery, in_progress, monitoring
     - Red: flagged, banned, closed_lost
   - Adjust column widths for readability

5. **Add Data Validation** (Optional but recommended)
   - Status columns → Data → Data validation → Dropdown from a range
   - Platform columns → Dropdown: Twitter, Instagram, YouTube, TikTok, etc.
   - Niche columns → Dropdown: AI utilities, Faith streak, Fitness
   - Effort/ROI/Risk → Dropdown: Low, Medium, High

6. **Set Up Conditional Formatting** (Optional but powerful)
   - OUTREACH_PIPELINE Next_Touch: Highlight overdue dates in red
   - EXPERIMENTS_AB End_Date: Highlight when test should conclude
   - WARMUP_DEVICE_MATRIX flags: Highlight any value > 0 in red
