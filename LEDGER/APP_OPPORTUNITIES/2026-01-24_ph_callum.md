# Callum - AI Calendar Assistant

**Source:** Product Hunt

**Date Found:** 2026-01-24
**Upvotes:** 262+ (Day Rank on Jan 22)

---

## What It Does

AI calendar assistant for teams that:
- Supercharges calendar management
- Likely: natural language scheduling
- Team coordination features
- AI-powered meeting suggestions

**Tags:** Productivity, Calendar, Artificial Intelligence

---

## Clone Difficulty

**Rating:** MEDIUM

**Why:**
- Calendar API integration (Google Calendar, Apple Calendar)
- NLP for natural language date/time parsing
- Team sync requires backend
- OAuth flows for calendar access

**Tech Stack Estimate:**
- React Native (Expo)
- Google Calendar API / Apple EventKit
- Claude/GPT for natural language parsing
- Backend for team features (Supabase or Firebase)

**Existing Libraries:**
- Chrono.js (natural language date parsing)
- react-native-calendar-events

## Monetization Model

**Primary:** Subscription
- Free: Personal calendar, basic AI
- Pro ($5.99/mo): Multiple calendars, advanced AI, team features
- Team ($9.99/user/mo): Admin controls, analytics, integrations

**Secondary:**
- Enterprise contracts
- Integration marketplace
- White-label for specific industries

## Competitive Landscape

- **Calendly** (scheduling links, not AI assistant)
- **Reclaim.ai** (AI calendar blocking)
- **Motion** (AI project management + calendar)
- **Clara** (AI scheduling assistant, enterprise)

**Differentiation opportunity:** Mobile-first (competitors are desktop-focused), niche-specific (parent coordination, church planning), simpler pricing

---

## Implementation Priority

**Score:** 6/10

**Reasons:**
- Medium complexity (calendar APIs can be tricky)
- Crowded space but mobile opportunity
- Strong retention (daily use)
- Team features add complexity
- Niche versions could differentiate

---

## Simpler Alternative

Instead of full calendar assistant, build:
- **"Text to Event"** - Screenshot or voice to calendar event
- Focused feature, easier to build
- Natural upgrade path to full assistant

---

## Next Steps

1. Start with "Text to Event" MVP (screenshot or voice)
2. Choose one niche (recommend: Parents or Students)
3. Add smart suggestions in v2
4. Team/sharing features in v3
5. Consider starting as widget/extension before full app
