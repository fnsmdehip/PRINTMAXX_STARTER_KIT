## Technical Considerations

**iOS WidgetKit:**
- Timeline-based updates
- Multiple widget sizes (small, medium, large)
- Lock screen widgets (iOS 16+)
- Interaction limitations (tap to open app)

**Quote Database:**
- Can start with public domain quotes
- Attribute properly
- Mix famous + original quotes
- Categorization is key
