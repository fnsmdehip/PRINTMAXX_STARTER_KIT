## Technical Considerations

**iOS Limitations:**
- Background alarm reliability
- Need to use local notifications creatively
- Can't truly prevent phone shutdown
- Screen time API for anti-snooze features

**Workarounds:**
- Use aggressive notification permissions
- Persistent audio playback
- Widget reminders
- Apple Watch integration for backup
