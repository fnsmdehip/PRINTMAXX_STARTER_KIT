## CRITICAL: How to Use LEDGER When Pursuing an Op

**Before starting ANY money method or op, query these files in order:**

### Step 1: Check Method Tracker
```
MONEY_METHODS_TRACKER.csv → Find method_id, status, priority, revenue model
```

### Step 2: Check Cross-Pollination
```
CROSS_POLLINATION_MATRIX.csv → Find synergy_partners, shared_audience, cross_sell_products
```

### Step 3: Check GTM Priorities
```
GTM_OPTIMIZATION_PRIORITIES.csv → ASO/SEO/GEO requirements for this method
```

### Step 4: Check Recent Alpha
```
ALPHA_STAGING.csv → Filter by category, find recent tactics that apply
```

### Step 5: Check Niche Details
```
NICHES.csv → Target demo, content themes, offer stack
```

### Step 6: Check Marketing Channels
```
MARKETING_CHANNELS_MASTER.csv → Distribution channels, costs, conversion rates
```

1. `MONEY_METHODS_TRACKER.csv` → MM007 status

3. `OUTBOUND_OPPORTUNITIES/` → Latest tactics
4. `CROSS_POLLINATION_MATRIX.csv` → Stack with MM004, MM005, AI001
5. `ALPHA_STAGING.csv` → Filter for OUTBOUND category

1. `MONEY_METHODS_TRACKER.csv` → CF001-CF010 options

4. `MARKETING_CHANNELS_MASTER.csv` → Distribution
