## Deliverability Comparison (Most Important)

### Instantly

**Deliverability Features:**
- **SISR (Server and IP Sharding and Rotation):** Auto-distributes sending across multiple IPs and servers, rotating and replacing if performance drops
- **4.2M+ warmup network:** Real accounts for warmup engagement
- **Inbox Placement Tests:** Automated checks against real mailbox providers before launch

**Performance:**
- Solid inbox placement
- Auto-rotation prevents burnout
- Good for scale (1000+ sends/day)

### Smartlead

**Deliverability Features:**
- **SmartSenders:** Automates SPF, DKIM, DMARC setup (eliminates human error)
- **AI-Powered Warmup:** Gradually builds trust with email providers
- **Unlimited Sender Addresses:** Strong reputation management across many inboxes

**Performance:**
- **Reply rate: 0.96%** (vs Lemlist 0.9%)
- Higher open rate = better deliverability
- Best for high-volume sending (10K+/day)

**Winner:** Smartlead edges out on pure deliverability metrics

### Lemlist

**Deliverability Features:**
- **Lemwarm:** Automated warmup with sender reputation tips
- **IP Rotation:** Built-in
- **DNS Verification:** Guided setup

**Performance:**
- Reply rate: 0.9%
- Stable inbox placement under 2024+ Gmail/Yahoo policies
- Lacks advanced features like inbox placement tests

**Source:** [Smartlead vs Instantly 2026](https://sparkle.io/blog/smartlead-vs-instantly/)

## Deliverability Infrastructure Deep Dive

### Instantly SISR (Light Speed Plan)

**How It Works:**
- Auto-distributes sends across multiple IPs and servers
- Monitors performance per IP/server
- Rotates out underperforming infrastructure
- Replaces burned IPs automatically

**Why It Matters:**
- Prevents IP burnout
- Maintains deliverability at scale
- Less manual management

**Cost:** $358/mo (Light Speed plan)

### Smartlead SmartSenders

**How It Works:**
- Automates SPF, DKIM, DMARC setup
- Guides you through DNS configuration
- Verifies setup before sending
- Eliminates human error in authentication

**Why It Matters:**
- Non-technical users can set up properly
- Reduces deliverability issues from misconfiguration
- Faster time to launch

**Cost:** Included in all plans

### Lemlist Lemwarm

**How It Works:**
- Automated warmup with engagement network
- Sender reputation tips and monitoring
- IP rotation built-in
- DNS verification guidance

**Why It Matters:**
- Reduces manual warmup work
- Maintains sender reputation
- Prevents spam folder placement

**Cost:** Included in all plans

## Migration Considerations

**From Gmail to Cold Email Tool:**
- Export contacts → CSV import
- Set up custom domains (don't use personal Gmail)
- 2-4 week warmup before sending
- Start with 20-30 sends/day, scale to 50-70

**From One Tool to Another:**
- Export sequence templates
- Export contact lists
- Reconfigure DNS (SPF/DKIM/DMARC) for new tool
- Re-warmup inboxes (even if previously warmed)

**Cost of Switching:** 2-4 weeks lost to warmup + setup time
