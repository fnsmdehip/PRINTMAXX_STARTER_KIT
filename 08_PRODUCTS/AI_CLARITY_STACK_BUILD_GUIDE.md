### Page 5: A - Assess (Metrics Tracker)

**Database:** Workflow Performance

**Properties:**
- Workflow (relation)
- Date Run (date)
- Time Spent (number)
- Quality Score (select: 1-5)
- Output (text: what was created)
- Learnings (text)
- Keep/Kill (select)

**Auto-calculated:**
- Avg quality per workflow
- Total time saved
- Kill candidates (quality <3, run <2 times in 30 days)

**Dashboard View:**
- Chart: Time saved over time
- Table: Workflows by ROI
- Gallery: Recent outputs
