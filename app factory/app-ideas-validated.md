## NICHE-DOWN STRATEGY

Take any proven app category and add:
- **Christian** version
- **Women-only** version
- **Over 40** version
- **Specific diet** version (carnivore, vegan, keto)
- **Specific aesthetic** (minimalist, dark mode, premium)

Examples:
- Brain Rot → Christian Screen Time ("Guard Your Eyes")
- Looksmax → Women's Glow-Up Tracker
- Cal AI → Carnivore AI (meat-only calorie tracking)
- Habit Tracker → Catholic Virtue Tracker

## QUICK VALIDATION CHECKLIST

Before building, confirm:

- [ ] Search TikTok for niche - are videos getting 100K+ views?
- [ ] Check Google Trends - is interest stable or growing?
- [ ] Find 3+ competitor apps on App Store
- [ ] Confirm competitors making $5K+/month (Sensor Tower/Appfigures)
- [ ] Identify the psychological trigger you're exploiting
- [ ] Have a clear answer to: "Why would someone pay weekly for this?"

If you can't check all boxes, pick a different idea.

## THE ALGORITHM

```
1. Pick psychological trigger (what deep need?)
2. Find trending content on TikTok (validated demand)
3. Check if apps in space are making money (Sensor Tower)
4. Find angle: niche down or improve existing
5. Build MVP in 1-2 weeks
6. Ship with hard paywall ($4.99-9.99/week)
7. Post daily content about the journey
8. After 30 days: kill if <$500 revenue, double down if growing
9. Repeat with next idea
```

## REVENUE BENCHMARKS

**What "Success" Looks Like:**

| Revenue | Status | What It Means |
|---------|--------|---------------|
| $0-1K/mo | Testing | Keep iterating or kill |
| $1-5K/mo | Traction | Double down on marketing |
| $5-10K/mo | Validated | Optimize paywall, add paid ads |
| $10-30K/mo | Growing | Consider hiring help |
| $30-100K/mo | Scaling | Build team, expand features |
| $100K+/mo | Winner | You've made it - protect and grow |

**Timeline Reality:**
- First $1K: Usually 1-3 months
- First $10K: Usually 3-6 months  
- First $100K month: Usually 6-18 months (if it happens)

Most indie apps never hit $10K/month. That's why portfolio approach works - build 10+, let data decide winners.
