## 🎨 Screenshot Guidelines

### Design Principles
1. **Clean backgrounds** - Use gradient or solid colors
2. **Clear text** - High contrast, readable fonts
3. **Feature focus** - Highlight one key feature per screenshot
4. **Mobile-first** - Optimize for small screens
5. **Consistent branding** - Same colors and fonts

### Content Strategy
1. **Screenshot 1:** Daily verse & streak tracking
2. **Screenshot 2:** Full Bible reader with search
3. **Screenshot 3:** Progress milestones & stats
4. **Screenshot 4:** Community sharing features
5. **Screenshot 5:** Premium features (AI helper, notes)
6. **Screenshot 6:** Faith Letters & personalization

### Technical Requirements
- PNG format only
- No transparency (solid backgrounds)
- 72 DPI minimum
- No device frames (App Store adds them)
- English text only (for global App Store)

## 🛠️ Creation Process

### Step 1: Capture Screenshots
1. Run app on iPhone 15 Pro Max simulator
2. Navigate to each screen
3. Use Xcode → Debug → View UI Hierarchy → Screenshot
4. Or use Simulator menu: Device → Screenshot

### Step 2: Edit in Canva/Photoshop
1. Open screenshot in design tool
2. Add text overlays (white text, black shadows)
3. Ensure text is readable at small sizes
4. Export as PNG with transparent background removed

### Step 3: Test on Device
1. Upload to App Store Connect test
2. Check appearance on different devices
3. Verify text readability
4. Get feedback from beta testers

### Step 4: Final Polish
1. Consistent font usage (SF Pro preferred)
2. Brand color integration
3. Remove any UI artifacts
4. Compress file size (under 4MB each)
