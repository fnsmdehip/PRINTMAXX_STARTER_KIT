## 🛠️ Development

### Tech Stack
- React Native (Expo)
- TypeScript
- Zustand (state management)
- AsyncStorage (local storage)
- RevenueCat (subscriptions)
- AdMob (ads)

### Key Features
- ✅ Daily verse delivery (KJV)
- ✅ Streak tracking
- ✅ Full Bible with search
- ✅ AI Bible helper (premium)
- ✅ Verse notes
- ✅ Referral system (3 shares = 50% off)
- ✅ Faith Letters (handwritten notes)
- ✅ Church partner discounts
- ✅ Push notifications

### Environment Setup
```bash
# Copy template
cp env.template .env

# Add your keys:
EXPO_PUBLIC_REVENUECAT_APPLE_API_KEY=your_key
EXPO_PUBLIC_ADMOB_APP_ID=your_admob_id
EXPO_PUBLIC_ADMOB_BANNER_ID=your_banner_id
EXPO_PUBLIC_OPENROUTER_API_KEY=your_ai_key  # Optional for premium AI
```

### Build Commands
```bash
npm install
npx expo start --ios
npx expo build:ios
```

## 📈 Growth Strategy

### Primary Channels (90% of effort)
1. **Church Partnerships** - Organic, high trust, bulk users
2. **Micro-Influencers** - $100-500 posts, targeted reach
3. **Organic Social** - TikTok/Reels daily content

### Secondary Channels (10% of effort)
4. **Paid Ads** - Only after proving organic conversion
5. **Podcast Sponsorships** - High cost, high trust

### Budget Allocation
- Month 1: $500 (influencers + tools)
- Month 2: $1,000 (influencers + church commissions)
- Month 3: $2,000 (influencers + paid ads)
