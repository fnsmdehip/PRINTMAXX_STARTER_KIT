# 📱 Scripture Streak - Master Launch & Operations Checklist

## 🎯 EXECUTIVE SUMMARY

**App:** Scripture Streak - Daily Bible habit building app
**Target:** Christians seeking consistent Bible reading
**Monetization:** Freemium (ads + $1.99 lifetime / $0.99 monthly)
**Launch Goal:** 1,000 downloads in Month 1, 10,000 in Month 3

---

## 📅 PHASE 1: PRE-LAUNCH SETUP (Week 1-2)

### ✅ App Store Submission
- [ ] Create Apple Developer account ($99/year)
- [ ] Set up App Store Connect account
- [ ] Generate app icons (1024x1024, 20x20, etc.)
- [ ] Write App Store description (see docs/APP-STORE-LISTING.md)
- [ ] Create 6.7" iPhone screenshots (6 total)
- [ ] Test app on physical device
- [ ] Submit for App Store review
- [ ] Prepare rejection response templates
- [ ] **Timeline:** 3-7 days for approval

### ✅ Payment Integration Setup
- [ ] Create RevenueCat account
- [ ] Set up app in RevenueCat dashboard
- [ ] Create products: "Lifetime" ($1.99), "Monthly" ($0.99)
- [ ] Configure entitlements (premium features)
- [ ] Test purchase flow in sandbox
- [ ] Set up shared secret for verification
- [ ] **Timeline:** 1 day

### ✅ Ad Integration Setup
- [ ] Create AdMob account
- [ ] Create Scripture Streak app in AdMob
- [ ] Create banner ad unit
- [ ] Get AdMob App ID and Banner ID
- [ ] Add to .env file
- [ ] Test ads in development
- [ ] **Timeline:** 1 day

### ✅ Marketing Infrastructure Setup
- [ ] **Cold Email:** Sign up for Instantly.ai ($37/mo)
- [ ] **Domain:** Buy scripturestreakpartners.com ($12/year)
- [ ] **Google Workspace:** Set up email ($6/mo)
- [ ] **Social Media:** Create TikTok, Instagram, Twitter accounts
- [ ] **Analytics:** Set up App Store Connect analytics
- [ ] **CRM:** Set up Google Sheets for tracking
- [ ] **Timeline:** 2-3 days

### ✅ Content Creation
- [ ] Create 20 TikTok/Reels videos (5 days worth)
- [ ] Design Instagram graphics (10 posts)
- [ ] Write Twitter thread templates (5 threads)
- [ ] Prepare email newsletter templates
- [ ] Create app demo video (15-30 seconds)
- [ ] **Timeline:** 3-4 days

### ✅ Contact Database Building
- [ ] Import church contacts (marketing/church-contacts.csv)
- [ ] Import influencer contacts (marketing/influencer-contacts.csv)
- [ ] Clean and verify email addresses
- [ ] Segment contacts by priority/size
- [ ] **Timeline:** 1-2 days

### App Store Rejection
- [ ] Read rejection reason carefully
- [ ] Fix issues within 24 hours
- [ ] Submit appeal with detailed explanation
- [ ] Have backup app description ready
- [ ] **Timeline:** 24-48 hours to resubmit

### Payment Issues
- [ ] Check RevenueCat dashboard
- [ ] Verify shared secret configuration
- [ ] Test purchase flow on device
- [ ] Contact RevenueCat support if needed
- [ ] **Timeline:** 4-8 hours to resolve

### Email Deliverability Problems
- [ ] Check spam folder placement
- [ ] Verify domain authentication (SPF, DKIM, DMARC)

- [ ] **Timeline:** 1-2 days to diagnose

- [ ] Scale up posting frequency

- [ ] **Timeline:** Immediate response

## 🎯 SUCCESS METRICS DASHBOARD

### Acquisition Metrics
- **Downloads:** Daily, weekly, monthly
- **Cost per Install:** By channel
- **Conversion Rate:** Free → Premium
- **Retention:** Day 1, Day 7, Day 30
- **Churn Rate:** Monthly subscribers

### Revenue Metrics
- **Monthly Recurring Revenue:** Subscription revenue
- **Average Revenue per User:** Lifetime value
- **Payback Period:** Marketing spend recovery
- **Customer Acquisition Cost:** Total marketing spend ÷ new customers
- **Lifetime Value:** Average revenue per user

### Engagement Metrics
- **Daily Active Users:** App opens per day
- **Session Length:** Average time in app
- **Feature Usage:** Which features are most used
- **Streak Distribution:** How many users maintain streaks
- **Verse Completion:** Daily verse reading completion

### Marketing Metrics
- **Email Metrics:** Open rate, click rate, reply rate
- **Social Metrics:** Engagement rate, follower growth
- **Influencer ROI:** Revenue ÷ influencer spend
- **Church Partnership ROI:** Revenue ÷ partnership spend
- **Ad Performance:** CPM, CTR, conversion rate
