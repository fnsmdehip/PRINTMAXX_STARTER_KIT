## How to Verify and Register

For each "LIKELY AVAILABLE" handle:

1. Go to https://twitter.com/i/flow/signup
2. During the username selection step, type the handle
3. Twitter will show a green checkmark if available, red X if taken
4. If taken (could be suspended), try the alternatives listed above
5. Register with a unique email per account
6. Immediately set bio + profile pic to avoid looking like a bot

**Rate limit warning:** Don't create more than 2-3 accounts from the same IP in the same session. Space them out or use different IPs.
