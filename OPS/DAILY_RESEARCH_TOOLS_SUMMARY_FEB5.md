## 3. Platform algorithm changes detected

### X/Twitter (January 2026 - MAJOR UPDATE)

**Source:** [Typefully analysis](https://typefully.com/blog/x-algorithm-open-source) | [SocialWick](https://www.socialwick.com/decoding-the-new-x-algorithm-to-stay-visible-in-2026)

| Signal | Weight | Action |
|--------|--------|--------|
| Bookmarks | 5x multiplier (+10 pts) | Create bookmark-worthy content (save-able tips, lists, frameworks) |
| Likes | +0.5 pts | Still matters but 20x less than bookmarks |
| Premium subscribers | +4 to +16 pts | Get X Premium immediately |
| Video tweets | 10x engagement | Add video to every post |
| Short tweets (71-100 chars) | +17% engagement | Keep tweets punchy |
| Positive tone | Algorithmic boost | Grok penalizes negativity even with high engagement |
| Negative/combative tone | SUPPRESSED | Avoid outrage bait, even if it gets replies |

**Key change:** X released full For You algorithm on Jan 20, 2026. Grok-powered transformer replaced manual features. Tone monitoring is new and significant.

### TikTok (2026 - FOLLOWER-FIRST SHIFT)

**Source:** [OpusClip](https://www.opus.pro/blog/tiktoks-new-algorithm-2026) | [Buffer](https://buffer.com/resources/tiktok-algorithm/) | [Socialync](https://www.socialync.io/blog/tiktok-algorithm-secrets-2026)

| Signal | Weight | Action |
|--------|--------|--------|
| Follower-first testing | PRIMARY | Build real follower base before expecting virality |
| Completion rate | 70% bar | Create content people watch to the end |
| Shares + saves | Above likes | Make content people send to friends |
| Rewatch rate | MASSIVE boost | Create loop-worthy content with reveals |
| Keywords in captions | +20-40% visibility | SEO-optimize every caption |
| Content length | 60-180 sec sweet spot | Longer than before, more storytelling room |

**Key change:** TikTok now tests with your followers first before wider distribution. This penalizes accounts with fake/bought followers and rewards genuine audience building.

### Instagram (2026 - ORIGINALITY PREMIUM)

**Source:** [Buffer](https://buffer.com/resources/instagram-algorithms/) | [Hootsuite](https://blog.hootsuite.com/instagram-algorithm/) | [SproutSocial](https://sproutsocial.com/insights/instagram-algorithm/)

| Signal | Weight | Action |
|--------|--------|--------|
| Originality | REWARDED | Create original content, don't repost/aggregate |
| Sends per reach | TOP 3 signal | Make content people DM to friends |
| Watch time | TOP 3 signal | Hook immediately, maintain attention |
| Likes per reach | TOP 3 signal | Still matters |
| 3-min Reels | Now viable | Long-form Reels no longer penalized |
| Aggregator accounts | PENALIZED | Content farm strategy needs original content, not reposts |
| Views | Primary metric | Unified across all formats (Reels, Stories, Photos, Carousels) |

**Key change:** Instagram decisively penalizing content aggregators. Our content farm accounts MUST create original content, not repost from other platforms.

### YouTube (2026 - SATISFACTION OVER ENGAGEMENT)

**Source:** [SocialBee](https://socialbee.com/blog/youtube-algorithm/) | [Miraflow](https://miraflow.ai/blog/youtube-shorts-algorithm-update-january-2026)

| Signal | Weight | Action |
|--------|--------|--------|
| Satisfaction signals | PRIMARY | Quality over clickbait |
| Shorts up to 3 min | NEW | More storytelling room in Shorts |
| Shorts-to-longform flow | BOOSTED | Use Shorts as funnel to longer content |
| Completion rate | Affects RPM | Higher completion = better monetization |
| Search filter: exclude Shorts | NEW | Long-form search results now separate from Shorts |

**Key change:** YouTube added user filter to exclude Shorts from search. This separates the two content types more. Shorts are increasingly a funnel to longform, not a standalone format.
