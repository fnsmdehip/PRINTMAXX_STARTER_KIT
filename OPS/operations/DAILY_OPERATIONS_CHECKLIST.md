# PRINTMAXX Daily Operations Checklist

Daily, weekly, monthly, and quarterly operations for solopreneur content distribution.

---

## Daily Operations Checklist

### Morning Routine (First Hour)

- [ ] Check platform analytics (engagement rate, follower delta)
- [ ] Review overnight engagement (comments, DMs, mentions)
- [ ] Check for urgent messages requiring immediate response
- [ ] Review scheduled content for today
- [ ] Quick competitor scan (2-3 accounts, 5 min max)
- [ ] Check ALPHA_STAGING.csv for pending items
- [ ] Review LEDGER/FUNNEL_METRICS.csv for anomalies

### Content Operations (Core Work Block)

- [ ] Post scheduled content across platforms
- [ ] Respond to comments from last 24 hours
- [ ] Create 1 story/ephemeral piece per active platform
- [ ] 90-min content creation block (batch for week)
- [ ] Schedule next 24-48 hours of content
- [ ] Update LEDGER/CONTENT_PIPELINE.csv with new items
- [ ] Check content queue depth (maintain 7-day buffer)

### Community Management

- [ ] Respond to all DMs within 24 hours
- [ ] Engage meaningfully on 5-10 comments
- [ ] Post 1 community contribution (subreddit, Discord, forum)
- [ ] Send 3 collaboration outreach messages
- [ ] Review and approve any UGC/mentions

### Growth Activities

- [ ] Engage authentically on 10 target accounts
- [ ] Research 3-5 new hashtags/keywords
- [ ] Monitor 2-3 trending topics in niche
- [ ] Send 5 network outreach messages
- [ ] Check HIGH_SIGNAL_SOURCES.csv accounts for alpha
- [ ] Add any findings to ALPHA_STAGING.csv

### Business Operations

- [ ] Follow up on leads older than 48 hours
- [ ] Respond to client/customer messages
- [ ] Log revenue to LEDGER/FUNNEL_METRICS.csv
- [ ] Update task list for tomorrow
- [ ] Check email inbox, process to zero
- [ ] Review MASTER_TASKS.md, update status

### End of Day (15 min)

- [ ] Log daily metrics to tracking sheet
- [ ] Prep tomorrow's content (thumbnails, captions ready)
- [ ] Review content performance, note winners
- [ ] Write 1-2 sentence win/learning for the day
- [ ] Set top 3 priorities for tomorrow

## Weekly Operations Checklist

### Monday: Planning

- [ ] Review last week's metrics (engagement, growth, revenue)
- [ ] Set 3 weekly goals (specific, measurable)
- [ ] Plan content themes for the week
- [ ] Schedule all recurring content
- [ ] Review competitor moves from past week
- [ ] Check LEDGER files for anything stale
- [ ] Run /daily-research for alpha scan

### Tuesday-Thursday: Execution

- [ ] Complete daily checklist
- [ ] Progress on weekly goals
- [ ] Create 2-3 pieces of evergreen content
- [ ] Test 1 new content format or platform feature
- [ ] Document what's working in OPS/logs/

### Friday: Content Batch

- [ ] Batch create next week's content (4-6 hours)
- [ ] Edit and polish all draft content
- [ ] Schedule content for weekend
- [ ] Create weekend engagement prompts
- [ ] Prep any automated posts

### Weekend: Light Touch

- [ ] 15-min morning check (engagement only)
- [ ] Respond to high-priority DMs only
- [ ] Post 1 casual/personal piece
- [ ] NO content creation (rest)
- [ ] Light competitor scroll for inspiration

## Emergency Protocols

### Viral Content (>10x normal engagement)

- [ ] Respond to comments within 2 hours
- [ ] Pin best comment or your CTA
- [ ] Create follow-up content immediately
- [ ] Push relevant offer/lead magnet
- [ ] Document what made it work
- [ ] Cross-post to other platforms

### Negative PR/Backlash

- [ ] Don't respond emotionally
- [ ] Screenshot everything
- [ ] Assess severity (troll vs legitimate)
- [ ] Prepare measured response if needed
- [ ] Hide/delete only if violates policy
- [ ] Document in OPS/logs/

### Platform Down

- [ ] Confirm it's platform-wide (not just you)
- [ ] Don't panic about missed posts
- [ ] Focus on other platforms
- [ ] Resume normal ops when back
- [ ] Note any data loss

### Burnout Warning Signs

- Dreading content creation
- Engagement feels like a chore
- Comparing yourself to others constantly
- Skipping daily checklists for 3+ days

**Response:**
- [ ] Take 2-3 day break
- [ ] Batch-schedule essential content only
- [ ] Review why you started
- [ ] Consider simplifying operations
