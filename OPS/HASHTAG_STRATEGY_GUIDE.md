## Platform Strategies

### Instagram

**Hashtag Limits:** 30 max, 5-15 optimal

**Placement:**
- In caption (slightly higher reach)
- First comment (cleaner look)
- Both perform similarly now

**Mix Strategy (15 hashtags):**
- 3 large (1M+ posts) - reach
- 5 medium (100k-1M) - discovery
- 5 small (10k-100k) - ranking
- 2 niche (<10k) - community

**Instagram Rules:**
- Don't repeat same set on every post
- Avoid banned hashtags (check regularly)
- Match hashtags to content
- Track which sets perform best

**When to Post:**
- Reels: Morning (6-9am) or evening (7-9pm)
- Carousels: Lunch (12-2pm)
- Stories: Anytime (ongoing presence)

---

### TikTok

**Hashtag Limits:** Unlimited, but 3-5 optimal

**Placement:** In caption only

**Strategy:**
- 1-2 broad (#fyp, #foryou)
- 1-2 niche-specific
- 1 trending (if relevant)

**TikTok Rules:**
- Less is more (algorithm-driven)
- Trending hashtags boost exposure
- Niche hashtags build community
- #fyp still works (despite myths)

**Trending Hashtag Sources:**
- Discover page
- Competitor analysis
- TikTok Creative Center

---

### Twitter/X

**Hashtag Limits:** 2 max (readability)

**Placement:** End of tweet or integrated

**Strategy:**
- 1-2 relevant hashtags max
- Focus on topic, not reach
- Skip hashtags on threads
- Use for topic trends only

**Twitter Rules:**
- Hashtags hurt engagement if overused
- Better to skip than force
- Use for joining conversations
- Monitor trending for opportunities

---

### LinkedIn

**Hashtag Limits:** 3-5 optimal

**Placement:** End of post

**Strategy:**
- 3 niche professional hashtags
- 1-2 broader industry hashtags
- Follow hashtags in your niche

**LinkedIn Rules:**
- Professional tone only
- Industry-specific hashtags
- Check follower count of hashtag
- Don't use Instagram-style hashtags

---

### YouTube

**Hashtag Limits:** 3 max (above title)

**Placement:** Video description (first 3 appear above title)

**Strategy:**
- 1 branded hashtag
- 2 topic hashtags
- First 3 in description shown above title

**YouTube Rules:**
- Tags (separate from hashtags) more important
- Hashtags mainly for branding
- Keep them clean and professional
- Include in description for discovery

## Hashtag Research Process

### Finding Hashtags

1. **Search platform** - Type keyword, see suggestions
2. **Check competitors** - See what top accounts use
3. **Use tools** - Hashtagify, RiteTag, Display Purposes
4. **Check post counts** - Balance reach vs. competition
5. **Test and track** - Measure which sets perform

### Evaluating Hashtags

| Factor | Check |
|--------|-------|
| Relevance | Does it match your content? |
| Size | Right balance of reach/competition? |
| Recent posts | Are people actively using it? |
| Quality | Are top posts high quality? |
| Banned | Is it restricted/banned? |

### Building Sets

1. Create 3-5 sets per niche
2. Rotate sets (don't repeat daily)
3. Track performance weekly
4. Replace underperformers
5. Add trending when relevant

---

## Tracking Performance

### What to Track

| Metric | Why |
|--------|-----|
| Reach from hashtags | Direct hashtag performance |
| Engagement rate | Content + hashtag combo |
| Follower growth | Long-term hashtag ROI |
| Top hashtags | Double down on winners |

### Tools

- **Instagram:** Native Insights (reach from hashtags)
- **TikTok:** Creator Tools (video analytics)
- **General:** Later, Sprout Social, Metricool

### Weekly Review

1. Check which sets drove most reach
2. Note any underperforming hashtags
3. Look for new trending hashtags
4. Update sets accordingly
5. Document in content calendar

---

## Common Mistakes

1. **Using same hashtags every post** - Looks spammy, reduces reach
2. **Only using large hashtags** - Too competitive, posts get buried
3. **Irrelevant hashtags** - Hurts algorithm trust
4. **Banned hashtags** - Can limit reach entirely
5. **Too many on Twitter** - Kills engagement
6. **Not tracking** - Can't improve what you don't measure
