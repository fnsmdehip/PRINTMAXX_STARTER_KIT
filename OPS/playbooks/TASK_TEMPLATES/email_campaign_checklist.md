# Email Campaign Checklist

**Use for:** Newsletter sends, promotional campaigns, sequences
**Time estimate:** 1-2 hours per email

---

## Pre-Writing (10 min)

### Define Purpose
- [ ] Campaign goal clear (sales, engagement, traffic, nurture)
- [ ] Target segment identified
- [ ] Single CTA decided
- [ ] Success metric defined

### Context
- [ ] Previous email performance reviewed
- [ ] Audience's current state considered
- [ ] Timing appropriate (not holiday, major event)
- [ ] Follows logically from last communication

---

## Writing (30-60 min)

### Subject Line
- [ ] Under 50 characters
- [ ] Specific benefit or curiosity hook
- [ ] No spam trigger words (FREE, ACT NOW, etc.)
- [ ] Tested: Would YOU open this?
- [ ] A/B test version ready (if supported)

### Preview Text
- [ ] Complements subject line (not repeats)
- [ ] Under 90 characters
- [ ] Adds context or intrigue

### Email Body

**Opening:**
- [ ] First line hooks immediately
- [ ] No "I hope this finds you well"
- [ ] Gets to point in first 2 sentences
- [ ] Personal tone (like texting friend)

**Middle:**
- [ ] One clear idea/topic
- [ ] Short paragraphs (1-3 sentences)
- [ ] Benefits over features
- [ ] Specific examples or stories
- [ ] Scannable format

**Closing:**
- [ ] Clear single CTA
- [ ] CTA button AND text link
- [ ] No desperate urgency (unless genuine)
- [ ] Sign-off matches brand voice

**Optional elements:**
- [ ] PS line (only if adds value)
- [ ] Social proof (testimonial, results)
- [ ] Image (only if enhances message)

---

## Quality Check (15 min)

### Copy Review
- [ ] Zero AI vocabulary (leverage, unlock, etc.)
- [ ] No em dashes
- [ ] Direct, specific language
- [ ] One hedge per sentence max
- [ ] Would PRINTMAXXER send this?

### Technical Review
- [ ] All links work (clicked each one)
- [ ] Link tracking/UTMs added
- [ ] Images load with alt text
- [ ] Unsubscribe link present
- [ ] Sender name/email correct
- [ ] Reply-to address monitored

### Compliance Review
- [ ] CAN-SPAM compliant (physical address, unsubscribe)
- [ ] Affiliate disclosures if applicable
- [ ] No false/misleading claims
- [ ] GDPR compliant if applicable

### Deliverability Check
- [ ] Not too many images
- [ ] No spam trigger words
- [ ] Tested to multiple email providers
- [ ] List is clean and engaged

---

## Pre-Send (10 min)

### Final Checks
- [ ] Preview in desktop view
- [ ] Preview in mobile view
- [ ] Send test to yourself
- [ ] Check test in spam folder
- [ ] Verify segment/list correct
- [ ] Confirm scheduled time

### Scheduling
- [ ] Time optimized for audience
- [ ] Timezone considered
- [ ] Not conflicting with other campaigns
- [ ] Backup send time if needed

---

## Post-Send (5-30 min)

### Immediate (5 min)
- [ ] Confirm send completed
- [ ] Check for bounce alerts
- [ ] Monitor replies (first hour)

### 24 Hours Later
- [ ] Check open rate
- [ ] Check click rate
- [ ] Review replies
- [ ] Note any unsubscribes

### 48-72 Hours Later
- [ ] Final metrics capture
- [ ] Update tracking sheet
- [ ] Archive for reference
- [ ] Document lessons learned

---

## Email Types Quick Reference

### Newsletter
- **Goal:** Engagement, trust
- **Length:** 300-500 words
- **CTA:** Read more, reply, share
- **Frequency:** Weekly or bi-weekly

### Promotional
- **Goal:** Sales, signups
- **Length:** 200-400 words
- **CTA:** Buy, sign up, book
- **Frequency:** As needed, not too often

### Nurture Sequence
- **Goal:** Build relationship
- **Length:** 200-300 words
- **CTA:** Varies by stage
- **Frequency:** Every 2-3 days

### Welcome Sequence
- **Goal:** Onboard, set expectations
- **Length:** 200-400 words
- **CTA:** Get started, reply
- **Frequency:** Days 0, 1, 3, 5, 7

### Re-engagement
- **Goal:** Win back inactive
- **Length:** 100-200 words
- **CTA:** Confirm interest, update
- **Frequency:** After 60-90 days inactive

## Email Metrics Benchmarks

| Metric | Poor | Average | Good | Great |
|--------|------|---------|------|-------|
| Open Rate | <15% | 20-25% | 30-40% | >40% |
| Click Rate | <1% | 2-3% | 4-5% | >5% |
| Reply Rate | <0.5% | 1-2% | 3-5% | >5% |
| Unsubscribe | >0.5% | 0.2-0.3% | <0.2% | <0.1% |

## Subject Line Formulas

**Number-based:**
- "3 mistakes killing your [result]"
- "I made $X with [method]"
- "5 minutes to [benefit]"

**Question-based:**
- "Are you making this [topic] mistake?"
- "What if you could [outcome]?"
- "Why do most [audience] fail at [thing]?"

**Direct benefit:**
- "How to [achieve result]"
- "The [adjective] way to [outcome]"
- "[Outcome] without [pain point]"

**Curiosity:**
- "I was wrong about [topic]"
- "This surprised me"
- "The [topic] nobody talks about"

**Avoid:**
- ALL CAPS
- Excessive punctuation!!!
- Emoji-only subjects
- "Quick question" (overused)
- "Hey" (too casual for cold)
