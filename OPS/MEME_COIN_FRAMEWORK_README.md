## Risk Management Rules

### Position Sizing

**Maximum per coin:**
- PAT001 (AI Trend Viral): 1%
- PAT002 (Viral Fitness): 2%
- PAT003 (AI Agent Platform): 0.5%
- PAT004 (AI Agent Original): 2%
- PAT005 (AI Agent Creative): 1.5%
- PAT006 (AI Agent Absurdist): 1%
- PAT007 (AI Agent Social): 1.5%

**Portfolio limits:**
- Max 2% per coin
- Max 10% total across all meme coins
- Max 3 active positions simultaneously

### Entry Rules

**Pre-entry checklist:**
- [ ] Pattern matches PAT001-PAT007
- [ ] Entry window IMMEDIATE or CLOSING (not MISSED)
- [ ] Confidence MEDIUM or HIGH (not LOW on solo)
- [ ] Holder count >200 in first hour
- [ ] Liquidity >$50K
- [ ] Top 10 holders <40% supply
- [ ] No obvious rug pull red flags

### Exit Rules

**Exit discipline (non-negotiable):**
1. **Ladder exits:** Never sell entire stack at once
2. **Time-based stops:** Each pattern has max hold time
3. **Trailing stops:** Set once 3x achieved
4. **Rug detection:** Dev wallet activity → exit immediately
5. **Pattern failure:** No 2x in expected window → exit

### Drawdown Stops

- Portfolio down 30% → stop new entries
- Portfolio down 50% → exit all positions
- 30-day cooldown after 50% drawdown

## Current Limitations

### Data Limitations

1. **Survivorship Bias:** Only successful coins analyzed. For every success, 50+ failures.
2. **Small Sample Size:** 8 coins total, patterns have 1-2 examples each.
3. **Recent Data:** October 2024 - February 2026 only. Short timeframe.
4. **Meta Dependency:** AI agent meta was hot. May not repeat.

### Pattern Limitations

1. **Category Creators Hard to Spot:** PAT004 requires identifying "first mover" (subjective)
2. **Quality Subjective:** PAT005 requires assessing AI content quality
3. **High-Profile Endorsements:** PAT001/PAT003 depend on unpredictable events
4. **Whale Manipulation:** PAT003 has extreme manipulation risk

### System Limitations

1. **Manual Social Monitoring:** Script doesn't auto-scrape Twitter/Reddit (requires manual context input)
2. **No Live Price Data:** No integration with DEX APIs for real-time tracking
3. **No Automated Trading:** Script only scores signals, doesn't execute trades
4. **No Contract Analysis:** Doesn't scan for rug pull code patterns

## Key Insights from Backtest

### Timing is Everything

- AI Trend Viral peaks in 19h (enter first 12h)
- Viral Fitness peaks in 61h (enter first 36h)
- AI Agent Platform peaks in 32h (enter first 18h)
- AI Agent Original peaks in 132h (enter first 48h)

**Miss the entry window → pass on the trade.**

### Social Proof Matters

High-profile mentions (Elon Musk, Marc Andreessen, Naval Ravikant) correlate with 200-7,000% ROI. Anonymous projects with no social proof = extreme rug risk.

### Pattern Types

- **Event-Driven** (PAT001, PAT002): Require external catalyst (AI release, viral video)
- **Platform-Driven** (PAT003, PAT007): Require platform launch + endorsement
- **Category-Driven** (PAT004): Require first-mover identification
- **Creative-Driven** (PAT005, PAT006): Require quality/personality assessment

Event-driven patterns are most objective. Creative-driven are most subjective.

### Exit Discipline

All successful cases required EXITING before the crash:
- MOLT crashed 75% from peak
- Most coins lose 80-90% from peak within 7 days
- Trailing stops essential (50-60% from peak)

**Greed kills. Exit targets are non-negotiable.**

## Critical Warnings

1. **This is speculative gambling, not investing.** Only use capital you can afford to lose entirely.

2. **100% win rate is survivorship bias.** We only studied successful coins. Real win rate unknown (likely 20-40%).

3. **Small sample sizes.** Most patterns have 1 example. Not statistically significant.

4. **Meta can die.** AI agent meta was hot Oct 2024 - Jan 2026. May cool off.

5. **Regulation risk.** SEC/CFTC may crack down on meme coins anytime.

6. **Rug pulls are common.** Most new meme coins are scams. Check liquidity locks.

7. **Whale manipulation.** PAT003 especially prone to coordinated pumps/dumps.

8. **Psychological warfare.** FOMO and greed will tempt you to break rules. Discipline = survival.
