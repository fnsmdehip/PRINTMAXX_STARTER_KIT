### 2.4 Backend-as-a-Service

**Supabase** (Open Source)
- Firebase alternative built on PostgreSQL
- Auth, database, storage, real-time
- Auto-generated REST/GraphQL APIs
- Row-Level Security (RLS)
- **Clone opportunity:** Build apps faster with managed backend

**Appwrite** (50K+ stars, Open Source)
- Multi-language SDK support
- JavaScript, Python, PHP, Dart, Ruby, Go
- Self-host or cloud
- Built-in messaging (SMS/email/push)
- **Clone opportunity:** Mobile/web apps with custom backends

**Neon** (Serverless Postgres)
- Acquired by Databricks for ~$1B
- Compute/storage separation
- Database branching (like git)
- 80%+ databases created by AI agents
- **Clone opportunity:** AI-first database applications

## Section 4: MCP Servers and Claude Tools

### 4.1 Official MCP Resources

**GitHub's Official MCP Server**
- URL: https://github.com/github/github-mcp-server
- Connect AI tools to GitHub context
- Compatible with VS Code 1.101+, Claude Desktop, Cursor, Windsurf
- Toolsets flag for selective functionality
- **Use case:** GitHub integration for AI workflows

**Model Context Protocol Servers (Anthropic)**
- URL: https://github.com/modelcontextprotocol/servers
- Pre-built servers for:
  - Google Drive
  - Slack
  - GitHub
  - Git
  - Postgres
  - Puppeteer
- **Use case:** Enterprise system integrations

### 4.2 Community MCP Servers

**claude-code-mcp**
- URL: https://github.com/steipete/claude-code-mcp
- Run Claude Code in one-shot mode
- Permissions bypassed automatically
- Agent-in-agent architecture
- **Use case:** Nested agent workflows

**n8n-MCP**
- URL: https://github.com/czlonkowski/n8n-mcp
- 1,084 n8n nodes documentation
- 537 core + 547 community nodes
- Works with Claude Desktop/Code/Windsurf/Cursor
- **Use case:** AI-assisted workflow automation

**ClaudeMCP.com**
- Community hub for MCP
- Server directory
- Client information
- Documentation
- **Use case:** Discovery of MCP ecosystem
