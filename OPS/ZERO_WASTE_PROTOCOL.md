```

    ↓

    ↓
2. CONTENT TEASER → Social post concepts (partial sauce, withhold full method)
    ↓
3. REPLY BAIT / DM FUNNEL → "DM me [KEYWORD] for full breakdown"
    ↓
4. INFO PRODUCT → Gumroad/course concept ($7-47 price point)
    ↓
5. HIGH-TICKET SERVICE → Done-for-you implementation ($500-2K)
    ↓
6. ONE-ON-ONE CONSULTING → Personal walkthrough ($200-500/hr)
```

When you discover alpha, TAG with:
- `applicable_methods`: Which MM/CF/AI methods can use this?
- `applicable_niches`: Faith/fitness/tech angles?
- `content_concepts`: Post/product/newsletter angles (list ideas, don't write full content)
- `cross_pollination_tags`: Synergies with existing alpha?
- `product_opportunity`: Gumroad/course/service concept?
- `repurposing_map`: Twitter → Medium → Newsletter → Gumroad → High-ticket flow
- `implementation_priority`: NOW / SOON / BACKLOG

## Reply bait formula

```
HOOK: "[Surprising result]. here's what I found:"
BODY: 3-4 bullet insights (enough sauce, not enough to fully replicate)
CTA: "reply 'FUNNEL' and I'll send you the full breakdown"
    OR "full 50-page breakdown on my Gumroad (link in bio)"
    OR "thread below" → self-reply → final tweet = product CTA
```

Creates 3 things simultaneously:
1. Algorithm signal (replies boost reach 10x vs likes)
2. Lead qualification (repliers = warm leads for DM funnel)
3. Content distribution (every reply = notification to replier's followers)

## Bot detection + earnings skepticism (mandatory)

**Engagement check:**
- 10K likes with 3 comments = botted
- Only generic comments ("fire", emojis) = suspicious
- Account age vs follower count way off = bought followers
- Mark: `engagement_authenticity: SUSPICIOUS`

**Earnings check:**
- Round numbers ($50K vs $47,382) = likely inflated
- Selling to audience = extra skeptical
- No screenshot = unverified
- Mark: `earnings_verified: FALSE`

**STILL extract the method even if numbers are BS.** Tactic may be real even if proof isn't.
