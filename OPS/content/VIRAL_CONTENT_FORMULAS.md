## Core Principles

### The 3-Second Rule

70% of viewers decide to watch or scroll within 3 seconds. Your hook must:
- Stop the scroll immediately
- Create curiosity or tension
- Promise specific value

### Retention Over Reach

In 2026, all major platforms prioritize:
- **Watch time** > views
- **Completion rate** > impressions
- **Rewatches** > shares
- **Comments** > likes

1 person watching 3x > 3 people watching 1x.

### The Algorithm Formula

```
Viral Potential = (Hook Strength x Retention) + Engagement Signals

Where:
- Hook = First 3 seconds
- Retention = % who watch to end
- Engagement = Comments, saves, shares (weighted by platform)
```

## Hook Formulas

### Category 1: Curiosity Gap Hooks

**H001: The Forbidden Knowledge**
```
"Nobody is talking about this..."
"The industry doesn't want you to know..."
"This is borderline illegal how much this helps..."
```
*Why it works:* Creates exclusivity. Implies insider info.

**H002: The Wait For It**
```
"Wait for it..."
"Watch until the end..."
"The last one is insane..."
```
*Why it works:* Promises payoff. Boosts completion rate.

**H003: The Question Hook**
```
"Why is everyone suddenly doing X?"
"What happens when you do X?"
"Can you spot what's wrong here?"
```
*Why it works:* Triggers mental engagement. Viewers want answers.

### Category 2: Contrarian/Controversy Hooks

**H004: The Hot Take**
```
"Unpopular opinion: [statement]"
"This will make people angry but..."
"Everyone is doing X wrong..."
```
*Why it works:* Forces agreement/disagreement. Drives comments.

**H005: The Stop/Start**

*Why it works:* Direct command. Creates urgency.

**H006: The Myth Bust**

*Why it works:* Challenges existing beliefs. Triggers curiosity.

## Content Structure Patterns

### Pattern 1: Problem-Agitate-Solve (PAS)

**Structure:**
```
0-3s:  HOOK (identify problem)
3-10s: AGITATE (make problem feel painful)
10-20s: SOLVE (present solution)
20-25s: RESULT (show outcome)
25-30s: CTA
```

**Example:**
```
HOOK: "Stop wasting 5 hours on content every week"
AGITATE: "You're stuck in a content hamster wheel. Create. Post. Repeat. And still no results."
SOLVE: "Here's the 30-minute system that replaced my entire workflow..."
RESULT: "Now I batch 2 weeks of content in one session"
CTA: "Comment SYSTEM for the full breakdown"
```

### Pattern 2: Before/After

**Structure:**
```
0-3s:  BEFORE state (relatable pain)
3-8s:  THE SHIFT (what changed)
8-20s: AFTER state (desirable outcome)
20-25s: HOW (brief explanation)
25-30s: CTA
```

### Pattern 3: Listicle Speed

**Structure:**
```
0-2s:  HOOK ("3 things you need...")
2-8s:  ITEM 1 (fastest point)
8-14s: ITEM 2 (build value)
14-20s: ITEM 3 (strongest point)
20-25s: BONUS (unexpected extra)
25-30s: CTA
```

*Tip:* Put your strongest point LAST. Viewers remember endings.

### Pattern 4: Tutorial Speed

**Structure:**
```
0-3s:  HOOK ("Copy this exact workflow")
3-8s:  STEP 1 (visual demonstration)
8-15s: STEP 2 (continue showing)
15-22s: STEP 3 (complete the task)
22-28s: RESULT (show finished outcome)
28-30s: CTA
```

### Pattern 5: Story Arc

**Structure:**
```
0-3s:  HOOK (tease the payoff)
3-10s: SETUP (context/situation)
10-20s: CONFLICT (what went wrong/challenge)
20-28s: RESOLUTION (how it ended)
28-30s: LESSON + CTA
```

### Pattern 6: Reaction/Commentary

**Structure:**
```
0-3s:  ORIGINAL CONTENT (show what you're reacting to)
3-8s:  INITIAL REACTION (emotional response)
8-20s: ANALYSIS (your take/explanation)
20-25s: CONCLUSION
25-30s: CTA (ask for their opinion)
```

### Pattern 7: Rewatch Loop

**Structure:**
```
0-3s:  HOOK (reference something "coming")
3-25s: CONTENT (build to payoff)
25-28s: PAYOFF (callback to hook)
28-30s: LOOP BACK (seamless transition to start)
```

*Why:* Rewatches count as multiple views. 70%+ completion required for viral 2026.

### Pattern 8: Hub Content (Carousels/Slides)

**Structure:**
```
Slide 1: HOOK (title + visual grab)
Slide 2: PROBLEM
Slides 3-8: SOLUTION STEPS
Slide 9: SUMMARY
Slide 10: CTA + BRAND
```

*Why:* Hub-and-spoke model. One comprehensive piece breaks into 10-20 shorts.

## Platform-Specific Tactics

### TikTok 2026

**Algorithm Priorities:**
1. Watch time (most important)
2. Completion rate (70%+ = viral potential)
3. Rewatches
4. Shares
5. Comments
6. Saves

**Optimal Video Length:**
- Tutorials: 50-60 seconds (76% watch-through rate)
- Entertainment: 15-30 seconds
- Story: 30-45 seconds

**Posting Strategy:**
- New accounts: 3-5 posts/day for first 2 weeks
- Established: 1-3 quality posts/day
- Best times: 6-9 AM, 12-2 PM, 7-10 PM (local time)

**Critical Tactics:**
1. **Reply videos:** Respond to comments with new videos. Simplest growth lever in 2026.
2. **Text overlays with keywords:** TikTok AI reads text. Include searchable terms on screen.
3. **First 60 minutes:** Reply to EVERY comment in first hour. Signals active creator.
4. **Follower-first testing:** New content shown to followers for 2-3 days before wider push.
5. **Use search:** TikTok is a search engine now. Optimize titles for keywords people search.

**Hashtag Strategy:**
- 3-5 relevant hashtags max
- Mix: 1 broad (1M+ posts), 2 medium (100K-1M), 2 niche (<100K)
- Include trending hashtag if genuinely relevant

### X/Twitter 2026

**Algorithm Priorities (from X algorithm release):**
1. Replies (HIGHEST weight - indicates conversation)
2. Quote tweets (HIGH - thoughtful engagement)
3. Reposts (HIGH - distribution signal)
4. Dwell time (HIGH - content is compelling)
5. Profile clicks (HIGH - interest in author)
6. Likes (MEDIUM - low-effort engagement)

**Post Optimization:**
- Get Twitter Blue/Premium (2-3x impression boost)
- Space posts 2-4 hours apart (author diversity scoring)
- Optimize for replies > likes
- Drive profile clicks with bio tease

**Best Performing Formats:**
1. Single insight with number
2. Threads (5-7 tweets max)
3. Screenshot/image with context
4. Polls (engagement bait)
5. Quote tweets of trending content

**Posting Times:**
- Weekdays: 8-10 AM, 12-1 PM, 5-6 PM EST
- Weekends: 9-11 AM EST
- Avoid: 3-4 AM, late night

**Thread Structure:**
```
Tweet 1: Hook (strongest claim + thread indicator)
Tweets 2-6: Value points (one idea per tweet)
Tweet 7: Summary + CTA
```

**Self-Reply Funnel:**
```
Main tweet: Hook with value
Self-reply: CTA with link/product
```
*Why:* First reply often most visible. Proven funnel structure.

### YouTube Shorts 2026

**Algorithm Priorities:**
1. Watch time
2. Engagement (comments, likes, shares)
3. Subscriber conversion
4. CTR from thumbnails (yes, they matter now)

**Optimal Length:**
- 50-60 seconds = 76% average watch-through rate
- Under 30 seconds for pure entertainment

**Critical Tactics:**
1. **Trending audio:** Use in first 5 seconds = 21% reach increase
2. **Post volume:** 18-22 Shorts monthly = top creator benchmark
3. **Titles matter:** Include keywords people search
4. **Loop content:** Videos that seamlessly loop get rewatch bonus

**Thumbnail (Yes, for Shorts):**
- YouTube now shows Shorts thumbnails in search
- Clear, readable text overlay
- Consistent branding

### Instagram Reels 2026

**Algorithm Priorities:**
1. Shares (to DMs and Stories)
2. Saves
3. Comments
4. Watch time
5. Likes

**Critical Differences from TikTok:**
- Shares to DMs weighted heavily
- Cover image matters more (grid aesthetic)
- Longer captions acceptable
- Hashtags in first comment works

**Posting Strategy:**
- 5-7 Reels per week
- Best times: 11 AM-1 PM, 7-9 PM
- Tuesday-Friday perform best

**Cover Image:**
- Custom cover with text overlay
- Match grid aesthetic
- Clear value proposition visible

**Caption Strategy:**
- Hook in first line
- Value or story in middle
- CTA at end
- Hashtags in first comment (debated, test both)

## Engagement Triggers

### Trigger 1: Controversy (Without Being Offensive)

**Safe Controversy Topics:**
- Industry debates (tools, methods, opinions)
- "Unpopular opinion" takes on common practices
- "This vs That" comparisons
- Challenging conventional wisdom

**Avoid:**
- Politics (unless that's your niche)
- Personal attacks
- Sensitive social issues

**Example:**
```
"Unpopular opinion: Cold email is dead in 2026"
```
Invites debate. Not offensive. Drives comments.

### Trigger 2: FOMO (Fear of Missing Out)

**Patterns:**
- "While you're scrolling, others are..."
- "The window for X is closing"
- "This won't work forever"
- "Already 500 people have..."

### Trigger 3: Curiosity Gaps

**Patterns:**
- "The result surprised me..."
- "What happened next changed everything"
- "Most people miss this part..."
- "Watch until the end to see..."

### Trigger 4: Relatability

**Patterns:**
- "POV: [common experience]"
- "Tell me you're a [X] without telling me"
- "This is so [profession/identity]"
- "When you [common situation]..."

### Trigger 5: Aspiration

**Patterns:**
- "This is your sign to..."
- "The version of you that wins..."

### Trigger 6: Social Proof

**Patterns:**
- Numbers: followers, revenue, results

## A/B Testing Framework

### The A/B Post Strategy (Proven by @aftermagics)

**How it works:**
Post the same topic twice with opposite takes. Track which performs better.

**Example:**
```
Post A: "start using ai to write your replies" -> 1K views
Post B: "stop using ai to write your replies" -> 1.9K views
```
Contrarian "stop" got 2x engagement.

### What to Test

**Hook Variables:**
- Affirmative vs negative framing
- Question vs statement
- Number vs no number
- "You" vs "I" perspective

**Format Variables:**
- Video length (15s vs 30s vs 60s)
- Text overlay style
- With vs without music
- Face vs no face

**CTA Variables:**
- Comment trigger word
- DM vs Comment vs Bio
- Specific vs generic

### Testing Cadence

**Weekly:**
- 2-3 hook variations per topic
- Track: views, completion rate, engagement rate

**Monthly:**
- Review top 20% performing content
- Identify patterns
- Double down on winners

### Metrics to Track

| Metric | Target | Platform |
|--------|--------|----------|
| Completion rate | >70% | TikTok, Reels |
| Engagement rate | >5% | All |
| Save rate | >2% | Reels, TikTok |
| Comment rate | >1% | All |
| Share rate | >0.5% | All |
| Profile click rate | >3% | X |

**Weekly Sprint (2-3 hours):**

3. Write 10 hook variations per theme
