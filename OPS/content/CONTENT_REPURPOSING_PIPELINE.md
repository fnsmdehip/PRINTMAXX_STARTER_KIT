# Content Repurposing Pipeline

**Purpose:** Transform ONE piece of content into 7+ platform-specific assets.
**Time to execute:** 45-60 minutes per piece (batch for efficiency)
**ROI:** 1 content piece → 7x distribution → 7x potential reach

---

## The Core Process

```
Source Content (Twitter viral, article, insight)
    ↓
Stage 1: Twitter Thread (native expansion)
    ↓
Stage 2: LinkedIn Post (professional reframe)
    ↓
Stage 3: TikTok/Reels Script (short-form video)
    ↓
Stage 4: YouTube Shorts Script (educational angle)
    ↓
Stage 5: Newsletter Section (deep dive)
    ↓
Stage 6: Blog Post Outline (SEO play)
    ↓
Stage 7: Carousel/Infographic (visual learners)
```

---

## Stage 1: Twitter Thread Template

**Format:** 7-tweet thread (hook + 5 value + CTA)

### Tweet 1 - The Hook
```
[CONSEQUENCE/RESULT] in [TIMEFRAME].

here's the exact [METHOD/SYSTEM/FRAMEWORK]:
```

**Example:**
```
408K views on a single thread about AI agents.

here's the exact system I used to write it:
```

### Tweet 2 - The Problem
```
most people [COMMON MISTAKE].

they [WHAT THEY DO WRONG].

but [COUNTER-INTUITIVE TRUTH]:
```

### Tweet 3 - The Framework (Part 1)
```
step 1: [ACTION]

[1-2 sentence explanation]

[specific tool/method if applicable]
```

### Tweet 4 - The Framework (Part 2)
```
step 2: [ACTION]

[why this matters]

[proof point or example]
```

### Tweet 5 - The Framework (Part 3)
```
step 3: [ACTION]

this is where most people stop.

but [WHAT SEPARATES WINNERS]:
```

### Tweet 6 - The Result/Proof
```
I used this to [SPECIFIC RESULT].

[NUMBER] [METRIC] in [TIMEFRAME].

[optional: screenshot or link]
```

### Tweet 7 - CTA
```
want the full breakdown?

[SOFT CTA - follow, like, bookmark]

building more of these. follow if you want to see them.
```

## Stage 3: TikTok/Reels Script Template

**Format:** 30-60 seconds, hook in first 3 seconds

### Script Structure
```
[SECOND 0-3: HOOK]
"[CONTROVERSIAL STATEMENT OR RESULT]"

[SECOND 3-10: CONTEXT]
"here's what I mean..."
[Quick explanation of why viewer should care]

[SECOND 10-30: THE VALUE]
"[step 1/insight 1]"
[B-ROLL: screen recording, text overlay, or demonstration]

"[step 2/insight 2]"
[B-ROLL: continuation]

"[step 3/insight 3]"
[B-ROLL: final demonstration]

[SECOND 30-45: THE RESULT]
"when I did this, [SPECIFIC OUTCOME]"
[Show proof if possible]

[SECOND 45-60: CTA]
"follow for more [NICHE] content"
[or "comment [WORD] and I'll send you the full guide"]
```

### B-Roll Suggestions
- Screen recordings of tools/process
- Text overlays with key points
- Quick cuts between face and content
- "POV" style content
- Before/after demonstrations

## Stage 4: YouTube Shorts Script Template

**Format:** 45-60 seconds, more educational than TikTok

### Script Structure
```
[SECOND 0-5: HOOK + PROMISE]
"[NUMBER] [THINGS/STEPS/SECRETS] that [RESULT]"
"by the end of this video, you'll know exactly how to [OUTCOME]"

[SECOND 5-15: CONTEXT/CREDIBILITY]
"I [CREDENTIAL/PROOF POINT]"
"and I'm going to show you [WHAT THEY'LL LEARN]"

[SECOND 15-45: THE CONTENT]
"number one: [POINT]"
[10 second explanation with visual]

"number two: [POINT]"
[10 second explanation with visual]

"number three: [POINT]"
[10 second explanation with visual]

[SECOND 45-60: CTA + LOOP]
"if this helped, subscribe for more"
"and watch this next video about [RELATED TOPIC]"
[Point to end screen]
```

### YouTube Shorts Best Practices
- Vertical format (9:16)
- Text on screen for key points
- Loop-friendly endings
- End screen with subscribe CTA
- First frame = thumbnail (make it count)

**Format:** 200-300 words, actionable, personal

### Structure
```
## [CATCHY SECTION TITLE]

[PERSONAL HOOK - 1-2 sentences about why you're sharing this]

[THE INSIGHT - 2-3 sentences explaining the core concept]

**Here's what I learned:**

- [BULLET 1 - Most important insight]
- [BULLET 2 - Practical application]
- [BULLET 3 - Counter-intuitive element]

**The actionable part:**

[2-3 sentences on exactly what reader should do with this information]

[OPTIONAL: Link to full resource, tool, or related content]

---
```

### Newsletter CTA Options
- "Reply and tell me: [QUESTION]"
- "Try this and let me know how it goes"
- "Forward this to someone who needs it"
- Link to deeper content (blog, video, product)

## Stage 7: Carousel/Infographic Template

**Format:** 5-10 slides, visual-first

### Slide Structure
```
SLIDE 1 - COVER
[BOLD STATEMENT or QUESTION]
[Your handle/name]
[Optional: "Swipe →" indicator]

SLIDE 2 - THE PROBLEM
[Relatable pain point]
[Simple visual or icon]
"Sound familiar?"

SLIDE 3 - THE SOLUTION INTRO
"Here's what actually works:"
[Transition to content]

SLIDES 4-7 - THE CONTENT
[One point per slide]
[Numbered: 1, 2, 3, 4]
[Icon or simple visual for each]
[2-3 lines of text max]

SLIDE 8 - SUMMARY
"Quick recap:"
[Bullet points of all main ideas]

SLIDE 9 - CTA
"Save this for later"
"Follow @[handle] for more"
"Share with someone who needs this"
```

### Design Specs
- **Size:** 1080x1350 (Instagram) or 1080x1920 (Stories/TikTok)
- **Font:** Bold, readable sans-serif
- **Colors:** 2-3 max, high contrast
- **Template tools:** Canva, Figma, or Photoshop templates

**Weekly content batching:**
1. Monday: Identify 3-5 source pieces
2. Tuesday: Create all Twitter threads
3. Wednesday: Adapt all LinkedIn posts
4. Thursday: Write all video scripts
5. Friday: Draft newsletter sections + blog outlines
