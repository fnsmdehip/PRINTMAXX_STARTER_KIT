# Content Repurposing Guide

## The 1:20 Principle

One piece of pillar content becomes 20+ pieces across platforms.

```
1 Long-Form Video (YouTube)
├── 5 Short clips (TikTok/Reels/Shorts)
├── 1 Blog post (SEO)
├── 1 Newsletter issue
├── 3 Twitter threads
├── 5 Twitter posts
├── 3 LinkedIn posts
├── 2 Instagram carousels
└── Podcast episode (audio)
```

## Platform Adaptation

### YouTube → TikTok/Reels/Shorts

**What Changes:**
- Duration: 60 sec max (30-45 optimal)
- Format: Vertical (9:16)
- Pace: Faster, hook in 1-3 seconds
- Captions: Always (80% watch muted)
- Music: Trending sounds help

**Process:**
1. Identify best moments from long video
2. Export vertical crop or reframe
3. Add captions (auto or manual)
4. Add trending sound (optional)
5. Hook text overlay at start

### Blog → Twitter Thread

**What Changes:**
- Length: Each tweet under 280 chars
- Format: Numbered, scannable
- Hook: First tweet must stop scroll
- CTA: Last tweet drives action

**Process:**
1. Extract 5-10 key points
2. Write hook tweet (promise/curiosity)
3. Each point = one tweet
4. Add examples/data where possible
5. End with CTA or summary

**Thread Structure:**
```
1/ [Hook - bold claim or question]
2/ [Context - why this matters]
3/ [Point 1]
4/ [Point 2]
5/ [Point 3]
...
10/ [Summary + CTA]
```

### Blog → LinkedIn Post

**What Changes:**
- Tone: More professional
- Format: Line breaks every 1-2 sentences
- Length: 1300 chars optimal
- Hook: First line critical (before "see more")

**Process:**
1. Extract one key insight
2. Write hook (bold statement or story)
3. Expand with 3-5 supporting points
4. Add personal take/experience
5. End with question for engagement

### Blog → Instagram Carousel

**What Changes:**
- Visual: Heavy on design
- Slides: 5-10 per carousel
- Text: Minimal per slide
- Hook: First slide = cover

**Process:**
1. Extract key takeaways
2. Design cover slide (hook)
3. One point per slide
4. Simple visuals, large text
5. Last slide = CTA

**Carousel Structure:**
```
Slide 1: Hook/Title (bold, curiosity)
Slide 2: Problem/Context
Slides 3-8: Key points (one each)
Slide 9: Summary
Slide 10: CTA (Follow, Save, Share)
```

### Video → Podcast Episode

**What Changes:**
- Remove visual references
- May need intro/outro
- Can keep or edit for audio flow

**Process:**
1. Extract audio from video
2. Add podcast intro/outro
3. Clean up audio (noise reduction)
4. Add chapter markers
5. Publish to podcast platforms

## Repurposing Workflows

### Weekly Content Workflow

**Monday: Create Pillar**
- Record long-form video OR write blog post
- This is your "content bank" for the week

**Tuesday: Transform**
- Video → Extract 3-5 short clips
- Blog → Write Twitter thread

**Wednesday: Adapt**
- Thread → LinkedIn version
- Blog → Instagram carousel

**Thursday: Distribute**
- Schedule all content across platforms
- Cross-link where appropriate

**Friday: Engage**
- Respond to comments
- Identify what performed well
- Note ideas for next week

### Batch Processing Workflow

**Record Day:**
- Film 4 videos in one session
- Same setup, different topics
- Bank content for the month

**Edit Day:**
- Edit all long-form
- Export all shorts
- Create thumbnails

**Write Day:**
- All blog posts
- All threads
- All captions

**Design Day:**
- All carousels
- All graphics
- All thumbnails

**Schedule Day:**
- Load everything into scheduler
- Set optimal posting times
- Prepare engagement plan

## Repurposing by Content Type

### From Video Content

```
10-min YouTube Video
├── 3x 60-sec TikToks (best moments)
├── 5x 30-sec Reels (single tips)
├── 1x Blog post (transcript + edit)
├── 1x Newsletter (summary + insights)
├── 1x Twitter thread (key takeaways)
├── 3x Twitter clips (video posts)
├── 1x LinkedIn post (professional angle)
├── 1x LinkedIn video (30-sec clip)
├── 1x Instagram carousel (step-by-step)

```

```
2000-word Blog Post
├── 1x Twitter thread (main points)

├── 1x LinkedIn post (summary + take)
├── 1x Instagram carousel (visual guide)
├── 1x Newsletter feature
├── 1x YouTube video (talk through it)
├── 3x Short videos (individual points)
└── 1x Quora answer (adapt for question)
```

```
45-min Podcast Episode

├── 1x Blog post (transcript edit)
├── 1x Twitter thread (highlights)
├── 5x Quote graphics
├── 1x Newsletter (summary)
└── 3x LinkedIn posts (different angles)
```

## Quality Control

### Before Repurposing

- [ ] Original content is high quality
- [ ] Key points are clearly identified
- [ ] Evergreen vs. time-sensitive noted
- [ ] Platform-specific adaptations planned

### During Adaptation

- [ ] Hook optimized for each platform
- [ ] Length appropriate for format
- [ ] Visuals match platform aesthetic
- [ ] CTA makes sense in context

### Before Publishing

- [ ] No typos or errors
- [ ] Links work (where applicable)
- [ ] Captions/alt text added
- [ ] Scheduled at optimal times

## Advanced Repurposing

### Content Recycling

Bring back old content:
- Update with new data
- New hook, same core
- "One year later" follow-up
- Seasonal relevance

### Cross-Platform Chains

```
Twitter Thread (test)
→ If viral, expand to Blog
→ Blog → YouTube video
→ Video → All platforms
```

Test on fastest platform (Twitter), double down on winners.

### Evergreen Library

Build a library of content that can be reshared:
- Timeless tips
- Fundamental concepts
- Popular pieces

Schedule reshares of top performers every 3-6 months.

## Common Mistakes

1. **No adaptation** - Same content, different platform doesn't work
2. **Ignoring platform norms** - Each has different expectations
3. **No tracking** - Can't improve what you don't measure
4. **Over-repurposing** - Quality drops with too many variants
5. **Wrong sequencing** - Long-form first, then atomize
