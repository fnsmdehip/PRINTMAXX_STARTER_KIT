# Voiceover Script Templates

Templates for faceless videos, explainers, and AI voiceover content.

---

## Template 1: Faceless TikTok/Reels (30-60 sec)

**Style:** Fast-paced, punchy, visual-first

**Structure:**
```
[0-3s] HOOK - Stop scroll statement
[3-25s] CONTENT - Main points with visual support
[25-28s] TWIST - Unexpected insight
[28-30s] CTA
```

**Script Format:**

```
HOOK:
"[Bold statement or question]"
[Visual: Eye-catching related image]

CONTENT:
"[Point 1]"
[Visual: Supporting graphic/footage]

"[Point 2]"
[Visual: Supporting graphic/footage]

"[Point 3]"
[Visual: Supporting graphic/footage]

TWIST:
"But here's what most people miss: [insight]"
[Visual: Emphasis graphic]

CTA:
"Follow for more [niche] content."
[Visual: Logo or handle]
```

**Example (Business Facts):**

```
HOOK:
"Apple makes more money in 1 hour than you'll make in your lifetime."
[Visual: Apple logo, money graphics]

CONTENT:
"Their revenue: $394 billion per year."
[Visual: Revenue chart]

"That's $45 million every hour."
[Visual: Clock + money]

"$750,000 every minute."
[Visual: Counting animation]

TWIST:
"And they started in a garage in 1976."
[Visual: Old Apple garage photo]

CTA:
"Follow for more insane business facts."
```

## Template 2: Explainer Video (2-5 min)

**Style:** Educational, clear, professional

**Structure:**
```
[0-15s] INTRO - What and why
[15-45s] CONTEXT - Background needed
[45-3:00] EXPLANATION - Core content
[3:00-4:00] EXAMPLES - Real applications
[4:00-4:30] SUMMARY - Key points
[4:30-5:00] CTA
```

**Script Format:**

```
INTRO:
"Today we're breaking down [topic].
By the end of this video, you'll understand exactly how [outcome]."
[Visual: Title card]

CONTEXT:
"First, some context.
[Background information]
[Why this matters now]"
[Visual: Supporting graphics, timeline, or footage]

EXPLANATION:
"Here's how it works.
[Step 1 or concept 1]
[Visual: Diagram or demonstration]

[Step 2 or concept 2]
[Visual: Diagram or demonstration]

[Step 3 or concept 3]
[Visual: Diagram or demonstration]"

EXAMPLES:
"Let's look at real examples.
[Example 1 with visuals]
[Example 2 with visuals]"

SUMMARY:
"To recap:
[Key point 1]
[Key point 2]
[Key point 3]"
[Visual: Summary graphic]

CTA:
"If this helped, like and subscribe for more [topic] content."
```

## Template 3: Story/Documentary Style (5-10 min)

**Style:** Narrative, engaging, cinematic

**Structure:**
```
[0-30s] COLD OPEN - Dramatic hook
[30s-1:30] INTRO - Set the stage
[1:30-8:00] NARRATIVE - Chronological story
[8:00-9:00] ANALYSIS - What it means
[9:00-10:00] CLOSE
```

**Script Format:**

```
COLD OPEN:
"[Dramatic statement or moment]
[Pause]
But to understand how we got here, we need to go back to [beginning]."
[Visual: Dramatic imagery]

INTRO:
"This is the story of [subject].
From [starting point] to [ending point].
Here's what happened."
[Visual: Title sequence]

NARRATIVE:
"It all started in [time/place].
[Event 1]
[Visual: Period-appropriate footage/images]

[Event 2]
[Visual: Supporting footage]

[Event 3]
[Visual: Supporting footage]

Then, something changed.
[Turning point]
[Visual: Dramatic shift in visuals]

[Resolution]
[Visual: Outcome footage]"

ANALYSIS:
"Looking back, this story tells us [lesson].
[Broader implications]
[Why it matters today]"
[Visual: Modern context]

CLOSE:
"And that's the story of [subject].
Subscribe for more stories like this."
[Visual: End card]
```

## Template 4: Listicle/Ranking (3-7 min)

**Style:** Fast-paced, engaging, visual variety

**Structure:**
```
[0-20s] INTRO - What list, why watch
[20s-6:00] LIST ITEMS - Each with visual
[6:00-6:30] WINNER/SUMMARY
[6:30-7:00] CTA
```

**Script Format:**

```
INTRO:
"[Number] [things] that [outcome/description].
Number [first] will [tease]."
[Visual: Title card with list preview]

LIST ITEMS:
"Number [X]: [Item Name]
[Description]
[Why it's on the list]
[Visual: Item footage/images]"

[Repeat for each item, building to #1]

WINNER:
"And the number one [thing]: [Winner]
[Why it wins]
[Visual: Celebratory/emphasis graphics]"

CTA:
"Which is your favorite? Comment below.
Subscribe for more [category] rankings."
```

## Writing for AI Voiceover

### Best Practices

**Punctuation:**
- Use periods for natural pauses
- Use commas for brief pauses
- "..." for dramatic pauses
- No complex punctuation

**Sentence Structure:**
- Short sentences work best
- 10-15 words maximum
- One idea per sentence
- Active voice always

**Pronunciation:**
- Spell out numbers (three, not 3)
- Use phonetic spelling for tricky words
- Avoid abbreviations (write "versus" not "vs")
- Test proper nouns

**Pacing:**
- Fast content: fewer pauses
- Emotional content: more pauses
- Tutorial content: medium pace

### Tone Markers

Add in script notes:

```
[UPBEAT] "Here's the exciting part..."
[SERIOUS] "But there's a problem..."
[CURIOUS] "What if I told you..."
[DRAMATIC] "Everything changed..."
```

### Common AI Voice Issues

| Issue | Fix |
|-------|-----|
| Monotone | Add punctuation variety |
| Too fast | Add periods and pauses |
| Mispronunciation | Use phonetic spelling |
| Robotic | Shorter sentences |
| Unnatural emphasis | Restructure sentence |

## Visual Pairing Guidelines

### Match Words to Visuals

| Word Type | Visual Type |
|-----------|-------------|
| Statistics | Charts, numbers |
| Actions | B-roll footage |
| Concepts | Diagrams, icons |
| Emotions | Faces, reactions |
| Products | Product shots |
| Places | Location footage |

### Timing

- Visual should appear 0.5s before related VO
- Hold visuals 2-4 seconds minimum
- Change visuals on new topics
- Don't let visuals lag behind VO

### Visual Sources

| Source | Use For | Notes |
|--------|---------|-------|
| Pexels | B-roll | Free, commercial use |
| Unsplash | Images | Free, commercial use |
| Pixabay | Mixed | Free, commercial use |
| Envato | Premium | Paid, high quality |
| Storyblocks | Video | Subscription, unlimited |

## Music Selection

### By Content Type

| Content | Music Style |
|---------|-------------|
| Motivational | Orchestral build, inspiring |
| Educational | Soft ambient, lo-fi |
| Story/Documentary | Cinematic, emotional |
| Fast-paced facts | Upbeat electronic |
| Relaxation | Ambient, peaceful |

### Music Sources

- Epidemic Sound (subscription)
- Artlist (subscription)
- YouTube Audio Library (free)
- Uppbeat (free with credit)
- Pixabay Music (free)

### Volume Mixing

- Music: -18 to -24 dB under VO
- VO: -6 to -3 dB
- Music up during pauses
- Fade in/out, never cut

## Scripting Checklist

### Before Writing
- [ ] Define target audience
- [ ] Know the one key message
- [ ] Gather visual ideas
- [ ] Choose music direction

### While Writing
- [ ] Hook in first 3 seconds
- [ ] One idea per sentence
- [ ] Visual notes included
- [ ] Pronunciation guides added
- [ ] Pacing markers included

### After Writing
- [ ] Read aloud for timing
- [ ] Test in AI voice tool
- [ ] Adjust for natural flow
- [ ] Pair with visual plan
