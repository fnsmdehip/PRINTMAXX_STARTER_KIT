# Email List Launch Checklist

## Phase 1: Foundation (Week -2)

### Platform Selection

| Platform | Best For | Starting Price |
|----------|----------|----------------|
| ConvertKit | Creators, courses | Free to 1k |
| Beehiiv | Newsletters, growth | Free to 2.5k |
| Mailchimp | E-commerce, general | Free to 500 |
| Substack | Writers, paid subs | Free (10% on paid) |
| Buttondown | Developers, minimal | Free to 100 |

### Platform Setup
- [ ] Account created
- [ ] Domain authenticated (SPF, DKIM, DMARC)
- [ ] Sending address configured
- [ ] Reply-to address set
- [ ] Logo/branding uploaded

### Technical
- [ ] SPF record added to DNS
- [ ] DKIM record added to DNS
- [ ] DMARC policy configured
- [ ] Test email delivered to inbox (not spam)
- [ ] Unsubscribe link working

## Phase 2: Lead Magnet (Week -2 to -1)

### Lead Magnet Creation
- [ ] Specific problem identified
- [ ] Format chosen (PDF, video, template, tool)
- [ ] Content created
- [ ] Professionally formatted
- [ ] Delivered automatically

### Lead Magnet Types (Pick One)

| Type | Best For | Conversion Rate |
|------|----------|-----------------|
| PDF Guide | How-to content | 30-50% |
| Checklist | Action-oriented | 40-60% |
| Template | Done-for-you | 35-55% |
| Mini-Course | Education | 25-40% |
| Free Tool | Interactive | 40-70% |
| Swipe File | Examples | 35-50% |

### Lead Magnet Delivery
- [ ] Automation created (signup → delivery)
- [ ] Download link working
- [ ] Delivery email copy written
- [ ] Fallback if automation fails

## Phase 3: Opt-In Setup (Week -1)

### Landing Page
- [ ] Clear headline (benefit-focused)
- [ ] Brief description of lead magnet
- [ ] Preview/mockup of what they get
- [ ] Simple form (email only if possible)
- [ ] Privacy statement
- [ ] Mobile responsive

### Form Placement Options
- [ ] Dedicated landing page
- [ ] Homepage section
- [ ] Blog sidebar
- [ ] Blog post footer
- [ ] Exit intent popup
- [ ] Scroll-triggered popup

### Opt-In Copy Formula
```
Headline: Get [Specific Result] in [Timeframe]
Subhead: [What they get] + [Why it's valuable]
CTA: [Action verb] + [What they get]
```

Example:
```
Headline: Double Your Email Open Rates in 7 Days
Subhead: Free checklist with 23 proven tactics used by top newsletters
CTA: Get the Free Checklist
```

## Phase 4: Welcome Sequence (Week -1)

### Essential Emails

**Email 1: Delivery (Immediate)**
- [ ] Lead magnet attached/linked
- [ ] Brief introduction
- [ ] What to expect from emails
- [ ] One quick win they can implement

**Email 2: Introduction (Day 1)**
- [ ] Your story/credibility
- [ ] Why you send emails
- [ ] What value they'll get
- [ ] Invite to reply

**Email 3: Quick Win (Day 3)**
- [ ] Actionable tip
- [ ] Related to lead magnet topic
- [ ] Easy to implement
- [ ] Builds trust

**Email 4: Resource/Tool (Day 5)**
- [ ] Share something useful
- [ ] Could be your content or curated
- [ ] Position as helpful
- [ ] Soft CTA

**Email 5: Engagement (Day 7)**
- [ ] Ask a question
- [ ] Get them to reply
- [ ] Segment based on response
- [ ] Build relationship

## Email Metrics to Track

| Metric | Good | Great | Action if Low |
|--------|------|-------|---------------|
| Open Rate | 30%+ | 45%+ | Better subject lines |
| Click Rate | 3%+ | 5%+ | Better CTAs, content |
| Unsubscribe Rate | <0.5% | <0.2% | Check frequency, relevance |
| Bounce Rate | <2% | <0.5% | Clean list |
| Reply Rate | 1%+ | 3%+ | Ask more questions |

## Subject Line Formulas

**Curiosity:**
- "I was wrong about [topic]"
- "This changed everything for me"
- "The [thing] nobody talks about"

**Benefit:**
- "How to [achieve result] in [timeframe]"
- "[Number] ways to [benefit]"
- "The fastest way to [result]"

**Personal:**
- "Quick question for you"
- "Can I ask you something?"
- "Thought you'd like this"

**Urgency:**
- "[Time] left to [action]"
- "Last chance: [offer]"
- "Don't miss this"

## Deliverability Best Practices

### Do:
- [ ] Authenticate domain (SPF, DKIM, DMARC)
- [ ] Use consistent sending address
- [ ] Include text version
- [ ] Make unsubscribe easy
- [ ] Send consistently
- [ ] Monitor metrics

### Don't:
- [ ] Buy email lists (ever)
- [ ] Use all caps or excessive punctuation
- [ ] Use spam trigger words excessively
- [ ] Send without permission
- [ ] Ignore bounce notifications
- [ ] Change sending address frequently

## Monetization Timeline

### 0-500 subscribers
- Focus on growth and engagement
- Test lead magnets
- Build relationship

### 500-2,000 subscribers
- Affiliate links (relevant products)
- Soft product mentions
- Build case studies

### 2,000-10,000 subscribers
- Sponsorships ($50-200 CPM)
- Digital products
- Premium tier option

### 10,000+ subscribers
- Multiple sponsors per issue
- Higher CPM rates
- Product launches
- Community/membership
