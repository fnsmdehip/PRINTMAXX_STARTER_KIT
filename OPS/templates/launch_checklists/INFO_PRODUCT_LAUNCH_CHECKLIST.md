# Info Product Launch Checklist

## Phase 1: Product Creation (Week -4 to -2)

### Validation
- [ ] Survey existing audience (if any)
- [ ] Find pain points in Reddit/Twitter discussions
- [ ] Check competitor products (Gumroad search, Udemy)
- [ ] Validate willingness to pay ($$ comments, course sales)
- [ ] Define specific transformation/outcome

### Content Planning
- [ ] Outline all modules/sections
- [ ] Define deliverables (videos, templates, worksheets)
- [ ] Estimate creation time per module
- [ ] Set realistic scope (2-4 modules for MVP)

### Format Decision

| Format | Best For | Pricing Range |
|--------|----------|---------------|
| PDF Guide | Quick wins, checklists | $19-47 |
| Video Course | Complex skills, processes | $47-297 |
| Templates/Swipe Files | Done-for-you assets | $27-97 |
| Community/Cohort | Accountability, networking | $297-997 |

---

## Phase 2: Content Creation (Week -2 to -1)

### Core Content
- [ ] Create Module 1 content
- [ ] Create Module 2 content
- [ ] Create Module 3 content
- [ ] Create bonus content (if applicable)
- [ ] Record videos (if video course)

### Production Quality
- [ ] Audio clean (no echo, background noise)
- [ ] Video 1080p minimum
- [ ] Consistent lighting/background
- [ ] Screen recordings clear and zoomed appropriately
- [ ] Edit out ums, pauses, mistakes

### Supplementary Materials
- [ ] Worksheets/templates created
- [ ] Checklists formatted
- [ ] Resource links compiled
- [ ] Bonus files organized

---

## Phase 3: Platform Setup (Week -1)

### Hosting Platform Choice

| Platform | Fee | Best For |
|----------|-----|----------|
| Gumroad | 10% | Quick launch, digital downloads |
| Teachable | $39+/mo | Video courses, drip content |
| Podia | $39+/mo | All-in-one, memberships |
| Kajabi | $149+/mo | Full business suite |
| Self-hosted | Varies | Maximum control |

### Platform Setup
- [ ] Account created
- [ ] Payment connected (Stripe/PayPal)
- [ ] Product created in platform
- [ ] Content uploaded
- [ ] Pricing set
- [ ] Checkout flow tested

### Sales Page
- [ ] Headline (specific outcome)
- [ ] Pain points addressed
- [ ] Transformation described
- [ ] Module breakdown listed
- [ ] Social proof (if available)
- [ ] Pricing with anchor
- [ ] FAQ section
- [ ] Guarantee (30-day minimum)
- [ ] CTA buttons (3+ on long pages)

---

## Phase 4: Pre-Launch (Week -1 to 0)

### Email Sequence
- [ ] Email 1: Story/problem (Day -7)
- [ ] Email 2: Solution tease (Day -5)
- [ ] Email 3: Social proof/case study (Day -3)
- [ ] Email 4: Launch announcement (Day 0)
- [ ] Email 5: Reminder (Day 2)
- [ ] Email 6: Last chance (Day 5)
- [ ] Email 7: Cart close (Day 7)

### Social Content
- [ ] Launch announcement post drafted
- [ ] Behind-the-scenes content prepared
- [ ] Testimonial graphics created
- [ ] FAQ content ready

### Technical
- [ ] Test purchase flow (use test card)
- [ ] Verify delivery/access works
- [ ] Check mobile experience
- [ ] Set up analytics (UTMs)

---

## Phase 5: Launch Day (Week 1)

### Morning Launch
- [ ] Send launch email
- [ ] Post on Twitter/X
- [ ] Post in relevant communities
- [ ] Update link in bio everywhere
- [ ] DM warm leads

### Throughout Day
- [ ] Monitor sales dashboard
- [ ] Respond to questions quickly
- [ ] Share social proof as it comes
- [ ] Fix any technical issues immediately

### Engagement
- [ ] Thank early buyers publicly
- [ ] Screenshot and share sales (social proof)
- [ ] Engage with comments
- [ ] Answer DMs

---

## Phase 6: Launch Week (Days 1-7)

### Daily Tasks
- [ ] Send scheduled emails
- [ ] Post daily content related to product
- [ ] Engage with comments/questions
- [ ] Track conversion metrics

### Mid-Week Push (Day 3-4)
- [ ] Share customer testimonials
- [ ] Address common objections
- [ ] Offer bonus for quick action (optional)

### Final Push (Day 5-7)
- [ ] Urgency messaging (cart closing)
- [ ] Last testimonials
- [ ] Final email sequence
- [ ] Countdown timer on page

---

## Phase 7: Post-Launch (Week 2+)

### Customer Success
- [ ] Onboarding email sequence
- [ ] Check-in after 7 days
- [ ] Request testimonials after 14 days
- [ ] Gather feedback for V2

### Optimization
- [ ] Analyze conversion rate
- [ ] Review email open/click rates
- [ ] Identify drop-off points
- [ ] A/B test sales page elements

### Evergreen Transition
- [ ] Set up evergreen funnel
- [ ] Create automated email sequence
- [ ] Add retargeting (optional)
- [ ] Build affiliate program (optional)

## Pricing Strategy

### Price Anchoring
```
"Value: $497"
"Today: $97"
```

### Bundle Pricing
- Core product: $47
- Core + templates: $67
- Core + templates + community: $97

### Launch Discount
- 20-30% off standard pricing
- Creates urgency
- Rewards early adopters

## Metrics to Track

| Metric | Target | Notes |
|--------|--------|-------|
| Conversion Rate | 2-5% | Of landing page visitors |
| Email Open Rate | 40%+ | For launch sequence |
| Email Click Rate | 5%+ | To sales page |
| Refund Rate | <10% | If higher, product issue |
| Customer Feedback | 4.5+ stars | For testimonials |

---

## Common Mistakes to Avoid

1. **Perfectionism** - Ship V1, improve based on feedback
2. **No audience** - Build list before product
3. **Feature overload** - Focus on one transformation
4. **Weak positioning** - Be specific about who it's for
5. **No urgency** - Open cart forever = low conversions
6. **Ignoring feedback** - Customer insights = V2 roadmap
