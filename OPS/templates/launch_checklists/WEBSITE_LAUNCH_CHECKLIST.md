# Website Launch Checklist

## Phase 1: Planning (Week -2)

### Goals & Strategy
- [ ] Primary goal defined (leads, sales, info)
- [ ] Target audience documented
- [ ] Key pages listed
- [ ] Conversion actions defined
- [ ] Success metrics established

### Technical Decisions
- [ ] Platform chosen (Next.js, WordPress, Webflow, etc.)
- [ ] Hosting selected (Vercel, Netlify, WPEngine)
- [ ] Domain purchased
- [ ] SSL certificate confirmed
- [ ] Analytics tool selected

---

## Phase 2: Content Creation (Week -2 to -1)

### Core Pages
- [ ] Homepage copy written
- [ ] About page written
- [ ] Product/service pages written
- [ ] Contact page set up
- [ ] Blog/content hub structure

### SEO Content
- [ ] Title tags for all pages (<60 chars)
- [ ] Meta descriptions for all pages (<160 chars)
- [ ] H1 tags optimized (one per page)
- [ ] Image alt text added
- [ ] Internal linking strategy

### Legal Pages
- [ ] Privacy policy
- [ ] Terms of service
- [ ] Cookie policy (if EU traffic)
- [ ] Refund policy (if applicable)
- [ ] Affiliate disclosure (if applicable)

---

## Phase 3: Design & Build (Week -1)

### Visual Design
- [ ] Logo finalized
- [ ] Color palette defined
- [ ] Typography selected
- [ ] Image assets gathered
- [ ] Favicon created

### Page Structure
- [ ] Navigation clear and simple
- [ ] Footer with essential links
- [ ] Mobile responsive
- [ ] Consistent layout across pages
- [ ] White space appropriate

### Forms & CTAs
- [ ] Lead capture form working
- [ ] Email integration tested
- [ ] CTA buttons prominent
- [ ] Thank you page created
- [ ] Form validation working

---

## Phase 4: Technical Setup (Week -1)

### Performance
- [ ] Images optimized (WebP, compressed)
- [ ] Lazy loading enabled
- [ ] Code minified
- [ ] CDN configured
- [ ] Page speed tested (aim <3s load)

### SEO Technical
- [ ] Sitemap.xml generated
- [ ] Robots.txt configured
- [ ] Schema markup added
- [ ] Canonical URLs set
- [ ] 404 page created

### Analytics & Tracking
- [ ] Google Analytics installed
- [ ] Google Search Console connected
- [ ] Conversion goals configured
- [ ] Heatmaps (optional: Hotjar, Clarity)
- [ ] UTM tracking ready

### Security
- [ ] SSL certificate active
- [ ] Forms protected (honeypot, reCAPTCHA)
- [ ] No exposed credentials
- [ ] Backups configured
- [ ] Security headers set

---

## Phase 5: Pre-Launch Testing (Day -3 to -1)

### Functionality Testing
- [ ] All links work (no 404s)
- [ ] Forms submit correctly
- [ ] Email notifications arrive
- [ ] Payment flows work (if applicable)
- [ ] Search works (if applicable)

### Cross-Browser Testing
- [ ] Chrome
- [ ] Safari
- [ ] Firefox
- [ ] Edge

### Device Testing
- [ ] Desktop (multiple sizes)
- [ ] Tablet
- [ ] Mobile (iOS and Android)

### Content Review
- [ ] No typos
- [ ] No lorem ipsum
- [ ] Images all loading
- [ ] Links go to correct places
- [ ] Phone numbers/emails correct

---

## Phase 6: Launch Day

### DNS & Deployment
- [ ] Final code deployed
- [ ] DNS pointed to hosting
- [ ] SSL working on new domain
- [ ] Old redirects in place (if migrating)
- [ ] www and non-www both work

### Post-Deploy Checks
- [ ] Site loads correctly
- [ ] All pages accessible
- [ ] Forms working
- [ ] Analytics tracking
- [ ] No console errors

### Announcements
- [ ] Social media posts
- [ ] Email to list
- [ ] Update all profile links
- [ ] Notify partners/affiliates

---

## Phase 7: Post-Launch (Week 1-4)

### Monitor
- [ ] Check analytics daily first week
- [ ] Monitor error logs
- [ ] Review user feedback
- [ ] Check Core Web Vitals

### SEO
- [ ] Submit sitemap to Google
- [ ] Submit to Bing Webmaster Tools
- [ ] Request indexing for key pages
- [ ] Build initial backlinks

### Optimize
- [ ] Identify high-bounce pages
- [ ] A/B test CTAs
- [ ] Improve page load speed
- [ ] Fix any usability issues

---

## SEO Checklist Detail

### On-Page SEO
- [ ] Primary keyword in title tag
- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Secondary keywords in H2s
- [ ] Internal links (3-5 per page)
- [ ] External links (1-2 to authoritative sources)
- [ ] Image alt text with keywords (naturally)

### Technical SEO
- [ ] Mobile-friendly (Google test)
- [ ] Page speed >90 (PageSpeed Insights)
- [ ] No duplicate content
- [ ] HTTPS everywhere
- [ ] Clean URL structure

### Schema Markup
- [ ] Organization schema on homepage
- [ ] Article schema on blog posts
- [ ] Product schema on product pages
- [ ] FAQ schema where applicable
- [ ] Breadcrumb schema

## Page Speed Targets

| Metric | Target | Tool |
|--------|--------|------|
| First Contentful Paint | <1.8s | PageSpeed Insights |
| Largest Contentful Paint | <2.5s | PageSpeed Insights |
| Total Blocking Time | <200ms | PageSpeed Insights |
| Cumulative Layout Shift | <0.1 | PageSpeed Insights |
| Overall Score | >90 | PageSpeed Insights |

## Common Launch Issues

| Issue | Fix |
|-------|-----|
| SSL not working | Wait 24hrs for propagation, or check DNS |
| Images broken | Check file paths, clear CDN cache |
| Forms not sending | Check email integration, spam folders |
| Slow loading | Optimize images, enable caching |
| 404 errors | Check redirects, update internal links |
