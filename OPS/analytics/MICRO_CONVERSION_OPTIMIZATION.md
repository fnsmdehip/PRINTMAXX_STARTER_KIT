# Micro-Conversion Optimization

Optimize every step of the funnel. Profile → Follow → Engage → Trust → Purchase. Small improvements compound.

---

## The 8-Stage Funnel

Every customer journey has micro-conversions. Optimize each stage.

```
Impression → Click → Land → Engage → Trust → Consider → Convert → Retain
```

| Stage | Micro-Conversion | Key Metric |
|-------|------------------|------------|
| 1. Impression | Saw content | Reach, Impressions |
| 2. Click | Clicked through | CTR |
| 3. Land | Stayed on page | Bounce rate |
| 4. Engage | Consumed content | Time on page, scroll depth |
| 5. Trust | Believed claims | Social proof clicks |
| 6. Consider | Explored offer | Pricing page views |
| 7. Convert | Took action | Conversion rate |
| 8. Retain | Came back | Retention, LTV |

## Stage 1: Impression → Click

### Goal: Get the click

**Platform-Specific CTR Optimization**

| Platform | CTR Lever | Optimization |
|----------|-----------|--------------|
| TikTok/Reels | First 0.5s | Hook immediately, movement |
| YouTube | Thumbnail + Title | Curiosity gap, faces, contrast |
| Twitter/X | First line | Strong hook, no fluff |
| LinkedIn | First line + image | Professional curiosity |
| Email | Subject line | Personalization, curiosity |
| Google | Title + meta | Keyword + benefit |
| App Store | Icon + screenshots | Color pop, clear value |

### Hook Formulas That Convert

**Curiosity Gap:**
- "Why [unexpected thing] works better than [expected thing]"
- "The [thing] that [big result] in [time]"
- "What [authority] doesn't want you to know about [topic]"

**Proof-First:**
- "[Number] [result] in [time]"
- "How I [achieved result] (exact process)"
- "[Specific result] without [common pain]"

**Pattern Interrupt:**
- Start with counter-intuitive claim
- Open with a controversial take
- Lead with unexpected format

## Stage 2: Click → Land

### Goal: Keep them on the page

**Above-the-Fold Essentials**

Must include in first viewport:
1. **Clear headline** - What is this?
2. **Value proposition** - Why should I care?
3. **Visual** - Product/result image
4. **CTA** - What do I do next?

**Bounce Rate Optimization**

| Issue | Fix |
|-------|-----|
| Slow load | Optimize images, lazy load |
| Confusing headline | Match ad/content promise |
| No clear next step | Add prominent CTA |
| Overwhelming content | Simplify, white space |
| Wrong audience | Improve targeting |

**Landing Page Speed Targets**

| Metric | Target | Tool |
|--------|--------|------|
| LCP | <2.5s | PageSpeed Insights |
| FID | <100ms | PageSpeed Insights |
| CLS | <0.1 | PageSpeed Insights |
| TTFB | <200ms | WebPageTest |

## Stage 3: Land → Engage

### Goal: Get them consuming content

**Engagement Metrics to Track**

| Metric | What It Tells You |
|--------|-------------------|
| Time on page | Content resonance |
| Scroll depth | Content consumption |
| Click map | What draws attention |
| Video watch % | Video effectiveness |
| Pages per session | Site stickiness |

**Content Consumption Optimization**

**Text content:**
- Short paragraphs (2-3 sentences)
- Subheadings every 300 words
- Bullet points for lists
- Bold key phrases
- Visual breaks (images, quotes)

**Video content:**
- Hook in first 3 seconds
- Pattern interrupts every 30s
- Clear chapter markers
- Captions always on
- Strong CTA at 80% mark

**Interactive elements:**
- Quizzes (high engagement)
- Calculators (value + engagement)
- Progress indicators
- Expandable sections

## Stage 4: Engage → Trust

### Goal: Build credibility

**Trust Signals Hierarchy**

| Signal | Trust Level | Implementation |
|--------|-------------|----------------|
| Reviews/ratings | High | Display prominently |
| Testimonials | High | Specific results, real photos |
| Case studies | Very high | Detailed success stories |
| Press mentions | Medium-high | Logo bar |
| Social proof numbers | Medium | "Join 10,000+ users" |
| Authority badges | Medium | Certifications, awards |
| Security badges | Medium | SSL, payment security |
| Money-back guarantee | High | Clear, no-questions policy |

**Testimonial Optimization**

Strong testimonials include:
1. Specific result (not vague praise)
2. Full name and photo
3. Relevant context (job, company)
4. Before/after or timeline
5. Emotional component

**Example:**
> "Went from 500 to 50,000 followers in 6 months. PrayerLock's daily reminders made consistency effortless." - Sarah M., Worship Leader

**Trust Sequence on Page**

1. **Early** - Social proof numbers ("10,000+ users")
2. **Middle** - Detailed testimonials
3. **Near CTA** - Guarantee + security badges
4. **After CTA** - FAQ addressing objections

## Stage 5: Trust → Consider

### Goal: Get them evaluating the offer

**Consideration Signals**

| Behavior | Indicates |
|----------|-----------|
| Pricing page view | Ready to evaluate |
| FAQ clicks | Objection research |
| Multiple visits | Comparison shopping |
| Feature comparison | Evaluating options |
| Demo/trial request | High intent |

**Pricing Page Optimization**

| Element | Best Practice |
|---------|---------------|
| Price anchoring | Show highest tier first |
| Feature comparison | Clear checkmarks |
| Recommended tier | Visual highlight |
| Social proof | Per-tier testimonials |
| FAQ | Below pricing table |
| CTA | Per-tier, clear action |

**Reducing Friction**

Remove barriers:
- Account creation requirements
- Unnecessary form fields
- Complex checkout
- Hidden pricing
- Unclear refund policy

Add clarity:
- What happens next
- Time to value
- Refund/cancel ease
- Support availability

## Stage 6: Consider → Convert

### Goal: Close the deal

**Conversion Rate by Channel**

| Channel | Benchmark CVR |
|---------|---------------|
| Direct traffic | 3-5% |
| Email list | 2-5% |
| Organic search | 2-3% |
| Paid search | 2-4% |
| Social organic | 1-2% |
| Social paid | 1-3% |
| Referral | 3-5% |

**CTA Optimization**

| Element | Test Variables |
|---------|----------------|
| Copy | "Start free trial" vs "Get started free" |
| Color | Contrast with page, A/B test |
| Size | Larger = more visible |
| Placement | Above fold, after value, fixed |
| Urgency | Countdown, limited offer |

**Form Optimization**

Every field costs ~7% conversion:
- 1 field = 100% baseline
- 2 fields = 93%
- 3 fields = 86%
- 5 fields = 72%

**Minimum viable form:**
- Email only (best conversion)
- Email + name (if personalization needed)
- Add fields after conversion if needed

**Checkout Optimization**

| Tactic | Expected Lift |
|--------|---------------|
| Guest checkout | +10-20% |
| Progress indicator | +5-10% |
| Multiple payment options | +5-10% |
| Trust badges | +5-15% |
| Exit-intent offer | +5-10% |
| Abandoned cart email | +10-15% |

## Stage 7: Convert → Retain

### Goal: Keep them coming back

**Retention Metrics**

| Metric | Formula | Target |
|--------|---------|--------|
| D1 retention | Users Day 1 / Users Day 0 | >40% |
| D7 retention | Users Day 7 / Users Day 0 | >20% |
| D30 retention | Users Day 30 / Users Day 0 | >10% |
| Monthly churn | Lost users / Starting users | <5% |

**Retention Optimization Tactics**

| Timeframe | Action |
|-----------|--------|
| Immediately | Welcome email/push |
| Day 1 | First value achievement |
| Day 3 | Feature discovery |
| Day 7 | Habit formation nudge |
| Day 14 | Milestone celebration |
| Day 30 | Renewal reminder (if applicable) |

**Retention Hooks**

| Hook | Example | Effect |
|------|---------|--------|
| Streaks | "7 day prayer streak!" | Habit formation |
| Progress | "50% to goal" | Completion drive |
| Social | "John also uses this" | FOMO |
| Fresh content | "New guided prayers" | Return visits |
| Notifications | "Time for your session" | Re-engagement |

## Funnel Analysis Framework

### Finding Leaks

1. **Map the funnel** - Define each step
2. **Measure drop-off** - % lost at each step
3. **Identify biggest leak** - Largest % drop
4. **Diagnose cause** - Why are they leaving?
5. **Prioritize fix** - ICE score the optimization
6. **Test solution** - A/B test the fix
7. **Iterate** - Move to next biggest leak

### Drop-Off Diagnosis

| Drop Point | Likely Causes |
|------------|---------------|
| High bounce | Slow load, mismatch, confusing |
| Low scroll | Content not engaging |
| Pricing page exit | Price shock, unclear value |
| Cart abandonment | Friction, hidden costs |
| Form abandonment | Too many fields, privacy concerns |
| Checkout drop | Payment issues, trust |

### Quick Wins by Funnel Stage

**Top of funnel:**
- Better thumbnails/hooks
- Clearer headlines
- Faster page load

**Middle of funnel:**
- Add social proof
- Simplify navigation
- Improve content quality

**Bottom of funnel:**
- Reduce form fields
- Add urgency/scarcity
- Remove friction
- Clear CTAs

## Testing Framework

### What to Test at Each Stage

| Stage | Test Elements |
|-------|---------------|
| Click | Headlines, thumbnails, hooks |
| Land | Hero, value prop, CTA placement |
| Engage | Content format, length, visuals |
| Trust | Testimonials, proof, guarantees |
| Convert | Pricing, forms, CTAs |
| Retain | Onboarding, emails, notifications |

### Testing Priorities

Focus testing resources on:
1. Highest traffic pages
2. Biggest drop-off points
3. Highest potential impact
4. Easiest to implement

### Minimum Sample Sizes

| Baseline CVR | Lift to Detect | Sample Needed |
|--------------|----------------|---------------|
| 2% | 20% | 15,000/variant |
| 5% | 20% | 6,000/variant |
| 10% | 20% | 3,000/variant |
| 20% | 20% | 1,500/variant |

**Rule:** If you can't get sample size in 2 weeks, test bigger changes.

## Quick Reference

### Conversion Rate Formula
```
CVR = Conversions / Visitors × 100
```

### Revenue Impact Calculator
```
Revenue Lift = Current Revenue × CVR Improvement %

Example:
$10,000/mo × 20% CVR lift = $2,000/mo additional
```

### Prioritization
```
Impact = Traffic Volume × CVR Improvement × Revenue per Conversion
```

Focus on high-traffic, low-converting stages first.
