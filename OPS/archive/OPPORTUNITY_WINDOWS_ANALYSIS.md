## URGENT: Execute NOW (Window Closing)

### 1. Vibe Coding Roblox Games
**Window:** 3-6 months
**Saturation:** LOW (emerging)
**Viability:** 8/10

**Why NOW:**
- Roblox just released official Studio MCP server
- Multiple community MCPs available (vibe-blocks-mcp, robloxstudio-mcp)
- Claude + Cursor can now directly edit Roblox games
- Low competition - most devs haven't adopted yet
- Roblox economy is massive ($3B+ annual revenue)

**Technical Stack:**
- [Roblox Studio MCP Server](https://github.com/Roblox/studio-rust-mcp-server) - Official
- [Vibe Blocks MCP](https://github.com/majidmanzarpour/vibe-blocks-mcp) - Community
- [Lobehub Roblox MCP](https://lobehub.com/mcp/zubeidhendricks-roblox-studio-mcp-claude-code)

**Limitations (from HuggingFace research):**
- Works well for: simple platformers, basic physics, rendering
- Struggles with: interaction systems, inventory, multiplayer, combat
- Best approach: Niche simple games, not complex systems

**Execution Path:**
1. Week 1: Set up Roblox Studio + MCP server + Claude
2. Week 1-2: Build 3-5 simple games (obby, simulator, tycoon clones)
3. Week 2-3: Test which gets traction
4. Week 3+: Double down on winner, iterate with AI
5. Monetize: GamePasses, Robux, Premium Payouts

**Differentiation:**
- Faith games (Christian/Muslim kids market)
- Educational games (study + Roblox hybrid)
- Fitness games (move to unlock in-game)
- Target underserved demographics (boomers gaming with grandkids)

**Risk:** Roblox platform dependency, AI-generated content policies unclear
