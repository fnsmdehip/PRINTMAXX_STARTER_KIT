# SEO/GEO/ASO Research Summary - January 2026

## Executive Summary

SEO/GEO/ASO landscape in 2026:
- **60%+ zero-click searches** - Featured snippets and AI overviews dominate
- **E-E-A-T (Experience)** - First-hand experience now critical ranking factor
- **GEO priority** - ChatGPT, Perplexity, Claude citations drive discovery
- **Video SEO** - 70%+ SERPs show video, 41% higher CTR
- **Mobile-first** - Only mobile version indexed by Google
- **ASO screenshot text** - OCR indexed for keyword ranking
- **Topic clusters** - Hub-and-spoke beats single keyword pages
- **Core Web Vitals** - INP replaces FID, performance is ranking factor

## High-Priority Immediate Actions

### 1. GEO Optimization (HIGHEST ROI)

**ALPHA223 - GEO Citation Patterns**
- AI engines (ChatGPT, Perplexity, Claude) cite sources with clear structure
- Perplexity favors technical depth
- ChatGPT favors conversational confidence

**Implementation:**
- Add dates to all content
- Cite 3-5 authoritative sources per article
- Use tables for data
- Update content quarterly
- Add "Last updated" timestamp

**Effort:** MEDIUM | **ROI:** HIGHEST

---

### 2. Featured Snippet Optimization (HIGH ROI)

**ALPHA221 - Zero-Click Search Dominance**
- 60%+ searches are zero-click
- Optimize for position zero not just #1
- Answer boxes drive brand awareness

**Implementation:**
- Structure content for featured snippets
- Use definition format in first 100 words
- Add tables for comparison content
- Use numbered/bulleted lists
- Include FAQ schema

**Effort:** LOW | **ROI:** HIGH

---

### 3. Video SEO Integration (HIGHEST ROI)

**ALPHA227 - Video SEO Dominance**
- Video results in 70%+ of SERPs
- Video thumbnails get 41% higher CTR
- YouTube is 2nd largest search engine

**Implementation:**
- Create YouTube video for every blog post
- Add video schema to pages
- Optimize YouTube SEO (title/description/tags)
- Add transcript to page
- Embed video above fold

**Effort:** HIGH | **ROI:** HIGHEST

---

### 4. E-E-A-T Experience Signals (HIGH ROI)

**ALPHA220 - Google E-E-A-T 2026**
- Experience added to E-A-T in 2022
- Critical for YMYL (Your Money Your Life) content
- First-hand experience signals matter

**Implementation:**
- Add author bio with credentials to all content
- Include first-person experience
- Add specific numbers/dates from personal use
- Include real photos not stock
- Link to author social profiles

**Effort:** MEDIUM | **ROI:** HIGH

---

### 5. ASO Screenshot Text Optimization (HIGH ROI, LOW EFFORT)

**ALPHA225 - Screenshot Text Indexed**
- iOS App Store OCR scans screenshot text
- Text overlay contributes to keyword relevance
- Strategic use boosts ranking

**Implementation:**
- Add keyword-rich text to first 2 screenshots
- Make text readable at thumbnail size
- Include primary keyword in screenshot 1
- Test keyword ranking with/without
- Don't keyword stuff, readability matters

**Effort:** LOW | **ROI:** HIGH

---

### 6. App Store First 2 Seconds (HIGHEST ROI, LOW EFFORT)

**ALPHA230 - First 2 Seconds CVR**
- Users decide within 2 seconds of seeing listing
- Icon and first screenshot are everything
- 70% never scroll past screenshot 2

**Implementation:**
- A/B test icon weekly
- Put core value in screenshot 1
- Use benefit-driven text overlay
- Test with 5-second test
- Heatmap icon in search results

**Effort:** LOW | **ROI:** HIGHEST

## Medium-Priority Optimizations

### 7. Topic Cluster Architecture (HIGHEST ROI)

**ALPHA228 - Topic Clusters Over Keywords**
- Pillar page + cluster content outranks single pages
- Internal linking shows topic authority
- Hub-and-spoke model for each niche

**Implementation:**
- Create pillar page for main topic
- Write 10+ cluster articles linking to pillar
- Interlink all cluster content
- Use descriptive anchor text
- Update pillar page with cluster links

**Effort:** HIGH | **ROI:** HIGHEST

---

### 8. Core Web Vitals Monitoring (HIGH ROI)

**ALPHA222 - Core Web Vitals 2026**
- LCP <2.5s, FID <100ms, CLS <0.1
- INP (Interaction to Next Paint) replaces FID
- Mobile-first indexing is default

**Implementation:**
- Audit Core Web Vitals monthly
- Optimize images with WebP/AVIF
- Implement lazy loading
- Monitor INP for interactions
- Test on real mobile devices

**Effort:** HIGH | **ROI:** HIGH

---

### 9. Helpful Content Update Compliance (HIGHEST ROI)

**ALPHA226 - Helpful Content System**
- Continuous not one-time update
- People-first content over SEO-first
- Signals: satisfies intent, expertise, original insights

**Implementation:**
- Audit content for thin pages
- Add original research/data
- Remove AI-generated fluff
- Add author expertise signals
- Track user engagement metrics

**Effort:** MEDIUM | **ROI:** HIGHEST

---

### 10. Programmatic SEO for Scale (HIGHEST ROI)

**ALPHA231 - Programmatic SEO Patterns**
- Auto-generate 1000s of pages from data
- Works for comparison, location, product pages
- Zapier has 25k+ programmatic pages

**Implementation:**
- Identify data source (CSV/API)
- Create template with unique content
- Add user-generated content section
- Implement filters/sorting
- Submit to Search Console

**Effort:** HIGH | **ROI:** HIGHEST | **Risk:** MEDIUM

## SEO Foundation Requirements

### 14. Mobile-First Indexing (HIGH ROI)

**ALPHA232 - Mobile-First Mandatory**
- Google only uses mobile version for indexing
- Desktop-only content is invisible
- Mobile site speed critical

**Implementation:**
- Audit mobile experience
- Check mobile PageSpeed score
- Test mobile-first features
- Ensure all content on mobile
- Optimize for thumb-friendly navigation

**Effort:** MEDIUM | **ROI:** HIGH

---

### 15. Link Building Velocity (HIGH ROI)

**ALPHA234 - Link Velocity Signals**
- Natural link growth is steady not spiky
- Sudden backlink spikes trigger spam filters
- Consistent month over month

**Implementation:**
- Build 5-10 links per month consistently
- Vary anchor text naturally
- Focus on relevant sites
- Track link velocity in GSC
- Avoid link farms

**Effort:** MEDIUM | **ROI:** HIGH | **Risk:** MEDIUM

---

- Optimize for "near me" searches

**Implementation:**
- Add city to app subtitle if local
- Create location landing pages
- Optimize for "[category] near me"
- Add location schema
- Build local backlinks

**Effort:** MEDIUM | **ROI:** MEDIUM
