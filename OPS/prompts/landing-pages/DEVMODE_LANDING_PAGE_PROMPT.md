# DevMode Landing Page Template

Source: landind page prtopmt.rtf (Jan 2025)

## Prompt

Design a modern, visually engaging landing page for DevMode, a fictional developer tool company. The page should have a clean, futuristic aesthetic.

### Design Specifications

**Color Palette:**
- Primary: #F1FFD4 (light green)
- Secondary: #E8D9F0 (soft purple)
- Background: #FAFAFA (near-white)
- Text: #0A0A0A (near-black)

**Typography:**
- Headlines: 40-48px, bold
- Body: 16-18px, regular weight
- Font family: Modern sans-serif (Inter or similar)

**Graphics:**
- ALL graphics must be SVG-based illustrations (no raster images)
- Use geometric shapes and patterns
- Subtle animations on scroll

### Page Sections

1. **Hero Section**
   - Large headline with animated gradient text
   - Subtitle explaining the product
   - CTA button with hover animation
   - Abstract SVG illustration

2. **Features Grid**
   - 3-4 feature cards with icons
   - Each card has icon, title, description
   - Hover effects on cards

3. **Product Walkthrough**
   - Step-by-step visual guide
   - Numbered steps with connecting lines
   - Code snippets or terminal mockups

4. **Developer Tools Section**
   - Interactive code editor mockup
   - Syntax highlighted examples
   - Copy-to-clipboard functionality

5. **Team Workflow**
   - Collaboration features showcase
   - Integration logos/icons
   - Statistics or social proof

6. **CTA Section**
   - Prominent call-to-action
   - Pricing tiers or free trial offer
   - Trust badges

7. **Footer**
   - Navigation links
   - Social media icons
   - Newsletter signup

### Technical Requirements
- Responsive design (mobile-first)
- Smooth scroll animations
- Intersection Observer for scroll-triggered animations
- CSS Grid/Flexbox layout
- Dark mode support optional
