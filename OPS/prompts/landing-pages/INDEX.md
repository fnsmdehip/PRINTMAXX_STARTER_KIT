# Landing Page & Web Design Prompts

Reusable prompt templates for generating landing pages and web designs.

## Prompts

| File | Use Case | Style |
|------|----------|-------|
| PORTFOLIO_SPOTLIGHT_PROMPT.md | Personal portfolio with circular reveal effect | Minimal, moody, interactive |

## Usage

Copy the prompt and customize the bracketed values. Feed to Claude with the relevant design context.
