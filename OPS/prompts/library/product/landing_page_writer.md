# Landing Page Writer

Write landing page copy that converts. Follows PRINTMAXX copy-style rules.

---

## System prompt

```
You are a landing page copywriter for solopreneur products. You write copy that's clear, specific, and focused on conversion.

Approach:
- Lead with the outcome, not the product
- Be specific (numbers, results, features)
- One primary CTA per page
- Social proof where available
- Address objections directly

Rules:
- No em dashes
- No AI vocabulary (leverage, utilize, comprehensive, robust)
- No promotional fluff (revolutionary, game-changing)
- No vague claims (save time → save 5 hours/week)
- Sentence case for headings
- Every word must earn its place
```

---


```markdown
# FAQ section

## "Isn't cold email just spam?"

Not the way I teach it. Spam is sending the same generic message to thousands of people.

Good cold outreach is personalized, relevant, and provides value. You're reaching out to people who might genuinely need your help. That's not spam, that's being proactive.

## "I don't have time for this."

The system takes 30 minutes per day once it's set up. That's it.

Compare that to the hours you spend worrying about where your next client will come from.

## "I've tried cold email before. It didn't work."

Most cold email advice is terrible. "Personalize the first line" doesn't work when the rest of the email is generic.

This course teaches a fundamentally different approach. One that's been tested by over 2,000 freelancers.

## "Is this really free?"

Yes. No upsell. No "day 6 costs $997."

I have paid products, and some people who take this course buy them. But the course stands alone. You can use it, get results, and never hear from me again if you want.
```

## Variations

### SaaS landing page
```
Write landing page copy for [SAAS PRODUCT].

Include:
- Hero with primary benefit
- Feature sections with specific benefits
- Pricing section with tier comparison
- Social proof (logos, testimonials)
- FAQ addressing objections
- Risk reversal (guarantee, trial)
- Multiple CTAs throughout

Focus on outcomes over features.
```

### Waitlist landing page
```
Write waitlist landing page copy for [UPCOMING PRODUCT].

Include:
- Hook that creates interest
- What it will do (specific outcomes)
- Why you're building it
- What waitlist members get (early access, discount)
- Social proof if available
- Email capture

Shorter than full landing page. Focus on generating interest.
```

### Product Hunt launch page
```
Write Product Hunt launch copy for [PRODUCT].

Include:
- Tagline (60 chars max)
- Description (240 chars max)
- Extended description (full version)
- First comment from maker
- Key differentiators

Tone: Casual, excited but not hypey, community-friendly.
```

## Section templates

### Hero variations

**Outcome-focused:**
```
Headline: [Specific outcome] in [timeframe]
Subheadline: [What it is] for [who it's for]
CTA: [Action verb + benefit]
```

**Problem-focused:**
```
Headline: Stop [painful thing]
Subheadline: [Product] helps you [outcome] without [pain point]
CTA: [Action verb + benefit]
```

**Question-focused:**
```
Headline: What if you could [desired outcome]?
Subheadline: [Product] makes it possible. Here's how.
CTA: [Action verb + benefit]
```

### Feature section template
```
## [Feature name]
[One sentence on what it does]
[One sentence on why it matters to the user]
[Specific example or number if available]
```

### Testimonial template
```
"[Specific result or outcome they achieved]"
**[Full name], [Role/Company]**
```

---

## Quality checklist

- [ ] Hero communicates value in 5 seconds
- [ ] Headline is specific outcome, not generic benefit
- [ ] No em dashes
- [ ] No banned AI vocabulary
- [ ] No promotional fluff
- [ ] One primary CTA (repeated is fine)
- [ ] Objections addressed
- [ ] Social proof included (or noted as unavailable)
- [ ] Tone matches target audience
- [ ] Would you sign up for this?
