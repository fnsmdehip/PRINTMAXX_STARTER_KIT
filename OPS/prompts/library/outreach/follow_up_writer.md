# Follow-Up Writer

Write follow-up messages that get responses. Follows PRINTMAXX copy-style rules.

---

## System prompt

```
You are a follow-up specialist. You write follow-ups that are persistent without being annoying, adding value with each touch.

Approach:
- Every follow-up adds something new
- Different angle or new information each time
- Respect for their time and attention
- Clear and graceful exits
- No guilt-tripping

Rules:
- No "just checking in" or "just following up"
- No "I know you're busy, but..."
- Add value with each message
- Give easy outs
- Max 3-4 follow-ups
- Space them appropriately
```

---

## User prompt template

```
Write follow-up messages for [CONTEXT].

Original outreach:
- What I sent: [ORIGINAL_MESSAGE]
- When I sent it: [DATE]
- What I was asking for: [ASK]

Follow-up context:
- Response so far: [RESPONSE - none, positive, neutral, objection]
- Relationship: [RELATIONSHIP - cold, warm, existing]
- Urgency: [URGENCY - time-sensitive, evergreen]

New information to include:
[NEW_INFO - updates, new angles, additional value]

Number of follow-ups needed: [COUNT]
```

## Example output

**Input:**
```
Write follow-up messages for freelance project proposal.

Original outreach:
- What I sent: Proposal for website redesign project
- When I sent it: 5 days ago
- What I was asking for: Meeting to discuss the project

Follow-up context:
- Response so far: None
- Relationship: Cold (found them through mutual contact)
- Urgency: Evergreen (no deadline)

New information to include:
- Just completed similar project for another company
- Have availability opening up next month
- Could offer a quick audit as alternative to full project

Number of follow-ups needed: 3
```

**Output:**

```
## Follow-up sequence: Freelance project proposal

---

### Follow-up 1 (Day 5-7 after original)

**Subject:** Re: Website redesign proposal

```
Hi [Name],

Wanted to share something relevant. Just finished a redesign for [similar company] that improved their conversion by 23%.

The challenges were similar to what I noticed on your site. Happy to share the approach we used if it's helpful.

Still interested in chatting if the timing works on your end.

[Your name]
```

**What's added:** New proof point, makes the value more concrete.

---

### Follow-up 2 (Day 10-14 after original)

**Subject:** Quick alternative idea

```
Hi [Name],

One more thought before I stop bothering you.

If a full redesign feels like too much right now, I could do a quick 30-minute audit of your current site. I'd record a video walking through what I'd prioritize.

No commitment, no cost. Just useful feedback whether we work together or not.

Let me know if that's interesting. If not, no worries at all.

[Your name]
```

**What's added:** Lower-commitment alternative, reduces friction.

---

### Follow-up 3 (Day 21 after original)

**Subject:** Closing the loop

```
Hi [Name],

I'll assume the timing isn't right for a website project. Totally understand.

I'll stop following up, but if things change in the future, I'm always happy to chat. [Company name] seems like it's doing great things.

Best of luck with everything.

[Your name]
```

**What's added:** Graceful exit, leaves door open, no guilt.

## Spacing summary

| Follow-up | Timing | Purpose |
|-----------|--------|---------|
| Original | Day 0 | Initial pitch |
| #1 | Day 5-7 | Add proof point |
| #2 | Day 10-14 | Offer alternative |
| #3 | Day 21 | Close gracefully |

---

## Notes

- Don't send all three if you get a response
- Adjust timing based on urgency (faster for time-sensitive)
- Each email should work even if they didn't see the previous ones
```

## Variations

### Meeting no-show follow-up
```
Write a follow-up for when someone missed a scheduled meeting.

Tone: Understanding, not passive-aggressive
Give benefit of the doubt
Offer easy reschedule
One attempt to reschedule, then let it go
```

Example:
```
Hi [Name],

I was on the call but didn't see you join. No worries, things come up.

Here's my calendar if you want to reschedule: [link]

If circumstances have changed and this isn't a priority anymore, also totally fine. Just let me know either way.

[Name]
```

### Post-meeting follow-up
```
Write a follow-up after a successful meeting.

Include:
- Brief recap of discussion
- Next steps (if agreed)
- Any promised deliverables
- Clear timeline

Keep it short. They were just in the meeting, they remember.
```

### Proposal follow-up
```
Write follow-ups after sending a proposal.

Structure:
1. (Day 3) Add social proof or new info
2. (Day 7) Address likely objection
3. (Day 14) Offer modified scope or timeline
4. (Day 21) Graceful close

Each follow-up should stand alone.
```

### Warm lead follow-up
```
Write follow-ups for someone who expressed interest but went quiet.

Different approach:
- They were interested, something changed
- Find out what
- Offer to help with whatever blocked them
- Accept if timing is wrong
```

## Follow-up timing guidelines

| Context | Follow-up 1 | Follow-up 2 | Follow-up 3 |
|---------|-------------|-------------|-------------|
| Cold outreach | 3-5 days | 7-10 days | 14-21 days |
| Warm lead | 2-3 days | 5-7 days | 10-14 days |
| Hot lead | 1-2 days | 3-4 days | 7 days |
| Time-sensitive | 1 day | 2-3 days | 5 days |

**Calendar placement:**
- Avoid Monday morning (buried in inbox)
- Avoid Friday afternoon (forgotten over weekend)
- Midweek mornings often work best
- Match their timezone

---

## What to add in each follow-up

**Follow-up 1:**
- New information (case study, result, news)
- Relevant resource
- Answer to a potential objection

**Follow-up 2:**
- Different angle or approach
- Lower-commitment alternative
- Address a different pain point

**Follow-up 3:**
- Graceful exit
- Summary of value
- Open door for future

**Never add:**
- Guilt ("I haven't heard back...")
- Pressure ("Time is running out...")
- Passive aggression ("Per my last email...")
- Desperation ("Please, I really need this...")

---

## Follow-up openers to avoid

- "Just following up..."
- "Just checking in..."
- "Circling back on this..."
- "Bumping this to the top of your inbox..."
- "I'm sure you're busy, but..."
- "I haven't heard back, so..."
- "Not sure if you saw my last email..."
- "Per my previous email..."

## Better openers

- "One more thought on [topic]..."
- "Something relevant came up..."
- "Quick update: [new info]..."
- "I had an idea that might help..."
- "Wanted to share [specific thing]..."

- [ ] Appropriate spacing
