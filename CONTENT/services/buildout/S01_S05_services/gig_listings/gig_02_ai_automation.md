- Daily scan of competitor sites/pricing pages → Claude summarizes changes → Slack alert with action recommendations

**Report Generation**
- Pull data from Google Analytics + Stripe + Notion → Claude writes weekly summary → auto-email to you every Monday 8AM

**Q: n8n vs Make.com — which should I use?**
A: n8n on a VPS = unlimited executions, one-time setup, ~$20/mo hosting. Make.com = easier to manage, better UI, but per-execution pricing gets expensive at volume. I'll recommend based on your use case.

**Q: Do I need a technical background?**
A: No. I build it, document it, and walk you through it on Loom. If something breaks, you message me.

**Q: What if my workflow needs an API that isn't pre-built?**
