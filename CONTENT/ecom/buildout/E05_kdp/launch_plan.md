# E05 KDP — Launch Plan

## Pre-Launch Setup (Before First Upload)

### Account Setup (30 minutes, one-time)
1. Go to kdp.amazon.com → Sign in with existing Amazon account
2. Complete tax information (W-9 for US) — required before receiving royalties
3. Set up bank account for royalty deposits (ACH direct deposit)
4. Create author pen name: "PrintMaxx Publishing" or personalized pen name per niche
5. Set up Amazon Author Central profile: authorcentral.amazon.com

### Design Stack (Set Up Before First Book)
**Cover design (free options):**
- Canva Pro ($13/mo): Best for non-designers, has KDP templates built in
- KDP Cover Creator: Free, built into KDP, very limited design options
- Adobe Express: Free tier available, more control than Cover Creator

**Cover specs (critical — wrong specs = rejection):**
- Ratio: 1.6:1 height-to-width (KDP standard)
- 6x9 paperback cover: 1800×2700px at 300 DPI minimum
- File format: JPG or TIFF
- Color space: RGB (NOT CMYK — KDP converts internally)
- Include spine: spine width = 0.0025 × page count (inches)
  - 120 pages = 0.3 inch spine
  - 200 pages = 0.5 inch spine
- Include back cover with brief description + ISBN barcode space

**Interior design (free tools):**
- Microsoft Word: KDP has .docx templates at kdp.amazon.com/en_US/help/topic/G200645680
- Google Docs: Download as PDF when done
- Canva: Design page by page, export as PDF
- Affinity Publisher ($70 one-time): Best for complex layouts if you're serious

**Interior specs:**
- 6x9 inch page size (most popular for planners/journals)
- Margins: 0.75 inch outside, 1 inch gutter (inside) for <300 pages
- Font: 10-12pt for body text
- File format: PDF (preferred) or .docx
- Bleed: 0.125 inch on all sides for images going to edge (optional for text-only)

## AMS (Amazon Marketing Services) Setup

### Campaign Structure per Book

**Campaign 1: Auto (run first 14 days)**
- Type: Sponsored Products — Auto
- Budget: $5/day
- Bid: $0.50 default (KDP suggests $0.45-0.65 for this niche)
- Duration: 14 days to collect data

**Campaign 2: Manual-Keyword (after week 2)**
- Type: Sponsored Products — Manual — Keyword targeting
- Targeting: Exact match only
- Keywords: Top 10-20 converters from auto campaign search term report
- Bid: $0.30-0.40 (usually lower than auto campaign bids)
- Budget: $3/day

**Campaign 3: Product Targeting (optional, month 2+)**
- Type: Sponsored Products — Manual — Product targeting
- Target: Your competitors' ASINs (put your ad on their book page)
- Bid: $0.25-0.35
- Budget: $2/day

**Total AMS budget per book: $5-10/day during launch, $3-5/day ongoing**
**10 books at maintenance: $30-50/day = ~$1,000-1,500/month AMS spend**
**Target RCOS: <40% = profitable AMS**

## Review Strategy

**Note: Do NOT violate Amazon's review policy. No incentivized reviews.**

### Compliant Review Tactics

**1. "Honest Review" Follow-Up**
- Set up a KDP A+ Content page (free — go to your book's page → A+ Content)
- Add a page at the end of your interior PDF:
  > "If this planner helped you, I'd appreciate an honest review on Amazon. It takes 60 seconds and directly supports a small publisher. [QR code or URL to your book's review page]"
- This is 100% compliant per Amazon TOS

**2. BookSirens ARC Distribution**
- Cost: ~$60 flat per book launch
- Sends to 50-200 readers who opt in to review in exchange for a free copy
- These are disclosed reviews (Amazon allows them with disclosure)
- Expected: 5-20 reviews per launch

**3. Social Distribution**
- Post your book launch on Twitter/X, Reddit (relevant subs), LinkedIn
- Offer free PDF excerpts as lead magnets to newsletter subscribers
- Ask newsletter readers to buy and review (don't offer anything in exchange)

**4. Author Copy Orders**
- Order 5-10 author copies ($2-4 each) to give to network personally
- Ask them to buy a copy on Amazon (full price) and review
- The buy-first requirement ensures Amazon counts it as a verified purchase review

### Review Velocity Targets

| Month | Target Reviews | Strategy |
|-------|---------------|----------|
| Month 1 | 5-10 | Network + ARC |
| Month 2 | 10-20 | Organic from AMS traffic |
| Month 3 | 20-30 | Compound organic |
| Month 6 | 40-60 | Full organic flywheel |

**10 reviews = Amazon starts showing your book in "Customers also bought" (massive organic lift)**
**25 reviews = Book starts appearing in email recommendation campaigns**

## Description Copy Template

KDP descriptions support basic HTML. Use it.

```html
<h2>Stop Using Generic Planners That Weren't Built for Your Business</h2>

<p>Every day without a system, you're losing money to distraction. The Solopreneur Daily Planner is built around one principle: revenue-producing tasks first, everything else second.</p>

<p><b>Inside this 90-day planner:</b></p>
<ul>
<li>Daily revenue goal tracking (not just to-do lists)</li>
<li>Time blocks designed around deep work, not meetings</li>
<li>Weekly review system that forces you to kill what's not working</li>
<li>Outreach tracking so you know your actual close rate</li>
</ul>

<p><b>Who this is for:</b></p>
<p>Freelancers, indie hackers, consultants, and side hustlers who are serious about the next 90 days being different from the last 90.</p>

<p><b>Specs:</b> 120 pages, 6x9 inch paperback, cream interior paper, premium matte cover.</p>

<p><b>Not for you if:</b> You want motivational quotes and blank lined pages. This is a working tool, not a journal.</p>
```

**Formula:** Problem → Solution → What's inside → Who it's for → Specs → Exclusion (creates desire via selective positioning)

## Template Multiplier Playbook

**Most powerful KDP strategy: 1 interior → multiple listings.**

### How to Execute

**Step 1:** Build one great interior (e.g., Habit Tracker 66-Day)
**Step 2:** Create 3-5 cover variations for adjacent niches
**Step 3:** Update cover + title + description only (interior stays identical)
**Step 4:** Upload as new ASIN (new listing)

### Proven Multiplier Sets

| Base Book | Variations | Additional Listings |
|-----------|-----------|---------------------|
| Habit Tracker | Sobriety, Fitness, Study, Prayer, Language Learning | 5 |
| Faith Journal | Catholic, Baptist, Non-Denominational, Youth | 4 |
| Budget Binder | Couples version, Business version, Teen version | 3 |
| IF Tracker | Keto, Carnivore, Mediterranean Diet, Water Fasting | 4 |
| Content Planner | Instagram, TikTok, YouTube, Podcast, LinkedIn | 5 |
| Stoic Journal | Minimalism, Entrepreneur Mindset, Daily Discipline | 3 |

**10 base books × 3 variations average = 30+ total ASINs from the same work.**

At 30 books each selling 5 copies/month at $5 royalty = **$750/month passive.**

## Monthly Operations After Launch

**Weekly (30 minutes):**
- Check BSR trends for each book (KDP dashboard)
- Review AMS campaign performance (Search Term Report)
- Adjust bids on underperforming keywords
- Check for new reviews — respond if appropriate (Author Central)

**Monthly (1 hour):**
- Review royalty payout (paid 60 days after month end)
- Identify top performers → consider creating variations
- Identify underperformers (BSR >200,000 with AMS spend) → reduce AMS budget or pause
- Add 1-2 new books to maintain portfolio growth

**Quarterly:**
- Refresh covers on underperforming books (A/B test cover concepts)
- Update descriptions based on what competitors are using
- Add new keyword field entries based on accumulated AMS data
- Evaluate going wide (Apple Books, Kobo) for any books outside KU
