## SHUTTERSTOCK CONTRIBUTOR GUIDE

### Earnings Tiers
| Lifetime Downloads | Rate ($/DL) | Notes |
|-------------------|-------------|-------|
| 0-999 | $0.25 | Starting rate |
| 1,000-9,999 | $0.33 | Resets Jan 1 each year |
| 10,000-24,999 | $0.36 | |
| 25,000-49,999 | $0.38 | |
| 50,000+ | $0.40 | |
| Enhanced license | $28.00 | Each Enhanced DL |
| On-demand pack | $0.15-0.25 | Lower rate |

**Strategy:** Upload high-volume (100+ clips) to hit 1K downloads fast and hit $0.33 tier.

### Upload Requirements
- Format: MP4 or MOV
- Resolution: Minimum 1920x1080 (4K preferred)
- Frame rate: 23.98, 24, 25, 29.97, or 30 FPS
- Bitrate: Minimum 10 Mbps (recommend 50+ Mbps for 4K)
- Color space: sRGB
- No letterboxing/pillarboxing
- Max file size: 4GB per clip
- Max duration: 60 seconds (10-30s optimal for search visibility)

### Metadata Formula (CRITICAL — drives 70% of discoverability)
```
Title: [Primary keyword] [secondary keyword] [descriptor], [use case]
Max 70 characters.

Example:
"abstract blue data network nodes connecting, technology background 4K"

Keywords: 50 keywords max. Lead with most specific, end with broad.
Example for Concept 1:
data network, abstract technology, digital connections, blue neon, glowing nodes, tech background, artificial intelligence, neural network, digital web, connected dots, tech wallpaper, blue abstract, data visualization, network graph, futuristic technology, sci-fi background, computer science, big data, cloud computing, internet of things, server network, data flow, cyber security, blockchain network, digital infrastructure, tech overlay, network topology, internet visualization, data transfer, information technology

Description: 200 characters max.
"Abstract blue glowing data nodes connecting across dark background. Network graph expanding in slow motion. Seamless loop. 4K footage."
```

### AI Disclosure Requirement
Shutterstock requires disclosure: check "AI-generated" box on upload.
- AI clips are reviewed separately
- Some categories restricted (realistic people, real brands)
- Abstract, nature simulations, motion graphics: approved routinely
- Tip: Avoid AI-generated realistic human faces (model release issues + platform restrictions)

## ADOBE STOCK CONTRIBUTOR GUIDE

### Earnings
- Video: 35% of license price
- Standard license: ~$79.99 = $28 to you
- Enhanced: ~$199.99 = $70 to you
- Subscription pool: $0.33-3.30/DL depending on subscription tier buyer uses

### Key Advantages Over Shutterstock
- Higher per-download rate for subscription buyers
- Adobe Creative Cloud integration = more professional buyers
- Better for "premium" clip positioning
- Buyers are willing to pay more (agencies, major brands)

### Upload Methods
- Via Adobe Bridge (bulk upload with metadata)
- Via Lightroom (photos, not video)
- Via adobe.com/stock/contributor directly
- **Recommended: Bridge for batch metadata editing**

### AI Policy
- Must disclose AI generation
- Use "Generated by AI" tag
- Cannot include copyrighted elements, real people, real brands
- Abstract, landscapes, backgrounds: generally accepted

## POND5 CONTRIBUTOR GUIDE

### Earnings
- Set your own price: 40% royalty
- Exclusive: 50% royalty
- Typical clip price: $10-200 (you set it)
- Recommended pricing: $15-30 for standard loops, $40-80 for premium original

### Best Content for Pond5
- Niche footage (underwater, aerial, historic, documentary)
- High-quality 4K originals
- Editorial content
- Longer clips (30-60 seconds)

### AI Policy
- Policy evolving — currently proceed with caution
- Best for: motion graphics, abstract, loops (less scrutinized)
- Avoid: realistic people, news/editorial style AI clips

## MOTION ARRAY GUIDE

### Earnings
- $1-5 per download depending on subscription tier
- Monthly payment via PayPal or check
- All content needs to be own work (AI counts if disclosed)

### Best Content for Motion Array

- Glitch effects
- Particle systems

### Pricing
- Accept Motion Array's standard pricing (they handle it)

## UPLOAD PIPELINE SOP

### Tools Needed
| Tool | Cost | Purpose |
|------|------|---------|
| Runway ML Gen-2 | $15/mo (Standard) | Generate 4-second clips |
| Kling AI | Free tier / $10/mo | Generate 5-second clips |
| Pika Labs | Free tier / $8/mo | Generate 3-second clips |
| Adobe Premiere / CapCut | Free (CapCut) | Extend clips, add loops |
| Adobe Bridge | Free (CC) | Batch metadata for Adobe Stock |
| ExifTool (CLI) | Free | Inject IPTC metadata for Shutterstock |
| Handbrake | Free | Re-encode to spec (10+ Mbps H.264) |

### Weekly Production Schedule
| Day | Task | Output |
|-----|------|--------|
| Monday | Generate 10 base clips (Runway/Kling) | 10 raw clips |
| Tuesday | Edit: extend, loop, color grade | 10 polished clips |
| Wednesday | Create variants (4 per clip) | 40 total clips |
| Thursday | Write metadata for all 40 clips | 40 ready-to-upload |
| Friday | Upload to SS + Adobe Stock | 40 clips live |
| Saturday | Upload Pond5 + Motion Array | 40 more live |

**Target: 40 new clips live per week = 160/month = 500+ by month 4**

### Metadata Batch Workflow
1. Generate 10 clips
2. Write metadata in Google Sheet: Title, Keywords (50), Description, Category
3. Export as CSV
4. Import via Adobe Bridge → apply to all clips
5. For Shutterstock: upload via web UI or SS API (requires approval for API access)
6. Use ExifTool to embed IPTC: `exiftool -IPTC:Keywords="keyword1,keyword2" video.mp4`
