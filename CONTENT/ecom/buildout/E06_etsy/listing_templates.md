## UNIVERSAL LISTING SETUP CHECKLIST

Before submitting any listing:

- [ ] Title: 140 chars, primary keyword in first 40 characters
- [ ] Photos: minimum 5 images, first = high-contrast mockup thumbnail
- [ ] Description: starts with consequence-first hook, includes "digital download" phrase
- [ ] Tags: all 13 used, no tag repeats what's already in title
- [ ] Category: deepest sub-category selected
- [ ] Variations: if offering multiple sizes, use Variations feature (not separate listings)
- [ ] Price: set launch price 15% below market average
- [ ] Instant download: toggle ON for digital files
- [ ] File upload: test download before publishing
- [ ] Materials: list "PDF", "Digital File", "Instant Download"
- [ ] Production time: set to "Instant" for digital

**Post-launch (day 1-3):**
- [ ] Share listing in 1 relevant Reddit community (value-first post, mention product)
- [ ] Pin to Pinterest with keyword-rich description
- [ ] Enable Etsy Ads at $1/day for 7-day test
- [ ] Set up auto-follow-up message thanking buyer + asking for review (Etsy Shop Manager > Messages > Auto-reply)

---

## SHOP-LEVEL SETUP CHECKLIST

One-time setup before first listing:

- [ ] Shop name: keyword-relevant + brand, e.g. "PrintMaxxDigitals" or "SoloPreneurShop"
- [ ] Shop icon: clean logo, works at 500x500px
- [ ] Shop banner: 3360x840px, shows 2-3 best sellers or brand statement
- [ ] About section: real story, who you are, what products you sell
- [ ] Shop policies: standard template — Etsy provides one, fill in all fields
- [ ] Announcement: current sale or featured products
- [ ] Free shipping threshold: $35+ gets Etsy's search boost for physical items (N/A for digital, but set up anyway for potential future physical products)
- [ ] Sections: organize by niche (e.g. "Faith & Prayer", "Business Tools", "Kids & Education")
- [ ] Social links: connect Instagram, Pinterest to shop
- [ ] Shop currency: set to USD (largest buyer pool)
