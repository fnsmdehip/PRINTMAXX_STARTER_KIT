## Export Sizes Required

| Platform        | Size (px)    | Notes                     |
|-----------------|-------------|---------------------------|
| App Store (iOS) | 1024 x 1024 | Master — no alpha          |
| iPhone Spotlight| 120 x 120   | @3x                       |
| iPhone App      | 180 x 180   | @3x                       |
| iPad App        | 167 x 167   | @2x                       |
| iPad Pro App    | 152 x 152   | @2x                       |
| Google Play     | 512 x 512   | 32-bit PNG with alpha OK   |
| Web Favicon     | 32 x 32     | .ico or .png               |
| Web Touch Icon  | 180 x 180   | apple-touch-icon           |

## Design Notes

- All icons should be recognizable at 29x29 (smallest iOS size)
- Avoid fine text or small details that disappear at small sizes
- Test in both light and dark mode wallpaper contexts
- Export as PNG (no JPEG for app icons)
