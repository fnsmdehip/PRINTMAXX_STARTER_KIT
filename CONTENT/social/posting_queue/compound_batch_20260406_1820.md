1/ I run 42 cron jobs that manage an autonomous revenue system.

Total cost: $0.22/day.

Here's the full architecture and what I learned building it over 62 days. Thread.

2/ The stack:
- Python scripts for all automation
- Claude Code as the orchestrator brain
- launchd (macOS) for process management
- cron for scheduling
- Filesystem as the state layer (not a database)

No AWS. No Docker. No Kubernetes. Just a MacBook and cron.

3/ The agent hierarchy:

L0: Human (me) - decisions only
L1: CEO Agent - daily orchestration, 16 phases
L2: Swarm Brain - audits every 4 hours, evaluates agent performance
L3: 8 venture agents - each manages a revenue lane
L4: 25 operational agents - scraping, scoring, content, leads
L5: Cron jobs - the heartbeat

4/ What actually runs daily:
- 5:00 AM: Morning DAG kicks off scrapers
- 5:15 AM: SEC + Crunchbase scanners
- 6:00 AM: Twitter + Reddit scrape cycle
- 7:00 AM: Tool scout + method discovery
- Every 4h: Swarm Brain audit cycle
- 10:00 PM: Auto-approve + integrate new alpha

5/ The cost story:

Week 1-4: $8-12/day (every agent calling Opus for everything)
Week 5-6: $2-4/day (model routing - Haiku for classification, Sonnet for code gen)
Week 7+: $0.22/day (killed redundant agents, consolidated scrapers, longer intervals)

97.5% reduction. Same pipeline output.

6/ What the system produces:
- 18,701 alpha intelligence entries (scraped from Twitter, Reddit, HN)
- 192,700 leads analyzed and scored
- 87 validated revenue methods
- 1,559 content pieces ready to post
- 4 iOS apps built and simulator-tested

7/ What I'd do differently:
- Start with 5 scripts, not 538. Most are dead weight.
- Test every script immediately after writing it (50%+ never ran)
- Don't build model routing until you need it. Haiku handles 80% of tasks.
- Filesystem > database for solo projects. git gives you versioning for free.

8/ The uncomfortable truth:

The system is technically impressive and functionally useless without human activation.

192K leads. 0 emails sent.
1,559 posts. 0 published.
87 methods. 0 launched.

Account creation (90 min) > another automation script.

Ship first. Automate second.
