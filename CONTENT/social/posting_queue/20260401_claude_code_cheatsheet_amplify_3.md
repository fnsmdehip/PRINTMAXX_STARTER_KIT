unpopular opinion: Claude Code is mostly a debugging tool, not a writing tool.

it's bad at knowing what to build.
it's very good at finding why something broke.

the prompt that actually works for debugging:

"the bug is somewhere in [filename]. don't look anywhere else. find it."

without that constraint it searches 12 files and finds nothing.

with that constraint it finds the bug in 90 seconds.

scope = speed.
