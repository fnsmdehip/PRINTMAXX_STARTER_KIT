- canvas fingerprint inconsistencies
- mouse movement patterns (bots move in smooth linear paths)
- typing cadence uniformity
- pause duration between actions (humans browse between actions, bots don't)
- timezone/language mismatches vs IP geolocation
- action velocity spikes (0 to 50 follows in an hour = sus)

this is why browser automation for engagement kills accounts. even with human-like delays, the micro-patterns give it away.
