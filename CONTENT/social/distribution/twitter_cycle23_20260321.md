Tweet 3/6:
What Ahrefs actually does better:
— Cleaner UI, less cognitive load
— Content Explorer is genuinely useful
— Rank Tracker is more granular
— Better YouTube keyword data

Most SEO agencies use Ahrefs. Most in-house marketers use SEMrush.

Tweet 4/6:
What free tools cover without paying either:
— Google Search Console (your actual ranking data)
— Ubersuggest free (keyword ideas, basic backlinks)
— Moz free (DA checker)
— Ahrefs free webmaster tools (your own site only)

For a new site under 10K monthly visitors, you probably don't need either.

Tweet 5/6:
The honest take:

If you're running paid ads AND SEO → SEMrush is worth the price.
If you're content-only → Ahrefs.
If you're bootstrapped and under $5K MRR → use Search Console + Ubersuggest and reinvest the $130.
