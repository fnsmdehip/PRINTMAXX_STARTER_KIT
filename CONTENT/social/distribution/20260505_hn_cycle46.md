## POST 1 — Show HN: cnsnt — offline consent form manager, AES-256-GCM, no server

**Title:** Show HN: cnsnt — offline consent form and document signing, AES-256-GCM encrypted, Tauri + PWA

**Body:**
I needed a consent form manager that keeps everything local for client-facing work. Existing solutions all require a cloud account and store your data on their servers.

cnsnt stores everything encrypted on your device:
- AES-256-GCM encryption for all stored documents
- PBKDF2 key derivation, 100,000 iterations
- HMAC-SHA-256 integrity checks on stored files
- Backup = export an encrypted file and store it wherever you want

11 templates: general consent, photo release, NDA, media release, freelance contract, medical consent, etc.

Two versions:
- Desktop: Tauri app (Rust + React), 3.4MB DMG — https://cnsnt-downloads.surge.sh
- Web PWA: https://cnsnt-web.surge.sh

The crypto uses Web Crypto API primitives throughout. The Web Crypto API was a deliberate choice over node-crypto because the PWA version runs entirely in the browser with no server-side component.

I'd welcome a review of the crypto implementation if anyone wants to look at the specifics. I'm not a security researcher and want to make sure I haven't made any subtle errors.

No telemetry, no analytics, no cloud unless you choose to back up.

## POST 2 — Show HN: dosewell — medication and supplement tracker, offline, no account

**Title:** Show HN: dosewell — track meds and supplements offline (PWA, local-only)

**Body:**
I have a large supplement stack and kept missing doses or couldn't remember if I'd taken the morning batch. Tried a few apps, they were either too complex or required creating an account.

dosewell is deliberately minimal:
- Log supplements and medications with AM/PM timing
- Streak tracking for adherence
- Local notifications for reminders
- Offline-first PWA, all data in localStorage
- No account, no sync to any server, no tracking

The goal was a 10-second logging experience. Open, tap what you took, close.

https://dosewell.surge.sh

Works as a PWA — add to home screen for best experience on mobile.

## POST 3 — Ask HN: Why do solo builders keep shipping before account setup?
