# Platform-Specific Optimization: IG Reels, FB Reels, TikTok, YouTube Shorts

## Why This File Exists

Same video. Four totally different metadata strategies.

Each platform has its own search index, hashtag culture, caption length sweet spot, and algorithm signals. Copy-pasting the same caption to all four is the fastest way to underperform everywhere.

---

## TikTok

### Video Specs
- Resolution: 1080x1920
- Max length: 10 min (focus on 30-60s for repurpose workflow)
- File size: up to 287MB
- Aspect ratio: 9:16 (required for FYP prominence)

### Caption Rules
- Max 2,200 characters, use 150-300 for best performance
- Hook your caption: first line visible without "more" click
- No paragraph breaks needed (people rarely read TikTok captions)
- Add 3-5 hashtags max: 1 broad (#makemoney) + 2 mid (#coldemailstrategy) + 1 niche (#solopreneurlife)
- Add 1 trending sound tag if relevant

### Hashtag Strategy
- Do NOT use 30 hashtags like it's 2018 Instagram
- TikTok recommends 3-5 relevant tags
- Trending hashtags on TikTok are pinned to challenges, use them only if content actually fits
- Create one brand hashtag (#printmaxxer) and use it every post

### TikTok-Specific Caption Templates

**Money method video:**
```
made $3,200 last month with this. took 4 hours to set up.

here's the exact method [link in bio]

#coldemail #solopreneur #makemoney2025
```

**Tool review video:**
```
the tool that replaced my VA. $29/month. here's what it does.

#aitools #automation #solopreneurlife
```

**Tips video:**
```
3 things I wish I knew before going full-time online

#entrepreneur #sidehustle #buildinpublic
```

### TikTok Algorithm Signals (Priority Order)
1. Completion rate (watch full video = biggest signal)
2. Shares (biggest virality multiplier)
3. Comments (reply to boost)
4. Likes
5. Profile clicks

First 15 minutes after posting: reply to every comment. This signals to TikTok that your content creates conversation.

## Instagram Reels

### Video Specs
- Resolution: 1080x1920 (or 1080x1350 for square-style Reels)
- Max length: 90 seconds for Reels (longer = limited distribution)
- File size: up to 1GB
- Bitrate: 5Mbps is enough (Meta recompresses to ~3.5Mbps)
- No TikTok watermark (IG suppresses watermarked content)

### Caption Rules
- Max 2,200 characters, sweet spot is 125-150 characters visible before "more"
- First sentence = hook (consequence-first)
- Add line breaks every 1-2 sentences for readability
- 3-10 hashtags in caption OR first comment (first comment is cleaner)
- Location tag if relevant (boosts local discovery)

### Hashtag Strategy
- 5-10 hashtags performs better than 20-30 in 2024-2025
- Mix: 2 broad (1M+ posts) + 3 mid (100K-1M) + 2 niche (<100K)
- Never use banned hashtags (check with Instagram's own hashtag search)

### IG-Specific Caption Templates

**Money method:**
```
$3,200 last month. 4 hours of setup. here's the breakdown.

cold email framework, $0 to start, 3 tools total.

drop "send it" in comments for the full guide

#coldemail #solopreneurlife #sidehustleideas #makemoney2025 #entrepreneurmindset
```

**Tool review:**
```
replaced my VA with this $29/month tool.

here's exactly what it automates [link in bio]

#aitools #automation #productivityhacks #solopreneur #buildinpublic
```

### Instagram Algorithm Signals (Priority Order)
1. Shares to Stories and DMs (highest weight)
2. Saves (strong signal for "valuable content")
3. Comments
4. Likes
5. Watch time / completion

Ask for saves: "Save this for when you're ready to start." This is the single most effective CTA on IG.

### Instagram-Specific Additions
- Add cover thumbnail: custom 1080x1920 with readable title (shows in grid)
- Reels in grid: use cover image so profile looks cohesive
- Tag collaborators if relevant (boosts their followers seeing it)
- Add to Stories within 1 hour of posting (drives initial velocity)

## Facebook Reels

### Video Specs
- Resolution: 1080x1920
- Max length: 90 seconds
- File size: up to 1GB
- Same specs as IG (both Meta, same upload pipeline)

### Caption Rules
- Max 63,206 characters (basically no limit)
- But sweet spot: 80-150 characters
- Facebook users read captions more than IG users, a slightly longer caption with a question at the end increases comments
- Hashtags matter less on FB than IG or TikTok, use 2-3 relevant ones max
- Links in captions are fine on FB (unlike IG where link in bio is required)

### Audience Difference
Facebook's Reels audience skews older: 25-45 vs TikTok's 18-30.

This matters for content strategy:
- More business-owner angle, less "here's my hustle path"
- Can reference more practical concerns: taxes, retirement, stability
- Less hype energy, more credibility signals

### FB-Specific Caption Templates

**Money method:**
```
How I made $3,200 last month with cold email, starting from zero.

Full breakdown in the comments. What part of cold email feels hardest for you?

#coldemail #onlinebusiness #entrepreneur
```

**Tool review:**
```
This $29/month tool replaced 8 hours of VA work per week. Here's how it works.

Link in comments for the full setup guide.

#aiautomation #smallbusiness #productivity
```

### FB Algorithm Signals
1. Comments + replies (Facebook rewards discussion threads)
2. Shares off-platform
3. Watch time
4. Reactions (any reaction counts, not just likes)

Ask a question at the end of every FB caption. FB's algorithm heavily weights comment threads.

### FB Distribution Note
FB Reels surface in three places:
1. FB Reels feed (main feed)
2. Watch tab
3. Suggested content for non-followers

Cross-posting IG Reels to FB via Meta Business Suite gets you all three placements in <2 minutes. Do this every time.

## YouTube Shorts

### Video Specs
- Resolution: 1080x1920 (must be vertical)
- Max length: 60 seconds for Shorts feed (up to 3 min with Shorts label)
- File size: up to 256GB (irrelevant, your files will be <500MB)
- Title: must be 100 characters max
- No watermarks from competing platforms (YouTube suppresses these)

### Title Rules (Shorts uses titles, not captions)
- Title IS your caption on YouTube Shorts
- 60-100 characters is the sweet spot
- Include the primary keyword people would search
- Title appears in search results, write for discoverability

### Description Rules
- Add a description (many creators skip this, it's SEO real estate)
- Include 2-3 keywords naturally in first sentence
- Add chapters if video is longer than 60s
- Add link to related video or channel
- Add 5-10 hashtags at bottom of description (not in title)

### Hashtag Strategy
- Add #Shorts in description (critical, tells algorithm to surface in Shorts feed)
- Add 4-7 niche hashtags in description
- Do NOT put hashtags in the title

### YouTube Shorts-Specific Templates

**Title examples:**
```
I made $3,200 last month with cold email (here's how)
This $29 AI tool replaced my VA [full breakdown]
3 things that changed my freelance income in 30 days
How to start cold email with $0 (step by step)
```

**Description template:**
```
Made $3,200 last month using this cold email framework. Zero experience needed, 3 free tools, works in any niche.

Full tutorial on my channel → [link]

#Shorts #coldemail #makemoneyonline #solopreneur #entrepreneur
```

### YouTube Shorts Algorithm Signals (Priority Order)
1. Swipe-away rate (how fast people skip, LOWEST is best)
2. Like/view ratio
3. Comments
4. Subscription rate from video (very high signal)
5. Watch percentage

First frame is critical on Shorts. YouTube shows the first frame as thumbnail in many placements. Make sure frame 1 has visible, readable text.

## Meta Business Suite Cross-Post Setup

1. Go to business.facebook.com
2. Select your Page
3. Go to Content → Create Reel
4. Upload 1080x1920 MP4
5. Write FB-specific caption
6. Toggle "Also share to Instagram" → add IG-specific caption
7. Schedule or post immediately

One upload. Two Meta placements. Takes 3 minutes.

```

[150 chars max, consequence-first hook, 3-5 hashtags]

INSTAGRAM:
[150 chars visible, save CTA, 5-8 hashtags in first comment]

FACEBOOK:
[150 chars, question at end, 2-3 hashtags, link in comment]

[60-100 chars, searchable keyword, hook]

[2-3 sentence description, 5-10 hashtags including #Shorts]
```

15 minutes of metadata prep per video. Never wing it.
