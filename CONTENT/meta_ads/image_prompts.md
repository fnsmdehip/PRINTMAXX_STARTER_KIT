## STYLE NOTES

- Dark backgrounds for text overlay compatibility

## USAGE NOTES

**Batch Generation:**
```bash
# Generate all 50 images (run from CONTENT/meta_ads/ directory)
# Rate limit: 1 concurrent request, wait 30s between
for i in $(seq 1 50); do
  # Extract and run each curl command from this file
  sleep 30
done
```

**Post-Processing:**
1. Generate raw image with Pollinations
2. Open in Canva or Figma
3. Add text overlay (headline + primary text first 2 lines)
4. Add logo/brand mark in corner
5. Add "Ad" disclosure label
6. Export at 1080x1080 (feed) and 1080x1920 (stories)

**Text Overlay Best Practices:**
- White or light text on dark images
- Maximum 20% text coverage (Meta's old rule is relaxed but less text still performs better)
- Headline in bold 48-72px
- Body text in 24-32px
- CTA button graphic at bottom

**Stories/Reels Crops:**
- Regenerate select prompts at 1080x1920 by changing dimensions in URL
- Or crop the 1080x1080 to center-focus for 9:16
