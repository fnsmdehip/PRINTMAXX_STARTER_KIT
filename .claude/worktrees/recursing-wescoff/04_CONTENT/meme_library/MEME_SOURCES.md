## Scraping Strategy

### Manual Process
1. Browse source accounts daily
2. Screenshot/save high-engagement posts (1k+ likes)
3. Save to `scraped/` folder with date
4. Add to queue for repurposing

### Curation Criteria
- High engagement (likes/RTs/views)
- Fits niche audience
- Not already oversaturated
- Can add value in quote tweet

### Repurposing Methods

| Method | Description | Platform |
|--------|-------------|----------|
| Quote Tweet | Add niche-relevant commentary | X |
| React Video | Film reaction to clip | TikTok/Reels |
| Slideshow | Compile multiple into slideshow | IG/TikTok |
| Thread Starter | Use as hook for thread | X |
