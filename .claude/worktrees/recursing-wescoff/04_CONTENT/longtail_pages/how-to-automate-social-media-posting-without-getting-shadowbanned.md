# How to automate social media posting without getting shadowbanned

## Quick Answer

Use official scheduling tools (Buffer, Later). Add 15-30 minute randomization between posts. Write posts manually or with AI, then review each one. Never use auto-reply bots. Post 1-2 times per day max.

## Why Platforms Shadowban

Platforms detect automation by looking for:
- Identical time intervals between posts (every hour on the hour)
- Repetitive content patterns
- High-volume posting from one account
- Bot-like behavior (instant replies, identical comments)
- Third-party tools that violate TOS

Shadowban = your posts don't show in feeds. No notification. You keep posting into the void.

## Safe Automation Rules

### 1. Use Official or API-Based Tools

**Safe:**
- Buffer (official API partnership)
- Later (official API partnership)
- Hootsuite (official API partnership)
- Platform native schedulers (X scheduler, Meta Business Suite)

**Risky:**
- Browser automation (Selenium, Puppeteer)
- Third-party Chrome extensions
- Mass-action tools (bulk follow/unfollow)

**Why:** Official tools use platform APIs. Platforms know you're scheduling. Browser automation looks like a bot.

### 2. Add Random Delays

Don't post at exact times. Add randomization.

**Bad pattern:**
- 9:00 AM, 12:00 PM, 3:00 PM, 6:00 PM every day

**Good pattern:**
- 9:14 AM, 12:27 PM, 3:03 PM, 6:19 PM (randomized)

Most tools let you add a 15-30 minute window. Use it.

### 3. Limit Post Volume

Platform limits vary:

| Platform | Safe Daily Limit | Danger Zone |
|----------|------------------|-------------|
| X/Twitter | 5-10 posts | 20+ posts |
| LinkedIn | 3-5 posts | 10+ posts |
| Instagram | 2-3 posts | 5+ posts |
| Facebook | 3-5 posts | 10+ posts |

More isn't better. Quality over volume.

### 4. Vary Content Format

Don't post the same type every time.

**Bad week:**
- Monday: Link post
- Tuesday: Link post
- Wednesday: Link post
- Thursday: Link post
- Friday: Link post

**Good week:**
- Monday: Link post
- Tuesday: Image post
- Wednesday: Text-only post
- Thursday: Poll
- Friday: Thread

Mix formats. Looks human.

### 5. Review Before Publishing

Never auto-publish without checking.

**Workflow:**
1. Generate/write posts
2. Load into scheduler
3. Review each post manually
4. Approve for scheduling
5. Let tool handle timing

Takes 10 minutes per day. Prevents disasters.

## Real Automation Stack

### For X/Twitter

**Tool:** Buffer ($15/mo)

**Setup:**
1. Write 5-7 posts Monday morning
2. Load into Buffer
3. Set schedule: Mon-Fri, randomized times
4. Review queue
5. Approve

**Daily time:** 30 min writing, 5 min review.

### For LinkedIn

**Tool:** Later ($25/mo) or LinkedIn native scheduler (free)

**Setup:**
1. Write 3 posts weekly
2. Schedule in Later or LinkedIn scheduler
3. Add images or PDFs
4. Set randomized times

**Weekly time:** 1 hour writing, 10 min scheduling.

### For Instagram

**Tool:** Later ($25/mo) or Meta Business Suite (free)

**Setup:**
1. Create 3-5 posts/week (images + captions)
2. Load into Later
3. Schedule with randomization
4. Approve

**Weekly time:** 2 hours content creation, 15 min scheduling.

## What NOT to Automate

**Never automate:**
- Replies to comments (manual only)
- DMs (manual only)
- Follows/unfollows (manual only)
- Likes (manual only)
- Shares/retweets (manual only)

These actions are high-risk. Platforms watch for bot behavior here.

## Red Flags That Trigger Shadowbans

1. Posting the same link 10+ times in a day
2. Using banned hashtags (#follow4follow, #like4like)
3. Editing posts immediately after publishing (looks like spam testing)
4. High unfollow rate (means people report you as spam)
5. Posting from multiple IPs (VPN + automation = red flag)

## How to Check If You're Shadowbanned

### X/Twitter
- Log out, search for your recent tweet by exact text
- If it doesn't appear, you're shadowbanned
- Or use shadowban.eu (checks your account)

### Instagram
- Post with a niche hashtag
- Ask a friend (who doesn't follow you) to search that hashtag
- If your post doesn't appear, you're shadowbanned

### LinkedIn
- Post something publicly
- Search LinkedIn for that post while logged out
- If it doesn't appear, you're limited

## Recovery From Shadowban

**If you're shadowbanned:**

1. Stop all automation for 7 days
2. Post manually only
3. Engage genuinely (comment, reply)
4. Delete any spammy posts
5. Remove third-party tool access
6. Request review (if platform allows)

Most shadowbans lift in 7-14 days if you stop violating TOS.

## Advanced: Using AI for Content

**Safe workflow:**
1. Use Claude to generate post ideas
2. Use Claude to write drafts
3. Edit each draft manually (add personal voice)
4. Remove AI tells (no "delve", "unlock", "leverage")
5. Load into scheduler
6. Review and approve

AI for drafts = fine. AI auto-publishing = risky.

## Pro Tips

- Post at different times each week (don't train platform to expect you at 9 AM)
- Engage with 5-10 posts before posting your own (looks human)
- Use different devices (mobile + desktop mix)
- Keep sessions realistic (don't post 10 times in 5 minutes)
- Monitor analytics for sudden drops (early shadowban signal)

## Common Mistakes

**Mistake 1:** Automating everything at once.

**Fix:** Start with scheduling only. Add features slowly.

**Mistake 2:** Using cheap third-party tools that promise "growth hacking."

**Fix:** Stick to official or API-based tools.

**Mistake 3:** Never engaging manually.

**Fix:** Reply to comments manually every day.

## Realistic Expectations

**With safe automation:**
- Saves 1-2 hours per week
- No shadowban risk
- Consistent posting schedule
- Still need human oversight

**What automation doesn't do:**
- Write perfect posts (still needs editing)
- Engage with audience (manual only)
- Guarantee virality (content quality matters)
