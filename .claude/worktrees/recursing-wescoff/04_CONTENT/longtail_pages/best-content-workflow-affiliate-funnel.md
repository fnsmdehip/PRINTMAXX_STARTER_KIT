# Best content workflow to post daily for affiliate funnel

## Quick Answer

Batch 5 pieces every week (2 hours work). Post 1 per day Mon-Fri. Mix 2 reviews, 1 tutorial, 1 roundup, 1 comparison. Disclose affiliate links clearly. Track clicks with UTM parameters. Goal: 100+ clicks per week to affiliate offers.

## Why This Workflow

Most affiliate marketers post randomly and blame algorithm. Algorithm doesn't care. Consistency matters. Posting 5x per week beats posting 20x when quality drops.

This workflow keeps you consistent and intentional.

## Content Mix (Weekly)

### Monday: Deep Review
Detailed look at one tool/product. Include:
- How it works
- Best for (who/what)
- Pricing
- Honest criticism
- Affiliate link with disclosure

Length: 800-1000 words

### Tuesday: Quick Tutorial
How-to post on using the product. Short and actionable.
- 5 steps to get started
- Where to find each setting
- Common mistakes
- Affiliate link in CTA

Length: 400-600 words

### Wednesday: Roundup
"Best X for Y" - compare 3 products side by side.
- Tool 1 | Tool 2 | Tool 3
- Price | Features | Best for
- Which I'd pick and why
- Affiliate links for each

Length: 600-800 words

### Thursday: Data/Comparison
"Tool A vs Tool B" - head-to-head.
- Specific features compared
- Cost per month
- Ease of use
- Who wins and when
- Affiliate links

Length: 500-700 words

### Friday: Story
Your experience using the product.
- Problem you had
- How the tool solved it
- Results (numbers if possible)
- Recommendation
- Affiliate link

Length: 400-600 words

## Workflow (2 hours per week)

**Monday, 10am (30 min ideation)**
- Brainstorm 5 products to feature
- Pick one for deep review
- Outline the post

**Tuesday, 10am (45 min writing)**
- Write deep review
- Add affiliate link with disclosure
- Save to schedule

**Wednesday, 10am (30 min)**
- Write quick tutorial
- Test the product while writing
- Schedule

**Thursday, 10am (30 min)**
- Write comparison piece
- Check current pricing
- Schedule

**Friday, 10am (25 min)**
- Write story
- Schedule all posts for next week

## FTC Disclosure (Critical)

Every post with an affiliate link needs:
- "I earn a commission when you buy through this link"
- Or: "Affiliate disclosure: I earn commissions from purchases"
- Put it BEFORE the link, not after

Example:
```
I use and recommend [Tool Name].
(Full disclosure: I earn a commission if you buy through this link.)
[Affiliate Link]
```

No disclosure = FTC fines. Not worth it.

## Scheduling

Use Buffer or Later:
- Write posts Tue-Fri
- Schedule for 10am Mon-Fri next week
- Set to post on your best day/time
- Track engagement

## Tracking Affiliate Clicks

Add UTM parameters to each link:

```
https://affiliate-link.com?utm_source=blog&utm_medium=content&utm_campaign=monday-review
```

In your affiliate dashboard, filter by:
- utm_campaign (which content)
- Date (when posted)
- Clicks and conversions

This tells you which content type converts best.

## Tools

- Google Docs (free): Write
- Buffer/Later ($15/mo): Schedule
- Canva ($10/mo): Create thumbnails
- Affiliate dashboard: Track clicks

Total: $25-30/mo.

## Red Flags

- Affiliate link in every post (looks spammy)
- No FTC disclosure (legal risk)
- Recommending products you haven't used (trust killer)
- Same affiliate link in all posts (loses credibility)

## Content Bank

Keep a running list:
- Tools you use weekly
- Products you've actually bought
- Services that saved you time
- Competitors doing similar work

When brainstorming, pick from this list. You'll write faster because you know the product.
