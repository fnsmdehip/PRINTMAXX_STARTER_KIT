speed and friction

PageScorer is one text input and a button. paste a URL, get your score. no account creation, no email verification, no captchas, no "please wait in queue." i've tested 50+ sites and never waited more than 10 seconds for results.

GTmetrix makes you create an account before you can do anything meaningful. free tier limits you to one test location (Vancouver). want to test from London or Tokyo? that's $14.95/mo minimum. want to test more than 3 pages? queue times start stacking up.

Bottom line: if you just need to check your page speed right now, PageScorer gets it done in 10 seconds. GTmetrix takes 3-5 minutes when you factor in login, queue, and navigation.

depth of analysis

winner: GTmetrix, if you need it
PageScorer
5.5/10
GTmetrix
9/10

GTmetrix gives you waterfall charts that show exactly which resource loaded when, how long DNS lookup took, how long the SSL handshake was. for performance engineers debugging specific bottlenecks, this data is gold.

PageScorer gives you what 90% of people actually need: a score, the key metrics (LCP, CLS, FID), and clear recommendations on what to fix. most website owners don't need to know their TTFB breakdown by millisecond. they need to know "is my site fast enough" and "what should i fix first."

Bottom line: GTmetrix is a scalpel. PageScorer is a health check. if you're debugging a specific performance issue on a CDN-backed enterprise site, use GTmetrix. if you need to know "is my site fast," PageScorer tells you in seconds.

PageScorer is free. not "free tier with limits" free. not "free trial" free. actually free. every feature, every test, every time. no credit card, no account, nothing.

GTmetrix free tier is usable but limited: one test location, basic analysis, slower queue. the Solo plan at $14.95/mo gets you priority, more locations, and API access. the Starter plan at $24.95/mo adds monitoring. the Growth plan at $49.95/mo adds team features. that's $180-600/year for a speed test tool.

PageScorer tells you exactly what to fix in plain language. "your images are too large, compress them." "your CSS is blocking page render, inline the critical styles." you don't need a computer science degree to understand the recommendations.

GTmetrix gives you Lighthouse-style audits with technical jargon. "eliminate render-blocking resources" is useful if you know what that means. if you're a solo founder building a landing page, you probably don't.

Bottom line: PageScorer recommendations are actionable for non-technical people. GTmetrix recommendations are accurate but assume you know web performance engineering.
