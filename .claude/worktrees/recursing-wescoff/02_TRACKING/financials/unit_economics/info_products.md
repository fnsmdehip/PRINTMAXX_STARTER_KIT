## LTV Calculations

### Single Product Buyer

```
LTV = Average Order Value x Purchase Frequency

For info products:
- AOV: $47 (typical)
- Frequency: 1.3 (some buy multiple)
- LTV = $47 x 1.3 = $61.10
```

### With Email Nurture Sequence

```
LTV = First Purchase + (Email List Value x Retention)

Components:
- First purchase: $47
- Email revenue: $1/subscriber/month (industry avg)
- Retention: 12 months avg
- LTV = $47 + ($1 x 12) = $59

Better email strategy:
- $2-5/subscriber/month possible with good sequences
- LTV = $47 + ($3 x 12) = $83
```

### Product Ladder LTV

```
If you have multiple products at different price points:

Product Mix:
- $27 product: 100 buyers
- $47 product: 40 buyers (40% upgrade)
- $97 product: 15 buyers (15% upgrade)
- $197 product: 5 buyers (5% upgrade)

Total Revenue: $2,700 + $1,880 + $1,455 + $985 = $7,020
Average LTV: $7,020 / 100 = $70.20
```

## Break-Even Analysis

### Per Product Break-Even

**Formula:**
```
Break-Even Sales = Fixed Costs / (Price - Variable Costs)

Example ($47 product):
- Creation cost: $500 (time + tools)
- Variable cost per sale: $4.70 (10% platform fee)
- Contribution margin: $42.30

Break-even: $500 / $42.30 = 12 sales
```

## ROI Formulas

### Content ROI

```
ROI = (Revenue - Cost) / Cost x 100

Example:
- 40 hours to create course
- Opportunity cost: $50/hr = $2,000
- Tool costs: $100
- Total investment: $2,100
- First year revenue: $5,000

ROI = ($5,000 - $2,100) / $2,100 x 100 = 138%
```

### Traffic ROI

```
Revenue per 1K Visitors = (Conversion Rate x Price x 1000)

Example:
- 2% conversion, $47 price
- RPV = 0.02 x $47 x 1000 = $940

If you can get 1K visitors for $200 (paid) or 10 hrs (organic):
- Paid ROI: ($940 - $200) / $200 = 370%
- Organic ROI: ($940 - $500) / $500 = 88% (at $50/hr)
```
