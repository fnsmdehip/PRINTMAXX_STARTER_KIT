# Brand Name Generator

Generate cohesive branded account names for the 3-niche PRINTMAXX system.

## Requirements

### AI Niche Accounts
- Theme: Productivity, automation, workflows
- Tone: Technical but accessible
- Examples: AIWorkflowPro, AutomateDaily, StackBuilder

### Faith Niche Accounts
- Theme: Daily devotion, spiritual productivity
- Tone: Encouraging, grounded
- Examples: DailyAnchor, FaithStack, PrayerFlow

### Fitness Niche Accounts
- Theme: Efficient fitness, minimal time
- Tone: No-BS, results-focused
- Examples: 3HourPhysique, MinimalFit, EffortStack

## Output Format

For each niche, provide:
1. Primary brand name (use across all platforms)
2. Handle variations for each platform (if primary taken)
3. Bio templates (platform-specific length)
4. Profile photo direction
5. Cover/banner concept

## Platforms

- X (Twitter): @handle, 160 char bio
- TikTok: @handle, 80 char bio
- Instagram: @handle, 150 char bio
- YouTube: channel name, about section

## Criteria

- Available on all platforms (check manually)
- Memorable, easy to spell
- Aligns with product names (AI Clarity Stack, Daily Anchor System, 3-Hour Physique)
- No trademark conflicts
