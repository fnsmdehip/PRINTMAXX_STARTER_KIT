Capital Genesis: Automated Recommendation Engine
5,277 Revenue Methods Scored on 7 Dimensions with Phase-Aware Weight Shifting
capital_genesis_ranker.py • 1,371 Lines • Daily 5:30 AM Cron

Capital Genesis ingests methods from 5 sources, deduplicates by ID, scores each on 7 weighted dimensions, assigns priority tiers (P0 through KILL), and outputs a daily priority stack consumed by the CEO Agent, Daily Tactical Engine, and KPI Executor.

7-Dimension Composite Scoring
composite = (revenue_potential * 0.25) // HIGHEST=10, HIGH=8, MED=5, LOW=2 + (speed_to_revenue * 0.20) // category baseline + text signals + (downside_risk * 0.15) // inverted: 10 = safest + (automation_potential * 0.15) // "FULL" automation forces min 8.0 + (synergy_score * 0.10) // cross-pollination +0.8/venture fed + (upfront_cost * 0.10) // inverted: $0=10, $500=4, $1K+=1 + (liability_risk * 0.05) // inverted: 10 = no liability

Weights auto-shift by revenue phase: at $0 the engine is tuned for "first dollar fast" (speed 0.30, cost 0.20). At $5K+ it shifts to protection (risk 0.20, liability 0.10). The system currently runs Phase 0 weights.
