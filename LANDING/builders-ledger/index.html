capital_genesis_ranker.py • 1,371 Lines • Runs Daily 5:30 AM Cron

The Capital Genesis Ranker is the brain of PRINTMAXX. It ingests methods from 5 sources (ALPHA_STAGING.csv, METHOD_DISCOVERY_LOG.csv, CAPITAL_GENESIS_LANE_STATUS.csv, 3,012 auto_ops stubs, and 182 master_ops from the xlsx bridge), deduplicates by method_id, scores each on 7 weighted dimensions, assigns priority tiers (P0/P1/P2/P3/KILL), and outputs a daily priority stack consumed by the CEO Agent and Daily Tactical Engine.

The 7-Dimension Scoring Formula
composite = (revenue_potential * 0.25) // HIGHEST=10, HIGH=8, MEDIUM=5, LOW=2 + (speed_to_revenue * 0.20) // 1-10, category baseline + text signals + (downside_risk * 0.15) // inverted: 10 = no risk + (automation_potential * 0.15) // 1-10, "FULL" auto forces min 8.0 + (synergy_score * 0.10) // cross-pollination bonus +0.8/venture + (upfront_cost * 0.10) // inverted: $0=10, $100=7, $500=4, $1K+=1 + (liability_risk * 0.05) // inverted: 10 = no liability

Dimension Scoring Detail

Speed to Revenue: Category baselines (DIGITAL_PRODUCTS=8, CONTENT_FARM=8, AFFILIATE=7, SAAS=3). Text signals: "immediate", "today" add +0.5 each. RPH ≥ $50 adds +1.0.

Downside Risk: Category baselines (DIGITAL_PRODUCTS=9, GROWTH_HACK=4). Red flags ("grey hat", "ban", "tos") subtract -1. Safe words ("proven", "organic") add +0.5.

Automation Potential: CONTENT_FARM=9, SCRAPING=9, BROKERING=8, COMMUNITY=3. "automate", "cron", "n8n", "claude" add +0.3.

Synergy: Cross-pollination bonus: +0.8 per venture fed (cap 4+). 30+ category-to-venture mappings. 11 ventures tracked.

Liability: Red flags ("ftc", "gdpr", "income claim", "gambling") subtract -0.5. Green flags ("open source", "mit license") add +0.3.
