# Section 1: Content Creation — 40 Production Prompts

## Twitter / X Prompts (1-15)

### Prompt 1: Viral Thread Opener
```
Write a Twitter thread hook about [TOPIC]. The first tweet must:
- Start with a bold, specific claim backed by a number
- Create an information gap (the reader MUST click to learn more)
- Be under 240 characters
- Not start with "I" or use words like "amazing," "incredible," or "game-changing"

Example energy: "I analyzed 1,247 landing pages that convert above 8%. They all break the same 3 'best practices.'"

Topic: [YOUR TOPIC]
Context: [WHO YOUR AUDIENCE IS AND WHAT THEY CARE ABOUT]
```

### Prompt 2: Twitter Thread Body (10 Tweets)
```
Write a 10-tweet thread about [TOPIC]. Structure:

Tweet 1: Hook (already written — use this: "[PASTE YOUR HOOK]")
Tweet 2: Set the stakes — what the reader loses by not knowing this
Tweet 3-8: Core content — one actionable point per tweet, with specific examples, numbers, or tools
Tweet 9: Summary — the 3 key takeaways in bullet points
Tweet 10: CTA — ask a question or point to a resource

Rules:
- Every tweet under 280 characters
- No thread numbering (no "1/10" or "Thread:")
- No emoji except in the CTA tweet
- Each tweet must stand alone — if someone screenshots one tweet, it should still make sense
- Use line breaks for readability
```

### Prompt 3: Engagement Bait (Value-First)
```
Write 5 standalone tweets about [TOPIC] designed to get replies. Each tweet must:
- Make a specific, slightly controversial claim
- End with a question that's genuinely interesting to answer
- Be under 200 characters
- NOT be ragebait — the controversy should be professional disagreement, not attacks

Format each as:
[TWEET TEXT]
---
Target replies: [what kind of replies this should generate]
```

### Prompt 4: "How I Did X" Story Tweet
```
Write a Twitter story post about how I [ACHIEVED RESULT] in [TIMEFRAME].

Facts to include:
- [FACT 1]
- [FACT 2]
- [FACT 3]

Structure: Start with the result (end state), then walk backward through the 3-4 key decisions that got there. End with the one insight nobody talks about.

Tone: Conversational, specific, zero humblebragging. If it sounds like a LinkedIn post, rewrite it.
```

### Prompt 5: Quote Tweet Response
```
I want to quote tweet this post: "[PASTE THE ORIGINAL TWEET]"

Write 3 quote tweet options:
1. Add a contrarian but respectful take
2. Add a specific data point or example that supports their point
3. Add a "here's what they're not telling you" angle

Each under 200 characters. The goal is to add value, not clout-chase.
```

### Prompt 6: Twitter Bio Generator
```
Write 5 Twitter bio options for someone who:
- Does: [WHAT YOU DO]
- For: [WHO YOU SERVE]
- Key result: [YOUR BEST METRIC OR ACHIEVEMENT]

Rules:
- Under 160 characters each
- No "| Speaker | Author | Thought Leader" format
- Must include what someone gets by following you
- One should be funny/irreverent, one should be dead serious, rest in between
```

### Prompt 7: Daily Tweet Calendar (7 Days)
```
Create a 7-day tweet calendar for a [NICHE] account.

Each day gets:
- 1 main tweet (value post, story, or insight)
- 1 engagement tweet (question, poll, or hot take)
- Suggested posting times (EST)

Mix the content types: Monday = insight, Tuesday = story, Wednesday = hot take, Thursday = how-to, Friday = resource share, Saturday = personal, Sunday = reflection.

My audience is [DESCRIBE AUDIENCE]. They care about [TOPICS].
```

### Prompt 8: Thread from Blog Post
```
Convert this blog post into a Twitter thread of 8-12 tweets:

[PASTE BLOG POST OR KEY POINTS]

Rules:
- Hook tweet must work as a standalone viral tweet
- Cut all filler, qualifiers, and "it depends" language
- Each tweet = one concrete idea with a specific example
- End with a CTA to read the full post at [URL]
- Don't say "In this thread" — just start teaching
```

### Prompt 9: Polarizing Take Framework
```
Write a tweet that takes a strong stance on [TOPIC].

The take: [YOUR ACTUAL OPINION]

Format options (pick the best fit):
A) "Unpopular opinion: [TAKE]. Here's why: [ONE SENTENCE REASON]"
B) "[THING EVERYONE DOES] is a waste of time. [ALTERNATIVE] works 3x better."
C) "Stop [COMMON ADVICE]. Start [CONTRARIAN ALTERNATIVE]. I tested both for [TIME PERIOD]."

The tweet must be defensible. Don't be wrong on purpose for engagement.
```

### Prompt 10: Twitter Carousel Script
```
Write a 6-slide Twitter carousel about [TOPIC].

Slide 1 (Cover): Bold headline + subtitle that creates curiosity
Slide 2-5: One key point per slide with a supporting detail or mini case study
Slide 6: Summary + CTA (follow for more, grab the free resource, etc.)

Design notes:
- Each slide: max 40 words
- Use bullet points, not paragraphs
- Headline per slide: 5-8 words
- Color scheme suggestion: [BRAND COLORS or "suggest one"]
```

### Prompt 11: Repurpose Replies into Posts
```
I had this exchange on Twitter:

Original tweet: "[ORIGINAL]"
My reply: "[MY REPLY]"
It got [NUMBER] likes.

Turn my reply into a standalone tweet or short thread that captures the same energy but doesn't need the original context. Add a hook and make it self-contained.
```

### Prompt 12: Trend Newsjacking
```
This is trending right now: [TRENDING TOPIC OR NEWS]

My niche is [YOUR NICHE].

Write 3 tweets that connect this trending topic to my niche in a way that's:
1. Relevant (not forced)
2. Valuable (adds a new angle)
3. Timely (would only work this week)

If the trending topic doesn't naturally connect to my niche, say so instead of forcing it.
```

### Prompt 13: Tweet Rewrite (Improve Performance)
```
This tweet got [NUMBER] impressions but only [NUMBER] engagements:

"[PASTE YOUR UNDERPERFORMING TWEET]"

Diagnose why it underperformed and write 3 improved versions. For each version, explain what you changed and why it should perform better. Keep the core message identical.
```

### Prompt 14: Testimonial Tweet
```
A customer said this about my product: "[PASTE TESTIMONIAL OR FEEDBACK]"

Turn this into a tweet that:
- Leads with the result/transformation, not the praise
- Includes the specific metric or outcome
- Doesn't sound like a paid ad
- Ends with a subtle CTA (not "buy now" — more like "DM me if you want the same")
```

### Prompt 15: Pinned Tweet
```
Write a pinned tweet for my Twitter profile.

Who I am: [BRIEF DESCRIPTION]
What I offer: [PRODUCT/SERVICE]
Best result: [YOUR BEST METRIC OR CASE STUDY]
CTA: [WHAT YOU WANT VISITORS TO DO — follow, DM, click link]

The pinned tweet should tell a first-time visitor exactly who I am, why they should care, and what to do next. Under 280 characters. No fluff.
```

## Newsletter Prompts (16-25)

### Prompt 16: Newsletter Subject Lines
```
Write 10 email subject lines for a newsletter about [TOPIC].

This week's content covers: [BRIEF SUMMARY OF WHAT'S IN THE EMAIL]

Rules:
- Under 50 characters each (mobile preview cutoff)
- No clickbait that doesn't deliver
- 3 should use numbers, 3 should use curiosity gaps, 2 should use urgency, 2 should be direct
- Test group: which 2 would you A/B test and why?
```

### Prompt 17: Newsletter Intro Paragraph
```
Write the opening paragraph for a newsletter about [THIS WEEK'S TOPIC].

My newsletter tone is: [CASUAL/PROFESSIONAL/IRREVERENT — pick one]
My audience is: [DESCRIBE THEM]
The key insight this week: [ONE SENTENCE SUMMARY]

The intro must:
- Hook in the first sentence (no "Hey everyone, hope you had a great week")
- Set up WHY this topic matters right now
- Transition naturally into the first section
- Be under 100 words
```

### Prompt 18: Newsletter from Raw Notes
```
Turn these raw notes into a polished newsletter section:

[PASTE YOUR ROUGH NOTES, BULLET POINTS, OR STREAM-OF-CONSCIOUSNESS THOUGHTS]

Target length: [300/500/800] words
Tone: [DESCRIBE YOUR NEWSLETTER VOICE]
Format: Use headers, bold key phrases, and short paragraphs (2-3 sentences max)
Add one actionable takeaway the reader can implement today.
```

### Prompt 19: Weekly Roundup Newsletter
```
Write a weekly roundup newsletter covering these items:

1. [ITEM 1 — brief description]
2. [ITEM 2]
3. [ITEM 3]
4. [ITEM 4]
5. [ITEM 5]

For each item, write:
- A one-line summary (what happened)
- A one-line analysis (why it matters to [YOUR AUDIENCE])
- A link placeholder: [LINK]

Add an intro that ties the items together with a theme, and a closing thought that gives the reader one thing to think about this week.
```

### Prompt 20: Newsletter CTA Block
```
Write a CTA section for my newsletter that promotes [PRODUCT/SERVICE/RESOURCE].

Details:
- What it is: [DESCRIPTION]
- Price: [PRICE or FREE]
- Key benefit: [ONE SENTENCE]
- URL: [LINK]

The CTA must:
- Not feel like an ad (integrate it naturally)
- Lead with the problem it solves, not the features
- Include a specific, time-bound reason to act now (if applicable)
- Be under 75 words
```

### Prompt 21: Welcome Email Sequence (3 Emails)
```
Write a 3-email welcome sequence for new subscribers to my [NICHE] newsletter.

Email 1 (immediate): Welcome + set expectations (what they'll get, how often) + deliver the lead magnet if applicable
Email 2 (Day 2): Your best/most popular piece of content, repackaged as "the one thing most people get wrong about [TOPIC]"
Email 3 (Day 4): Soft pitch for [PRODUCT/SERVICE] with a case study or result

My newsletter: [NAME AND BRIEF DESCRIPTION]
My product: [WHAT YOU SELL]
Tone: [YOUR VOICE]
```

### Prompt 22: Newsletter Growth Post
```
Write a tweet promoting my newsletter that doesn't sound like "subscribe to my newsletter."

Newsletter name: [NAME]
What subscribers get: [SPECIFIC VALUE]
Current subscriber count: [NUMBER]
Best issue topic: [TOPIC OF YOUR BEST NEWSLETTER]

The tweet should share a valuable insight FROM the newsletter, then mention the newsletter as the source. The value comes first, the plug comes last.
```

### Prompt 23: Sponsorship Pitch Email
```
Write a cold email pitching [BRAND/COMPANY] to sponsor my newsletter.

My newsletter:
- Name: [NAME]
- Subscribers: [NUMBER]
- Open rate: [PERCENTAGE]
- Niche: [TOPIC]
- Audience demographics: [WHO READS IT]

What I'm offering:
- [SPONSORSHIP FORMAT — dedicated email, banner, mention, etc.]
- [PRICE]

The email should be under 150 words, lead with why their product is relevant to my audience, and include one specific metric that proves the value.
```

### Prompt 24: Newsletter Content Repurpose
```
Take this newsletter issue and create:
1. A Twitter thread (8 tweets)
2. A LinkedIn post (150-200 words)
3. A Reddit post for r/[SUBREDDIT] (follows community norms — value first, no self-promotion in the post body)

Newsletter content:
[PASTE NEWSLETTER]

Each output should feel native to its platform, not like a copy-paste job.
```

### Prompt 25: Monthly Newsletter Review
```
Here are my newsletter stats for the last 4 issues:

Issue 1: Subject "[SUBJECT]" — Open rate: [%], Click rate: [%]
Issue 2: Subject "[SUBJECT]" — Open rate: [%], Click rate: [%]
Issue 3: Subject "[SUBJECT]" — Open rate: [%], Click rate: [%]
Issue 4: Subject "[SUBJECT]" — Open rate: [%], Click rate: [%]

Analyze the patterns:
- Which subject line style performed best and why?
- What's driving clicks vs. what's driving opens?
- Give me 3 specific recommendations for next month's subject lines and content strategy
- Suggest 4 subject lines for next month based on what the data shows works
```

## Blog / Long-Form Prompts (26-40)

### Prompt 26: Blog Post Outline
```
Create a detailed outline for a blog post titled "[TITLE]".

Target audience: [WHO]
Target keyword: [SEO KEYWORD]
Target length: [1500/2500/4000] words
Intent: [INFORMATIONAL / TRANSACTIONAL / COMPARISON]

Outline format:
- H1 (title)
- Intro (hook, problem statement, what they'll learn)
- H2 sections (4-6 main sections)
  - Key points under each (2-3 bullet points)
- H2: Frequently Asked Questions (3-5 questions)
- Conclusion with CTA

For each H2, suggest a specific example, stat, or case study to include.
```

### Prompt 27: Blog Intro That Hooks
```
Write the opening 150 words of a blog post about [TOPIC].

The reader searched for: "[SEARCH QUERY]"

The intro must:
- Acknowledge their problem in the first sentence
- Agitate it in the second sentence (what happens if they don't solve it)
- Promise the solution in the third sentence
- NOT start with a question, a definition, or "In today's fast-paced world"
- Include the target keyword naturally within the first 100 words
```

### Prompt 28: How-To Blog Post
```
Write a complete how-to blog post: "[HOW TO TITLE]"

Requirements:
- 1500-2000 words
- Step-by-step format with numbered steps
- Each step: what to do, why it matters, common mistake to avoid
- Include one real tool recommendation per step (with pricing)
- Screenshots/images marked as [IMAGE: description of what to capture]
- FAQ section at the end (4 questions)
- Target keyword: [KEYWORD] — use it in H1, first paragraph, one H2, and conclusion
```

### Prompt 29: Comparison Blog Post
```
Write a comparison post: "[PRODUCT A] vs [PRODUCT B]: Which Is Better for [USE CASE]?"

Cover these dimensions:
- Pricing (exact plans and prices)
- Ease of use (specific UX differences)
- Features (what each has that the other doesn't)
- Best for (who should pick which)
- Deal-breakers (when each product is the WRONG choice)

Include a comparison table. End with a clear recommendation, not "it depends." Take a stance.

My audience is [DESCRIBE THEM]. They care most about [TOP PRIORITY].
```

### Prompt 30: Listicle Blog Post
```
Write a listicle: "[NUMBER] Best [TOOLS/STRATEGIES/RESOURCES] for [USE CASE] in [YEAR]"

For each item include:
- Name and one-line description
- Pricing (exact — free, $X/month, etc.)
- Best feature (the ONE thing that makes it worth using)
- Limitation (the ONE thing that might be a deal-breaker)
- Who it's for (one sentence)

Order by: best overall first, budget option included in top 5, most niche options at the end.
Items: [NUMBER — between 7 and 15]
```

### Prompt 31: Case Study Blog Post
```
Write a case study blog post about how [CLIENT/USER/I] achieved [RESULT].

Facts:
- Starting point: [WHERE THEY WERE]
- End result: [WHERE THEY ARE NOW]
- Timeframe: [HOW LONG IT TOOK]
- Key actions: [3-5 SPECIFIC THINGS THEY DID]
- Tools used: [LIST TOOLS]

Structure: Situation > Problem > Solution > Results > Key Takeaways
Length: 1200-1800 words
Include direct quotes (write realistic ones based on the facts above).
End with a CTA relevant to the reader: "If you want similar results, [ACTION]"
```

### Prompt 32: SEO Meta Description
```
Write 3 meta description options for this blog post:

Title: [BLOG POST TITLE]
Target keyword: [KEYWORD]
Summary: [1-2 SENTENCE SUMMARY OF THE POST]

Each meta description must:
- Be 150-160 characters (hard limit)
- Include the target keyword
- Have a clear value proposition
- End with action-oriented language
- NOT start with "Learn how to" or "Discover"
```

### Prompt 33: Content Upgrade / Lead Magnet
```
I wrote a blog post about [TOPIC]. Suggest 3 content upgrade ideas that I can offer in exchange for an email address.

Each content upgrade should:
- Be directly related to the blog post topic
- Take me under 2 hours to create
- Provide immediate, tangible value
- Be deliverable as a PDF, spreadsheet, or template

For each suggestion, write the opt-in copy (headline + 2-sentence description + CTA button text).
```

### Prompt 34: Blog Post from Podcast/Video Transcript
```
Turn this transcript into a polished blog post:

[PASTE TRANSCRIPT OR KEY SECTIONS]

Rules:
- Remove all filler words, "ums," and conversational tangents
- Add structure: H2 headers every 300-400 words
- Keep the speaker's voice and personality — don't make it sound like a textbook
- Add a "Key Takeaways" section at the top (3-5 bullet points)
- Target length: [WORDS]
- Bold the most quotable sentences (these become social media pull quotes)
```

### Prompt 35: Blog Post Conclusion + CTA
```
Write the conclusion for a blog post about [TOPIC].

Main points covered in the post:
1. [POINT 1]
2. [POINT 2]
3. [POINT 3]

CTA goal: [WHAT YOU WANT THE READER TO DO — subscribe, buy, try, share]

The conclusion must:
- Summarize in 2-3 sentences (not repeat every point)
- Reinforce the one most important insight
- Transition naturally into the CTA
- NOT say "In conclusion" or "To wrap things up"
- Be under 150 words
```

### Prompt 36: Content Calendar (30 Days)
```
Create a 30-day content calendar for a [NICHE] blog.

Target audience: [WHO]
Publishing frequency: [X posts per week]
Content pillars: [3-4 TOPIC CATEGORIES]
Goal: [TRAFFIC / EMAIL SIGNUPS / PRODUCT SALES]

For each post, provide:
- Working title
- Target keyword (long-tail, suggest based on niche)
- Content type (how-to, listicle, case study, comparison, opinion)
- Estimated word count
- Internal linking opportunity (which other posts on the calendar it should link to)
```

### Prompt 37: Expert Roundup Post
```
I'm writing an expert roundup post: "[TITLE — e.g., 12 Experts Share Their Best Advice on X]"

Write the outreach email I should send to experts asking for their contribution:
- Under 100 words
- Specific question for them to answer
- Deadline
- What they get in return (link back, social promotion)

Then write the intro and conclusion for the roundup post (I'll fill in the expert quotes).
```

### Prompt 38: Pillar Page Outline
```
Create a pillar page outline for the topic: "[BROAD TOPIC]"

This pillar page should:
- Be 3000-5000 words
- Cover the topic comprehensively (someone should not need to read anything else)
- Link to 8-12 supporting blog posts (suggest titles for posts I should write)
- Include an FAQ section (8-10 questions)
- Have a table of contents
- Target the main keyword: [KEYWORD]

For each section, note which supporting blog post it should link to and what long-tail keyword that supporting post should target.
```

### Prompt 39: Product Description
```
Write a product description for [PRODUCT NAME].

Details:
- What it is: [DESCRIPTION]
- Price: [PRICE]
- Key features: [LIST 3-5 FEATURES]
- Target buyer: [WHO]
- Problem it solves: [PAIN POINT]

Write 3 versions:
1. Short (50 words) — for product cards and ads
2. Medium (150 words) — for product pages
3. Long (300 words) — for landing pages

Each version should lead with the benefit, not the feature. "Save 5 hours per week" not "Includes automation features."
```

### Prompt 40: Content Audit Prompt
```
I'm going to paste the titles and URLs of my last 20 blog posts. Analyze them and tell me:

1. Content gaps: What topics am I NOT covering that my audience needs?
2. Cannibalization: Are any posts competing for the same keyword?
3. Update candidates: Which posts likely have outdated information based on their titles?
4. Consolidation: Which thin posts could be merged into one comprehensive post?
5. Winners: Based on the titles alone, which 3 posts are most likely to perform well for SEO and why?

My blog posts:
[PASTE TITLES AND URLS]

My niche: [NICHE]
My audience: [AUDIENCE]
```
