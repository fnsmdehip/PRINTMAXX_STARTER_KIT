How to Set Up MCP in Claude Code (5 minutes)

1

Install Claude Code

Already have it? Skip. Install with: npm install -g @anthropic-ai/claude-code

2

Open your MCP config

Create or edit ~/.claude/mcp.json — this is where you add MCP servers

3

Add your first MCP server

Example for Playwright MCP (replaces Browserbase): "playwright": {"command": "npx", "args": ["@playwright/mcp@latest"]}

4

Restart and use

Claude Code loads MCP servers on startup. Say "use playwright to..." and it just works. No SaaS dashboard. No API key wrangling. No per-seat pricing.

5

Add the MCP Marketplace to find more
