1. `AUTO_STATUS_LIVE`
   - Scores each OP with runtime readiness, blocker key, signal count, lane, and next action.
2. `PRIORITY_AUTOMATION_EXEC`
   - Maps `PRIORITY LAUNCH` rows to live readiness and automation actions.
3. `ETC_EXPANSION_QUEUE`
   - Implements the CODEX `etc` expansion logic by surfacing adjacent ops in the same lane/category.

   - Embeds live artifact evidence (counts, blockers, approvals) at generation time.
5. `DEEP_PLAYBOOK_INDEX`
   - Structures the 2,999-row deep playbook into executable sections/steps with lane inference.
6. `ALPHA_THESIS_INDEX`
   - Structures LLM alpha opportunities into venture rows with blocker keys and command templates.
7. `VENTURE_AUTOMATION_MAP`
   - Merges priority launch ops + alpha thesis ventures into one execution map with command templates.
