# Community Monetization Playbook

**Method ID:** MM031 (Paid Communities)
**Revenue target:** $7,250/mo (5 communities x 50 members x $29/mo avg)
**Startup cost:** $0-$99/mo depending on platform
**Time to first dollar:** 14-30 days (if audience exists), 60-90 days (cold start)

---

## 1. Platform Comparison

### Skool

| Attribute | Details |
|-----------|---------|
| **Cost** | $99/mo flat (creator pays), members pay whatever you set |
| **Payout** | Stripe direct, no platform cut on membership fees |
| **Features** | Courses built-in, gamification (leaderboard, levels), calendar, community feed |
| **Discovery** | Skool has internal discovery (skool.com/discovery). Communities rank by engagement. Free marketing if you rank top 10 in category |
| **Affiliate** | 40% recurring commission on Skool referrals ($39.60/mo per referred creator) |
| **Limits** | No DM feature (forces public engagement). No native app (mobile web only). No custom domain on free plan |
| **Best for** | Course + community bundles, niches where gamification drives retention (fitness, learning, biz) |
| **Niche fit** | Fitness (leaderboards), Tech/Solopreneur (course + mastermind), Faith (accountability) |

**Real numbers:** Alex Hormozi's Skool community hit $2.5M/mo in 2024. Top non-celebrity Skool groups do $10K-$100K/mo. Median paid Skool community with 200+ members: ~$5K/mo.

### Discord (with Patreon/Whop gate)

| Attribute | Details |
|-----------|---------|
| **Cost** | Discord free. Whop takes 0% on first $500/mo, then 3%. Patreon takes 5-12% |
| **Payout** | Via Whop (Stripe) or Patreon (direct deposit, 1-5 day delay) |
| **Features** | Channels, voice chat, bots, roles, threads, screen share, stages. Most flexible platform |
| **Discovery** | Zero built-in discovery. Must drive all traffic yourself |
| **Limits** | High churn if content cadence drops. "Free Discord" stigma makes paid harder to sell. Requires bot setup for gating |
| **Best for** | Gaming, crypto, tech communities. Audiences already on Discord daily |
| **Niche fit** | Tech/Solopreneur (devs live on Discord), NSFW (Whop gating, age verification) |

**Real numbers:** Top paid Discords (crypto alpha groups) charge $99-$499/mo. Average indie paid Discord: 30-80 members at $19-$49/mo. Churn is 8-15% monthly (highest of all platforms).

### Telegram (Premium Channels)

| Attribute | Details |
|-----------|---------|
| **Cost** | Free to create. Payment via Whop, Launchpass, or manual (crypto/PayPal) |
| **Payout** | Via payment processor you choose. No native Telegram monetization in most regions |
| **Features** | Channels (broadcast), groups (discussion), voice chat, file sharing (up to 2GB), bots |
| **Discovery** | Zero. Telegram has no discovery. Pure push-traffic model |
| **Limits** | No built-in payments. No course hosting. Limited moderation tools. Spam is constant without good bots |
| **Best for** | Crypto signals, NSFW/adult (least moderated major platform), international audiences |
| **Niche fit** | NSFW/Findom (VIP tiers, least censored), Faith (Middle East/South Asian audiences use Telegram heavily) |

**Real numbers:** Crypto signal groups charge $49-$299/mo. Top ones: 500-2,000 members. NSFW Telegram VIPs: $9.99-$49.99/mo, 50-500 members typical. Churn 5-10% (lower than Discord because notifications are stickier).

### Circle

| Attribute | Details |
|-----------|---------|
| **Cost** | $89/mo (Basic), $199/mo (Professional), $399/mo (Business) |
| **Payout** | Stripe integration. Circle takes 0% of membership fees |
| **Features** | Spaces (channels), courses, events, member directory, custom domain, white-label, API |
| **Discovery** | None. Self-hosted feel. But custom domain = SEO benefit |
| **Limits** | Expensive. $89/mo minimum means you need ~$200/mo revenue just to break even. No free plan |
| **Best for** | Premium communities ($99+/mo), course creators who want professional branding |
| **Niche fit** | Tech mastermind (professional look), Faith (white-label "church community" feel) |

**Real numbers:** Circle communities avg $4,500/mo revenue (Circle's own data, 2024). Skews toward established creators with existing audiences. Not ideal for cold start.

### Geneva

| Attribute | Details |
|-----------|---------|
| **Cost** | Free |
| **Payout** | No native monetization. Must gate externally |
| **Features** | Rooms (text, audio, video), events, clean mobile UX, calendar |
| **Discovery** | Minimal. Some in-app browsing |
| **Limits** | No payments. No courses. Small user base. Company future uncertain |
| **Best for** | Free communities, local/IRL groups, creative communities |
| **Niche fit** | Not recommended for paid. Consider only for free-tier funnel top |

### Platform Decision Matrix

| Factor | Skool | Discord | Telegram | Circle |
|--------|-------|---------|----------|--------|
| **$0 startup** | No ($99/mo) | Yes | Yes | No ($89/mo) |
| **Built-in courses** | Yes | No | No | Yes |
| **Built-in discovery** | Yes | No | No | No |
| **Mobile app** | No (web) | Yes | Yes | No (web) |
| **Custom domain** | No | No | No | Yes |
| **Payment integration** | Built-in | Via Whop/Patreon | Via Whop/Launchpass | Built-in |
| **Best retention** | High (gamification) | Low-Medium | Medium-High | Medium |
| **Content moderation** | Moderate | Strict | Minimal | Moderate |
| **Cheapest to start** | $99/mo | $0 | $0 | $89/mo |

**Recommendation by budget:**
- **$0 budget:** Discord + Whop or Telegram + Launchpass
- **$99/mo budget:** Skool (best all-in-one, built-in discovery is free marketing)
- **$200+/mo budget:** Circle (if premium positioning matters)

## 3. Pricing Tiers (Universal Framework)

### Tier Architecture

| Tier | Price | Purpose | Margin | Typical Retention |
|------|-------|---------|--------|-------------------|
| **Free** | $0 | Growth, email capture, trust building | 0% (cost center) | N/A |
| **Basic** | $19-$29/mo | Core community access, async content | 85-95% | 4-6 months avg |
| **Premium** | $49-$99/mo | Live access, personalized elements | 90-95% | 6-10 months avg |
| **VIP** | $199-$499/mo | 1-on-1 elements, mastermind, custom | 80-90% | 8-14 months avg |

### Pricing Psychology

**Anchoring:** Always show VIP tier first. Makes $29/mo look trivial by comparison.

**Annual discount:** Offer 2 months free on annual ($29/mo = $290/yr instead of $348). Locks in retention and reduces churn math.

**Founding member pricing:** First 25 members get 40% off for life. Creates urgency, rewards early adopters, generates testimonials. "Founding member" badge is social proof inside community.

**Price increases:** Raise price every 50 members. Publicly announce: "Price goes to $49/mo at 100 members." Creates urgency for fence-sitters. Existing members keep old price (grandfathered).

### What Each Tier Must Include

**Free tier (growth engine):**
- 1 valuable post per day (teaser of paid content)
- Weekly free resource (template, checklist, guide)
- Community chat (limited channels/topics)
- Email capture on signup (feed to newsletter)

**Basic ($19-$29/mo):**
- Everything in free
- Full community access (all channels/spaces)
- Weekly live Q&A or workshop
- Content library (past recordings, guides, templates)
- Member directory (networking value)

**Premium ($49-$99/mo):**
- Everything in Basic
- Monthly group coaching call (small group, 8-12 people)
- Priority response on questions
- Accountability partner matching
- Exclusive content drops

**VIP ($199-$499/mo):**
- Everything in Premium
- Monthly (or bi-weekly) 1-on-1 call
- Direct message access to host
- Custom deliverable (audit, plan, strategy, content)
- "Inner circle" channel with other VIPs only

## 4. Growth Tactics

### Free Content to Paid Community Funnel

```
Twitter/X post (value bomb with specific numbers)
  -> "I share the full breakdown in my community" (bio link)
    -> Free tier signup (email captured)
      -> 7-day email drip showing community value
        -> Paid tier offer (founding member pricing, limited spots)
```

**Conversion benchmarks:**
- Free follower to free community member: 2-5%
- Free community member to paid: 5-15% (with proper nurture)
- Paid basic to premium upsell: 10-20% (after 2+ months)

### Scarcity and Exclusivity

**Capped membership:** "Only 50 spots in the Inner Circle." Real or artificial, it works. Announce countdown publicly: "47/50 spots filled."

**Application process:** For premium tiers ($99+/mo), require a short application (3 questions). Rejection rate of 10-20% increases perceived value. Questions: What's your current revenue? What's your goal? Why this community?

**Waitlist:** When "full," maintain a waitlist. Email waitlist when spot opens. Creates demand signaling.

### Founding Member Strategy

**Launch sequence (14 days):**
- Day 1-3: Announce community on all channels. "Building something. Details soon."
- Day 4-7: Share concept + founding member pricing (40% off for life, first 25 only)
- Day 8-10: Share behind-the-scenes setup. Social proof from beta testers if available
- Day 11-13: "X spots left" countdown posts
- Day 14: Launch. Close founding pricing at 25 members

**Founding member benefits:**
- Locked-in low price for life
- "Founding Member" badge (visible in community)
- Input on community direction (they feel ownership)
- First access to new features/content

### Cross-Promotion Between Our 5 Communities

| From | To | Mechanism |
|------|-----|-----------|
| Faith (Daily Anchor) | Sleep Lab | "Better sleep improves morning prayer focus" |
| Fitness (3-Hour Physique) | Sleep Lab | "Recovery is 80% sleep. Join Sleep Lab for the other half" |
| Sleep Lab | Fitness | "Energy from good sleep needs an outlet. 3 hours/week is all you need" |
| Tech (Inner Circle) | All others | Feature niche communities as case studies. "Here's how Daily Anchor hit $2K/mo in 60 days" |
| NSFW VIP | Tech (anonymized) | Business model case study for Inner Circle (anonymized revenue data) |

**Bundle pricing:** Access all 5 communities for $79/mo (vs $145/mo bought separately). The "PRINTMAXX All Access" pass. Increases LTV, reduces churn.

### Additional Growth Channels

**Affiliate program:** Pay existing members 20-30% recurring for referrals. A member paying $29/mo who refers 3 friends is net positive. Use Skool's built-in affiliate or FirstPromoter for Discord/Telegram.

**Guest expert collaborations:** Invite niche experts for a free live session. Their audience sees the community. 10-20% of attendees convert to free tier. 5% convert to paid within 30 days.

**Content repurposing:** Every live call = 3-5 tweet clips + 1 YouTube short + 1 newsletter section. Community content IS marketing content. Zero Waste Protocol applies.

**SEO play:** Skool communities index on Google. Optimize community name and description for "[niche] community" keywords. Circle with custom domain enables full SEO.

## 5. Content Cadence (Retention Engine)

### Daily (5-10 min creator time)

| Day | Content Type | Example (Fitness) | Example (Faith) |
|-----|-------------|--------------------|--------------------|
| Mon | Prompt/Question | "What's your training split this week?" | "What are you praying for this week?" |
| Tue | Quick tip | "Front squat mobility: 2 min wrist stretch" | "Try box breathing before morning prayer" |
| Wed | Resource drop | Workout PDF or video | Guided prayer audio |
| Thu | Win celebration | "Post your PRs from this week" | "Share one answered prayer" |
| Fri | Discussion | "Unpopular fitness opinion?" | "Which verse hit different this week?" |
| Sat | Free content (public) | Workout posted to Twitter (funnel top) | Prayer prompt posted publicly |
| Sun | Rest/reflection | "Recovery check: sleep, hydration, soreness" | "Weekly reflection: gratitude journal prompt" |

### Weekly (1-2 hours creator time)

- **1x live call** (45-60 min): Q&A, hot seat, workshop, or guest expert
- **1x content drop**: downloadable resource (template, guide, checklist, program update)
- **1x engagement prompt**: poll, challenge, or contest to spike activity

### Monthly (2-3 hours creator time)

- **Monthly recap**: wins from the community, metrics shared, top contributors highlighted
- **New content module**: fresh course/guide added to library
- **Member spotlight**: interview or feature 1-2 members (creates social proof + retention)
- **Price/tier update**: if adding spots, announce publicly. If raising price, notify existing members they're locked in

### Retention Benchmarks

| Metric | Target | Red Flag |
|--------|--------|----------|
| Monthly churn | <8% | >12% |
| Daily active (% of paid) | >30% | <15% |
| Weekly engagement (posts/comments) | >50% of members | <25% |
| Live call attendance | >40% of paid | <20% |
| NPS score | >50 | <30 |

**If churn exceeds 12%:** Survey churned members (automate with Typeform on cancellation). Common fixes: more live interaction, better onboarding (first 7 days critical), accountability matching, content variety.

**Onboarding sequence (first 7 days, automated):**
1. Day 0: Welcome message + intro prompt ("Tell us your goal")
2. Day 1: Point to 3 "start here" resources
3. Day 2: Accountability partner match (or pod assignment)
4. Day 3: First live call invitation
5. Day 5: Check-in DM ("How's your first week?")
6. Day 7: Ask for feedback + testimonial

## 7. Launch Sequence (Zero to First 25 Members)

### Week 1-2: Pre-Launch

- [ ] Choose platform per niche (see decision matrix above)
- [ ] Set up community (channels, welcome flow, content library with 5 starter resources)
- [ ] Create landing page (Carrd.co $19/yr or Skool's built-in page)
- [ ] Write 7-day email onboarding sequence
- [ ] Prepare founding member offer (40% off, first 25)
- [ ] Create 3 teaser posts for social

### Week 3: Soft Launch

- [ ] Invite 5-10 beta members (free, for testimonials and feedback)
- [ ] Run beta for 5 days, iterate on feedback
- [ ] Collect 3-5 testimonials from beta members
- [ ] Prepare launch content (5 tweets, 1 thread, 1 email)

### Week 4: Public Launch

- [ ] Post launch thread on X/Twitter
- [ ] Email full list with founding member offer
- [ ] Post in relevant subreddits (value-first, not promotional)
- [ ] DM 20 warm leads individually
- [ ] Cross-promote in newsletter
- [ ] Goal: 10-25 founding members in first 7 days

### Week 5+: Growth Mode

- [ ] Weekly content cadence running
- [ ] First live call done (record for library)
- [ ] Affiliate program activated (20-30% recurring)
- [ ] Guest expert booked for month 2
- [ ] First member spotlight published
- [ ] Raise price at 25 members (announce publicly)
