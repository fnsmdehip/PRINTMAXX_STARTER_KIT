Questions

Common questions.

What size company is this for?+

Most clients have 10-500 employees and $2M-$100M in revenue. If you have fewer than 5 employees, most of these agents won't save enough time to justify the cost. If you're a Fortune 500, we work with your IT and compliance teams, but the engagement starts the same way: one agent, one workflow, measured results.

Can we start with one agent and scale up later?+

That's what most clients do. Start with an agent install at Tier 1. Run it for a month, see the numbers, then decide if you want to add more agents or move to Tier 2/3. Each agent works independently, so you can add them one at a time. Some clients start with a single agent and grow to a full multi-agent system over 6-12 months.

What if our use case isn't listed?+

If a human does it on a schedule, follows a process, and makes decisions based on data, we can build an agent for it. The agents on our homepage are starting points. We've also built agents for financial reconciliation, regulatory compliance, supply chain coordination, recruiting pipelines, quality assurance, and data processing.

Is Business DNA worth the premium over standard agent install?+

It depends on complexity. If you know exactly what you want automated (e.g., "answer our phones"), a Tier 1 agent install is the right call. Business DNA is for organizations where the workflows are complex, undocumented, or involve institutional knowledge that nobody has written down. Agents built from DNA analysis handle edge cases that described-process agents miss, because they're modeled on what your people actually do, not what they say they do. For large operations where getting it wrong is expensive, the observation phase pays for itself.

What about data security for sensitive industries?+

We offer four deployment models. Standard cloud (encrypted, SOC 2 compliant). Private cloud (your AWS/GCP/Azure). On-premise with open-source models (data never leaves your network). And hybrid (AI reasoning runs in the cloud, but your sensitive data stays on-premise and never leaves your network). For finance, healthcare, legal, and defense, we also offer client-owned model training: we train a model on your data that you own completely. We don't retain copies. Your proprietary information never touches a third-party provider.

Is there a discount for bundling agents?+

Yes. Install 2+ agents together and the combined price drops 15%. Each agent after the first is faster to deploy since the core infrastructure is already in place.

Why not build this in-house?+

You can. Our agent OS, sovrun, is open-source and MIT licensed. If you have AI engineers on staff, start there. Most companies that come to us tried internal builds first and hit two problems: it took 3-4x longer than expected, and the agents worked in demos but broke in production. We've deployed enough agents to know the failure modes. That's what you're buying: fewer dead-end iterations. If you do build in-house, we'll still consult on architecture.

What's the cancellation policy on retainers?+

30 days notice. No penalties, no exit fees. We hand over all documentation, credentials, and monitoring access. Your agents keep running. You own everything. We designed it this way because we'd rather keep clients by being good than by making it hard to leave.
