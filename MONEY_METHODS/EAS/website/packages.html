Questions

Before you pick a tier.

Can we start with one agent and scale up later?+

That's what most clients do. Start with an AI Receptionist or Lead Machine at Tier 1. Run it for a month, see the numbers, then decide if you want to add more agents or move to Tier 2/3. Each agent works independently, so you can add them one at a time.

What if our situation doesn't fit these tiers?+

We scope custom projects regularly. The tiers cover 80% of engagements, but if you need something specific, we'll build a custom proposal. Same fixed-price approach, same delivery timeline. The tiers are starting points, not cages.

Is there a discount for bundling agents?+

Yes. If you install 2+ agents together, we reduce the combined price by 15%. The agents share infrastructure (same workflow engine, same database backend), so we pass the efficiency savings to you.

What's the cancellation policy on Managed Autonomy?+

30 days notice. No penalties, no exit fees. We hand over all documentation, credentials, and monitoring access. Your agents keep running. You own everything. We designed it this way because we'd rather keep clients by being good than by making it hard to leave.
