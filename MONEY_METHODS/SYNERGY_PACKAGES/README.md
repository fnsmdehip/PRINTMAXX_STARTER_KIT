## Pattern Recognition: What Makes a High-Value Synergy?

### Amplification Factors (20-30 points):
- **Double monetization:** Same asset, two revenue streams (TikTok Rewards + Shop)
- **Value ladders:** Ascending offers maximize LTV (newsletter → course → community → coaching)
- **Cross-selling:** Same customer, multiple products (voice AI → clinic SaaS)
- **Platform arbitrage:** Zero marginal cost distribution (cross-post to FB Reels)

### Implementation Efficiency (15-25 points):
- **Solo executable:** One person can run entire system
- **Automation-friendly:** AI, scripts, or tools handle heavy lifting
- **Reusable components:** Build once, deploy many (paywall optimization across 30 apps)
- **Low coordination:** No dependencies on others (vs agency model)

### Audience Overlap (15-20 points):
- **100% overlap:** Same buyer needs both products (clinic needs voice AI + SaaS)
- **Natural progression:** One offering leads to next (app → course → coaching)
- **Same pain point:** Multiple solutions for one problem (sleep content + app + affiliate)

### Risk Reduction (10-15 points):
- **Diversified revenue:** Multiple streams from one customer
- **Platform independence:** Not reliant on single platform (TikTok + FB + YouTube)
- **Niche flexibility:** If one niche fails, pivot to another (dental → physio → med spa)
- **Kill switches:** Can abandon quickly if not working (presale validation)

### Discovery Uniqueness (5-10 points):
- **Non-obvious combination:** Competitors haven't thought of it
- **First-mover advantage:** New platform or ecosystem (MCP, Bluesky)
- **Underserved niche:** Market exists but poorly served (autism apps, menopause tracking)
- **Novel application:** Known tactic applied to new context (clipper model for TikTok Shop)
