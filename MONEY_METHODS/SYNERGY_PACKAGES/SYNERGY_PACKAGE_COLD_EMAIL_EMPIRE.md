## Why This Stack Works

One cold email infrastructure serves multiple revenue streams. Apollo sources contacts. Clay enriches with intent signals (job changes, tech stack, funding). Instantly sends with domain rotation. LinkedIn warms via Dux-Soup.

The same pipeline targets: (1) local businesses for web redesign ($500-$2K one-time) then upsells to Voice AI ($500-$1,500/mo recurring), (2) B2B prospects who get nurtured into a newsletter then hit with flash sales, (3) government contractors identified via FOIA with redesign offers, (4) agency clients via FOIA + LinkedIn intel.

Build once, run for all services. The web redesign pipeline alone generates $3K-$5K/mo within 60 days.

## Implementation Sequence

### Week 1: Infrastructure Setup
1. Apollo account setup, build ICP lists (industry, company size, title, tech stack)
2. Clay workspace with intent triggers (job changes, tech stack signals, funding rounds)
3. Purchase 5+ domains for Instantly sending
4. Configure Instantly for warmup (21 days minimum)
5. Set up Dux-Soup for LinkedIn automation

### Week 2: First Sequences
6. Build 5-email sequence for web redesign: scrape bad websites, auto-generate AI preview
7. Build 3-email sequence for Voice AI: "You missed 23 calls last month"
8. Build 5 value-first email sequences (no selling, pure insight) for newsletter
9. Start sending at 20-30 emails/day per domain (warmup)
10. LinkedIn: connect with local business owners via Dux-Soup

### Week 3: Scale + Revenue
11. Increase sending to 50-100/day as domains warm
12. Close first web redesign deals at $500-$2K
13. During redesign onboarding, pitch Voice AI at $500-$1,500/mo
14. Newsletter list growing from cold email opt-ins
15. File first FOIA requests for government vendor contracts

### Week 4: Flash Sale + Expansion
16. At 200+ newsletter subscribers, create first digital product ($47-$97)
17. Send 15 value emails, then execute first flash sale
18. FOIA results arrive: cross-reference with Clay data
19. Build contractor directory with premium listings ($200-$500/mo)
20. Analyze results, build second product based on what sold
