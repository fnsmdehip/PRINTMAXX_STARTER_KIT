```

"Write a tactical post for [niche] about [topic].
Format: Hook → 3 bullet insights → CTA
Style: @pipelineabuser (specific numbers, no fluff)
Max 280 characters for X."
```

### Weekly Newsletter (AI-Generated):
```
Prompt Template:
"Write a 500-word newsletter for [niche] about [topic].
Structure: Story hook → Framework → Action steps
Include: Case study, specific numbers, 1 CTA (course/cohort/community)
Style: Conversational, tactical, no AI words."
```

### Weekly YouTube Video (AI Script):
```
Prompt Template:
"Write a 10-minute YouTube script for [niche] about [topic].
Structure: Hook (0-30s) → Problem (1-3min) → Framework (3-8min) → CTA (8-10min)
Include: On-screen text, B-roll suggestions, timestamps
Style: Educational, not salesy until final CTA."
```
