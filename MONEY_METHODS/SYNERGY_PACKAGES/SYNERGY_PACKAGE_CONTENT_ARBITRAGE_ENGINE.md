## Why This Stack Works

One piece of content gets posted to TikTok, Facebook Reels, and YouTube Shorts simultaneously. Same video, three platforms, zero marginal cost per additional platform. TikTok Shop affiliate links on every video add 10-30% commission on product sales.

n8n orchestrates the full pipeline: trending topic detected, Claude API reformats per platform spec, Opus Clip generates video clips, Buffer schedules across all accounts. One solopreneur managing 13 content niches across 4 platforms = 52 content streams.

A video getting 100K views across three platforms earns: $40-$600 from TikTok + $20-$400 from FB Reels + $1-$6 from Shorts + $200-$3,000 from TikTok Shop. Volume play: 3 videos/day across 3 platforms = 9 placements/day.

## Implementation Sequence

### Week 1: Content Engine Setup
1. Choose 3 content niches (motivation, finance tips, sleep/relaxation)
2. Set up n8n on $5/mo VPS
3. Build n8n workflow: RSS/trending input to Claude API reformatting
4. Create Buffer account, connect platforms
5. Apply for TikTok Creativity Program (1+ min format)

### Week 2: Production Pipeline
6. Batch create 30 videos using Kling AI + stock footage
7. Create 5 faceless TikTok accounts across niches
8. Apply for TikTok Shop affiliate on all accounts
9. Set up Opus Clip Pro for auto-clipping long-form
10. Connect n8n to Buffer API for automated scheduling

### Week 3: Launch + Distribution
11. Post 3x/day to TikTok, auto-crosspost to FB Reels and Shorts
12. Add TikTok Shop links to every video
13. Track RPMs per platform per content type
14. Build directory site for best-performing niche (100+ listings)
15. Launch weekly newsletter with top-performing content roundup

### Week 4: Optimize + Scale
16. Double down on highest-RPM content types
17. Kill underperforming accounts/niches
18. Add newsletter sponsorship slots after 1,000+ subscribers
19. Launch premium directory listings ($100-$500/mo)
20. Scale to 5-10 content accounts per platform
