## Risk Mitigation

### Technical Risks:
1. **Voice AI accuracy:** Start with narrow use case (hours, insurance, scheduling). Expand after validation.
2. **Integration complexity:** Use Zapier/Make for MVP integrations. Build custom later.
3. **Downtime:** SLA 99.5% uptime. Use Twilio's redundancy.

### Market Risks:
1. **Competition:** Large players (Zocdoc, Weave) are slow. You're fast and niche-focused.
2. **Regulation:** HIPAA compliance required. Use BAA with Vapi/Twilio. Store no PHI initially.
3. **Adoption resistance:** Start with tech-forward clinics (younger owners, recently opened).

### Financial Risks:
1. **Churn:** Offer 3-month pilot with exit clause. Track NPS. Fix issues fast.
2. **CAC payback:** Target 3-month payback. With $2.5K MRR per customer and $1K CAC, this works.
3. **Capital constraints:** Bootstrap to $10K MRR, then consider RBF ($50K-100K) to scale outbound.
