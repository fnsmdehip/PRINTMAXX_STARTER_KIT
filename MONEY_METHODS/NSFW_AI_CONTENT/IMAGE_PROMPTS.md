## Prompt Engineering Notes

**Base quality modifiers (append to ALL prompts):**
```
photorealistic, 85mm lens, natural lighting, shallow depth of field, high resolution, detailed skin texture, professional photography, shot on Canon R5
```

**Negative prompt (for SD/Flux, not needed for Midjourney):**
```
cartoon, anime, illustration, painting, sketch, deformed, distorted, disfigured, bad anatomy, extra limbs, blurry, low quality, watermark, text, signature
```

**Consistency tips:**
- Use the same seed number per character to maintain facial consistency
- Reference previous successful outputs with --sref (style reference) in Midjourney
- Save character reference images and use --cref for face/body consistency
- Midjourney v6.1+ has best photorealism. Use --v 6.1 or latest

**Aspect ratios:**
- Portrait: --ar 3:4 or --ar 4:5 (Instagram/Fanvue standard)
- Landscape: --ar 16:9 (Twitter header, cinematic shots)
- Square: --ar 1:1 (Twitter posts, profile photos)

## Prompt Modification Guide (for VA)

### Adjusting for Different Platforms

**Fanvue/Fansly (paid content):**
- Use prompts as-is for standard content
- For premium sets: add "editorial fashion photography" and "high-end studio lighting" to elevate quality
- For PPV: create themed variations by changing wardrobe/setting while keeping the character consistent

**Reddit (free teaser content):**
- Use lifestyle and casual prompts (less editorial, more candid)
- Output images will be watermarked "AI Generated" before posting
- Crop or select single images from sets for individual Reddit posts

**Twitter (engagement content):**
- Square crops (--ar 1:1) work best for Twitter feed
- Close-up portraits and selfie-angle shots perform best
- Add "viral social media photo" to prompts for a more casual feel

### Quick Wardrobe Swaps

Replace wardrobe descriptions in any prompt to create variety:
- **Workout:** "sports bra and leggings" / "running shorts and tank top" / "yoga outfit"
- **Casual:** "oversized sweater and jeans" / "sundress" / "crop top and shorts"
- **Going out:** "fitted dress" / "blazer and jeans" / "jumpsuit"
- **Cozy:** "oversized hoodie" / "pajama set" / "silk robe"

### Setting Swaps

Replace setting descriptions for fresh content from the same prompt:
- **Indoor:** "modern apartment" / "coffee shop" / "gym" / "studio"
- **Outdoor:** "beach" / "park" / "rooftop" / "city street"
- **Moody:** "candlelit room" / "rainy window" / "dark studio" / "neon-lit"
- **Bright:** "sunlit kitchen" / "poolside" / "flower field" / "white studio"

### Batch Generation Workflow

1. Open Midjourney or Flux interface
2. Set character reference (--cref) using best previous generation for that character
3. Run 5-10 prompts per session, selecting best outputs
4. Upscale winners to maximum resolution
5. Apply "AI Generated" watermark in bottom right corner (semi-transparent)
6. Organize into folders by character > content type > date
7. Repeat for next character
8. Full month of content (150 images) takes approximately 4-6 hours
