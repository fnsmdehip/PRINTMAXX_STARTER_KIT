## Character Development Rules

1. **Consistency is everything.** Each character has a defined personality. Never break character in posts or DMs.
2. **Visual continuity.** Use consistent Midjourney/SD parameters per character. Same face, same body type, same distinguishing features.
3. **Hair/wardrobe changes are fine** but core features (face, body type, skin tone, distinguishing marks) must stay consistent.
4. **Voice must match visual.** Don't use a bubbly voice for Elara or a mysterious tone for Kai.
5. **No real-world claims.** Never imply these characters can meet subscribers in person, have real jobs, or exist outside the digital space.
6. **Disclosure on every new platform, profile, and first interaction.** No exceptions.
