
- [ ] Download `.ipa` file from Expo dashboard

### Transporter Upload

- [ ] Open Transporter app (macOS)

- [ ] Assign build to version 1.0.0

## Human Blockers (Manual Tasks Required)

### 1. Apple Developer Account
**Cost:** $99/year
**Action:** Enroll at https://developer.apple.com/programs/enroll/
**Timeline:** 24-48 hours for approval
**Required for:** App Store submission, TestFlight, push notifications

### 2. Privacy Policy URL
**Action:** Generate privacy policy and host at `https://biomaxx.app/privacy`

**Options:**
- Use privacy policy generator (freeprivacypolicy.com, termly.io)
- Manual hosting on GitHub Pages (free)
- Add to landing page (if you build one)

**Required disclosures:**
- Data collection: None (all local storage)
- Third-party services: None
- User rights: Data stays on device, deleted when app deleted
- Contact email for privacy questions

### 3. Terms of Service URL
**Action:** Generate ToS and host at `https://biomaxx.app/terms`

**Required terms:**
- Subscription auto-renewal policy
- Refund policy (Apple handles, but state policy)
- Acceptable use (no reverse engineering, etc.)
- Disclaimer: Not medical advice
- Limitation of liability

### 4. App Store Connect Setup
**Action:** Create app listing in App Store Connect

**Steps:**
1. Log in to https://appstoreconnect.apple.com
2. Go to "My Apps" > "+" > "New App"
3. Select iOS platform
4. Enter app name: `biomaxx`
5. Select primary language: English (U.S.)
6. Enter Bundle ID: `com.biomaxx.app` (must match Xcode project)
7. SKU: `biomaxx-001` (internal tracking ID)

### 5. Generate Screenshots
**Action:** Capture 6 screenshots from Expo app using iPhone 15 Pro Max simulator

**Steps:**
1. Run app in Xcode simulator (iPhone 15 Pro Max)
2. Navigate to each screen from screenshot descriptions above
3. Cmd+S to save screenshot (auto-saves to Desktop)
4. Resize if needed: 1290 x 2796 px
5. Add text overlays with header/subheader (use Figma, Canva, or Photoshop)
6. Export as PNG (max 500KB per file)

**Optional:** Use screenshot frames (https://www.screely.com or Figma mockups)

### 6. Subscription Configuration in App Store Connect
**Action:** Create subscription products

**Steps:**
1. App Store Connect > My Apps > biomaxx > Subscriptions
2. Create Subscription Group: `biomaxx Premium`
3. Add Product 1:
   - Reference Name: `biomaxx Monthly`
   - Product ID: `com.biomaxx.app.monthly`
   - Duration: 1 month
   - Price: $4.99
   - Free Trial: 7 days
4. Add Product 2:
   - Reference Name: `biomaxx Annual`
   - Product ID: `com.biomaxx.app.annual`
   - Duration: 1 year
   - Price: $29.99
   - Free Trial: 7 days
5. Set Subscription Display Name: `biomaxx Premium`
6. Add Subscription Description: `Unlock all circadian tracking features and science-backed protocols.`

### 7. Upload Build via Transporter
**Action:** Upload signed `.ipa` to App Store Connect

**Steps:**
1. Download Transporter from Mac App Store (free)
2. Download `.ipa` from Expo build dashboard
3. Open Transporter, sign in with Apple ID
4. Drag `.ipa` file into Transporter
5. Wait for validation and upload (5-15 min)
6. Check App Store Connect > TestFlight for build

### 8. Submit for Review
**Action:** Click "Submit for Review" in App Store Connect

**Timeline:**
- Review time: 24-48 hours (average)
- Rejection common on first try (respond to feedback, resubmit)
- Approval → live in 1-2 hours

## Common Rejection Reasons & Fixes

### 1. "Insufficient App Description"
**Fix:** Expand description to 1000+ characters, detail all features

### 2. "Subscription Terms Not Clear"
**Fix:** Add auto-renewal disclosure to app description (already included above)

### 3. "Privacy Policy Missing or Inadequate"
**Fix:** Ensure privacy policy URL is live and covers all App Store requirements

### 4. "App Crashes on Launch"
**Fix:** Test on real device (not just simulator) before submission

### 5. "Misleading Health Claims"
**Fix:** Remove "cure," "treat," "diagnose" language. Add researcher attribution.

### 6. "Subscription Paywall Too Aggressive"
**Fix:** Ensure users can browse features before hitting paywall (we already have 7-day trial)
