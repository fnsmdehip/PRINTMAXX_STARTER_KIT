## Phase 1: Roblox SEO (Day 1 - Free)

Roblox's algorithm prioritizes games with good metadata and early retention.

### Game Title Optimization

**Format:** [Hook] [Genre] [Unique Element]

**Good examples:**
- "AI Factory Tycoon ⚡ Build AI Empire"
- "Tech Tycoon 🤖 AI Company Simulator"
- "Quantum Computer Tycoon 💻 Get Rich"

**Why this works:**
- Hook = grabbing attention (emoji helps)
- Genre = clear expectation (tycoon)
- Unique = differentiates from other tycoons

**Bad examples:**
- "Tycoon Game" (too generic)
- "My Cool Game" (no genre signal)
- "aifactorytycoonv1" (no spaces, unclear)

### Game Description

**Template:**
```
[Hook sentence with benefit]

🎯 [Feature 1]
🚀 [Feature 2]
💰 [Feature 3]
⚡ [Feature 4]

[Social proof or update schedule]
```

**Example:**
```
Build your AI empire from a basic server to a quantum supercomputer. Start small, upgrade smart, dominate the leaderboard.

🎯 10+ unique upgrades (servers → quantum computers → AGI)
🚀 Compete on global leaderboard
💰 Buy gamepasses for 2x speed and auto-collect
⚡ New updates every week

Join 10K+ players building their AI companies!
```

**Why this works:**
- Benefit-first (what player gets)
- Scannable bullets (easy to skim)
- Social proof (10K players = social validation)
- Update promise (shows active development)

### Tags (Max 5)

Choose tags that match your game + have high search volume:

**Primary tags:**
1. Tycoon (highest volume for this genre)
2. Simulator (adjacent genre, more discovery)
3. Building (gameplay type)

**Secondary tags:**
4. Tech (theme differentiation)
5. Multiplayer (feature)

**Check tag performance:** Roblox search bar autocomplete shows popular terms.

### Thumbnail Design

**Dimensions:** 1920x1080 (16:9)
**File size:** <5MB

**Winning formula:**
1. Bright, contrasting colors (blue/yellow, green/purple)
2. Large text (readable on mobile)
3. Visual hierarchy (biggest = most important)
4. Show the gameplay (tycoon buildings, money, upgrades)
5. Avoid clutter (3-4 elements max)

**Example layout:**
- Background: Screenshot of tycoon with glowing machines
- Text: "AI TYCOON" (huge, bold, white with black outline)
- Subtext: "Build → Upgrade → Dominate" (smaller, bottom)
- Element: Money stack or leaderboard icon (top right)

**Tools:**
- Photoshop (professional)
- Figma (free alternative)
- Canva (easiest, templates available)

**A/B test thumbnails:** Change every 3-7 days, track CTR in Analytics.

### Icon Design

**Dimensions:** 512x512 (square)
**File size:** <2MB

**Winning formula:**
- Simple, recognizable shape (one object)
- High contrast (stands out in search results)
- Clear at small sizes (will be shown at 50x50)

**Example ideas:**
- Glowing blue server rack (tech theme)
- Robot head with circuit pattern (AI theme)
- Dollar sign made of circuits (money + tech)

**Test:** Shrink to 50x50 pixels. Still recognizable? Good. Blurry? Simplify.

## Phase 2: Roblox Advertising (Week 1 - $10-50 Budget)

Roblox native ads = highest ROI for cold start (better than TikTok ads for Roblox games).

### Ad Types

**Sponsored Games (best ROI):**
- Your game shows at top of Home/Discover page
- Bid per impression (CPM model)
- Budget: $10-20 initial test

**Banner Ads (lower ROI):**
- 728x90 banner on Roblox website
- Good for retargeting (later stage)
- Skip for now

### How to Run Sponsored Game Ad

**Step 1: Create Ad Campaign**
1. Go to roblox.com/develop
2. Click your game → Advertising
3. Select "Sponsored Games"
4. Choose targeting:
   - Age: 8-16 (core demo)
   - Gender: All
   - Device: All (but prioritize mobile - 60% of Roblox traffic)
5. Set daily budget: $10/day
6. Duration: 3 days ($30 total test)

**Step 2: Set Bid**
- Start with $0.01 CPM (cost per 1000 impressions)
- Increase if not getting impressions (check every 6 hours)
- Goal: 50K+ impressions for $30 = 150+ clicks

**Step 3: Track Performance**
- CTR (click-through rate): target 0.5-1%
- CPI (cost per install/visit): target $0.20-0.40
- ROAS (return on ad spend): track revenue from ad cohort

**Optimization:**
- If CTR <0.3%: change thumbnail (too generic)
- If CTR >1% but no retention: improve game (first 5 minutes critical)
- If retention good (30%+ D1): scale budget to $50-100/day

### Ad Schedule

**Best times to run ads:**
- Thursday-Sunday (highest Roblox traffic)
- 3pm-9pm EST (after school hours)
- Avoid: Monday-Tuesday morning (lowest traffic)

**Initial test schedule:**
- Friday 3pm - Sunday 9pm (54 hours)
- Budget: $30 total
- Expected: 100-300 new players

## Phase 3: Social Media (Week 1-4 - Free)

Drive external traffic to Roblox game. TikTok and YouTube Shorts = highest ROI for Roblox games.

### TikTok Strategy

**Content types that work:**

**1. Gameplay clips (15-30 seconds)**
- Show satisfying progression (upgrading, money collecting)
- Add trending sound
- Hook: "POV: you're building an AI empire in Roblox"
- CTA: "Game in bio" or "Search AI Factory Tycoon"

**2. Progression timelapses**
- Start: basic server rack
- End: massive quantum computer complex
- Trending sound: phonk or fast-paced EDM
- Text overlay: "$100 → $1M in 10 minutes"

**3. Gamepass "secrets"**
- Show VIP gamepass benefits
- "This gamepass makes you 2x faster"
- Creates urgency (FOMO)

**4. Competitor comparisons**
- "AI Tycoon vs Lumber Tycoon 2"
- Show why yours is better (faster progression, cooler theme)

**Posting schedule:**
- 2-3 clips per day
- Best times: 6pm-9pm (US time zones)
- Use hashtags: #Roblox #RobloxTycoon #AITycoon #GamingTikTok

**Expected results:**
- 100-500 views per video initially
- 1/20 videos goes viral (10K+ views)
- Viral video = 50-200 game visits

### YouTube Shorts Strategy

**Same content as TikTok, but:**
- Longer shelf life (Shorts recommended for weeks)
- Higher monetization (YouTube Shorts Fund)
- Better SEO (searchable)

**Video titles:**
- "I Built a $1M AI Company in Roblox"
- "This Roblox Tycoon Gamepass is OP"
- "Richest Player in AI Factory Tycoon"

**Thumbnails:**
- Bright colors, big text
- Show money or expensive upgrades
- Your avatar with shocked expression (classic YouTube)

**Posting schedule:**
- 1-2 Shorts per day
- Repost TikTok top performers

### Twitter/X Strategy

**Who to target:**
- Roblox dev community (#RobloxDev)
- Indie game devs (cross-promotion)
- Roblox players (search "roblox tycoon" and reply)

**Content:**
- Behind-the-scenes (code snippets, design decisions)
- Revenue updates ("$50 revenue in first week")
- Player milestones ("100 players online!")
- Ask for feedback (engage community)

**Growth tactic:**
- Reply to big Roblox dev accounts (AlvinBlox, TheDevKing)
- Share your game in #RobloxDev hashtag
- DM Roblox YouTubers offering free Robux for coverage

## Phase 5: Influencer Outreach (Week 3-4)

Get Roblox YouTubers to play your game = massive traffic spike.

### Target YouTubers

**Tier 1: Micro (10K-100K subs)**
- More likely to respond
- Cheaper rates
- Still drives 500-2K visits per video

**Tier 2: Mid (100K-1M subs)**
- Selective about games
- Expensive ($100-500 per video)
- Drives 5K-20K visits

**Tier 3: Macro (1M+ subs)**
- Very selective
- Expensive ($500-5K)
- Skip unless you have budget

### Outreach Template

**Email/DM:**
```
Hey [Name],

I'm the dev of AI Factory Tycoon, a new tycoon game on Roblox. I think your audience would love it (tech theme, fast progression).

Would you be interested in making a video? I can offer:
- [X] Robux for your time
- Early access to new updates
- Your name in credits

Here's the game: [link]
Let me know if interested!

[Your name]
```

**Rates:**
- Micro (10K-100K): $50-200 in Robux
- Mid (100K-1M): $200-500 in Robux

**Expected ROI:**
- 10K subscriber YouTuber → 500 visits → 10-30 paying players → $10-50 revenue
- Break even on micro influencers, profit on virality

## Phase 6: Retention Optimization (Ongoing)

Players must come back or revenue dies. Optimize for D1 retention (% who return next day).

### Metrics to Track

**In Roblox Analytics:**
- DAU (daily active users)
- D1 retention (target: 30%+)
- Average session time (target: 15+ minutes)
- Play time per user (target: 30+ minutes total)

**Benchmarks:**
| Metric | Poor | Good | Great |
|--------|------|------|-------|
| D1 Retention | <20% | 30-40% | 40%+ |
| D7 Retention | <10% | 15-25% | 25%+ |
| Avg Session | <10 min | 15-20 min | 20+ min |
| Conversion | <1% | 2-5% | 5%+ |

### Retention Tactics

**Daily Rewards:**
- Day 1: $5K cash
- Day 2: $10K cash
- Day 3: Free gamepass
- Creates habit loop (return daily)

**Time-gated content:**
- Upgrades take 30 seconds to build
- Forces player to return later
- Can skip with "Skip Timer" developer product

**Events:**
- "2x Money Weekend" (every 2 weeks)
- "New Upgrade Unlocked" (monthly updates)
- Creates urgency to play NOW

**Social features:**
- Visit friends' tycoons
- Leaderboard competition
- "Send Gift" (cash to friends)

**Push notifications:**
- "Your tycoon earned $50K while offline!"
- "New upgrade available!"
- Roblox handles this via their notification system

## Phase 7: Scaling Ads (Month 2+)

Once retention is good (30%+ D1), scale ad spend.

### When to Scale

**Required metrics:**
- D1 retention: 30%+
- Conversion rate: 2%+
- ROAS: 2x+ (earn $2 for every $1 spent on ads)

**If metrics not met:** Focus on game improvements, not ads. Don't spend on ads if retention is poor.

### Budget Progression

**Week 1-2:** $30 test (prove concept)
**Week 3-4:** $100 if ROAS >1.5x
**Month 2:** $300-500 if ROAS >2x
**Month 3+:** $1K+ if sustainable

### Diversify Channels

**After Roblox ads proven:**
- TikTok ads (target: Roblox players, gaming interest)
- YouTube pre-roll ads (target: Roblox videos)
- Influencer partnerships (revenue share deals)

**Avoid:**
- Google Display Network (low quality traffic)
- Facebook ads (wrong demo for Roblox)

## Phase 8: Content Updates (Ongoing)

Stale games die. Update weekly to stay in algorithm.

### Update Cadence

**Weekly (small updates):**
- New upgrade (add Button11, Button12, etc)
- Bug fixes
- Balance tweaks (adjust prices if too hard/easy)

**Monthly (big updates):**
- New feature (rebirth system, quests, pets)
- New area (expand tycoon map)
- Limited-time event (holiday theme)

**Announce updates:**
- Roblox group post
- Discord announcement
- In-game notification
- Social media (TikTok, Twitter)

### Feature Roadmap

**Month 1:** Launch, iterate on core loop
**Month 2:** Rebirth system (prestige for permanent multipliers)
**Month 3:** Quests/achievements
**Month 4:** Pets system (collect pets that boost earnings)
**Month 5:** Trading system (trade upgrades with friends)
**Month 6:** PvP mode (compete directly)

Each new feature = traffic spike from Roblox algorithm (shows in "Updated" section).

## Common Mistakes to Avoid

### Marketing Mistakes

1. **Spending on ads before retention is good**
   - Fix: Get 30%+ D1 retention BEFORE scaling ads

2. **Generic thumbnail/title**
   - Fix: Stand out with unique theme, bright colors, clear text

3. **No social proof**
   - Fix: Add "10K+ players" to description once you hit it

4. **Inconsistent content**
   - Fix: Post 2-3 TikToks daily for 30 days minimum

5. **Ignoring analytics**
   - Fix: Check Roblox Analytics daily, iterate on data

### Game Mistakes

1. **Tutorial too long**
   - Fix: Players should be collecting money within 10 seconds

2. **Progression too slow**
   - Fix: First 5 upgrades should be buyable in 5 minutes

3. **Monetization feels required**
   - Fix: Game should be beatable without spending (gamepasses = convenience)

4. **No clear goal**
   - Fix: Add leaderboard, richest player competition, final upgrade

5. **Bugs on launch**
   - Fix: Test with 5+ friends before public launch
