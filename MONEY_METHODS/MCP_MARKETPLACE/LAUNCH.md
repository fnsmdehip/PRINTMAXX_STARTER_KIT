## week 1 - launch

### day 1 - Product Hunt
- submit to Product Hunt
- title: "MCPHub - find the MCP server you need in 30 seconds"
- tagline: "the directory for Model Context Protocol servers"
- post at 12:01 AM PT (Product Hunt resets midnight PT)
- prepare 3-4 screenshots: hero, search, category filter, install command copy
- have 5-10 people ready to upvote + leave genuine comments in first hour

### day 1 - Hacker News
- submit as "Show HN: MCPHub, a searchable directory of MCP servers"
- post between 8-9 AM ET (peak HN traffic)
- don't shill. let the product speak.
- respond to every comment in first 2 hours

### day 1-2 - Reddit
- r/ClaudeAI - "i built a directory of 50+ MCP servers, searchable with one-click install commands"
- r/LocalLLaMA - same post, angle toward open-source/self-hosted servers
- r/artificial - broader AI audience
- r/SideProject - builder audience
- don't crosspost. write unique copy for each sub. match the tone.

### day 2-3 - Twitter/X
- thread: "i indexed 50+ MCP servers into a searchable directory. here are the 10 best ones nobody talks about."
- tag @AnthropicAI, @ClaudeAI
- engage in MCP conversations, drop the link naturally
- DM 10 MCP server authors, tell them their server is listed

### day 3-5 - Discord
- Claude Code Discord - share in appropriate channel
- Cursor Discord - MCP users there too
- AI/ML Discords - anywhere MCP is discussed

## week 2-4 - outreach

### server author outreach
- find every MCP server on GitHub (search topic:mcp-server, topic:mcp)
- email/DM each author: "hey, your server is listed on MCPHub. want to update the listing or add anything?"
- this does two things: validates the listing AND gets the author to share it
- target: 20 outreach messages per day

### content
- write 2 blog posts (host on mcphub.dev/blog or Medium):
  1. "the 10 most useful MCP servers for developers"
  2. "how to build an MCP server in 30 minutes"
- turn each into a Twitter thread
- submit blog posts to dev.to, Hashnode

### partnerships
- reach out to awesome-mcp list maintainers: "want to link to MCPHub as a searchable version?"
- reach out to AI newsletter authors: include MCPHub in their tools roundup
