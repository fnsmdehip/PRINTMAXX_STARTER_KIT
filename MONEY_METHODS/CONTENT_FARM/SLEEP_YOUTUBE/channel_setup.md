# Sleep YouTube Channel Setup Guide

## Channel Name Options

Test availability by searching YouTube and checking domain availability (.com):

1. **Deep Sleep Sounds** - Straightforward, SEO-friendly, trust signal
2. **8 Hour Sleep** - Numeric differentiation, clear value prop
3. **Pure Sleep Audio** - Clean, professional, quality signal
4. **Sleep Sound Library** - Expansive catalog feel, authority positioning
5. **Ambient Sleep Lab** - Slightly unique, sciencey trust signal

**Availability Check Pattern:**
- YouTube search: `"[name]" site:youtube.com`
- Must have <5 exact matches with significant subscribers
- Available .com domain is bonus (not required but good for credibility)

**Recommended:** Deep Sleep Sounds (clearest positioning, best SEO)

## Default Upload Settings

Configure these in YouTube Studio → Settings → Upload Defaults:

**Visibility:** Public (or Unlisted during batch upload phase, then switch to Public on schedule)

**Category:** People & Blogs (better monetization than Music for sleep content)

**Comments:** Hold potentially inappropriate comments for review (reduces spam moderation)

**Ratings:** Show how many viewers like/dislike this video (standard)

**Audience:** Not made for kids (sleep content is primarily for adults, enables all features)

**Automatic Chapters:** Off (manual chapters in description for better control)

**Allow Embedding:** Yes (increases reach)

**Publish to Subscriptions Feed:** Yes

**Default Tags (add these to all videos):**
