Complete list of sources for high-quality, legally usable ambient audio.

## Free Sources (Zero Cost)

### 1. Pixabay (Music & Sound Effects)
**URL:** https://pixabay.com/sound-effects/
**License:** Pixabay License (free for commercial use, no attribution required)
**Best For:** Rain, thunder, ocean waves, nature sounds, white noise

**Search Terms:**
- "rain sound"
- "ocean waves"
- "thunderstorm"
- "forest night"
- "white noise"
- "brown noise"
- "fireplace crackling"
- "airplane cabin"

**Pro Tips:**
- Filter by duration (select 1+ minute for easier looping)
- Download WAV format for highest quality
- Check "Sound Effects" category, not "Music"
- Sort by "Popular" to find best quality recordings

**Download Process:**
1. Search for sound type
2. Preview to verify quality
3. Click "Free Download"
4. No account required (but recommended for download history)

---

### 2. Freesound.org
**URL:** https://freesound.org/
**License:** Mix of CC0 (public domain), CC-BY (attribution required), CC-BY-NC (non-commercial)
**Best For:** Specific nature sounds, ambient recordings, field recordings

**Search Terms:**
- "rain forest"
- "ocean beach"
- "crickets night"
- "washing machine"
- "fan white noise"
- "library ambience"
- "airplane interior"

**Pro Tips:**
- FILTER BY LICENSE: Use "CC0" for no attribution, or "CC-BY" and add credit in description
- Avoid "CC-BY-NC" (non-commercial only - NOT suitable for monetized YouTube)
- Create free account (required for downloads)
- Download highest quality available (typically 192kbps+)
- Check "duration" filter for longer files (easier looping)

**Attribution Format (if using CC-BY):**
```
Sound: [Sound Name] by [Username] (freesound.org/people/[username]/sounds/[id])
License: CC-BY 4.0
```

## DIY: Record Your Own (Free, Fully Owned)

### 11. Record Real Sounds
**Equipment Needed:**
- Smartphone with decent microphone (iPhone, Android flagship)
- OR portable audio recorder ($50-200: Zoom H1n, Tascam DR-05X)
- Optional: Windscreen for outdoor recording

**What to Record:**
- Rain on your roof/window
- Thunderstorms from covered porch
- Fan in your room
- Washing machine/dryer
- Fireplace crackling
- Coffee shop ambience (with permission)
- Forest sounds during camping

**Recording Tips:**
- Record in WAV or highest quality MP3 (320kbps minimum)
- Record at least 5-10 minutes (easier to find perfect loop)
- Steady position (use tripod or stable surface)
- Avoid handling noise
- Test recording first (listen with headphones)

**Legal Note:**
- Your own recordings = you own full copyright
- No attribution needed
- Safe for monetization
- Can sell/license to others

### 12. Generate Your Own White/Brown/Pink Noise
**Tool:** Audacity (free audio software)
**URL:** https://audacityteam.org/

**How to Generate:**
1. Open Audacity
2. Generate → Noise
3. Select type: White, Pink, or Brown
4. Set duration (e.g., 10 minutes)
5. Export as WAV or MP3

**Customize:**
- Apply low-pass filter for softer sound
- Apply fade in/out for seamless looping
- Adjust amplitude (volume)
- Add subtle modulation (very gentle LFO)

**License:**
- Noise you generate = public domain
- Fully owned by you
- Safe for commercial use

## Legal Safety Checklist

Before using any audio:

- [ ] License allows commercial use (YouTube monetization = commercial)
- [ ] License allows redistribution in video format (most do)
- [ ] Attribution added to description if required
- [ ] No "non-commercial" (NC) licenses used
- [ ] No music samples from copyrighted songs
- [ ] If in doubt, use CC0 or public domain only

**Red Flags:**
- "Personal use only" = NOT OK
- "Non-commercial" = NOT OK for monetized YouTube
- No clear license = DON'T USE

---

## Quality Checklist for Downloaded Audio

Before using in video:

- [ ] No clipping or distortion
- [ ] Consistent volume throughout
- [ ] No abrupt cuts that will be obvious when looped
- [ ] No background noise (voices, traffic, etc.) unless intentional
- [ ] Bitrate 192kbps+ (320kbps or WAV preferred)
- [ ] Long enough for clean loop (5+ minutes ideal)

**Test Loop:**
1. Import into video editor
2. Place clip twice back-to-back
3. Play across the seam - should be seamless
4. Adjust start/end points if needed
5. Add crossfade (0.5-1 second) to smooth transition

Create unique sounds by layering:

**Example 1: Enhanced Thunderstorm**
- Layer 1: Heavy rain (base)
- Layer 2: Distant thunder (occasional)
- Layer 3: Wind through trees (subtle)

**Example 2: Cozy Cabin**
- Layer 1: Fireplace crackling

**Example 3: Perfect Sleep**
- Layer 1: Brown noise (base)
- Layer 2: Very quiet ocean waves (subliminal)

**Layering Tool:** Audacity (free)
1. Import first audio file
2. Tracks → Add New → Audio Track
3. File → Import → Audio (import second file)
4. Adjust volumes (one should be dominant, others subtle)
5. File → Export → Export as MP3

## Audio Editing Quick Guide

**Tool:** Audacity (free, cross-platform)

**Common Edits:**

**1. Create Seamless Loop:**
- Select last 1 second of audio
- Effect → Fade Out
- Select first 1 second of audio
- Effect → Fade In
- Export

**2. Remove Background Noise:**
- Select section with ONLY noise (no desired sound)
- Effect → Noise Reduction → Get Noise Profile
- Select all (Ctrl+A)
- Effect → Noise Reduction → OK
- Preview to ensure not too aggressive

**3. Normalize Volume:**
- Select all (Ctrl+A)
- Effect → Normalize → Set to -3 dB
- OK

**4. Generate Alarm Sound:**
- Generate → Tone → Sine wave → 440 Hz → 10 seconds
- Effect → Fade In (8 seconds)
- Effect → Amplify (-12 dB for gentleness)
- Export
