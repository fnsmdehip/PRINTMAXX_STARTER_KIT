found a $2k-5k per month revenue play: automated code audits for indie devs.

here's the setup:
- indie dev has 1-3 year old codebase
- no time to refactor
- scared of shipping new features (might break things)
- no budget for senior engineer ($10k/mo)

what they actually need: someone to find and fix 10-20 critical issues. that's it.

the play: offer "Code Audit + Fix Report" for $200-500 per repo. spend 1 hour using claude to:
1. read their codebase (upload to claude files api)
2. find actual bugs (not style issues)
3. generate patches
4. test them locally
5. deliver a fix report with diffs

most code audits charge $3k-10k because they use humans. you're competing on speed and price, not credentials.

time to first customer: 3 days (post on twitter + indie hacker communities). assuming 5 customers per month = $1k revenue. 10 customers = $2k. 20 customers = $4k.

the play scales because: every dev has legacy code. every dev is scared to touch it. there are 5M+ indie devs globally. you only need 10 per month.

margin is 80% (your time cost is ~$20/hr, you charge $200-500). this is better roi than saas.
