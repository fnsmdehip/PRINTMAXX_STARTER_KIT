# Post 004 — Quran App Stack

**Platform:** X (Twitter)
**Length:** Thread (4 tweets)
**Niche:** Faith

---

built 4 quran-focused apps in 3 weeks. using the same streak template, one per school of thought (hanafi, maliki, shafi'i, hanbali). live on app stores.

why separate apps? app store algorithm treats them as different apps. each one gets featured independently. combined, they're 4x more visible than one consolidated app.

each app:
- daily quranic verse
- fiqh (jurisprudence) for that school
- prayer time calculator
- streak tracker
- unlock premium at day 30 ($4.99)

cost: $0 (template + minor customization).
revenue: waiting on stripe. expected $200-500/month per app = $1k/mo across 4.

the ramadan window closes may 25. after that, demand drops 60%. but may-december is still solid ($100+/mo per app). the games are seasonal but sustained.

the play: target faith niches with seasonal spikes. build in the off-season. launch during the spike. repeat every year.
