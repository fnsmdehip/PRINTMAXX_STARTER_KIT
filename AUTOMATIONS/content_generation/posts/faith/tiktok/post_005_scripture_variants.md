# Post 005 — Scripture App Variants Strategy

**Platform:** TikTok
**Format:** Short video (45-60 sec)
**Niche:** Faith

---

**Voiceover:** "deployed 8 scripture apps. each targeting one Christian denomination. all using the same template."

**On-screen:**
- App store screenshots (8 apps: Catholic Scripture, Orthodox Scripture, Evangelical Scripture, etc.)
- Template code (highlighting shared codebase)
- Earnings dashboard (combined potential $800-2k/mo)

**Script:**

"most faith app builders create one app. i created 8.

each targets one denomination: catholic uses douay-rheims + catechism notes. evangelical uses kjv + topical prayer. orthodox uses septuagint.

why? app store treats them as independent apps. each gets featured separately. each gets ranked separately.

the code is 95% the same. swapping commentary and translation takes 1 hour per app.

revenue per app: $100-300/mo. 8 apps = $1.2k-2.4k/mo potential.

the play: don't build for 'everyone'. build for your tribe. people prefer apps made for their tradition."

**Hook:** "tribal apps convert better than generic apps"

**CTA:** "which community would you build for?"
