Date: 2026-04-02

## What
A curated marketplace where developers buy and sell skill files, CLAUDE.md configs, .cursorrules, Codex profiles, and agent configurations that make AI coding tools dramatically more effective for specific tasks. The GitHub ecosystem (Affitor/affiliate-skills at 224 stars, Railly/agentfiles at 272 stars, nagisanzenin/claude-code-production-grade-plugin at 133 stars) proves massive demand. Nobody has built the Shopify for this.

## Why Now
Claude Code, Cursor, Codex, and Windsurf all now use skill/rules files to customize behavior. Every serious user has built custom configs for their workflow. But sharing/selling these configs is fragmented across GitHub repos, Discord servers, and Twitter threads. The "agentfiles" Obsidian plugin (272 stars in days) shows people are already organizing these. A marketplace that lets you browse, preview, one-click install, and buy premium configs would capture the entire emerging category. This is like the Sublime Text package manager moment, or the VS Code extension marketplace moment, but for AI agent configs.
