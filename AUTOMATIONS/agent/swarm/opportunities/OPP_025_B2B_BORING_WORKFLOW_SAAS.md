## How

**MVP scope (Week 1-2):** Evidence screenshot tool
- Chrome extension that captures, labels, and exports screenshots from common compliance-relevant pages (AWS IAM console, GitHub settings, Okta admin, etc.)
- Auto-generates an evidence spreadsheet with: control ID, evidence type, date captured, screenshot URL
- Basic pricing: $49/mo per user (target: one "compliance champion" per company)

**Step 1 (Day 1-3):** Build Chrome extension
- Manifest V3 Chrome extension with popup UI
- "Capture for SOC2 Control CC6.1" button -- user clicks, screenshot is captured + labeled
- Evidence stored as base64 in IndexedDB, exportable as ZIP with spreadsheet

**Step 2 (Day 3-5):** AI enhancement
- Claude API: given a screenshot URL or text, identify WHICH compliance controls it satisfies
- Auto-labeling: user uploads an AWS config export → AI maps it to relevant SOC2 controls
- This is the real moat -- users don't know which control their AWS screenshot covers

**Step 3 (Day 5-7):** Launch
- Landing page with 1 case study ("how I cut my SOC2 evidence collection from 60 hours to 5")
- Target: r/sysadmin, r/netsec, Y Combinator's "Request for Startups" compliance channels
- LinkedIn cold outreach to "Compliance Officer" and "CISO at 20-100 person startup" personas
