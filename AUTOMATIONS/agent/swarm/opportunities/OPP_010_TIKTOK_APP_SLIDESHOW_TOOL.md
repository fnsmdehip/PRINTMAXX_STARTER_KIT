# OPP-010: TikTok Auto-Slideshow App Marketing Tool

**Score:** 8.0/10 (Fit: 9, Effort: 8, ROI: 7)
**Startup Cost:** $0
**Time to First Revenue:** 1 week (as service) / 2 weeks (as tool)

---

## What

Build or use auto-slideshow generation tools to create TikTok marketing content for our 7 apps (and sell as service to other app makers). @mattwelter's free tool (1000+ likes across 3 posts) generates TikTok slideshows from product URLs — validates massive demand.

## Why

- @mattwelter's tool got 1000+ likes (525+323+215) across 3 posts — AUTHENTIC engagement
- TikTok is the #1 app discovery channel for Gen Z (our PrayerLock/SleepMaxx target demo)
- Auto-slideshow = zero-effort content at scale
- We have 7 apps that need marketing content RIGHT NOW
- During Ramadan, Muslim community TikTok engagement peaks
- Could dual-purpose: market our apps + sell as service ($50-200/app for other indie hackers)
- Our Remotion pipeline (React-based video) + image_factory agent can power this

## How

1. **Phase 1: Use existing tool** — Run @mattwelter's tool on all 7 app URLs to generate free TikTok content immediately
2. **Phase 2: Build our own** — Python script: URL → screenshot key features → add captions/music → export as TikTok-format video (1080x1920, 15-30s)
3. **Phase 3: Productize** — Offer "5 TikToks for your app in 24 hours" service on Twitter/Indie Hackers. $99/batch.
4. **Content pipeline:**
   - Generate 5 TikToks per app = 35 videos
   - Post 2/day across accounts = 17 days of content
   - A/B test hooks: "this app changed my sleep" vs "POV: you finally sleep 8 hours"

## Expected ROI

### Internal (app marketing):
- 35 TikToks → avg 5,000 views each = 175,000 impressions
- 2% click-through = 3,500 app landing page visits
- 5% install rate = 175 installs (free marketing)

### External (service):
- 10 clients/mo x $99 = $990/mo
- Costs: ~$0 (automated pipeline)
- Scale: Add premium tier ($199 for 10 TikToks + analytics report)

## First 3 Steps

1. Test @mattwelter's tool on PrayerLock URL — generate 3 TikTok slideshows today
2. Post first TikTok on test account — measure views/engagement over 48 hours
3. If >1K views, batch-generate for all 7 apps and create dedicated TikTok posting schedule

## Risk

- TikTok algorithm is unpredictable (mitigate: volume play — 35 videos means some will hit)
- Auto-generated content may look generic (mitigate: custom hooks + trending audio)
- @mattwelter's tool may have limitations (mitigate: build our own with Remotion if needed)
