# OPP-009: Shopify Merchant Migration Cold Outreach

**Score:** 8.0/10 (Fit: 8, Effort: 7, ROI: 9)
**Startup Cost:** $0
**Time to First Revenue:** 1-2 weeks

---

## What

Target Shopify merchants angry about price hikes with cold outreach offering migration services or alternative storefront setup. Validated demand from Reddit (90 score, 67 complaints on r/smallbusiness).

## Why

- Shopify price hike backlash generating active complaints on Reddit, Twitter, forums
- 67+ comments of frustration on r/smallbusiness (ALPHA17033)
- Merchants actively searching for alternatives = warm leads without warming
- Our cold email pipeline (MM007_COLD_OUTBOUND) is already set up with 87.4% margins
- We have Next.js expertise for building alternative storefronts
- Migration services command $500-5,000 per client
- Recurring revenue potential: host their new store on our infrastructure ($50-200/mo)

## How

1. **Lead generation:** Scrape Reddit threads about Shopify complaints → extract usernames → find their stores → find contact info
2. **Cold email sequence:** "Saw you're frustrated with Shopify's pricing. Here's what a migration to [Medusa/Saleor/custom Next.js] looks like: same features, $0/mo platform fee, you own everything."
3. **Service tiers:**
   - Basic migration: $500 (move products + design to new platform)
   - Custom storefront: $2,000-5,000 (Next.js + Stripe + custom design)
   - Hosting + maintenance: $100-200/mo recurring
4. **Differentiation:** Show them the math — Shopify at $79-299/mo vs self-hosted at $0-20/mo. 12 months of savings pays for the migration.

## Expected ROI

- 100 cold emails/week → 5% response rate = 5 conversations
- 20% close rate = 1 client/week
- Average deal: $1,500 one-time + $150/mo recurring
- Month 1: 4 clients = $6,000 + $600/mo recurring
- Month 3: $6,000/mo one-time + $2,400/mo recurring = $8,400/mo

## First 3 Steps

1. Scrape r/smallbusiness Shopify complaint threads → build prospect list of 50 merchants with store URLs and contact info
2. Write 3-email cold sequence using @pipelineabuser framework (consequence-first hook about Shopify costs)
3. Send first batch of 25 cold emails via existing outbound infrastructure

## Risk

- Migration projects are labor-intensive (mitigate: build templates/scripts for common migrations)
- Shopify may reverse pricing (unlikely — they've done this before)
- Need to actually deliver quality (our Next.js skills are strong)
