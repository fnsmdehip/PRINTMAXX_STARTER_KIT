## Why

- Perplexity serves 15M+ daily queries. ChatGPT web browsing is active for ~100M users. Bing Copilot is baked into Windows 11
- AI search engines cite specific pages. A page that gets cited in Perplexity for "best X tool" drives traffic with zero Google ranking needed
- AIO content structure is: Q&A format, first-person authority signals, numbered lists, specific tool comparisons with prices and features -- NOT the blog post format that Google prefers
- Live example from r/juststart: a site got 43 Microsoft Copilot citations on a new domain after one optimized article. No Google DA required.
- Our existing programmatic SEO pipeline (600 pages at printmaxx-seo.surge.sh) uses the wrong format for AI search -- this creates a parallel pipeline with the RIGHT format

## How

**Step 1 (Day 1):** Keyword research for AI-cited affiliate topics
- Target: "best [AI tool] for [use case]" questions that show up in ChatGPT/Perplexity answers
- Tools: Perplexity itself (search the question, see what gets cited), SerpAPI for AI Overview presence
- Priority keywords: AI writing tools, project management AI, AI coding assistants, AI email tools
- 50 priority question keywords in 1 day using Python + SerpAPI

**Step 2 (Day 2-3):** AIO content template
- Template structure proven to get cited:
  ```
  # [Tool Name] Review 2026: [Specific Use Case]
  > Quick answer: [Tool] is best for [X] because [specific reason with data point]
  - Price: $X/mo
  - Best for: [persona]
  - Affiliate link: [link]
  [Detailed Q&A section addressing 5-7 follow-up questions]
  [Comparison table with 3 competitors]
  [Specific limitations section -- AI engines trust balanced reviews]
  ```
- Claude prompt to generate 10 pages/day in this format

**Step 3 (Day 3-4):** Distribution
- Deploy as separate site (aio-reviews.surge.sh or similar) -- separate from existing SEO site
- Submit sitemap to Bing (Copilot reads Bing index aggressively)
- Manually submit 5 best pages to Perplexity's "suggest a source" form
- Verify citations: run target queries in Perplexity weekly, track which pages get cited

**Step 4:** Wire affiliate links
- AI writing: Jasper (30% recurring), Copy.ai (45% recurring), Writesonic (30%)
- Productivity: Notion AI, Linear, ClickUp affiliate programs
- Dev tools: Cursor ($20/signup), GitHub Copilot, Tabnine
