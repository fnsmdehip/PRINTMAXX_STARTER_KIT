## What

## How

**Target vertical pick (choose one):**
Option A -- **Compliance MCP (SOC2/ISO27001):** Connect Claude to NIST, CIS, ISO frameworks as structured data. Let compliance teams ask "what evidence does SOC2 CC6.1 require?" and get accurate answers with citations.
Option B -- **Legal Research MCP:** Connect Claude to free legal sources (CourtListener, Harvard Caselaw API) for case law lookups. Target solo attorneys ($79/mo, they pay $200+/mo for Westlaw).
Option C -- **Accounting Standards MCP:** Parse FASB/GAAP/IFRS standards into Claude's context. Target CPAs and accountants drafting memos.

**Recommended: Option A (Compliance MCP)** -- frameworks are stable text documents, no ongoing data licensing, fastest to build.

**Step 1 (Day 1-2):** Ingest compliance frameworks
- Download NIST 800-53, SOC2 TSCs, ISO27001 controls as structured JSON
- Build semantic search index (SQLite + simple BM25 or Chroma for vector search)
- Expose as MCP tool: `search_controls(query, framework)` → returns relevant controls with citations

**Step 2 (Day 2-3):** MCP server packaging
- Use Anthropic's `mcp` Python SDK
- Package as `pip install compliance-mcp` OR as a hosted HTTP endpoint (FastAPI)
- Test end-to-end: Claude Desktop connects to local server, queries return accurate results

**Step 3 (Day 3-4):** Stripe gating + distribution
- Stripe Payment Link: $29/mo individual, $79/mo team (3 seats)
- Gate: API key check in MCP server -- valid key = full access
- Launch in r/cybersecurity, r/sysadmin, r/compliance, LinkedIn compliance groups
- Write 1 article: "I built a Claude MCP server for SOC2 -- here's how it works" → submit to HN
