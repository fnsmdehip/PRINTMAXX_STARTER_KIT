All comparison/affiliate pages that had `"author":{"@type":"Person","name":"PRINTMAXX"}` were corrected to `"author":{"@type":"Organization","name":"PRINTMAXX","url":"https://printmaxx.surge.sh/"}`.

### 2. OG Image SVG→PNG Migration — 28 pages fixed

Replaced inline SVG data URIs in `og:image` and `twitter:image` tags with Pollinations.ai PNG URLs.

**Why this matters:** Facebook, Twitter/X, LinkedIn, and Slack all REJECT inline SVG data URIs as OG images. Pages with SVG OG images show as blank/missing thumbnails when shared on any social platform. Since social sharing is a key distribution channel, this directly blocks one of the main organic traffic paths.

**Pollinations.ai URL pattern used:**
`https://image.pollinations.ai/prompt/{ENCODED_PROMPT}?width=1200&height=630&nologo=true&seed={N}`

These URLs serve permanent PNG images from Pollinations CDN. No deployment required.
