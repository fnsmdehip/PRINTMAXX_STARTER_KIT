## New Website Weakness Patterns Confirmed This Cycle

### Pattern 1: HTTP-Only Sites in 2026 (qpsaz.com)
Still exists. Confirmed via site: search showing all pages as http://. In 2026 this is extremely rare and extremely credible as a pitch angle. The customer sees "Not Secure" in Chrome before reading a single word.

**Future search pattern:** Search `site:DOMAIN.com` and look for http:// (not https://) in the URLs. If a business still runs HTTP in 2026, their site is deeply neglected — every other signal will also be bad.

### Pattern 2: Split Domain (.net + .com)
Callaway Outdoor owns both callawayoutdoor.com and callawayoutdoor.net. This confuses Google's authority signals — backlinks and SEO power are distributed across both domains instead of consolidated. Business owners usually don't know this is happening.

**Future search pattern:** Search business name + domain variations. If a business shows up on .net, check if .com also exists. Split domain = instant pitch angle.

### Pattern 3: Instagram-Only (No Website)
Lauren Baer Photography has zero Google-indexed web presence. Booking via Instagram DM. Spring wedding photographers with Instagram-only presence are losing all organic search traffic to photographers with even basic portfolio sites.

**Future search pattern:** Search "wedding photographer [city]" and note which results are Instagram links (not websites). Any photographer appearing ONLY via Instagram profile = no website = warm lead.

### Pattern 4: GoDaddy-Confirmed via ZoomInfo
ZoomInfo company profiles include technology stack data. Dr. Hans Gaul's GoDaddy hosting was confirmed via ZoomInfo listing. This is a reliable secondary source for confirming weak hosting without visiting the site.

**Future search pattern:** ZoomInfo search of any small business name shows tech stack. If GoDaddy appears — that's a confirmed signal.

## Cycle 8 Intelligence Notes

### Pool Service Market Intelligence
AZ pool market note: Arizona pool companies compete on proximity and chemical quality, not SEO. Most small operators (sub-10 employees) have never had a website audit. The HTTP site at qpsaz.com is a perfect example: owner Rustin Genet has been in business long enough to have built a site before HTTPS was standard, and never updated it.

**Highest-ROI pitch for pool companies:** Review schema markup. Every pool company has Google reviews. Almost none of them have the schema markup that makes those stars appear in search results. It's a 30-minute technical fix that changes how the listing looks in search — and it's provably different before/after.

### Wedding Photographer Market
Solo photographers with Instagram-only presence = fastest-growing lead segment for website builds. They're making $2K-6K per wedding, seasonal income, and know they need a web presence — they just haven't prioritized it. Price point: $400-800 for a portfolio site is trivially affordable.

**Best outreach channel:** Instagram DM, not email. They live on Instagram. Message should be 3-4 sentences max — short like an Instagram comment, not a cold email.
