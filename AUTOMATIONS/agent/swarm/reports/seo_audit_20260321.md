## Critical Bugs Fixed

### 1. Broken JSON-LD — best-ai-tools-2026
File: LANDING/affiliate-pages/best-ai-tools-2026/index.html
Bug: Nested opening `<script type="application/ld+json">` tag inside another JSON-LD block (line 29). Google rejects all JSON-LD from that point forward. FAQPage and ItemList were invisible to rich result generation.
Fix: Removed duplicate tag. Now 3 clean blocks (Article + FAQPage + ItemList). FAQPage now eligible for rich results.
