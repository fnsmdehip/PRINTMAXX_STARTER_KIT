QUICK START COMMANDS
─────────────────────────────────────────────────────────────────────────────

# 1. Install dependencies (one time)
pip install requests beautifulsoup4 tqdm

# 2. Run pipeline with sample data
cd AUTOMATIONS
python3 local_biz_pipeline.py --urls-file sample_local_biz_urls.csv --dry-run

# 3. View results
cd output/pipeline_run/landing_pages
python3 -m http.server 8000
# Open: http://localhost:8000/index.html

PIPELINE WORKFLOW
─────────────────────────────────────────────────────────────────────────────

INPUT (CSV) → SCRAPE (analyze sites) → SCORE (0-100) → FILTER (HIGH priority)
    ↓             ↓                        ↓              ↓
  URLs       Mobile/SEO/SSL/etc     LOW = needs help   score < 40
               ↓
           GENERATE (landing pages) → HOST (deploy) → OUTREACH (emails)
               ↓                         ↓               ↓
           HTML files             Vercel/Netlify    Instantly.ai CSV
