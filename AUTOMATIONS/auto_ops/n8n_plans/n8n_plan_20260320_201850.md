# n8n Workflow Plan

**Created:** 2026-03-20T20:18:50.903681
**Description:** DAG pipeline that: (1) catalogs overpriced SaaS tools by category, (2) builds n8n workflow replacements for the highest-value targets, (3) packages each as a sellable template with README + setup guide, (4) generates content (tweets, threads, landing copy) documenting each build, (5) creates cold outreach targeting businesses paying for these tools. We already have n8n at localhost:5678 — this method is zero-cost to execute and produces both digital products AND service leads.

## Steps

- Research top 20 overpriced SaaS tools replaceable by n8n (email marketing, social scheduling, lead scoring, CRM connectors)
- Build 4 n8n workflows at localhost:5678 replacing the specific tools mentioned ($299 email, $199 social, $149 lead scoring, $99 form+CRM)
- Export workflow JSON, write setup READMEs, screenshot key nodes
- Create digital product listings: individual workflows $29-79, full bundle $149
- Generate 4 Twitter threads with exact cost-savings math (hook: 'I replaced $746/mo in SaaS with free n8n workflows')
- Build 4 SEO comparison pages: '[Tool] alternative using n8n' targeting buyer-intent longtail
- Cold email 50 businesses visibly using these overpriced tools (scrape from G2 reviews, Twitter complaints)
- Post in r/n8n, r/selfhosted, n8n community forum as free template contributor to build authority
- Weekly cron: scan for new SaaS tools with pricing complaints, add to replacement queue
