# Platform/Growth Research Loop

## Context
You are scanning for new growth tactics, platform changes, and automation opportunities. Focus on what's working NOW and safe automation limits.

## Task
1. Check state.md for the next uncompleted category
2. Research that category using web search and available sources
3. Write findings to disk at the specified output location
4. Mark the category as complete in state.md
5. Output `<promise>COMPLETE</promise>` when all categories done

## Sources to Scan
- Platform announcement accounts (Twitter, Instagram, TikTok official)
- BlackHatWorld growth sections (filter for legal tactics)
- @tatealax, @wesocialgrowth - Growth hacking
- r/growthhacking, r/socialmediamarketing
- Product Hunt - New growth tools

## Research Categories
1. **Platform changes** - New features, algorithm updates, API changes
2. **Automation limits** - Safe thresholds per platform (follows, likes, DMs)
3. **Growth hacks** - Viral mechanics, referral tactics, engagement tricks
4. **Services** - Growth services worth using, pricing
5. **Tools** - New automation tools, scheduling apps, analytics
6. **Ban patterns** - What's getting accounts banned, how to avoid

## Output Format
For each opportunity found, create a file at:
`LEDGER/GROWTH_OPPORTUNITIES/[date]_[platform]_[tactic].md`

Include:
- Platform and tactic type
- Specific limits or thresholds
- Evidence (screenshot, post, documentation)
- Risk assessment (low/medium/high)
- Implementation steps
- Cost if applicable

## Quality Filter
APPROVE if:
- Has specific numbers or limits
- Recent (last 30 days)
- From reliable source

DIG DEEPER if:
- Older tactic (may be outdated)
- Aggressive limits claimed
