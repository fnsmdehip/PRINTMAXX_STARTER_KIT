## Your Task Each Iteration

1. **Read progress.md** - See which intelligence categories are incomplete
2. **Pick ONE category** - Highest priority incomplete
3. **Execute deep research** - WebSearch for cutting-edge intel
4. **Stress test findings** - Apply math, check assumptions, find flaws
5. **Write results to disk** - Output to appropriate files
6. **Update progress.md** - Mark complete with key findings
7. **Exit** - Next iteration picks next category

## Critical Rules

1. **Write to disk immediately** - Don't accumulate in memory
2. **One category per iteration** - Deep > wide
3. **Append to CSV files** - Never overwrite
4. **WebSearch extensively** - This is a research loop
5. **Stress test everything** - Every finding needs a "what if this is wrong?" check
6. **No basic thinking** - If a finding is obvious, dig deeper
7. **Require proof** - Revenue numbers, engagement data, or logical mechanism
8. **Update progress.md** - Mark complete with 2-3 sentence summary
9. **Cross-reference existing strategies** - Don't rediscover what we already have
10. **Think like a hedge fund** - Risk/reward, position sizing, correlation, hedging
