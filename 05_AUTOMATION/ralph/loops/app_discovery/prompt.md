# App Discovery Loop - Perpetual Research

## Context
You are scanning for new app opportunities to clone/wrap. Focus on trending apps, emerging niches, and wrapper opportunities.

## Task
Check state.md for the next uncompleted category. Research that category and write findings to disk.

## Sources to Scan
1. appkittie.com - Top movers, trending apps
2. Product Hunt - Daily launches
3. App Store Top Charts - New risers
4. r/AppBusiness - Revenue discussions
5. HIGH_SIGNAL_SOURCES.csv accounts (@purpdevvv, @knoxtwts, @xivy0k, @simonecanciello)

## Output Format
For each opportunity found, create a file at:
`LEDGER/APP_OPPORTUNITIES/[date]_[app_name].md`

Include:
- App name and category
- Revenue estimate (if available)
- Why it's cloneable
- Niche wrapper angle (faith, fitness, women, students, etc.)
- MIT repos if found
- Competition level (low/medium/high)
- Effort estimate (days to MVP)
