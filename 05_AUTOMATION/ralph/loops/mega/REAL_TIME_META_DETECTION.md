## PHASE 3: VELOCITY DETECTION (Catch explosions early)

### Metrics that signal "THIS IS META RIGHT NOW":

**GitHub:**
- 10K+ stars in <1 week = explosive
- 1K+ stars in <1 day = investigate immediately
- Trending tab appearance = confirm it's real

**X/Twitter:**
- 10K+ likes in <24 hours from tech account = meta
- Quote tweet ratio >50% = controversial/viral
- Multiple high-signal accounts posting same thing = confirm

**Reddit:**
- 1K+ upvotes in <6 hours on tech subreddit = hot
- Gilded 5+ times = community validation
- Cross-posted to 3+ subreddits = spreading fast

**Product Hunt:**
- #1-3 ranking on launch day = worth investigating
- 500+ upvotes first day = significant
- Comments mentioning specific use cases = actionable

## ALPHA QUALITY TEST (Before Adding to ALPHA_STAGING.csv)

### Minimum Requirements (ALL must be true):

1. **Specific numbers:**
   - GitHub stars, upvotes, views, revenue, users
   - "Trending" without numbers = REJECT

2. **Named tools/platforms:**
   - "AI tool" = REJECT
   - "OpenClaw" or "Moltbook" = APPROVED

3. **Time-bounded:**
   - "100K stars in 2 months" = APPROVED
   - "Popular repo" = REJECT

4. **Proof type (at least ONE):**
   - Live GitHub repo with stars
   - Revenue screenshot
   - Media coverage from TechCrunch/Fortune/etc
   - Simon Willison or similar credible source mentioned it
   - Personal testimonial with specific results

5. **Actionability (can we do THIS WEEK):**
   - Fork repo = YES
   - "Build AI" = NO (too vague)

6. **Not already in LEDGER:**
   - grep source_url before adding
   - grep title for duplicates
   - Similar angle OK if different source
