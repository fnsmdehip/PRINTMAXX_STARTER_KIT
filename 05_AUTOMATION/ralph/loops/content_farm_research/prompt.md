# Content Farm Research Loop

## Mission
Research content farm tactics, algorithm updates, platform monetization changes, and emerging content formats.

## What to Research Each Iteration

Pick ONE of these categories (rotate through them):

### 1. YOUTUBE_ALGORITHM
- Latest algorithm changes (Shorts, longform, community posts)
- Monetization threshold changes
- RPM trends by niche
- Faceless channel tactics that work now
- Content format innovations

### 2. TIKTOK_ALGORITHM
- Current algorithm meta (what gets pushed)
- Creator Fund / Creativity Program payouts
- TikTok Shop integration tactics
- Content format trends (stitch, duet, slides)
- Automation limits and safety

### 3. INSTAGRAM_ALGORITHM
- Reels vs Feed performance data
- Threads growth tactics
- Collab feature usage
- Carousel engagement rates
- Story vs Reel reach comparison

### 4. FACEBOOK_REELS
- RPM comparison (FB Reels vs TikTok vs YT Shorts)
- Monetization requirements
- Cross-posting strategy
- Content that performs on FB specifically
- Bonus programs and creator incentives

### 5. CONTENT_FORMATS
- New viral content formats emerging
- Template/structure patterns that get engagement
- Hook patterns with highest retention
- Trending audio/sound strategies
- Visual style trends

### 6. MONETIZATION_UPDATES
- Platform payout changes (any platform)
- New monetization features
- Ad revenue optimization
- Sponsorship rate benchmarks
- Affiliate integration opportunities

## Output Format

For each finding, append to LEDGER/ALPHA_STAGING.csv:
```
alpha_id,source,source_url,category,title,description,actionable_steps,effort_level,roi_potential,risk_level,applies_to_niches,status,reviewed_date,reviewer_notes
```

Category: CONTENT_FORMAT or PLATFORM_ARBITRAGE
Status: PENDING_REVIEW

## State File
Read `.ralph/progress.md` first to see what categories have been covered.
Update it after each iteration.

## Rules
1. ONLY append to CSV files, never overwrite
2. Check for duplicates before adding (grep source_url)
3. Each finding needs specific numbers or proof
4. Focus on actionable tactics, not general trends
5. Note platform-specific limits and safety thresholds
