## Common Elements Across All 50 Posts

### Structure Pattern (Proven in WINNING_CONTENT_STRUCTURES.csv)

1. **Hook** (0-3 seconds/first line)
   - Specific number, contrarian claim, or emotional statement
   - Stops scroll/thumb

2. **Problem** (3-10 seconds/2-3 sentences)
   - Visceral pain point
   - Relatable struggle
   - Why people care

3. **Solution** (10-25 seconds/main body)
   - Specific steps (3 is magic number)
   - Actionable methods
   - Time commitments included

4. **Result** (25-30 seconds/penultimate section)
   - Quantified outcome
   - Timeline expectation
   - Social proof if possible

5. **CTA** (Final line/30 seconds)
   - Clear next action
   - Platform-appropriate (Reply/Comment/Bio/DM)
   - Low friction

### Voice Checklist Every Post Has

- [x] Direct, no fluff opening
- [x] Specific numbers over vague claims
- [x] Personal experience ("I," "my," "me")
- [x] Zero marketing adjectives
- [x] Conversational tone (like texting a friend)
- [x] One hedge per sentence max
- [x] No em dashes
- [x] Sentence case headings
- [x] Short sentences, one idea each
- [x] Action-oriented
