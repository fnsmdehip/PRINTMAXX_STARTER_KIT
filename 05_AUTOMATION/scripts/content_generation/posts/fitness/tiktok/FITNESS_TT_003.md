# TikTok Post - FITNESS003

**Platform:** TikTok/Reels | **Niche:** Fitness | **Structure:** CS003 - Listicle | **CTA:** Comment | **Length:** 30-45 seconds

---

## Hook (0-2 seconds)

3 exercises that look boring but build real muscle

## Exercise 1 (2-10 seconds)

Barbell rows

Everyone wants to bench. Rows are boring. Rows build the back.

Back strength = entire physique looks bigger.

3 sets x 5 reps. Heavy.

## Exercise 2 (10-18 seconds)

Goblet squats

Squats suck. Everyone hates them. Everyone who does them gets the best legs.

2 weeks in: You feel it.
8 weeks in: Everyone sees it.

3 sets x 8 reps.

## Exercise 3 (18-26 seconds)

Deadlifts

Looks terrifying. Is actually the simplest lift.

Full body work in one movement.

1 set x 5 reps. That's enough.

## CTA (26-30 seconds)

These 3 boring exercises > 20 fancy exercises

Comment which one you're trying first.
