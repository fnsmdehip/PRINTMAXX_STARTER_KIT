# Social scheduling SOP template for solopreneurs

## Quick Answer

Create one master document with posting schedule, templates, and checklist. Takes 20 minutes per week. Publish 3 times daily across platforms. This SOP eliminates decisions, saves time, and keeps consistency.

## Why You Need an SOP

Without process:
- Forget to post
- Inconsistent voice
- Ad-hoc scheduling
- Burns mental energy

With SOP:
- Automated (n8n triggers)
- Consistent (same format every time)
- Predictable (know what goes out when)
- 20 minutes to manage all week

## The SOP Template

```
SOCIAL SCHEDULING SOP v1

PURPOSE
Post 3x daily across Twitter, LinkedIn, Instagram
Keep consistent voice and message flow
Minimal manual work

SCHEDULE
- 8 AM: Thread kickoff (Twitter)
- 12 PM: Quick tip (LinkedIn + Instagram)
- 6 PM: Engagement bait (Twitter + LinkedIn)

PLATFORMS COVERED
- Twitter/X: 2 posts daily
- LinkedIn: 1 post daily
- Instagram: 1 post daily

TIME ALLOCATION: 20 minutes/week
```

## Weekly Content Buckets

Create 3 posts per day by rotating these:

### 8 AM - Thread Kickoff (Twitter)

Format: 5-8 tweets on one topic

Template:
```
1/ [Hook - problem or insight]
2/ [Why this matters]
3/ [First step]
4/ [Second step]
5/ [Third step]
6/ [Common mistake]
7/ [Results or payoff]
8/ [Link to blog or CTA]
```

Time: 10 minutes to write

**Example:**
```
1/ Most solopreneurs schedule social media wrong.
They post at random times. No strategy.
Here's how to fix it in 5 minutes.

2/ First: Identify your audience timezone.
Look at your analytics.
When do they post/engage?
```

### 12 PM - Quick Tip (LinkedIn + Instagram)

Format: Single image + caption (200-300 words)

Template:
```
[Opening hook]

[Quick tip/insight]

[Why this matters]

[Call to action or related link]
```

Time: 8 minutes (with image)

**Example:**
```
Most automation tools break after month 1.

Here's why: Solopreneurs set it up, forget to monitor.

The fix: 5-minute weekly check
- Are emails sending?
- Are leads captured?
- Is data correct?

[Set a calendar reminder]
```

### 6 PM - Engagement Bait (Twitter + LinkedIn)

Format: Question or poll + discussion

Template:
```
[Question related to audience struggle]

[A) Option 1]
[B) Option 2]
[C) Option 3]

[Follow-up comment to spark replies]
```

Time: 5 minutes

**Example:**
```
What breaks your automation stack first?

A) Rate limits
B) Database sync issues
C) Webhook failures
D) Forgot password

I usually hit A within 2 days of launch...
```

## Weekly Plan (60 Minutes Total Setup)

Monday morning:
- Pick 5 topics (5 min)
- Write 5 threads (50 min)
- Schedule all in Buffer/Later (5 min)

Done. Posts go out automatically all week.

## Tool Setup

**Option 1: n8n automation** (Free, self-hosted)
- Write posts in spreadsheet
- n8n picks them up
- Posts automatically

**Option 2: Buffer** ($15/mo)
- Write in Buffer dashboard
- Schedule all at once
- Post automatically

**Option 3: Later/Hootsuite** ($25-50/mo)
- Full calendar view
- Analytics built-in
- Collaboration features

Recommendation: Start with Buffer. Switch to n8n if you build custom flows.

## Voice Guidelines

When writing posts:

- Short sentences
- One idea per tweet
- Ask questions
- Use specifics (not vague claims)
- Admit failures/tradeoffs
- No AI adjectives (seamless, revolutionary, etc.)
- Tag relevant people 1-2 times per week

## Measurement

Track in spreadsheet:
- Date posted
- Platform (Twitter, LinkedIn, Instagram)
- Topic/thread name
- Impressions
- Engagement rate
- Clicks

Review monthly. Double down on what works.

## Common Mistakes

**Don't:** Schedule every day at same time (algorithm notices)
**Do:** Vary times slightly (7:55 AM, 8:05 AM, etc.)

**Don't:** Post without checking replies (seems robotic)
**Do:** Reply to 5-10 comments daily

**Don't:** Recycle same post each week (boring)
**Do:** Rotate 4-5 different angles

**Don't:** Post more than 3x daily (noise)
**Do:** Stick to 3x daily schedule

## Scaling Up

When to add capacity:
- More than 3 posts daily → Use scheduling tool
- Multiple accounts → Use multi-account tool
- Team posting → Use collaborative tool
- Niche-specific themes → Use content calendar
