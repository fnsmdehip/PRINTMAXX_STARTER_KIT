# Social Scheduling SOP Template for Solopreneurs

A standard operating procedure for consistent social media posting without burning out.

## Quick Answer

- Batch create content weekly (2-3 hours)
- Schedule in advance (7 days minimum)
- Use templates for consistency
- Track what works, cut what doesn't
- Automate everything except engagement

## The Weekly SOP

### Monday: Content Planning (30 min)

**9:00 AM - Review last week**
- [ ] Check analytics for top performers
- [ ] Note what flopped
- [ ] Update content themes

**9:30 AM - Plan this week**
- [ ] Choose 7 content themes
- [ ] Assign themes to days
- [ ] Note any time-sensitive content

### Tuesday: Content Creation (2 hours)

**Create 7 posts minimum:**
- [ ] 3 value posts (tips, frameworks, how-tos)
- [ ] 2 engagement posts (questions, polls)
- [ ] 1 personal post (story, lesson learned)
- [ ] 1 promotional post (soft sell)

**For each post:**
- [ ] Write hook (first line)
- [ ] Add body (2-5 lines)
- [ ] Include CTA (soft)
- [ ] Add media if needed

### Wednesday: Scheduling (30 min)

**9:00 AM - Load into scheduler**
- [ ] Add all posts to scheduling tool
- [ ] Set optimal posting times
- [ ] Double-check all content
- [ ] Schedule for 7 days

### Daily: Engagement (15 min)

**Check twice daily:**
- [ ] Reply to comments (within 2 hours)
- [ ] Like relevant posts in niche
- [ ] Reply to 3-5 posts in niche
- [ ] Check DMs

## FAQ

**Q: How many posts per day?**
A: 1-2 per platform. Consistency beats volume.

**Q: Best posting times?**
A: Test and measure. General: 8-10 AM and 6-8 PM local time.

**Q: Should I cross-post everywhere?**
A: Adapt content per platform. Same idea, different format.
