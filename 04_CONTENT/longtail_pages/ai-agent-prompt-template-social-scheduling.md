# AI agent prompt template for social scheduling

Writing 300+ social posts yearly kills most founders. AI can generate them, but only with the right prompt.

Here's the template that works.

## The master prompt

```
You are a social media expert for [NICHE].

Generate 5 posts for [PLATFORM] about [TOPIC].

Each post should:
- Start with a hook (problem or question)
- Include one specific number or fact
- End with engagement (question or call to action)
- Be [TONE] in voice
- Target [AUDIENCE]

Format:
Post 1: [content]
Post 2: [content]
etc.

Platform constraints:
- Twitter: max 280 chars
- LinkedIn: 100-200 words preferred
- Email: 150-250 words

Return ONLY the posts. No explanations.
```

## The platform-specific prompt

**For Twitter (280 chars max):**

```
Write 5 tweets about [TOPIC].

Each tweet:
- Hook (problem or stat)
- One key insight
- Call to action
- EXACTLY under 280 characters

Make them go viral. Use power words (surprising, wrong, etc.).
```

**For LinkedIn (100-300 words):**

```
Write 5 LinkedIn posts about [TOPIC].

Each post:
- Personal story or observation
- Business insight
- Specific number or example
- 150-250 words
- End with question to drive comments
```

**For email (150-250 words):**

```
Write 5 email hooks about [TOPIC].

Each email hook:
- Subject line (under 50 chars)
- Opening line (why they should read)
- Body (150-200 words)
- One clear CTA

Make each one unique. Different angles.
```

## Batch generation workflow

**Monday morning (15 minutes):**
1. Open Claude
2. Generate 20 post ideas (4 prompts × 5 posts)
3. Save to Notion

**Tuesday (5 minutes):**
1. Pick 3 ideas
2. Run platform-specific prompt
3. Get 15 posts

**Wednesday (5 minutes):**
1. Pick 3 different ideas
2. Run platform-specific prompt
3. Get 15 posts

**Total:** 25 minutes for 30 posts. 0.8 minutes per post.

At this rate: 600+ posts per month from 5 hours of work.

## Quality gates

Not all AI outputs work. Add this to every prompt:

```
Before returning posts, rate each on:
- Clarity (1-10)
- Engagement potential (1-10)
- Truthfulness (1-10)

Only return posts scoring 8+ on all metrics.
```

This filters out weak content before you see it.

## Customizing voice

Your brand voice matters. Add this:

```
Your voice is [VOICE_DESCRIPTION].

Examples of posts in this voice:
[PASTE 2-3 REAL POSTS YOU'VE WRITTEN]

Match this style exactly.
```

AI learns from examples. Feed it your best posts. It replicates the style.

## Common mistakes

**Mistake 1: Vague prompts.** "Generate posts about marketing."
Output: Generic garbage.

**Mistake 2: No examples.** AI guesses your voice.
Output: Doesn't match brand.

**Mistake 3: No quality gate.** All outputs are used.
Output: Some posts tank.

**Mistake 4: Same template forever.** Audience gets bored.
Output: Engagement drops after month 2.
