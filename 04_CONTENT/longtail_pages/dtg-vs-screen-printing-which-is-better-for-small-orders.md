# DTG vs screen printing: which is better for small orders

## Quick answer

DTG (direct-to-garment) wins for orders under 25 units because there's no setup fee. Screen printing wins at 50+ units because the $50-150 setup fee gets spread across more shirts, dropping per-unit cost below DTG.

## Cost comparison by order size

| Order Size | DTG Total | Screen Print Total | Winner |
|------------|-----------|-------------------|--------|
| 1 shirt | $15 | $65 ($50 setup + $15) | DTG |
| 10 shirts | $120 | $150 ($50 setup + $10/ea) | DTG |
| 25 shirts | $275 | $250 ($50 setup + $8/ea) | Tie |
| 50 shirts | $500 | $350 ($50 setup + $6/ea) | Screen |
| 100 shirts | $900 | $500 ($50 setup + $4.50/ea) | Screen |

The crossover point is 25-30 units for most printers.

## When DTG wins

**Use DTG when:**
- Order is under 25 units
- Design has many colors (8+ colors)
- Design has gradients or photorealistic images
- You need fast turnaround (same-day possible)
- Testing new designs before committing

**DTG strengths:**
- No setup fee
- Unlimited colors at same price
- Photo-quality prints
- Quick production (minutes per shirt)
- Easy to print one-offs

## When screen printing wins

**Use screen printing when:**
- Order is 50+ units
- Design has 1-4 colors
- You want maximum durability
- Printing on dark fabrics
- Same design, multiple shirt colors

**Screen printing strengths:**
- Lowest per-unit cost at scale
- Most durable print (50+ washes)
- Vibrant colors on dark fabrics
- Specialty inks (metallic, glow, puff)
- Industry standard for merch

## Quality comparison

| Factor | DTG | Screen Print |
|--------|-----|--------------|
| Color accuracy | Excellent | Good |
| Durability | Good (30+ washes) | Excellent (50+ washes) |
| Feel on fabric | Soft | Slightly raised |
| Detail level | High | Medium-high |
| Dark fabric results | Requires white underbase | Excellent |

## Turnaround time

**DTG:**
- Setup: 5 minutes
- Print time: 2-5 minutes per shirt
- Same-day delivery possible
- No minimum order

**Screen printing:**
- Setup: 30-60 minutes per color
- Print time: 30 seconds per shirt (once set up)
- Typical turnaround: 3-7 days
- Setup only makes sense at 25+ units

## FAQ

**Which lasts longer after washing?**

Screen printing typically lasts 50+ washes vs 30+ for DTG. Both are fine for normal use. Screen printing ink sits on top of the fabric, DTG ink soaks into it.

**Can DTG print white on black shirts?**

Yes, but it requires a white underbase layer first. This adds $3-5 per shirt and slightly increases print thickness. Screen printing handles dark fabrics better.

**Which is better for photorealistic images?**

DTG, hands down. Screen printing requires separating images into distinct colors. DTG prints continuous tones like an inkjet printer.

**What about sublimation?**

Sublimation only works on polyester (white or light colors). It's the best option for all-over prints on poly fabrics but won't work on cotton.

## Decision flowchart

1. **Order under 25 units?** Go DTG

3. **Order 50+ units, 1-4 colors?** Go screen
4. **Need metallic/specialty ink?** Go screen
5. **Printing on dark fabric at scale?** Go screen
6. **One-off or small test?** Go DTG
