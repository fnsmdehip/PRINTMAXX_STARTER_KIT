# What tools do indie hackers use for GEO/AI-SEO

Indie hackers need cheap tools that work. Here's what they actually use for AI overview optimization.

## Tier 1: The essentials (free)

**Google Search Console.** You need to know if you're ranking. Free. Non-negotiable.

**Claude.** Content generation. Fast enough for bulk pages. $0 if you have API access.

**Markdown editor.** VS Code or Obsidian. Free. Version control with Git (free).

**GitHub.** Host your content. Free tier works for 1000+ pages.

Total: $0

## Tier 2: The multipliers ($50-100/month)

**Playwright.** Test if your pages load fast. Monitor for AI crawlers. ~$10/month VPS + free Playwright.

**Airtable.** Track which pages rank where. Free tier for 1200 rows. $20/month if you go pro.

**SendGrid.** Notify you when pages rank. Free tier: 100 emails/day.

Total: ~$50/month

## Tier 3: Optional (if you're serious)

**Semrush.** Keyword research and ranking tracking. $120/month.

**Browserbase.** Render pages for AI crawlers. $50-200/month.

**Custom domain email.** Increases ranking authority. $12/year.

Most indie hackers skip this tier until they see traction.

## What actually works

Indie hacker stack that generates $10k/month:

- Search Console (free) - track rankings
- Claude API ($50/month) - generate 50+ pages
- Airtable free ($0) - manage content calendar
- GitHub ($0) - version control
- Cron job ($10/month) - automate publishing

Total: $60/month. No other tools needed.

## Common tool mistakes

**Mistake 1: Too many tools.** You buy Semrush, Ahrefs, Surfer, ContentStudio, etc. Cost: $500/month. Result: information overload.

**Mistake 2: Waiting for perfect tools.** You spend 2 months researching which tool. Meanwhile, competitors rank. Start with Search Console + Claude.

**Mistake 3: Tool addiction.** You optimize tools instead of writing content. Writing > tools.

## The lean stack

Start with this for 3 months:

1. Google Search Console - free
2. Claude - $20/month
3. Google Sheets - free
4. GitHub - free
5. Cron job on any VPS - $10/month

Total: $30/month. Enough to validate your idea.

## When to upgrade

After 3 months, if you're getting:
- 1k+ monthly organic visitors
- 20+ pages ranking

Then consider:
- Airtable ($20) for better tracking
- Playwright ($10) for monitoring
- Semrush ($120) for competitive analysis

Only add tools that answer "Will this help me make money?" If yes: upgrade. If no: skip.
